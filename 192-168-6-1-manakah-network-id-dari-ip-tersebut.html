12. Network id, broadcast id, range ip, jumlah host dari ip 192.168.45.1/24


IP address : 192.168.45.1/24 atau

= 192.168.45.1 255.255.255.0

Cara menghitung :

Ubah IP Address dan Netmask kedalam bentuk biner, 192.168.45.1 IP Address tersebut berada di kelas C,  ambil octet ketiga dan ke empat untuk dihitung, maka :

IP address : x.x.0010 1101. 0000 0001 = 45.1

Netmask : x.x.1111 1111. 0000 0000 = 255.0

Jawab:

1. Network ID

Network ID = IP Address AND Netmask

IP address : x.x.0010 1101. 0000 0001

Netmask    : x.x.1111 1111. 0000 0000

——————————————————

AND           : x.x.0010 1101. 0000 0000 = 45.0

Sehingga Network ID nya adalah 192.168.45.0

2. Broadcast ID

Broadcast ID = IP Address OR ~Netmask

IP address : x.x.0010 1101. 0000 0001

~Netmask  : x.x.0000 0000 . 1111 1111

———————————————————

OR : x.x.0010 1101. 1111 1111 = 45.255

Sehingga Broadcast Id nya adalah 192.168.45.255

3. Range IP

Range IP = Network ID hingga(-) Broadcast ID

Sehingga Range IP nya adalah 192.168.45.0 – 192.168.45.255

4. Jumlah Host

Jumlah Host = [tex]2^{y} -2[/tex]

dimana y adalah jumlah biner 0 pada octet 3 dan 4 netmask

/24 bila di-binerkan menjadi:

1111 1111. 1111 1111. 1111 1111. 0000 0000

255         255        255            0

Sehingga Jumlah Host

= [tex]2^{8} - 2[/tex]

= [tex]256 - 2[/tex]

= [tex]254[/tex]

25. Berapakah network id dan broadcast id dari IP address 160.132.34.23/23?


Pertama, jadikan network prefix menjadi subnet mask dalam binary:
11111111.1111111.11111110.00000000
Setelah itu, ubah Ip Addressnya menjadi binary juga:
10100000.10000100.00100010.00010111

AND kan kedua bilangan tsb:
[tex]11111111.11111111.11111110.00000000 \\ 10100000.10000100.00100010.00010111 [/tex]
Hasil:
[tex]10100000.10000100.00100010.00000000[/tex]
Ubah ke dalam bentuk Decimal akan menjadi
160.132.34.0
Maka 160.132.34.0 adalah network id nya.

Untuk broadcast:
Cukup memflip(menginvert) subnet mask dan ORkan dengan Network ID:
[tex]00000000.00000000.00000001.11111111 \\ 10100000.10000100.00100010.00000000[/tex]
Hasil:
[tex]10100000.10000100.00100011.111111111[/tex]
Ubah ke dalah bentuk decimal akan menjadi 160.132.35.255


Maka:
Network address dan broadcast address dari 160.132.24.23/23 adlaah:
160.132.34.0 dan 160.132.35.255.
