[tex] \frac{ {9}^{7} \times {9}^{3} }{ {9}^{2} } [/tex]

[tex] = \frac{ {9}^{7 + 3} }{ {9}^{2} } [/tex]

[tex] = \frac{ {9}^{10} }{ {9}^{2} } [/tex]

[tex] = {9}^{10} \div {9}^{2} [/tex]

[tex] = {9}^{10 - 2} [/tex]

[tex]{ \boxed{ \tt = {9}^{8} }}[/tex]

= [tex]9^{7+3-2}[/tex]

Pangkat nya bisa ditambah karena sifat dari perpangkatan. Dengan syarat bilangan pokok nya sama.

Kalau ada a²xa³=a⁵

Karena kalau dijabarkan

a²= a x a

a³= a x a x a

a² x a³= a x a x a x a x a = a⁵

Begitu pula dengan pembagian,

Kalau ada a³ ÷ a² , hasilnya adalah a¹= a

Karena,

a³= a x a x a

a²= a x a

a³÷a²= a x a x a ÷ a x a= a¹
