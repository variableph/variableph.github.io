1. struktur 2-etil-2,2,4,6-tetrametiloktana


Struktur 3-etil-2,2,4,6-tetrametiloktana:

      CH₃ C₂H₅

         |     |

CH₃ - C - CH - CH - CH₂ - CH - CH₂ - CH₃

         |             |                 |

       CH₃        CH₃             CH₃

Pembahasan:

Nama senyawa hidrokarbon ini adalah 3-etil-2,2,4,6-tetrametiloktana.  

Pada rantai utama, senyawa ini memiliki rantai utama berupa senyawa berupa oktana yang terdiri dari 8 atom karbon, dengan rantai lurus dengan hanya ikatan tunggal (alkana).

Pada rantai utama ini tidak terdapat adanya ikatan ganda (seperti pada alkena) atau rangkap tiga (pada alkuna)

Pada rantai utama ini, di atom karbon ke-3 melekat gugus etil (-C₂H₅).  

Kemudian di rantai utama ini juga melekat pada gugus metil (-CH₃) atom karbon ke-2 sebanyak dua buah dan pada atom karbon ke-4 dan ke-6, masing-masing 1 buah.

Sehingga, rumus bangun 3-etil-2,2,4,6-tetrametiloktana adalah:

      CH₃ C₂H₅

         |     |

CH₃ - C - CH - CH - CH₂ - CH - CH₂ - CH₃

         |             |                 |

       CH₃        CH₃             CH₃

Untuk memastikan rumus bangun ini, dapat diperiksa bahwa setiap atom karbon memiliki 4 ikatan, baik dengan atom karbon lain atau dengan atom hidrogen

2. 3-etil-2,2,4,6-tetrametiloktana


Rumus struktur dari 3-etil-2,2,4,6-tetrametiloktana dapat diamati pada gambar terlampir.

Pembahasan

Pada persoalan ini diminta untuk menuliskan rumus struktur dari 3-etil-2,2,4,6-tetrametiloktana.

Mengenai aturan penamaan alkana menurut IUPAC (International Union of Pure and Applied Chemistry) dapat disimak pada pranala berikut https://brainly.co.id/tugas/2860668.

Dilihat dari namanya, 3-etil-2,2,4,6-tetrametiloktana termasuk jenis hidrokarbon alkana. Alkana adalah senyawa jenuh yaitu senyawa organik yang hanya memiliki ikatan tunggal antaratom karbon dalam molekulnya. Deret homolog alkana adalah [tex]\boxed{ \ C_nH_{2n+2} \ }.[/tex]

Hal-hal yang perlu diketahui dalam pembuatan rumus struktur 3-etil-2,2,4,6-tetrametiloktana adalah sebagai berikut:

Rantai induk terpanjang adalah oktana dengan delapan buah atom C.Pada nomor urut 2 terdapat dua buah metil, nomor urut 4 dan 6 terdapat masing-masing sebuah metil sebagai cabang-cabang. Total ada empat buah metil.Pada nomor urut 3 terdapat cabang etil.

Sekadar pengingat, seluruh gugus alkil yang ada sebagai cabang adalah [tex]\boxed{ \ CH_3 = metil \ } \ dan \ \boxed{ \ C_2H_5 = etil \ }[/tex]

23. buatlah struktur dari nama senyawa berikut1.2,4 dimetil pentana2. 3 etil ,3.5 dimetil heptana3. 2.4 dietil,2 metil heksana4. 3.3.4 trimetil heksana5. 2.3 dietil 4 propil oktuna6. 3 etil 4 metil 4 propil 2 nonena​


Struktur dari senyawa berikut adalah

1)  2,4-dimetilpentana

Struktur :

          CH₃             CH₃  

           |                   |  

CH₃ − CH − CH₂ − CH − CH₃  

2) 3-etil-3,5-dimetilheptana

Struktur :

                    C₂H₅         CH₃  

                     |                 |  

CH₃ − CH₂ − C − CH₂ − CH − CH₂ − CH₃  

                     |

                    CH₃

3) 2,4-dietil-2-metilheksana

Struktur :

          CH₃          C₂H₅

           |                |  

CH₃ − C − CH₂ − CH − CH₂ − CH₃  

           |

          C₂H₅

Penamaan struktur diatas tidak tepat. Penamaan yang tepat sesuai dengan aturan IUPAC adalah 3-etil-5,5-dimetilheptana.

4) 3,3,4-trimetilheksana

Struktur :

                    CH₃ CH₃

                     |       |  

CH₃ − CH₂ − C  − CH − CH₂ − CH₃  

                     |

                    CH₃

5) 2,3-dietil-4-propil-1-oktena

Struktur :

               C₂H₅

                 |

CH₂ = C − CH − CH − CH₂ − CH₂ − CH₂ − CH₃  

           |               |

          C₂H₅        CH₂ − CH₂ − CH₃

6) 3-etil-4-metil-4-propil-2-nonena​

Struktur :

                         CH₂ − CH₂ − CH₃

                          |

CH₃ − CH = C − C − CH₂ − CH₂ − CH₂ − CH₂ − CH₃  

                    |      |

                  CH₂ CH₃

                    |

                  CH₃
