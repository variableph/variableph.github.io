5. akar persamaan dari 3x2-12=0 adalah....


Akar persamaan dari 3x² - 12 = 0 adalah -2 dan 2.

Pembahasan

Persamaan kuadrat adalah persamaan polinomial dengan suku terdingginya berderajat 2.

Bentuk umum dari persamaan kuadrat adalah ax² + bx + c, dengan a, b dan c adalah koefisiennya, x² dan x adalah peubahnya.

Untuk mengetahui akar-akar dari persamaan kuadrat dengan cara memfaktorkan.

Penyelesaian soal

Cara 1

3x² - 12 = 0

Kita faktorkan

(3x - 6)(x + 2) = 0

3x - 6 = 0     atau   x + 2 = 0

3x = 6                          x = -2

x = 6 : 3

x = 2

HP = {-2, 2}

Cara 2

3x² - 12 = 0

3x² = 12

x² = 12 : 3

x² = 4

x = ± √4

x = ± 2

HP = {-2, 2}

6. Nilai diskriminan dari -3x2+ 8x – 12 = 0 adalah…​


~Persamaan Kuadrat

_____________________

Nilai diskriminan dari - 3x² + 8x - 12 = 0 adalah - 80

---

Soal

Nilai diskriminan dari - 3x² + 8x - 12 = 0 adalah ...

Pembahasan

Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0

dimana:

a = koefisien x²b = koefisien xc = konstanta

Untuk kasus ini dminta untuk menentukan diskriminan dari persamaan kudrat tersebut, Untuk menentukan diskriminan atau yang disimbolkan sebagai D gunakan rumus D = b² - 4ac

____________

Diketahui

Persamaan kuadrat: - 3x² + 8x - 12 = 0

didapat:

a = - 3b = 8c = - 12

Ditanya

Nilai diskriminan ( D )

Solusi

D = b² - 4ac

D = 8² - 4( - 3 )( - 12 )

D = 64 - ( - 12 )( - 12 )

D = 64 - 144

D = - 80

Jadi, Nilai diskriminan dari - 3x² + 8x - 12 = 0 adalah - 80
