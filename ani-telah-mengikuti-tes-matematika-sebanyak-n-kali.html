Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya ia mendapatkan nilai 83 dengan nilai rata rata adalah 80. Tetapi, jika nilai tesnya adalah 67, maka nilai rata rata nya adalah 76. Maka nilai n adalah 3.

Penyelesaian Soal :

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya ia mendapatkan nilai 83 dengan nilai rata rata adalah 80" yaitu :

X = ∑X/ n

Keterangan : X = Nilai rata rata  

                      ∑X = Jumlah nilai

                      n = banyaknya data

Maka perhitungannya yaitu :

X = ∑X/ n

80 = (x + 83) / (n + 1)

80 (n + 1) = x + 83

80n + 80 = x + 83

x - 80n = 80 - 83

x - 80n = -3      ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "jika nilai tesnya adalah 67, maka nilai rata rata nya adalah 76" yaitu :

X = ∑X/ n

76 = (x + 67) / (n + 1)

76 (n + 1) = x + 67

76n + 76 = x + 67

x - 76n = 76 - 67

x - 76n = 9       ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai n dengan menggunakan cara sebagai berikut :

x - 80n = -3

x - 76n = 9

___________ -

   -4n = -12

      n = [tex]\frac{12}{4}[/tex]

      n = 3

∴ Kesimpulan nilai n adalah 3.

Penjelasan dengan langkah-langkah:

Nilai mtk 15 orang = 6,6

Nilai mtk 16 orang = 6,7

6,6 x 15 = 99

6,7 x 16 = 107,2

Maka nilai Ani adalah 107,2 - 99 = 8,2

Jawab:

Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data. Jika nilai n₁ orang rata-ratanya x₁, nilai n₂ orang rata-ratanya x₂ dan nilai n₃ orang rata-ratanya x₃, maka rata-rata gabungannya adalah

x = \frac{jumlah \: data}{banyaknya \: data} = \frac{\sum x_{i}}{n}

Rumus rata-rata

x = \frac{jumlah \: data}{banyaknya \: data} = \frac{\sum x_{i}}{n}

Pembahasan

Diketahui

Nilai rata rata tes matematika 15 siswa adalah 6,6.

n₁ = 15 dan x₁ = 6,6

Nilai ani disertakan, maka nlai rata ratanya menjadi 6,7

n₂ = 1, x₂ = nilai Dinda dan

x_{gab} = 6,7

Ditanyakan

Nilai ani dalam tes matematika

Jawab

x_{gab} = \frac{n_{1} . x_{1} + n_{2} . x_{2}}{n_{1} + n_{2}}

6,7 = \frac{15(6,6) + 1(x_{2})}{15 + 1}

6,7 = \frac{99 + x_{2}}{16}

6,7 (16) = 99 + x₂

107,2 = 99 + x₂

107,2 - 99 = x₂

8,2 = x2

Jadi nilai ani dalam tes matematika tersebut adalah 8,2

15. Seorang siswa telah mengikuti tes sebanyak n kali dan memperoleh nilai rata-rata 80. Berapakah nilai yang harus dia peroleh pada tes selanjutnya agar nilai rata ratanya 82


Seorang siswa telah mengikuti tes sebanyak n kali dan memperoleh nilai rata-rata 80. Nilai yang harus dia peroleh pada tes selanjutnya agar nilai rata ratanya 82 adalah (2n + 82). Hasil tersebut diperoleh dengan menggunakan rumus rata-rata gabungan. Jika nilai n₁ orang rata-ratanya x₁, nilai n₂ orang rata-ratanya x₂ dan nilai n₃ orang rata-ratanya x₃, maka rata-rata gabungannya adalah

[tex]{\bar x}_{gab} = \frac{n_{1}.x_{1} + n_{2}.x_{2} + n_{3}.x_{3}}{ n_{1} + n_{2} + n_{3}}[/tex]

Pembahasan

Diketahui

Banyaknya tes yang telah diikuti (n₁) = n Rata-rata (x₁) = 80 Rata-rata gabungan setelah mengikuti satu tes lagi (n₂ = 1) adalah [tex]{\bar x}_{gab}[/tex] = 82

Ditanyakan

Nilai tes berikutnya (x₂) = …. ?

Jawab

[tex]{\bar x}_{gab} = \frac{n_{1}.x_{1} + n_{2}.x_{2}}{ n_{1} + n_{2}}[/tex]

[tex]82 = \frac{n(80) + 1.x_{2}}{ n + 1}[/tex]

82(n + 1) = 80n + x₂

82n + 82 – 80n = x₂

2n + 82 = x₂

Jadi nilai berikutnya yang harus diperoleh adalah  

x₂ = 2n + 82  

dengan n adalah banyak tes yang telah diikuti sebelumnya

Berarti nilai tes berikutnya tergantung dari nilai n nya

Nilai berikutnya yang harus diperoleh, jika ia sudah mengikuti tes sebanyak

jika mendapat nilai 83 :

[tex] \frac{nx + 83}{n + 1} =80 \\ \\ > > nx + 83 = 80 (n + 1) \\ > > \: \: \: \: \: \: \: \: \: \: nx = 80(n + 1) - 83.....(i) [/tex]

jika mendapat nilai 67 :

80(n+1) - 83 = 76(n+1) - 67

26. Nilai yang diberikan pada setiap tes matematika merupakan bilangan bulat dari 0 sampai dengan 10 . Jika seorang siswa mengikuti lima kali tes matematika dengan median dari nilai tes adalah 8, maka rata rata terkecil yang mungkin untuk tes matematika siswa tersebut adalah


Rata-rata terkecil yang mungkin terjadi pada tes matematika tersebut adalah 4,8. Soal tersebut merupakan soal tentang rata-rata.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal matematika yang membahas tentang rata-rata. Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data.

Untuk menyelesaikan soal tersebut kita harus menjumlahkan semua data pada soal dan menghitung banyaknya data. 

Penyelesaian soal

Diketahui:

Terdapat 5 kali ujian dengan median 8Nilai tes tersebut terdiri dari bilangan bulat 0 sampai 10

Ditanyakan:
Tentukan rata-rata terkecil dari 5 ujian Matematika tersebut!

Jawab:

Terdapat 5 kali ujian dengan median 8 ⇒ 0 ≤ x1, x2 ≤ 8 dan 8 ≤ x4, x5 ≤ 10. sehingga untuk mendapatkan rata-rata terkecil kita harus mengganti x1 dan x2 dengan 0, mengganti x4 dan x5 dengan 8.

Sehingga

[tex]x terkecil = \frac{0+0+8+8+8}{5}[/tex][tex]xterkecil = \frac{24}{5}[/tex][tex]xterkecil = 4,8[/tex]

Jadi, rata-rata terkecil yang mungkin terjadi pada tes matematika tersebut adalah 4,8.

28. Ani telah mengikuti tes matematika sebanyak ???? kali. Pada tes berikutnya ia memperoleh nilai 83 sehingga nilai rata- rata Ani adalah 80. Tetapi jika nilai tes tersebut adalah 67, maka nilai rata-ratanya adalah 76. Nilai ???? adalah ….2


Dimisalkan : Ani telah mengikuti tes sebanyak P kali. Dan rata-rata awal nilai Ani = Q.

Jika Ani memperoleh nilai 83 pada tes berikutnya, maka rata-rata nilainya menjadi 80.

<==>

( [Q x P] + 83 ) / ( P + 1 ) = 80

( PQ + 83 ) = 80 . ( P + 1 )

PQ + 83 = 80P + 80

PQ = 80P + 80 - 83

PQ = 80P - 3 (i)

Jika Ani memperoleh nilai 67 pada tes berikutnya, maka rata-rata nilainya menjadi 76.

<==>

( [Q x P] + 67 ) / ( P + 1 ) = 76

( PQ + 67 ) = 76 . ( P + 1 )

PQ + 67 = 76P + 76

PQ = 76P + 76 - 67

PQ = 76P + 9 (ii)

Nilai PQ pada (i) = nilai PQ pada (ii). Maka :

80P - 3 = 76P + 9

80P - 76P = 9 + 3

4P = 12

P = 12 ÷ 4

P = 3

Jadi nilai P (????) = 3
