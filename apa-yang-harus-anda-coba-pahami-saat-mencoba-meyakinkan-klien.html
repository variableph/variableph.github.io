20. sebuah server mempunyai IP Addreass 200.100 4.65 agar klien dapat mengakses server tersebut IP Adreass klien adalah


Sebuah server mempunyai IP Address 200.100.4.65 agar klien dapat mengakses server tersebut maka IP Address klien adalah 200.100.4.66 atau seterusnya. Pada proses klien server dalam suatu jaringan alamat IP haruslah dalam berada class yang sama. Namun pemberian alamat IP Address juga tidak boleh sama dengan yang lain karena dapat menimbulkan IP Conflict dalam suatu jaringan.

Untuk memastikan apakah klien sudah dapat mengakses ke server dapat menggunakan perintah ping. Penulisan perintah ping digunakan pada command prompt. Makan penulisan perintahnya adalah ping 200.100.4.65. Kemudian perhatikan hasilnya.
