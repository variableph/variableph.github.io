3. Jika laptop di restart, apakah semua file akan terhapus? Dan proses restart adalah proses?


Jawaban:

Jika laptop di restart, maka file tidak akan terhapus, kecuali file tersebut belum disimpan. Dan proses restart adalah proses pengulangan kembali dimana komputer yang tadinya menyala, dimatikan kemudian dinyalakan kembali.

Penjelasan:

File yang sudah disimpan di laptop tidak akan hilang hanya karena laptop itu di restart, akan tetapi jika file tersebut belum pernah disimpan sebelumnya, maka jika laptop di restart ulang, maka file tersebut akan hilang, jadi sebaiknya sebelum merestart laptop pastikan semua file sudah disimpan.
