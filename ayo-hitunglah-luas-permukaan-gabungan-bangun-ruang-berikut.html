Penjelasan dengan langkah-langkah:

3. Diketahui : Sisi kubus = 12 cm

                      panjang balok = 20 cm

                      lebar balok = 12 cm

                      tinggi balok = 12 cm

   Ditanya : L gabungan = ?

   L kubus = S² = 12² = 144 cm²

   L balok = 2 (pl + pt + lt)

                = 2 (20X12 + 20X12 + 12X12)

                = 2 (240 + 240 + 144)

                = 2 (624) = 1.248 cm²

Luas gabungan = 144 cm² + 1.248 cm² = 1.392 cm²

4. Diketahui : Prisma segitiga : L alas = 1/2 X 8 X 9 = 36 cm²

                                                            t = 8 cm

   L prisma segitiga = 2 X L alas + Keliling alas X tinggi

                                 = 2 X 36 + (8 + 9 + 8) X 8

                                 = 2 X 36 + 25 X 8

                                 = 776 cm²

  Balok : p = 16 cm

              l = 12 cm

              t = 8 cm

L balok = 2 X (pl + lt + pt)

            = 2 X (16X12 + 12X8 + 16X8)

            = 2 (192 + 96 + 128) = 832 cm²

L gabungan = L prisma segitiga + L balok

                    = 776 cm² + 832 cm² = 1.608 cm²

5. Diketahui : Tabung : t = 20 cm

                                      r = 27-20 = 7 cm

                          Bola : r = 7 cm

 Ditanya : L gabungan = ?

 L tabung = 2[tex]\pi[/tex]r (r+t)

                 = 2 X 3,14 X 7 (7+20)

                 = 1.186,92 cm²

 L setengah bola = 2[tex]\pi[/tex]r²

                             = 2 X 3,14 X 7²

                             = 307,72 cm²

L gabungan = L tabung + L setengah bola

                    = 1.186,92 cm² + 307,72 cm²

                    = 1.494,64 cm²
