1. 2 mol Ba(OH2) direaksikan dengan P2O5 menurut reaksi:Ba(OH)2+P2O5->Ba3(PO4)2+H2O. Berapa gram P2O5 yg dibutuhkan agar Ba(OH)2 tepat habis bereaksi?


Diketahui :
n Ba(OH)2 = 2 mol

Ditanya : Berapa gram P2O5 yang dibutuhkan agar Ba(OH)2 tepat habis bereaksi ?

Jawab :

Reaksi :
3Ba(OH)2 + P2O5 --> Ba3(PO4)2 + 3H2O

n P2O5 :
= Koefisien P2O5 / Koefisien Ba(OH)2 × n Ba(OH)2
= 1 / 3 × 0,2 mol
= 0,67 mol

• Mencari massa P2O5 yang dibutuhkan :

Mr P2O5 :
= 2.Ar P + 5.Ar O
= 2.31 + 5.16
= 62 + 80
= 142 gram/mol

massa P2O5 :
= n . Mr
= 0,67 mol. 142 gram/mol
= 95,14 gram




2. 2 mol Ba(OH2) direaksikan dengan P2O5 menurut reaksi:Ba(OH)2+P2O5->Ba3(PO4)2+H2O. Berapa gram P2O5 yg dibutuhkan agar Ba(OH)2 tepat habis bereaksi?


3 Ba(OH)2 + P2O5 => Ba3(PO4)2 + 3 H2O
mol Ba(OH)2 = 2 mol
perbandingan koefisien reaksi = 3 : 1 : 1 : 3

mol P2O5 = 1/3 x mol Ba(OH)2 = 1/3 x 2 mol = 0,67mol
Mr P2O5 = 142
massa P2O5 = 0,67 mol x 142 gr/mol = 95,14 gram

12. Diketahui 200 mL larutan Ba(OH)2 memiliki PH=12. Massa Ba(OH)2 dalam larutan tersebut adalah ...? (Mr Ba(OH)2 = 171)


pH =12 pOH =14-12=2 [OH-]=10^-2 pd Ba(OH)2 utk [OH-] = 2×M 10^-2 = 2×M M= 0,5×10^-2 M=(gr/Mr)×(1000/ml) 0,5×10^-2=(gr/171)×(1000/200) gr=171×10^-3=0,171 grDiketahui :
V larutan Ba(OH)2 = 200 ml = 0,2 L
pH = 12
Mr = 171 gram/mol

Ditanya : Massa Ba(OH)2 dalam larutan ?

Jawab :

pOH :
= 14 - pH
= 14 - 12
= 2

Mencari [OH-] :
- log [OH-] = pOH
- log [OH-] = 2
- log [OH-] = - log 10-²
[OH-] = 10-²

Molaritas Ba(OH)2 :
= [OH-] / b
= 10-² M / 2
= 0,005 M
= 0,005 mol/L

n(mol) Ba(OH)2 :
= M .V
= 0,005 mol/L . 0,2 L
= 0,001 mol

Massa Ba(OH)2 :
= n . Mr
= 0,001 mol . 171 gram/mol
= 0,171 gram

29. pH suatu larutan Ba(OH)2 adalah 13+log4 maka konsentrasi larutan Ba(OH)2 tersebut adalah...​


Jawaban:

Jadikan jawaban terbaik terima kasih

Penjelasan:

13+log 4 = pH

pOH = 1 - log 4

Konsentrasi = 4 x 10^-1 atau 0,4

Jawaban:

0,2 Molar

Penjelasan:

pH = 13 + log 4

MENGHITUNG pOH

pOH = 14 - pH

= 14 - ( 13 + log 4 )

= 1 - log 4

MENGHITUNG KONSENTRASI ION [OH-]

pOH = -log [OH-]

[OH-] = 4 x 10^(-1) M

Ba(OH)2 merupakan basa kuat valensi 2.

Ba(OH)2 ---> Ba2+ + 2OH-

valensi = 2

MENGHITUNG KONSENTRASI Ba(OH)2

M = [OH-] / valensi

= 4 x 10^(-1) / 2

= 2 x 10^(-1)

= 0,2 M
