Software tersebut manghasilkan output berupa penyajian data, jika datam data (berupa makalah) diperoleh kata-kata,kalimat ataupun paragraf yang sama yang tidak menhasilkan sumber (plagiat) software akan memberikan tanda seseuai dengan fitur yang ditawarkan oleh software tersebut.

27. cara kerja perangkat lunak pengolah browser adalah​


Perangkat Lunak Browser sendiri merupakan sebuah software yang memiliki fungsi untuk memberikan tampilan dengan file yang disediakan oleh jaringan web pusat. Yang ditampilkan sendiri dapat berupa halaman web, gambar, video, atau jenis konten tampilan yang lainnya. Dokumen tersebut terkumpul menjadi sebuah jaringan yang disebut sebgai Web, sehingga perangkat lunak browser juga sering disebut dengan Web Browser.

Pembahasan

Cara kerja dari web browser :

Agar dapat menggunakan web browser tertentu, maka harus terlebih dahulu menginstal software atau aplikasi web browser ke dalam perangkatnya. Hanya dengan cara inilah sebuah web browser dapat dijalankan di sebuah perangkat baik komputer, laptop ataupun gadget.

Setelah software dari web browser tersebut berhasil terinstal dengan baik, selanjutnya pengguna dapat membuka aplikasi tersebut. Setelah membukanya, pengguna dapat mengetikan alamat website pada bagian address bar di laman web browser yang biasanya terletak di bagian atas.

Kemudian setelah memasukkan alamat website dan eksekusi lalu enter, maka aplikasi web browser akan bekerja dan mengambil data dari website yang dituju, biasanya dalam kode html. Selanjutnya yakni browser akan diarahkan pada halaman yang dituju, misalnya memasukkan google.co.id maka browser akan mengakses dan membuka website Google setelah melalui verifikasi.

30. Jelaskan apa perangkat lunak antiplagiasi


Jawaban:

Adalah perangkat lunak untuk mendeteksi adanya plagiasi didalam sebuah karya.

Penjelasan:

Plagiasi adalah cara seseorang dalam mengambil karya orang lain tanpa izin dan menganggap bahwa karya tersebut adalah karyanya sendiri. Contoh: Mencantumkan atau membuat lagu didalam video yang dibuat tanpa adanya royalti, lalu mengakui bahwa itu hasil karyanya sendiri. Terdapat juga contoh plagiasi sebagian dimana orang tersebut sudah melakukan parafrase namun masih terdapat beberapa kalimat yang identik dengan sumber aslinya dan tidak mencantumkan sitasi.  Untuk menghindari plagiasi dapat dilakukan dengan cara mendownload aplikasi anti plagiarisme yang sudah tersedia di google atau play store.
