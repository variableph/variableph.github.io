❃ Penggunaan mikro-organisme ragi

Saccharomyces cerevisiae atau biasa disebut ragi (yeast) adalah mikro-organisme yang bisa digunakan untuk membuat anggur, roti, kue, dan bir dari jaman dulu. Ragi dapat berespirasi secara anaerobik. Saat ragi berespirasi, ragi akan menghasilkan alkohol (etanol) dan karbon dioksida (lihat kotak di atas)

❃ Pembuatan roti

Roti terbuat dari tepung, dimana dibuat dari menggiling biji-bijian tanaman sereal. Kebanyakan roti terbuat dari tepung gandum.Tepung mengandung banyak pati dan juga protein khususnya protein dengan nama gluten (perekat). Untuk membuat roti, tepung dicampur dengan air dan ragi untuk membuat adonan.Enzim amilase menghancurkan beberapa pati di dalam adonan untuk membuat glukosa dan maltosa, dimana ragi akan dipakai untuk respirasi aerobik. Proses respirasi ini menghasilkan gelembung karbon dioksida dan gelembung ini akan terperangkap di dalam adonanya. Gluten (perekat) membuat adonanya jadi elastis dan gelembung karbon dioksidanya membuat adonanya jadi mengembang. Respirasi aerobik juga bisa untuk membuat alkohol, tapi alkoholnya bakal ada saat ortinya sudah dipanggang. Saat roti dipanggang, raginya bakal mati.

Sebuah toko roti ingin membuat dua macam roti yaitu roti unyil dan roti imut. Untuk membuat roti unyil dibutuhkan 50 gram tepung terigu dan 30 gram keju. Sedangkan untuk membuat roti imut dibutuhkan 40 gram tepung terigu dan 40 gram keju. Persediaan tepung terigu sebanyak 8 kg, keju sebanyak 6 kg dan bahan - bahan lainnya mencukupi. Jika harga jual satu roti unyil Rp1.000,00 dan roti imut Rp1.200,00 maka hasil penjualan maksimum dari kedua roti tersebut adalah ....  

Pembahasan :  

Untuk mempermudah membuat model matematikanya, kita buat dalam bentuk tabel

Bahan | Roti unyil (x) | Roti imut (y) | Persediaan
Terigu |     50 gr         |       40 gr       | 8 kg = 8.000 gr
Keju    |    30 gr          |      40 gr        | 6 kg = 6.000 gr
Harga | 1.000x          |     1.200y       | ...

50x + 40y ≤ 8.000
5x + 4y ≤ 800 => diarsir ke bawah
jika x = 0 maka y = 200 => (0, 200)
jika y = 0 maka x = 160 => (160, 0)  

30x + 40y ≤ 6.000
3x + 4y ≤ 600 => diarsir ke bawah
jika x = 0 maka y = 150 => (0, 150)
jika y = 0 maka x = 200 => (200, 0)  

x ≥ 0, y ≥ 0  

Fungsi sasaran :
F(x, y) = 1.000x + 1.200y  

Titik potong kedua garis :
5x + 4y = 800
3x + 4y = 600
------------------ -
2x = 200
x = 100  

3x + 4y = 600
3(100) + 4y = 600
300 + 4y = 600
4y = 300
y = 75
titik potong = (100, 75)  

Buat grafiknya (bisa dilihat pada lampiran)
Diperolehlah titik – titik sudutnya yaitu : (0, 150), (160, 0) dan (100, 75)
Kita substitusikan ke fungsi sasaran :
f(x, y) = 1.000x + 1.200y
f(0, 150) = 1.000(0) + 1.200(150) = 180.000
f(160, 0) = 1.000(160) + 1.200(0) = 160.000
f(100, 75) = 1.000(100) + 1.200(75) = 190.000  

Jadi hasil penjualan maksimum dari kedua roti tersebut adalah Rp190.000,00 (100 roti unyil dan 75 roti imut)
