1. angka 1 sampai 20 dalam bahasa inggris?


Number

JAWABAN:

Perhatikan angka  1 - 20 dalam bahasa inggris berikut ini!

1 => one 2 => two 3 => three 4 => four 5 => five 6 => six 7 => seven 8 => eight 9 => nine 10 => ten 11 => eleven 12= > twelve 13 => thirteen 14 => fourteen 15 => fifteen 16 => sixteen 17 => seventeen 18 => eighteen 19 => nineteen 20 => twenty

PEMBAHASAN:

Pelajaran tentang adalah (number) adalah materi dasar yang harus dipelajari oleh pemula untuk menambah perbendaharaan kata. Dalam pelajaran bahasa inggris ada dua jenis penomeran (number), yaitu Cardinal number dan Ordinal number. Cardinal number atau angka kardinal (satu, dua, tiga, dsb.) seperti angka yang ada di atas adalah kata sifat yang merujuk ke jumlah atau kuantitas. Ordinal number atau bilangan bertingkat (pertama, kedua, ketiga, dsb.) merujuk ke distribusi. Agar kamu lebih paham perhatikan bentuk cardinal number berikut ini!

1 => first 2 => second 3 => third 4 => fourth 5 => fifth 6 => sixth 7 => seventh 8 => eighth 9 => ninth 10 => tenth 11 => eleventh 12 => twelfth 13 => thirteenth 14 => fourteenth 15 => fifteenth 16 => sixteenth 17 => seventeenth 18 => eighteenth 19 => nineteenth 20 => twentieth
