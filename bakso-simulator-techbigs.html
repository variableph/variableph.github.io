8. tentukan Permutasi Dari• Sakura• School• Simulator​


Jawaban :

1. banyaknya susunan katanya dari sakura adalah ( 360 ) susunan kata.

2. banyak susunan kata dari school adalah (360 ) susunan kata.

3. banyakkan susunan kata dari Simulator adalah ( 362.880) susunan kata

Pendahuluan :

=>

Dalam matematika, Faktorial dari bilangan bulat positif dari n yang dilambangkan dengan n!, adalah produk dari semua bilangan bulat positif yang kurang dari atau sama dengan n:

n! = n × ( n - 1 ) × ( n - 2 ) × ( n - 3 )

Operasi faktorial digunakan sebagai bidang matematika, terutama di :

kombinatorikaljabardan analisis matematika.

Penggunaannya yang paling dasar menghitung kemungkinan urutan dan permutasi dari n yang berada di objekk yang berbeda.

=>

Permutasi adalah metode penyusunan kembali angka/objek ke dalam berbagai susunan dimana tidak ada pengulangan yang terjadi. Dalam permutasi ini, urutan menjadi salah satu perhatian, dimana tidak ada anggota yang bertukar tempat.

dengan unsur ganda :

p = n! / k!

tanpa unsur ganda :

p = n!

=>

Kombinasi bisa didefinisikan sebagai suatu metode untuk melakukan susunan terhadap benda/objek, dimana terdapat peluang terjadinya pengulangan. Dalam kombinasi, urutan tidak menjadi prioritas, dimana kemungkinan terdapat anggota yang bertukar tempat.

( n, r ) = n! / ( n! - r! )

17. jelaskan yang kalian ketahui tentang program simulator simrobot​


Jawaban:

Istilah simulator robotika dapat merujuk pada beberapa aplikasi simulasi robotika yang berbeda. Misalnya, dalam aplikasi robotika seluler , simulator robotika berbasis perilaku memungkinkan pengguna membuat dunia sederhana dari objek kaku dan sumber cahaya serta memprogram robot untuk berinteraksi dengan dunia ini. Simulasi berbasis perilaku memungkinkan tindakan yang lebih bersifat biologis jika dibandingkan dengan simulator yang lebih biner, atau komputasi. Selain itu, simulator berbasis perilaku dapat "belajar" dari kesalahan dan mampu menunjukkan kualitas antropomorfik dari keuletan.

Salah satu aplikasi paling populer untuk simulator robotika adalah untuk pemodelan 3D dan rendering robot dan lingkungannya. Perangkat lunak robotika jenis ini memiliki simulator yaitu robot virtual, yang mampu meniru gerakan robot sebenarnya dalam amplop kerja nyata. Beberapa simulator robotika menggunakan mesin fisika untuk menghasilkan gerakan robot yang lebih realistis. Penggunaan simulator robotika untuk pengembangan program kontrol robotika sangat disarankan terlepas dari apakah robot sebenarnya tersedia atau tidak. Simulator memungkinkan program robotika untuk dengan mudah ditulis dan di- debug secara off-line dengan versi terakhir dari program yang diuji pada robot yang sebenarnya. Ini terutama berlaku untuk robot industri aplikasi saja, karena keberhasilan pemrograman off-line bergantung pada seberapa mirip lingkungan nyata robot dengan lingkungan simulasi.

Tindakan robot berbasis sensor jauh lebih sulit untuk disimulasikan dan / atau diprogram secara off-line, karena gerakan robot bergantung pada pembacaan sensor seketika di dunia nyata.

18. Jelaskan fungsi dari cpu simulator!


Jawaban:

CPU Simulator adalah lingkungan simulasi komputer sederhana yang mana dikembangkan untuk membantu para siswa memahami arsitektur Komputer. Jadi fungsinya adalah untuk membantu kita, para siswa memahami arsitektur Komputer

Jawaban✨

Fungsi CPU simulator :

Memiliki peran penting dalam berjalannya keseluruhan pengoperasian komputer. Fungsi CPU mirip dengan kalkulator dengan daya pemrosesan yang lebih cepat dan kuat.Fungsi CPU yang paling utama adalah memproses dan mengolah semua kalkulasi, perintah dan melakukan operasi aritmatika dan logika terhadap data yang diambil dari memori atau dari informasi yang dimasukkan melalui perangkat keras lainnya untuk menunjang pengoperasian komputer.Menjalankan program-program yang disimpan dalam memori utama dengan cara menerjemahkan instruksi mengeksekusinya satu persatu sesuai alur perintah. CPU dikontrol menggunakan sekumpulan instruksi perangkat lunak komputer.

Pembahasan✨

CPU adalah sebuah perangkat keras yang memiliki andil besar dalam pengoperasian sebuah sistem komputer. Pada komputer, CPU biasanya berupa perangkat keras yang dikoneksikan dengan layar monitor untuk mendapatkan perintah-perintah sistem. Pada Laptop atau notebook, CPU sudah terpasang didalam perangkat laptop atau notebook itu sendiri.

CPU merupakan singkatan dari Central Processing Unit, dalam bahasa Indonesia CPU juga disebut Unit Pemroses Sentral atau UPS. Istilah lain dari CPU adalah prosessor.

Simulator [ si•mu•la•tor ]

kata nomina ( kata benda)

1) program yang berfungsi untuk menyimulasikan suatu peralatan, tetapi kerjanya agak lambat daripada keadaan yang sebenarnya;

2) alat untuk melakukan simulasi; alat yang dapat menyimulasikan

Fungsi simulator :

Simulator merupakan suatu alat yang digunakan sebagai media pembelajaran yang mempunyai bentuk dan fungsi sama seperti alat atau unit yang aslinya.
