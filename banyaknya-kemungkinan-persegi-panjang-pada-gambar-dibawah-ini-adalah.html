15. bangun gabungan persegi dan persegi panjang dengan keliling 28 cm gambar lah 2 kemungkinan bentuknya


Misalkan:

panjang sisi persegi = s

panjang persegi panjang = p

lebar persegi panjang = l

Kita pilih 2 kemungkinan gambar bangun sebagai berikut. (Gambar terlampir)

1. Jika sisi persegi berimpit dengan sisi panjang persegi panjang, di mana panjang sisi persegi = panjang persegi panjang (s = p).

Akibatnya:

keliling bangun = s + s + s + p + l + l

↔ 28 = 3s + p + 2l

↔ 28 = 3s + s + 2l

↔ 28 = 4s + 2l

Pilih nilai s = 5 cm maka

28 = 4(5) + 2l

↔ 28 = 20 + 2l

↔ 28 – 20 = 2l

↔ 2l = 8

↔ l = 4 cm

Akibatnya luas bangun = luas persegi + luas persegi panjang
= (s × s) + (p × l) 
= (5 × 5) + (5 × 4)
= 25 + 20 
= 45 cm²

2. Jika sisi persegi berimpit dengan panjang persegi panjang, di mana panjang sisi persegi ≠ panjang persegi panjang.

Akibatnya:

keliling bangun = s + s + s + (p – s ) + p + l + l

↔ 28 = 2s + 2p + 2l

↔ 28 = 2s + 2p + 2l

↔ 28 = 2s + 2p + 2l

Pilih nilai s = 3 cm  dan p = 6 cm maka

28 = 2(3) + 2(6) + 2l

↔ 28 = 6 + 12 + 2l

↔ 28 – 6 – 12 = 2l

↔ 2l = 10

↔ l = 5 cm
Akibatnya luas bangun = luas persegi + luas persegi panjang
= (s × s) + (p × l) 
= (3 × 3) + (6 × 5)
= 9 + 30 
= 39 cm²
