8. 10. Perhatikan gambar berpola berikut! Banyaknya lingkaran pada pola ke-20 adalah .....A. 170 B. 198 C. 210 D. 222​


Banyaknya lingkaran pola ke-20 adalah 231 (tidak ada dipilihan jawaban).

PEMBAHASAN

Barisan aritmatika merupakan suatu barisan bilangan dimana beda tiap suku bilangannya adalah tetap. Salah satu jenis barisan aritmatika adalah barisan artimatika tingkat 2 dimana rumus untuk mencari suku ke-n adalah :

[tex]\displaystyle{u_n=a+(n-1)b+\frac{(n-1)(n-2)c}{2}}[/tex]

dengan :

a = suku pertama

b = beda barisan pertama untuk tingkat 1

c = beda barisan untuk tingkat 2

.

DIKETAHUI

Bnayak lingkaran pola 1, u₁ = 3

Banyak lingkaran pola 2, u₂ = 6

Banyak lingkaran pola 3, u₃ = 10

Banyak lingkaran pola 4, u₄ = 15

.

DITANYA

Tentukan banyaknya lingkaran pada pola ke-20.

.

PENYELESAIAN

Banyaknya lingkaran pada tiap pola :

[tex]3\smile 6\smile 10\smile 15[/tex]

[tex].~~~3\smile4~\smile~5\\[/tex] → tingkat 1

[tex].~~~~~~~1\smile1[/tex] → tingkat 2 (beda barisan sudah sama)

Banyaknya lingkaran tiap pola membentuk barisan aritmatika tingkat 2, dengan :

Suku pertama = 3 → a = 3

Beda barisan pertama untuk tingkat 1 = 3 → b = 3

Beda barisan untuk tingkat 2 = 1 → c = 1

.

Rumus suku ke-n :

[tex]\displaystyle{u_n=a+(n-1)b+\frac{(n-1)(n-2)c}{2} }[/tex]

[tex]\displaystyle{u_n=3+(n-1)3+\frac{(n-1)(n-2)1}{2} }[/tex]

[tex]\displaystyle{u_n=3+3n-3+\frac{n^2-2n-n+2}{2} }[/tex]

[tex]\displaystyle{u_n=3n+\frac{1}{2}n^2-\frac{3}{2}n+1 }[/tex]

[tex]\displaystyle{u_n=\frac{1}{2}n^2+\frac{3}{2}n+1 }[/tex]

.

Maka banyaknya lingkaran pola ke-20 :

[tex]\displaystyle{u_{20}=\frac{1}{2}(20)^2+\frac{3}{2}(20)+1 }[/tex]

[tex]\displaystyle{u_{20}=200+30+1 }[/tex]

[tex]\displaystyle{u_{20}=231 }[/tex]

.

KESIMPULAN

Banyaknya lingkaran pola ke-20 adalah 231 (tidak ada dipilihan jawaban).

23. banyaknya lingkaran pada pola ke-20 adalah …


Jawaban:

B. 381

Penjelasan dan jawaban dengan langkah-langkah:

1, 3, 7, 13

2, 4, 6

2, 2

Dijawab:

2a = 2

a = 1

3a + b = 2

3(1) + b = 2

b = 2 - 3

b = -1

a + b + c = 1

1 - 1 + c = 1

c = 1

Rumus pola bilangan tingkat 2:

Un = an² + bn + c

U20 = 1(20²) - 1(20) + 1

U20 = 1(400) - 20 + 1

U20 = 400 - 20 + 1

U20 = 381

Jika ditanya S20, maka:

Rumus jumlah suku pertama barisan aritmatika:

Sn = n/2 (a + Un)

S20 = 20/2 (1 + 381)

S20 = 10 (382)

S20 = 3820
