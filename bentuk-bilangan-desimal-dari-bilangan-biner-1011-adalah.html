Penjelasan:

Cara Kerja Bilangan Biner:

Bilangan ke-n dari kanan setelah koma bernilai [tex]2x^{n-1}[/tex]

Bilangan paling kanan (ke-1 dari kanan), bernilai [tex]2^{0}[/tex] = 1

Bilangan paling kanan (ke-2 dari kanan), bernilai [tex]2^{1}[/tex] = 2

Bilangan paling kanan (ke-3 dari kanan), bernilai [tex]2^{2}[/tex] = 4

Bilangan paling kanan (ke-4 dari kanan), bernilai [tex]2^{3}[/tex] = 8

1011:

(1 x [tex]2^{3}[/tex]) + (0 x [tex]2^{2}[/tex]) + (1 x [tex]2^{1}[/tex]) + (1 x [tex]2^{0}[/tex]) = 8 + 0 + 2 + 1 = 11

1010:

(1 x [tex]2^{3}[/tex]) + (0 x [tex]2^{2}[/tex]) + (1 x [tex]2^{1}[/tex]) + (0 x [tex]2^{0}[/tex]) = 8 + 0 + 2 + 1 = 10


13. Konversikan bilangan biner ke desimal 1011(2)=......10


Hasil konversi bilangan biner tersebut ke desimal adalah 11₍₁₀₎. Bilangan tersebut didapatkan dengan mengalikan masing-masing angka bilangan biner dengan bilangan pangkat berbasis dua. Hasil perkalian tersebut dijumlahkan. Simak penjelasan berikut.

Penjelasan dengan langkah-langkah

Materi

Bilangan biner adalah serangkaian bilangan berbasis dua, yaitu hanya terdiri dari angka 1 dan 0. Sistem atau rangkaian bilangan seperti ini digunakan dalam ilmu komputer. Berikut beberapa bilangan lain serta jumlah basisnya.

Bilangan desimal → berbasis 10, yaitu 0 sampai 9.Bilangan oktal → berbasis 8, yaitu 0 sampai 8.Bilangan heksadesimal → berbasis 16.

Cara konversi bilangan biner ke desimal:

Mengalikan setiap angka bilangan biner dari belakang (kanan) dengan bilangan pangkat berbasis 2 dimulai dari pangkat nol.Menjumlahkan hasilnya.

Diketahui

Bilangan biner 1011₍₂₎.

Ditanya

Konversikan bilangan tersebut ke bilangan desimal!

Jawab

1011₍₂₎ = (1 × 2⁰) + (1 × 2¹) + (0 × 2²) + (1 × 2³)

1011₍₂₎ = (1 × 1) + (1 × 2) + (0 × 4) + (1 × 8)

1011₍₂₎ = 1 + 2 + 0 + 8

1011₍₂₎ = 11₍₁₀₎

Kesimpulan

Jadi, hasil konversi bilangan biner tersebut ke desimal adalah 11₍₁₀₎.

20. 1. konveksikan bilangan desimal 10244 ke bilangan biner2. konveksikan bilangan biner 1011 ke bilangan desimal3. konveksikan bilangan oktal 234 ke bilangan desimal​


Jawab:

1. 10244 (desimal) = 10100000000100 (biner)

2. 1011 (biner) = 11 (desimal)

3. 234 (oktal) = 156 (desimal)

Penjelasan dengan langkah-langkah:

1. Cara konversinya ialah:

- membagi 2 bilangan desimal itu terus, hingga hasilnya 1

- menuliskan sisa hasil pembagiannya secara urut.

- bila hasil akhirnya 1, maka konversinya ditulis dari bawah ke atas.

- untuk lebih jelasnya, bisa dilihat di gambar lampiran.

2. Perhitungan konversi biner ke desimal:

1011 = (1 x 2³) + (0 x 1²) + (1 x 2¹) + (1 x 2⁰) = 8 + 0 + 2 + 1 = 11

Jadi 1011 (biner) = 11 (desimal)

3. Perhitungan konversi oktal ke desimal:

234 = (2 x 8²) + (3 x 8¹) + (4 x 8⁰) = 128 + 24 + 4 = 156

Jadi 234 (oktal) = 156 (desimal)

29. Buktikan bahwa 11 dalam bilangan biner= 1011


Besaran merupakan suatu pengukuran dalam fisika dan satuan adalah unit yang terukurnya .

I.Besaran : Sesuatu yang dapat diukur, memiliki nilai dan satuan  

Pembagian Besaran  

1.Berdasar Satuan  

a.Besaran Pokok : Besaran yang satuan sudah terdefinisikan dan berdiri sendiri  

Contoh :  

( J I Wa Ku S M P )  

Jumlah zat ( mol ), Intensitas cahaya ( Cd ), Waktu ( s ), Kuat arus ( A ), Suhu ( K ), Massa ( Kg ) dan Panjang ( m )  

b.Besaran Turunan : Besaran yang satuannya diturunkan dari satuan besaran pokok  

Contoh :  

Luas ( m2 ), Volume ( m3 ), Massa jenis ( Kg/m3 ), Kecepatan ( m/s ), Percepatan ( m/s2 ), Gaya ( N = kgm/s2 ), Energi ( J = Nm = Kgm2/s2 ), Daya ( Watt = J/s = Kgm2/s3 ), Tekanan ( N/m2 = Kg/ms2 ), Momentum ( Kgm/s ), Impuls ( Kgm/s ), Usaha (J)  

2.Berdasar Arah  

a.Besaran Skalar : Besaran yang hanya memiliki nilai saja  

Contoh : Semua besaran pokok termasuk besaran skalar, Luas, Volume, Kelajuan, Perlajuan, Massa Jenis, Energi, Daya dsb  

b.Besaran Vektor : Besaran yang memiliki nilai dan arah  

Contoh : Perpindahan, Kecepatan, Percepatan, Gaya, Momentum dsb  

II.Dimensi  

Merupakan gambaran bagaimana besaran tersebut tersusun atas besaran-besaran pokok.  

Fungsi dimensi : untuk menguji benar / setara tidaknya suatu persamaan ( Persamaan yang benar secara dimensi belum tentu benar secara fisis )  

Dimensi dari besaran – besaran pokok :  

Jumlah zat [ N ], Intensitas cahaya [ J ], Waktu [ T ], Kuat arus [ I ], Suhu [ θ ], Massa [ M ] dan Panjang [ L ]

Pembahasan  

Pada pengukuran bilangan biner atau bilangan dua pangkat, dapat dikerjakan sesuai dengan gambar

[tex]1011=1*2^3+0*2^2+1*2^1+1*2^0\\1011=8+0+2+1\\1011=11[/tex]

maka terbukti bahwa 1011 adalah 11
