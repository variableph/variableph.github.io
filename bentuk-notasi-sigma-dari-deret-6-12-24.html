7. Nyatakan deret berikut dalam bentuk notasi sigma 2+4+6+8+10+12+14+16


Notasi sigma 2 + 4 + 6 + 8 + 10 + 12 + 14 + 16

Pendahuluan

Notasi sigma adalah sebuah tanda yang digunakan untuk menuliskan suatu penjumlahan secara singkat.

Secara umum, notasi sigma didefinisikan sebagai berikut

[tex]U_{1} +U_{2} + U_{3} + ... + U_{n} =   \sum\limits^n_{i=1} {U_{i}}[/tex]

Pelajari lebih lanjut : Notasi sigma dari : 28 + 33 + 38 + 43 + 48 + .... + 48 → https://brainly.co.id/tugas/24109207

Pembahasan

Menentukan rumus suku umum

2 + 4 + 6 + 8 + 10 + 12 + 14 + 16

U₁ = a = 2

Un = 16

b = 4 - 2 = 2

n = 8

Rumus suku ke-n deret

Un = a + (n - 1) b

     = 2 + (n - 1) 2

     = 2 + 2n - 2

Un = 2n

Bentuk notasi sigma

Batas bawah notasi sigma adalah k = 1 karena 2 merupakan suku pertama deret tersebut.

Batas atas sigma adalah 8 karena banyak suku n = 8

Notasi sigma [tex]\sum\limits_{k~ = ~1}^8 2k[/tex]

Jadi bentuk notasi sigma 2 + 4 + 6 + 8 + 10 + 12 + 14 + 16 adalah [tex]\sum\limits_{k~ = ~1}^8 2k[/tex]

9. Bentuk notasi sigma dari deret 2 + 6 + 10 + 14 + ...+ 142 adalah​


S = 2 + 6 + 10 + 14 + ... + 142

S = 2(1 + 3 + 5 + 7 + ... + 71)

S = 2((1+0) + (2+1) + (3+2) + (4+3) + ... + (36+35))

S = 2((1 + 2 + 3 + ... + 36) + (1 + 2 + 3 + ... + 36))

[tex]S = 2(\sum^{36}_{k = 1} k+ \sum^{35}_{k = 1}k)[/tex]

[tex] \frac{S}{2} = \sum^{36}_{k = 1} k + \sum^{35}_{k = 1}k[/tex]

[tex]\frac{S}{2} + 36= \sum^{36}_{k = 1} k + \sum^{35}_{k = 1}k + 36[/tex]

[tex]\frac{S}{2} + 36= \sum^{36}_{k = 1}k + \sum^{36}_{k = 1} k[/tex]

[tex]\frac{S}{2} + 36=2 \sum^{36}_{k = 1} k[/tex]

[tex]\frac{S}{2} + 36= \sum^{36}_{k = 1} 2k[/tex]

[tex]\frac{S}{2} = ( \sum^{36}_{k = 1} 2k ) - 36[/tex]

[tex]\frac{S}{2} = ( \sum^{36}_{k = 1} 2k - 1 ) [/tex]

[tex]S =2 ( \sum^{36}_{k = 1} 2k - 1 ) [/tex]

[tex]S = \sum^{36}_{k = 1} 4k - 2 [/tex]

15. Notasi sigma dari deret penjumlahan 1+5+9+....+401 akan memiliki bentuk notasi sigma...


1 + 5 + 9 + ... + 401

=> deret aritmafika : suku awal = 1 & beda = +4.

Maka :

Un = 1 + (n - 1).(4)

Un = 1 + 4n - 4

Un = 4n - 3

Menghitung nilai n untuk Un = 401 :

4n - 3 = 401

4n = 401 + 3

4n = 404

n = 101

Sehingga :

[tex] \boxed{ \boxed{1 + 5 + 9 + ... + 401 = \sum \limits^{101}_{k = 1} \: 4k - 3}}[/tex]

30. tentukan notasi sigma dari Deret 3+6+12+...+6144​


Notasi Sigma dari deret 3+6+12+...+6144 adalah

[tex]\Sigma ^{12}_{n = 1} \: 3( {2}^{n - 1} ) \\ [/tex]

Pembahasan

Diketahui:

3+6+12+...+6144

Ditanya:

Notasi sigma dari 3+6+12+...+6144

Penyelesaian:

3+6+12+...+6144

Suku pertama = 3

Rasio = 2

Un = 3*2^(n-1)

6144 = 3*2^(n-1)

2^(n-1) = 2048

2^(n-1) = 2¹¹

n - 1 = 11

n = 12

Sn = 3(2ⁿ - 1)

Notasi sigmanya dari 3+6+12+...+6144 adalah

[tex]\Sigma ^{12}_{n = 1} \: 3( {2}^{n - 1} ) \\ [/tex]
