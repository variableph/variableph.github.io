Bentuk sederhana dari 2√6 / 3+√3 adalah √6 - √2

.

Pembahasan

Pecahan irasional adalah pecahan yang memiliki penyebut yang irasional, contohnya:

[tex] \frac{1}{ \sqrt{4} } [/tex]

[tex] \frac{3}{2 \sqrt{2} + 9} [/tex]

.

Yang dimaksud dengan penyebut yang irasional adalah penyebut yang menggunakan bilangan irasional sebagai penyebutnya. Contoh bilangan irasional seperti:

[tex] \sqrt{9} [/tex]

[tex] \sqrt{27 + 8} [/tex]

[tex]3 \sqrt{9} [/tex]

.

Cara merasionalkan suatu penyebut adalah dengan melakukan perkalian sekawan, contohnya;

[tex] = \frac{x}{ \sqrt{x} - \sqrt{y} } [/tex]

[tex] = \frac{x}{ \sqrt{x} - \sqrt{y} } \times \frac{\sqrt{x} + \sqrt{y}}{\sqrt{x} + \sqrt{y}} [/tex]

[tex] = \frac{x( \sqrt{x} + \sqrt{y} )}{( \sqrt{x} - \sqrt{y} )( \sqrt{x} + \sqrt{y}) } [/tex]

[tex] = \frac{x \sqrt{x} +x \sqrt{y} }{ {( \sqrt{x} )}^{2} - {( \sqrt{y} )}^{2} } [/tex]

[tex] = \frac{x \sqrt{x} +x \sqrt{y} }{ x - y } [/tex]

.

Catatan:

[tex] {x}^{2} - {y}^{2} = (x - y)(x + y)[/tex]

[tex] \sqrt{a} \times \sqrt{a} = {( \sqrt{a} )}^{2} = a[/tex]

.

PenyelesaianDiketahui

[tex] \frac{2 \sqrt{6} }{3 + \sqrt{3} } = [/tex]

.

Ditanya

Bentuk sederhana dari 2√6 / 3+√3 adalah...

.

Jawab

[tex] = \frac{2 \sqrt{6} }{3 + \sqrt{3} } [/tex]

[tex] = \frac{2 \sqrt{6} }{3 + \sqrt{3} } \times \frac{3 - \sqrt{3} }{3 - \sqrt{3} } [/tex]

[tex] = \frac{2 \sqrt{6} (3 - \sqrt{3} )}{(3 + \sqrt{3} )( 3 - \sqrt{3} )} [/tex]

[tex] = \frac{6 \sqrt{6} - 2 \sqrt{18} }{ {3}^{2} - {( \sqrt{3} )}^{2} } [/tex]

[tex] = \frac{6 \sqrt{6} - 2 \sqrt{9 \times 2} }{ 9 - 3} [/tex]

[tex] = \frac{6 \sqrt{6} -( 2 \times 3) \sqrt{ 2} }{ 6} [/tex]

[tex] = \frac{6 \sqrt{6} - 6 \sqrt{2} }{6} [/tex]

[tex] = \sqrt{6 } - \sqrt{2} [/tex]

.

Jadi, bentuk sederhana dari 2√6 / 3+√3 adalah √6 - √2

.
