13. bagaimana cara mendeklarasikan array yang akan dipakai untuk menampung lima nama bunga


bagaimana cara mendeklarasikan array yang akan dipakai untuk menampung lima nama bungaPendahuluan

Dalam pemrograman, array adalah suatu struktur yang terdiri dari sejumlah elemen yang memiliki tipe data yang sama. Elemen-elemen array tersusun secara sekuensial dalam memory computer. Array dapat berupa satu dimensi, dua dimensi, tiga dimensi ataupun banyak dimensi (multi dimensi).

Array Satu Dimensi

Array Satu Dimensi tidak lain adalah kumpulan elemen-elemen identik yang tersusun dalam satu baris. Elemen-elemen tersebut memiliki tipe data yang sama, tetapi isi dari elemen tersebut boleh berbeda.

Pendeklaraian array:

{Tipe_Data} {Nama_Array}[n]={elemen0,elemen1,elemen2,…,n}

n=jumlah elemen  

Contoh:

int ArrayAngka[20];

Cara mengaksesnya:

ArrayAngka[0] = 5;

ArrayAngka[1] = 10;

dan seterusnya.

Array Dua Dimensi

Array Dua Dimensi adalah suatu array di dalam array. Array dua dimensi sering digambarkan sebagai sebuah matriks, merupakan perluasan dari array satu dimensi. Jika array satu dimensi hanya terdiri dari sebuah baris dan beberapa kolom elemen, maka array dua dimensi terdiri dari beberapa baris dan beberapa kolom elemen yang bertipe sama.

Pendeklarasian:

{Tipe_Data} {Nama_Array}[m][n];

Contoh:

int arrBuffer[3][2];    

arrBuffer[0][1] = 18

arrBuffer[2][0] = 1

arrBuffer[2][1] = 0

Artinya kita membuat suatu array dengan jumlah elemen 3 dan di setiap elemen mempunyai 2 elemen lagi di dalamnya, dan setiap elemen bisa diisi.

Pembahasan

Untuk mendeklarasikan array yang akan dipakai untuk menampung lima nama bunga, maka kita membutuhkan array dari string yang mampu menyimpan 5 buah string.

Kesimupulan

Dalam bahasa C#

String[] bunga = new String[5];

bunga[0] = "mawar";

bunga[1] = "melati";

bunga[2] = "kantil";

bunga[3] = "tulip";

bunga[4] = "kenanga";


Dalam bahasa C

String adalah array of char, maka:

char bunga[5][10] = {"mawar", "melati", "kantil", "tulip", "kenanga"};

26. Package java pada netbeans untuk membuat desain form interface adalah …..​


Jawaban:

Swing

Penjelasan:

Package java yang digunakan untuk membuat desain form interface pada NetBeans adalah javax.swing.package. Package ini menyediakan berbagai kelas dan metode yang dapat digunakan untuk membuat desain form interface yang menarik dan interaktif.

Package javax.swing.package pada NetBeans termasuk kelas-kelas seperti JFrame, JButton, JLabel, JTextField, JTextArea, dan masih banyak lagi. Anda dapat menggunakan kelas-kelas tersebut untuk membuat elemen-elemen interface seperti frame, tombol, label, text field, dan text area pada aplikasi Java Anda.

Untuk menggunakan package javax.swing.package pada NetBeans, Anda perlu menambahkan import statement berikut pada file Java Anda:

import javax.swing.*;

Setelah menambahkan import statement tersebut, Anda dapat menggunakan kelas-kelas yang tersedia di dalam package tersebut untuk membuat desain form interface pada aplikasi Java Anda.
