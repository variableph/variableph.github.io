1 ) Harga pokok ekspor dapat dihitung dengan rumus:

Harga Pokok Ekspor = (Jumlah Dus x Jumlah Botol per Dus x Harga per Botol) + Biaya Angkut + Biaya Sewa Kontainer

Untuk sambal ijo:

Jumlah dus = 35

Jumlah botol per dus = 24

Harga per botol = Rp. 22,000.0

Maka, Harga Pokok Ekspor sambal ijo = (35 x 24 x 22,000.0) + 800,000.0 + 2,500,000.0 = Rp. 18,166,000.0

Untuk sambal mata:

Jumlah dus = 21

Jumlah botol per dus = 24

Harga per botol = Rp. 24,000.0

Maka, Harga Pokok Ekspor sambal mata = (21 x 24 x 24,000.0) + 800,000.0 + 2,500,000.0 = Rp. 15,824,000.0

Untuk sambal pete:

Jumlah dus = 14

Jumlah botol per dus = 24

Harga per botol = Rp. 24,000.0

Maka, Harga Pokok Ekspor sambal pete = (14 x 24 x 24,000.0) + 800,000.0 + 2,500,000.0 = Rp. 10,772,000.0

Untuk sambal terasi:

Jumlah dus = 28

Jumlah botol per dus = 24

Harga per botol = Rp. 19,000.0

Maka, Harga Pokok Ekspor sambal terasi = (28 x 24 x 19,000.0) + 800,000.0 + 2,500,000.0 = Rp. 14,188,000.0

Jadi, Harga Pokok Ekspor total untuk semua produk adalah: Rp. 58,950,000.0

= (Rp. 58,950,000.0 + Rp. 800,000.0 + Rp. 2,500,000.0) x (11% + 2,5%)

= Rp. 7,640,250.0

3 ) Total cost yang harus disiapkan perusahaan untuk ekspor ini adalah:

Total Cost = Harga Pokok Ekspor + Biaya Angkut + Biaya Sewa Kontainer + Pajak
