Start jongkok adalah start yang dilakukan dengan posisi badan jongkok. Start jongkok biasa disebut dengan start (a) pendek. Start jongkok digunakan pada lari jarak pendek dan pelari pertama pada lari estafet.

Pembahasan:

Start merupakan salah satu teknik dasar yang harus dikuasai dalam beberapa cabang olahraga atletik. Start terdiri dari beberapa macam, salah satunya adalah start jongkok. Start jongkok adalah gerakan start yang dilakukan dengan posisi badan jongkok. Start jongkok terdiri dari tiga macam, yakni.

Start pendek. Start pendek adalah posisi start jongkok yang dilakukan dengan kedua kaki menapak pada papan start. Lutut kaki depan ditekuk dan di posisikan di depan dada. Sedangkan lutut kaki belakang diletakkan di tanah dengan posisi di depan jari-jari kaki depan.

Start menengah. Start menengah adalah posisi start jongkok yang dilakukan dengan kedua kaki menapak pada papan start. Lutut kaki depan ditekuk dan di posisikan di depan dada. Sedangkan lutut kaki belakang diletakkan di tanah dan sejajar dengan tumit kaki depan.

Start panjang. Start panjang adalah posisi start jongkok yang dilakukan dengan kedua kaki menapak pada papan start. Lutut kaki depan ditekuk dan di posisikan di depan dada. Sedangkan lutut kaki belakang diletakkan di tanah dan berjarak 1 jengkal dari tumit kaki belakang.
