Jawaban:

Mainboard atau Motherboard. Processor (CPU) Harddisk (HDD) RAM (Random Access Memory) Optical drive (CD / DVD) Floppy Disk. VGA CARD (Kartu Grafis) Sound Card.

Penjelasan:

Motherboard adalah bagian komputer yang paling utama karena berisi sistem BIOS (Basic input output system) , pengatur koneksi input-output(chipset), soket prosessor, soket memory (RAM),  soket kartu grafis (VGA card) dan soket kartu tambahan (additional cards seperti PCI, ISA). BIOS adalah bagian utama yang mengatur sistem input output pada komputer. Bagian yang termasuk input seperti : Keyboard dan mouse. Bagian yang termasuk output seperti Monitor dan printer. Prosessor adalah otak sentral dari sebuah komputer. Prosessor adalah yang mengerjakan semua perintah yang sudah terprogram dan disimpan dalam harddisk. Dalam prosessor dikenal istilah frekuensi clock, yaitu kecepatan sebuah prosessor untuk mengerjakan perintah program dalam satu detik. Satuan frekuensi Clock dinyatakan dalam Herts (Hz). Harddisk adalah media penyimpanan data permanen, jadi data tidak hilang meskipun listrik sudah dimatikan. Harddisk berisi sebuah cakram magnetik yang mampu menyimpan data. Ukuran harddisk dinyatakan dalam Byte (B), contoh: 160GB (160 milyar byte). Harddisk ditemukan  pertama kali oleh Reynold Johnson di tahun 1956. Harddisk pertama berukuran 4.4 MB.RAM adalah unit penyimpan data tidak permanen artinya data dalam RAM akan hilang jika listrik mati. Ukuran data RAM dinyatakan dengan Byte (B) dan kecepatan akses RAM dinyatakan dengan Hertz (Hz). Jadi dalam RAM tidak cuma data saja, namun ada parameter lain yaitu kecepatan RAM. Kecepatan RAM harus sesuai dengan spesifikasi soket RAM pada motherboard. Contoh: RAM 512MB PC667 mempunyai ukuran data 512MB dengan kecepatan akses 667 MHz.Optical Drive adalah alat pembaca untuk media penyimpan data berupa disk DVD / CD. DVD/ CD berupa kepingan cakram optik yang berisi data. Floppy disk adalah media pembaca untuk disket.Ssaat ini mungkin disket sudah tidak populer. Namun diakui atau tidak disket turut memajukan dunia komputer selama beberapa dekade. VGA adalah singkatan dari Video Graphics Array. VGA Card berfungsi mengeluarkan output grafis (gambar) untuk ditampilkan pada monitor. Ukuran VGA Card ditentukan dari ukuran RAM nya, semakin besar RAM sebuah VGA Card maka semakin halus gambar yang dihasilkan.Sound Card adalah bagian yang mendekode data data digital menjadi sinyal suara. Dengan penemuan soundcard maka perkembangan dunia multimedia pada komputer menjadi makin meluas.

18. Jelaskan apa itu hardware dan sebutkan 5 komponen hardware di komputer.


Jawaban:

Hardware merupakan komponen fisik yang terdapat pada komputer

contoh 5 hardware pada komputer :

- CPU (central processing unit)

- GPU (Graphics processor unit)

- RAM (Random Access Memory)

- HDD (hard disk drive)

- Mouse

- Keyboard

- Monitor

- SSD (Solid State Drive)

dan lain-lain...

Penjelasan:

Hardware adalah komponen fisik yang diperlukan komputer agar bisa dijalankan sebagaimana mestinya. Komponen hardware pada komputer ada banyak sekali dan berbagai macam fungsi. Berikut beberapa contoh hardware komputer dan fungsinya.

- CPU (Central Processing Unit)

Kalian sering dengar merk Int*l, AMD, dan kawan-kawannya? Ya, itulah CPU (central processing unit). Dari namanya, CPU merupakan pusat segala kegiatan yang dilakukan pada komputer, seperti mengetik, menggerakkan krusor mouse, menonton video, dsb. CPU ini berfungsi layaknya otak manusia, ialah yang memproses semua yang terjadi di komputer kalian.

- GPU (Graphics Processor Unit)

Sering dengar kata "RTX" ?, atau mungkin ada yang membicarakan semisal GTX seri sekian, RTX seri sekian. Itu adalah GPU (Graphics Processor Unit). GPU berfungsi sebagai kartu tambahan untuk membantu kinerja CPU dalam melakukan rendering, bermain game yang memerlukan process 3D render, mengedit video, membuat proyek, dsb.

- RAM (Random Access Memory)
Kalian sering perdebatkan kalau semakin banyak RAM, maka semakin bagus performa? Nyatanya, jawaban ini bisa jadi iya, bisa jadi tidak. Dikarenakan RAM memiliki fungsi untuk menyimpan data sementara dari program/aplikasi yang sedang kalian jalankan. Mengapa jawaban dari perdebatan itu adalah "iya", dikarenakan penggunaan RAM sendiri tergantung kebutuhan kita. Apabila kita sering melakukan multitasking, semisal membuka chrome, sambil mengerjakan tugas, sambil bermain game, maka kita memerlukan RAM yang besar untuk memenuhi kebutuhan tersebut agar tidak terjadi stutter atau bahkan crash. Karena semakin banyak program/aplikasi yg dijalankan sekaligus, memerlukan RAM yg semakin banyak juga. Sebaliknya, jika kita hanya menggunakan untuk satu program dalam satu waktu, kita tidak perlu memerlukan banyak RAM. Jadi, penggunaan RAM pada intinya disesuaikan dengan penggunaan.

- HDD (Hard Disk Drive)

Merupakan penyimpanan pada Komputer kalian. Tanpa ini, juga kalian tidak bisa menggunakan komputer kalian karena tidak ada tempat untuk menempatkan OS (Operating System) pada komputer kalian. Biasanya HDD memiliki ukuran yang besar, seperti 512GB, 1TB, 2TB, dst.

- Mouse

Tentu kalian sudah tau, bahwasannya mouse digunakan untuk menunjuk pada apa yang akan kita lakukan pada komputer kita, mengarahkan, dan melakukan sesuatu.

- Keyboard

Tentu kalian juga sudah tau, bahwasannya keyboard digunakan untuk mengetik pada komputer. Sebenarnya tanpa keyboard fisik, kalian tetap bisa mengetik menggunakan "On-screen keyboard". Akan tetapi pasti lebih mudah untuk menggunakan keyboard fisik daripada keyboard virtual. seperti waifu kalian yg virtual, hehe canda ;)

- monitor

Berfungsi untuk menunjukkan apa yang sedang kalian lakukan di komputer kalian. Tanpa monitor, sebenarnya komputer kalian tetap bisa berfungsi seperti biasa. Akan tetapi apakah kalian tetap bisa menggunakan komputer kalian dengan benar tanpa tahu apa yang sedang kalian klik, atau ketik karena tidak bisa melihatnya tanpa adanya monitor?

- SSD (solid state drive)

SSD tidak jauh beda dengan HDD, hanya saja SSD memiliki kecepatan write dan read lebih cepat, hingga 1GB/detik, bahkan pada SSD yang lebih mahal bisa mencapai 10GB/detik. SSD memiliki harga yang relatif mahal dikarenakan kecepatan transfer datanya tersebut. Karena itu, SSD memiliki penyimpanan yang tidak sebanyak HDD.
