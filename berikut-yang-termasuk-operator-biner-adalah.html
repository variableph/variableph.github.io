4. tanda > termasuk operator ... *


Materi: Pemrograman - Operator

Jawaban:

Tanda > termasuk operator pembanding (comparison operator).

Penjelasan:

Dalam bahasa pemrograman, tanda > digunakan sebagai operator pembanding berbagai kondisi atau variabel. Tanda > termasuk dalam operator pembanding atau disebut juga comparison operator.

Contoh penerapannya pada C++:

//Program dimulai

#include <iostream>

int main() {

   int angka1 = 15;

   int angka2 = 20;

   cout << (angka2 > angka1); //Menghasilkan output true/benar

   return 0;

}

//Program selesai

13. Yang termasuk operator biner​


Jawaban:

1. Operator Aritmatika

adalah operator yang digunakan untuk melakukan operasi aritmatika seperti penjumlahan, pengurangan, perkalian, bembagian, dll

2. Operator Logika

adalah operator yang digunakan untuk melakukan operasi dimana nilai yangdihasilkan dari operasi tersebut hanya berupa nilai benar (true) atau salah (false). Nilai ini disebut dengan nilai boolean.

3. Operator Relasional

merupakan operator yang digunakan untuk menentukan relasi atau hubungan dari dua buah operand.

4. Operator Bitwise

merupakan operator yang digunakan untuk melakukan operasi-operasi yang berhubungan dengan manipulasi bit. Operator bitwise hanya dapat dilakukan pada operand yang bertipe char dan int saja karena ini berkoresponden dengan tipe byte dan word didalam bit.

15. apa saja yang termasuk operator perbanadingan


berikut ini simbol-simbol operator perbandingan (dalam MS. Excel):
= (sama dengan)
< (lebih kecil dari)
> (lebih besar dari)

>= (lebih dari sama dengan)

1. sama dengan: =
2. tidak sama dengan: <>
3. kurang dari: <
4. lebih dari: >
5. kurang dari atau sama dengan: <=
6. lebih dari atau sama dengan: >=
