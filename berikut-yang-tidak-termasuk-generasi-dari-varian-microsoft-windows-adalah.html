Jawaban:

1. tekan tombol windows

2. ketik dxdiag

3. Klik dxdiag

3. akan muncul aplikasi dxdiag ( DirectX Diagnostic Tool )

Penjelasan:

DxDiag, kependekan dari DirectX Diagnostics, adalah layanan yang digunakan untuk menguji fungsionalitas DirectX dan memecahkan masalah video atau masalah perangkat keras terkait suara di OS Windows

#Semoga Membantu

Jawaban singkat:

Langkah 1: tekan tombol Window & tombol R di keyboard (bersamaan)

Langkah 2: saat tampil kotak dialog Run, ketikkan dxdiag

Langkah 3: tekan tombol Enter di keyboard

Penjelasan tambahan:

Langkah 1: tombol Window & tombol keyboard bisa dilihat di lampiran (pada gambar keyboard/ dalam kotak kuning)

Langkah 2: kotak dialog Run, ditunjukkan tanda panah merah

Pengetikan dxdiag ditunjukkan tanda panah biru

Langkah 3: tombol Enter bisa dilihat di lampiran (pada gambar keyboard/ dalam kotak pink)

Bila langkah ketiga sudah usai, maka tunggu munculnya tampilan seperti pada gambar soal.
