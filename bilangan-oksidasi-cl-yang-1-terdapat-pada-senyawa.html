10. tentukan bilangan oksidasi Cl dalam senyawa-senyawa KCLO


Bilangan oksidasi Cl dalam KClO adalah +1. Bilangan oksidasi K dalam KClO adalah +1. Bilangan oksidasi O dalam KClO adalah -2

Pembahasan

∴ Bilangan oksidasi Cl pada KClO adalah +1.

Untuk memperoleh bilangan oksidasi ( biloks) atom Cl dapat dilakukan dengan subtitusi dari bilangan oksidasi ( biloks) yang sudah diketahui pada senyawa KClO

∵Biloks total KClO adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  K adalah +1 ( karena bilangan oksidasi untuk unsur golongan IA ( Li, Na, K, Rb, Cs, Fr) adalah +1)

KClO = 0

(jumlah atom K x biloks atom K) + (jumlah atom Cl x biloks atom Cl) + (jumlah atom O x biloks atom O) = 0

(1 x 1) + (1 x biloks Cl) + (1x (-2)) = 0

1 + Biloks S - 2 = 0

Biloks Cl= 2-1

Biloks Cl = +1
