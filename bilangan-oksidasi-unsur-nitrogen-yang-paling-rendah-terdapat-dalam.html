19. Unsur nitrogen dapat menyusun beberapa senyawa dengan bilangan oksidasi yang berbeda. Urutan bilangan oksidasi unsur nitrogen dalam NH3, NO, KNO3, HNO2,NO2 dan N2 dari yang rendah


Pertanyaan yang diajukan adalah Unsur nitrogen dapat menyusun beberapa senyawa dengan bilangan oksidasi yang berbeda. Urutan bilangan oksidasi unsur nitrogen dalam NH₃, NO, KNO₃, HNO₂,NO₂ dan N₂ dari yang rendah

Jawaban

NH₃ , unsur hidrogen dalam senyawa bernilai = +1, sehingga unsur N dalam senyawa tersebut memiliki bilangan oksidasi = -3

NO, unsur oksigen dalam senyawa bernilai = -2, sehingga unsur N dalam senyawa tersebut memiliki bilangan oksidasi = +2

KNO₃ ⇄ K⁺ + NO₃⁻, bilangan oksidasi K = +1  

NO₃⁻, unsur oksigen dalam senyawa bernilai = -2, dan senyawa NO₃⁻ memiliki muatan total sebesar -1, sehingga unsur N dalam senyawa tersebut memiliki bilangan oksidasi = +3  

NO₃ = -1  

N + 3.(-2) = -1  

N = +5  

HNO₂  ⇄ H⁺ + NO₂⁻, bilangan oksidasi H = +1  

NO₂⁻, unsur oksigen dalam senyawa bernilai = -2, dan senyawa NO₃⁻ memiliki muatan total sebesar -1, sehingga unsur N dalam senyawa tersebut memiliki bilangan oksidasi = +3  

NO₂ = -1  

N + 2.(-2) = -1  

N = +3

NO₂, unsur oksigen dalam senyawa bernilai = -2, sehingga unsur N dalam senyawa tersebut memiliki bilangan oksidasi = +4

N₂ = 0, karena dalam bentuk molekul bebas.

Ururan Nitrogen yang memiliki bilangan oksidasi paling rendah NH₃< N₂ < NO < NO₂⁻ < NO₂ < NO₃⁻

Pembahasan

Bilangan Oksidasi atau yang biasa disingkat biloks didefinisikan sebagai jumlah muatan negatif dan positif dalam atom, yang secara tidak langsung menandakan jumlah elektron yang telah diterima atau diserahkan.  

ATURAN PENENTUAN BILANGAN OKSIDASI  

Dalam memudahkan penentuan bilangan oksidasi, harus dipahami terlebih dahulu mengenai aturan penentuan bilangan oksidasi yang dijabarkan sebagai berikut :  

1. Bilangan oksidasi untuk unsur - unsur bebas (dalam bentuk atom, atau molekul unsur) adalah 0 (nol).  

Unsur bebas berbentuk atom contohnya Ca = 0, Na = 0, Al = 0  Unsur bebas dalam bentuk molekul contohnya H₂ = 0; O₂ = 0 ; Cl₂ = 0  

2. Bilangan oksidasi logam dalam suatu senyawa bernilai positif. Nilainya bergantung pada jenis logam.  

Contohnya bilangan oksidasi Na pada NaCl, NaOH, Na₂SO₄ = +1  

3. Bilangan oksidasi unsur golongan VIA (O, S, Se, Te, Po) yang berikatan pada senyawa biner adalah -2, dan unsur golongan VIIA (F, Cl, Br, I, At) yang berikatan dalam senyawa adalah -1.’  

Bilangan oksidasi unsur Cl pada KCl, BaCl₂ adalah -1.  Bilangan oksidasi unsur S pada H₂S dan BaS adalah -2.  

4. Bilangan oksidasi unsur H pada bentuk senyawanya adalah +1.  

5. Bilangan oksidasi unsur O pada bentuk senyawanya adalah -2, kecuali  

Dalam senyawa yang berikatan dengan F, bilangan oksidasinya adalah +2.  Dalam senyawa peroksida, seperti H₂O₂, Na₂O₂ dan BaO₂ , bilangan oksidasinya adalah -1.  Dalam senyawa superoksida, seperti KO₂ dan NaO₂, bilangan oksidasinya adalah -½.  

6. Untuk atom lain yang memiliki deret bilangan oksidasi yang berbeda dapat ditentukan melalui perhitungan dari data aturan bilangan oksidasi yang sudah diketahui.

22. bilangan oksidasi unsur nitrogen dalam senyawa dinitrogen pentaoksida adalah


Bilangan Oksidasi unsur nitrogen dalam senyawa dinitrogen pentaoksida  ( [tex]N_{2}O_{5}[/tex] ) adalah +5. Bilangan oksidasi disebut juga biloks.

Pembahasan

∴ Biloks N dalam  [tex]N_{2}O_{5}[/tex] adalah +5.

Diperoleh dari subtitusi biloks yang sudah diketahui pada  [tex]N_{2}O_{5}[/tex] untuk memperoleh biloks [tex]N[/tex]

∵Biloks total  [tex]N_{2}O_{5}[/tex]  adalah 0 ( aturan biloks no 2), )dan biloks O adalah -2 ( aturan biloks no 9)

    Biloks    [tex]N_{2}O_{5}[/tex] = 0

    (jumlah atom N x biloks N) + (jumlah atom O x biloks atom O)  = 0

    (2 x biloks N) + ( 5 x (-2)) = 0

    2 Biloks N - 10 = 0

    Biloks Mn = [tex]\frac{10}{2}[/tex]

    Biloks Mn = +5

30. unsur nitrogen dapat menyusun beberapa senyawa dengan bilangan oksidasi yang berbeda. Urutkan bilangan oksidasi unsur nitrogen dalam NH3, NO, KNO3, HNO2, NO2, dan N2 dari yang terendah


NH3, N2 , NO, HNO2, NO2 , KNO3NH₃
biloks H = +1 (umumnya)
biloks N + 3 biloks H = biloks NH₃
bilks N + 3(+1) = 0
biloks N = -3

NO
biloks O = -2 (umumnya)
biloks N + biloks O = biloks NO
biloks N + (-2) = 0
biloks N = +2

KNO₃
biloks K = +1 (karena masuk golongan logam alkali)
biloks O = -2

biloks N = +4

HNO₂
biloks H = +1
biloks O = -2
biloks H + biloks N + 2 biloks O = biloks HNO₂
(+1) + biloks N + 2(-2) = 0
biloks N = +3

NO₂
biloks O = -2
biloks N + 2 biloks O = biloks NO₂
biloks N + 2(-2) = 0
biloks N = +4

biloks N₂ = 0 (karena merupakan unsur bebas)
