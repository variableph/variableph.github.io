Pertanyaan:

Diketahui:

Lynn berjalan menuruni anak tangga menuju sebuah ruangan untuk mengikuti kelas bahasa Inggris. Untuk sampai ke ruangan tersebut, ia harus menuruni 12 anak tangga. Ketika sampai di depan ruangan, ia baru sadar bolpoinnya terjatuh di anak tangga yang dituruninya tadi, Akhirnya Lynn memutuskan menaiki 7 anak tangga kembali untuk mengambil bolpoinnya.

Ditanya:

Soal: 1

Nyatakan benar atau salah pertanyaan berikut sesuai dengan bacaan diatas

Posisi Lynn sebelum menuju ke kelas bahasa Inggris berada di lantai dasar

= benar/salah

Bolpoin Lynn jatuh pada tangga ke lima

= benar/salah

Posisi Lynn saat menaiki tangga dinyatakan dengan bilangan - 7

= benar/salah

Untuk menentukan posisi Lynn dalam mengambil bolpoin yang jatuh dinyatakan dengan operasi hitung

-12 + 7 = -5

= benar/salah

_____________________

Soal: 2

Buatlah garis bilangan yang menunjukkan letak jatuhnya bolpoin Lynn tersebut

Cara penyelesaian:

Soal no 1.

--------------------------------------------

bag A.

Posisi Lynn sebelum menuju ke kelas bahasa Inggris berada di lantai dasar

= benar/salah

Jawaban: salah.

Pembuktian: kata "Lynn berjalan menuruni anak tangga" menunjukkan bahwa Lynn berada pada lantai atas.

--------------------------------------------

bag B.

Bolpoin Lynn jatuh pada tangga ke lima

= benar/salah

Jawaban: Benar.

Pembuktian:

→ Lynn menuruni 12 anak tangga

→ ia baru sadar bolpoinnya terjatuh

→ Akhirnya Lynn memutuskan menaiki 7

maka :

12 - 7 = 5 ( anak tangga ke 5 )

--------------------------------------------

bag C.

Posisi Lynn saat menaiki tangga dinyatakan dengan bilangan - 7

= benar/salah

Jawaban: salah

Pembuktian:

→ Posisi awal Lynn dinyatakan sebagai Bilangan Nol ( 0 )

→ Turun tangga dinyatakan sebagai Bilangan Negatif ( - )

→ Naik tangga dinyatakan sebagai Bilangan Positif ( + )

maka:

Posisi Lynn saat menaiki tangga dinyatakan dengan bilangan 7 ( positif )

--------------------------------------------

bag D.

Untuk menentukan posisi Lynn dalam mengambil bolpoin yang jatuh dinyatakan dengan operasi hitung

-12 + 7 = -5

= benar/salah

Jawaban: benar.

Pembuktian:

→ Posisi awal Lynn dinyatakan sebagai Bilangan Nol ( 0 )

→Turun tangga dinyatakan sebagai Bilangan Negatif ( - )

→ Naik tangga dinyatakan sebagai Bilangan Positif ( + )

maka:

-12 + 7 = -5

--------------------------------------------

Soal no 2.

ada pada gambar.
