16. sandang keret minangka sesukih cakra lan pepet tulodo kreteg​

Pembahasan

Sandhangan wyanjana kagolong ana 3 :

1. Cakra = sesulih ra ➡️ ꦿ

➡️➡️ tuladha :

Supra = ꦱꦥꦿCakra = ꦕꦏꦿGatra = ꦒꦠꦿSatriya = ꦱꦠꦿꦶꦪ

2. keret = sesulih cakra lan pepet ➡️ ꦽ

➡️➡️ tuladha :

sregep = ꦱꦽꦒꦼꦥ꧀

kremi = ꦏꦽꦩꦶ

greget = ꦒꦽꦒꦼꦠ꧀

trenggiling = ꦠꦽꦁꦒꦶꦭꦶꦁ

3. pengkal = sesulih ya ➡️ ꦾ

➡️➡️ tuladha :

setya = ꦱꦼꦠꦾ

madya = ꦩꦢꦾ

kyai = ꦏꦾꦲꦶ

Bagya = ꦧꦒꦾ

Jawaban:
