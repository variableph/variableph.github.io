25. cara melihat buku kerja di halaman web​


Jawaban:

Menyimpan semua atau sebagian buku kerja ke halaman Web statis

Excel untuk Microsoft 365 Lainnya...

Anda dapat menyimpan seluruh buku kerja, termasuk tab untuk menavigasi lembar kerja, atau sebagian buku kerja, seperti rentang atau bagan, untuk halaman Web statis sehingga pengguna dapat menampilkan data Microsoft Office Excel di World Wide Web. Sebagai contoh, jika memiliki angka penjualan di lembar kerja Excel, Anda dapat menyimpan angka penjualan bersama dengan bagan untuk membandingkan angka tersebut, dan pengguna dapat menampilkan informasi tanpa harus membuka Office Excel.

Pelajari selengkapnya tentang menyimpan data Excel ke halaman Web statis

Ketika menyimpan data Excel ke halaman Web statis, sangatlah penting untuk memahami fitur Penerbitan Ulang Otomatis, tempat file pendukung berada, cara menentukan hyperlink, dan perbedaan antara membuka kembali halaman Web yang disimpan dengan membuka kembali buku kerja asli.

Menggunakan fitur Penerbitan Ulang Otomatis

Setiap kali menyimpan buku kerja, Anda dapat menerbitkan ulang buku kerja secara otomatis ke Web dengan memilih kotak Penerbitan Ulang Otomatis setiap saat buku kerja ini disimpan dalam kotak dialog Terbitkan sebagai Halaman Web. (Di tab File (Microsoft Office Tombol ￼ di Excel 2007), klik Simpan Sebagai,pilih format halaman Web (.htm, .html, .mht, .mhtml), lalu klik Terbitkan.)

Namun, mungkin ada kalanya Anda tidak ingin menerbitkan ulang item dalam buku kerja secara otomatis, misalnya, ketika:

Telah memasukkan informasi rahasia atau sensitif yang seharusnya tidak disimpan ke lokasi publik

Untuk sementara waktu terputus dari jaringan atau lokasi Server web

Menyimpan buku kerja yang dibuat oleh seseorang yang bukan merupakan sumber tepercaya (Buku kerja dapat berisi makro yang mengumpulkan informasi dari lokasi jaringan menggunakan kredensial Anda, lalu menyimpan informasi tersebut ke lokasi publik.)

Jika diperlukan, Anda dapat membuat Penerbitan Ulang Otomatis tidak tersedia sementara waktu (untuk sesi buka dan tutup saat ini) atau secara permanen tidak tersedia (setiap saat Anda membuka, menyimpan, dan menutup).

Tentang file Web pendukung dan hyperlink

Halaman Web sering kali berisi gambar dan file pendukung lainnya serta hyperlink ke halaman Web lainnya. Ketika file pendukung atau halaman dipindahkan, halaman Web yang merujuk ke file dan halaman mungkin tidak lagi berfungsi dengan baik.

File pendukung

Ketika Anda menyimpan buku kerja atau lembar kerja Excel sebagai halaman Web (.htm, .html), Excel akan membuat folder pendukung bernama "file filename_" tempat menyimpan semua file pendukung halaman, seperti poin, tekstur latar belakang, dan grafik. Misalnya, jika file Anda dinamai Page1.htm, Excel subfolder bernama "file Page1_." Excel menetapkan nama, seperti nama image001.jpg dan image002.gif, ke file pendukung.

Jika memindahkan atau menyalin halaman Web ke lokasi lain, Anda juga harus memindahkan folder pendukung untuk mempertahankan semua tautan ke halaman Web. Ketika menyimpan kembali ke lokasi lain, Excel secara otomatis menyalin folder pendukung untuk Anda.

Misalnya, anggap saja Anda memiliki halaman Web: http://contoh.microsoft.com/Halaman1.htm. Halaman ini menyertakan poin, yang disimpan di folder pendukung: http://contoh.microsoft.com/file_ Halaman1. Jalur relatif untuk file poin adalah /file_ Halaman1/gambar001.gif dan /file_ Halaman1/gambar002.gif. Jika memindahkan Halaman1.htm ke lokasi baru, seperti http://contoh.microsoft.com/, Anda juga harus memindahkan folder file pendukung (file_Halaman1) ke http://contoh.microsoft.com/.

Hyperlink absolut dan relatif

Ketika membuat halaman Web, Excel secara otomatis mengelola file dan hyperlink terkait sehingga gambar muncul dan tautan berfungsi ketika halaman ditempatkan di Server web akhir.

Ketika semua file, seperti poin, tombol navigasi, tekstur latar belakang, grafik, dan halaman Web yang dituju oleh Hyperlink yang Anda buat, ditempatkan di, atau disimpan ke, server Web yang sama, Excel mempertahankan tautan sebagai tautan relatif.

Hyperlink ke situs Web di server lain, misalnya daftar tempat favorit Anda di Web, dipertahankan sebagai tautan absolut, yaitu lokasi file tetap.

Ketika menyimpan halaman Web ke lokasi berbeda, tautan yang tidak dapat diubah menjadi tautan relatif tetap sebagai tautan absolut.

Ketika menghapus elemen yang disimpan sebagai tautan relatif, Excel secara otomatis menghapus file pendukung yang terkait dari folder pendukung.

Nama bahasa-spesifik untuk file pendukung halaman Web

Akhiran muncul dalam bahasa versi Microsoft Office Sytem yang digunakan untuk menyimpan file sebagai halaman Web. Misalnya, anggap saja Anda menggunakan versi bahasa Belanda Microsoft Office System untuk menyimpan file bernama Halaman1 sebagai halaman Web. Nama default folder pendukung akan menjadi Halaman1_bestanden. Bagian berikut berisi daftar lengkap bahasa dan nama folder default.
