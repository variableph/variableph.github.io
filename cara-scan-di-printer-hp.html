1. bagaimana cara scan di printer mp 237


Bagi yang masih pemula menggunakan printer, mungkin hal yang satu ini pernah anda alami, bingung bagaimana cara melakukan scaning/scanner pada printter mp237. Adapun langkah-langkah untuk memulai : 1. pastikan driver mp237 anda sudah ter install dengan lengkap 2. nyalakan printer anda 3. tempatkan dokumen pada printer yang mau anda scan 4. klik start -> all programs -> canon utilities -> IJ scan utilities 5. setelah program terbuka, silahkan pilih mau set yang mana. kalo tidak mau ribet pilih aja auto. 6. setelah selesai proses, file akan otomatis ter simpan di my document. Berhasil...

10. Cara scan soal tanpa ngetik


Mengikuti perkembangan teknologi dan perangkatnya, scan bisa dilakukan tanpa mengetik. Cara melakukan scan tanpa mengetik adalah dengan cara menginstall beberapa pilihan aplikasi scanner pada smartphone dengan cara memfoto file yang akan di scan kemudian hasilnya akan langsung terproses.

Pembahasan

Untuk membuat arsip digital dari dokumen fisik, dokumen tersebut biasanya dipindai menggunakan pemindai yang terpasang pada komputer. Tetapi bagaimana jika pemindai anda rusak dan anda tidak dapat menggunakannya?Saat ini, alih-alih menggunakan ponsel atau ponsel pintar anda untuk memindai atau men scan dokumen, anda dapat menggunakan alternatif.

Tentunya hal ini membuat lebih hemat karena tidak perlu membeli scanner baru atau mencari teman yang memilikinya. Smartphone Android saat ini menyertakan aplikasi Google Drive sebagai bagian dari sistem operasi standar. Ternyata selain menyimpan file di Google cloud, aplikasi ini juga bisa kamu gunakan untuk memindai atau memindai dokumen.

Dokumen fisik jarang digunakan saat ini. Untuk aplikasi, banyak dokumen fisik dialihkan ke dokumen digital atau online karena berbagai alasan, salah satunya adalah kenyamanan. Namun, memindai dokumen atau scan bisa sangat merepotkan jika anda masih menggunakan perangkat keras OCR. Untuk mendapatkan hasil pemindaian terbaik, anda tidak hanya harus membeli perangkat, tetapi juga memahami prosedur pemindaian dengan cermat. Oleh karena itu, aplikasi pemindaian dokumen dimaksudkan untuk membantu anda dengan mudah memindai dokumen anda menjadi file digital. Bahkan, file tersebut dapat diedit sebelumnya untuk memasukkan informasi yang lebih lengkap jika diinginkan.
