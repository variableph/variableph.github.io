2. bagaimana cara paling mudah buat setting DNS server?


Perangkat lunak server nama yang paling umum, disebut bernama ("nama dee") disertakan dengan BIND, Domain Nama Internet Berkeley, yang juga mencakup pustaka penyelesai dan alat lainnya. Seperti tulisan ini BIND hingga versi 9.4.2. Versi 8.3.1 juga digunakan secara luas.

Alternatif utama untuk BIND adalah djbdns, sebuah paket dari pencipta qmail. Ini dirancang untuk modularitas dan keamanan (penulis memiliki tawaran tetap $ 500 kepada siapa saja yang menemukan lubang keamanan dalam perangkat lunak, yang telah tidak diklaim sejauh ini). Ini lebih kecil dan lebih cepat dari BIND, tetapi lisensi mengendalikan kode sumber lebih ketat, dan djbdns mendapat banyak kritik untuk itu. Ada perdebatan yang memikat tentang keutamaan kedua sistem tersebut dalam arsip milis pengguna BIND.

Dalam tutorial ini, saya berbicara banyak tentang BIND, meskipun saya benar-benar menganjurkan djbdns sebagai alternatif. Berikut ini panduan untuk beralih dari BIND ke djbdns.

Agaknya Anda memiliki, paling tidak, nama domain dan alamat IP, dan Anda ingin yang menunjuk ke yang lain. Jika Anda belum mencapai titik itu, Anda mungkin ingin berbicara dengan ISP untuk melihat tentang mendapatkan alamat IP, dan registrar untuk nama domain (Jika Anda butuh bantuan, konsultasikan dengan kami Pilih dan Daftarkan Domain Nametutorial).

Pertama lihat apakah Anda sudah MENGGUNAKAN BINDING pada sistem Anda. Mungkin ada di sana, di belakang layar yang tidak terduga. Ketik bernama -v pada baris perintah. Jika mengembalikan respons yang memberi tahu Anda versi BIND yang Anda jalankan, itu berarti Anda memilikinya di sistem Anda, dan tidak perlu menginstalnya - meskipun BIND adalah target utama serangan, dan jika Anda tidak memiliki versi terbaru Anda mungkin membiarkan diri Anda terbuka terhadap risiko keamanan.

Jika tidak, unduh sumber dan unzip, lalu instal dengan ./configure; buat; buat instal. Instruksi lengkap tentang konfigurasi build untuk sistem Anda tersedia dalam file INSTALL; di sini adalah langkah-langkah untuk Mac OS X.

Setelah menginstal BIND, Anda harus mengonfigurasinya. Kami tidak memiliki ruang di sini untuk masuk ke semua perubahan konfigurasi BIND. Ada seni untuk menyempurnakan segalanya sesuai keinginan Anda - heck, ada satu buku utuh tentang topik itu.

DNS howto akan memberi tahu Anda cara mengatur file named.conf, yang berisi informasi konfigurasi untuk server nama. File-file data zona yang berisi catatan DNS yang kita bahas - biasanya, satu untuk penggunaan lokal dan satu lagi untuk penggunaan eksternal - ditempatkan dalam direktori yang ditentukan. Maka sistem Anda harus diberitahu untuk menggunakan server nama yang Anda buat. Bagaimanapun, cara-cara di tldp.org menjelaskan semua itu lebih baik daripada yang saya bisa.

Dengan BIND diatur, server nama Anda siap untuk bergoyang. DNS how-to akan memberi tahu Anda cara mengatur file named.conf, yang berisi informasi konfigurasi untuk server nama. File harus mengatakan bahwa server nama Anda otoritatif untuk mydomain.dom untuk pencarian ke depan serta 40.168.192.in-addr.arpa untuk pencarian terbalik. File-file data zona yang berisi catatan DNS yang kami kunjungi biasanya, satu untuk penggunaan lokal dan yang lain untuk penggunaan eksternal ditempatkan dalam direktori yang ditentukan. Maka sistem Anda harus diberitahu untuk menggunakan server nama yang Anda buat. Bagaimanapun, cara menjelaskan semua itu lebih baik daripada yang saya bisa.Bayar orang yang terbiasa setting dns :)

8. 1. Sebutlah secara berurutan Cara-cara konek internet melalui NirkabelPada Perangkat Android!Jawaban Anda​


1. Melalui Jaringan

Cara memperoleh sambungan internet melalui jaringan maksudnya menghubungkan komputer Anda pada sebuah jaringan komputer yang telah terhubung dengan internet. Sebuah komputer dijadikan server dan terhubung dengan internet, kemudian komputer-komputer lain dalam jaringan dihubungkan dengan komputer server tersebut. Cara seperti ini umum digunakan di warnet, perusahaan dan kampus.

2. Melalui Jaringan TV Kabel

Umumnya mengakses internet menggunakan jaringan TV Kabel ini dapat Anda peroleh sepaket dengan berlangganan TV Kabel itu sendiri. Jaringan TV Kabel merupakan jaringan dengan komunikasi data yang lebar. Hal ini disebabkan TV Kabel harus melewati gambar bergerak dan suara jika digunakan internet. Itu sebabnya mengakses internet menggunakan jaringan TV Kabel akan membuat kita memperoleh akses internet yang cepat.

3. Melalui Dial Up Atau Saluran Telepon Langsung

Dial up merupakan salah satu cara memperoleh sambungan internet yang bisa Anda coba. Dial up adalah koneksi ke internet menggunakan jaringan telepon regular. Dial up biasanya cukup menggunakan komputer tunggal. Syaratnya Anda harus memiliki modem, kabel jaringan telepon serta ISP.

4. Melalui GPRS

GPRS atau General Packet Radio Service adalah koneksi internet menggunakan ponsel atau PDA (Personal Data Assistance) yang berfungsi sebagai modem melalui kabel data. Cara sambungan internet ini menggunakan operator ponsel sebagai ISP nya.

5. Melalui WI-FI

WI-FI atau wireless fidelity adalah cara memperoleh sambungan internet dengan teknologi tanpa jaringan kabel. WI-FI menggunakan frekuensi tinggi yaitu berada pada spectrum 2,4 GHz. Untuk terhubung dengan WI-FI, umumnya menggunakan sebuah notebook atau PDA yang sudah dilengkapi dengan kartu WI-FI.

Akses internet dengan WI-FI terbilang cukup cepat hingga kecepatan 11 mbps. Namun, syaratnya Anda harus berada pada wilayah atau lokasi yang terdapat sinyal WI-FI yaitu sejauh kurang lebih 100m dari titik akses.

6. Melalui V-SAT

Cara memperoleh sambungan internet yang bisa Anda coba berikutnya adalah melalui V-Sat. V-Sat menggunakan parabola yang diarahkan ke posisi satelit untuk mendapatkan sambungan internet. Koneksi ini merupakan solusi bagi mereka yang tinggal di daerah terpencil dan kesulitan mendapatkan jaringan telepon maupun ISDN.

7. Melalui Wireless Broadband (Wibro)

Akses internet menggunakan Wibro memungkinkan kita mengakses internet dengan jarak 1 km dari titik akses dan dengan kecepatan 512 kbps. Mengakses Wibro juga sukses dilakukan dalam kendaraan bergerak dengan kecepatan 60 km/jam.

8. USB Wireless Modem GSM/CDMA

Akses internet USB Wireless Modem adalah menggunakan perangkat USB Modem yang telah diisi dengan kartu seluler GSM atau CDMA.

11. Ada TP-LINK tl-wr840n.1. Sebutkan cara setting IP Network!2. Sebutkan cara setting IP LAN!3. Sebutkan cara setting routing!​


1. Cara setting IP Network pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router dengan mengetik alamat IP router (biasanya 192.168.0.1 atau 192.168.1.1) di URL.Masukkan username dan password default yang tercantum pada manual atau label pada router.Pilih "Network" atau "LAN" dari menu utama.Ubah pengaturan IP Network sesuai kebutuhan, misalnya dengan mengubah alamat IP, subnet mask, dan gateway.Klik "Save" atau "Apply" untuk menyimpan perubahan.

2. Cara setting IP LAN pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "LAN" dari menu utama.Ubah pengaturan IP LAN sesuai kebutuhan, misalnya dengan mengubah alamat IP dan subnet mask.Klik "Save" atau "Apply" untuk menyimpan perubahan.

3. Cara setting routing pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "Routing" dari menu utama.Pilih tipe routing yang dibutuhkan, seperti Static Routing atau Dynamic Routing.Masukkan informasi yang dibutuhkan untuk setiap tipe routing, misalnya destination IP, subnet mask, gateway, dan metric.Klik "Save" atau "Apply" untuk menyimpan perubahan.

note: Langkah-langkah ini mungkin berbeda tergantung dari versi firmware router. Pastikan untuk membaca manual pengguna atau dokumentasi produk untuk informasi lebih rinci.

21. apa kepanjangan dan pengertian dari : 1. DNS 2. DHCP 3. proxy


1.Pengertian Mail Server

Adalah Perangkat lunak program yang mendistribusikan file atau informasi sebagai respons atas permintaan yang dikirim via email, juga digunakan pada bitnet untuk menyediakan layanan serupa http://FTP.

2.Pengertian DHCP Server

DHCP merupakan singkatan dari Dinamyc Host Configuration Protocol adalah sebuah layanan yang secara otomatis memberikan nomor IP kepada komputer yang memintanya. komputer yang memberikan nomor IP inilah yang disebut sebagai DHCP server, sedangkan komputer yang melakukan request disebut DHCP Client.

fungsi DHCP

Seperti yang sudah diterangkan. fungsi DHCP ini adalah dapat memberikan nomor IP secara otomatis kepada komputer yang melakukan request.

3.Pengertian Web Server

Web Server adalah sebuah perangkat lunak server yang berfungsi menerima permintaan HTTP atau HTTPS dari klien yang dikenal dengan web browser dan mengirimkan kembali hasilnya dalam bentuk halaman-halaman web yang umumnya berbentuk dokumen HTML. Salah satu server web yang terkenal di linux adalah

Apache. Apache merupakan server web antar platform yang dapat berjalan di beberapa platform seperti linux dan windows. Web Server juga merupakan sebuah komputer yang menyediakan layanan untuk internet. Server disebut juga dengan host. Agar anda dapat memasukkan web yang anda rancang ke dalam

internet, maka anda harus memiliki ruangan terlebih dahulu dalam internet, dan ruangan ini disediakan oleh server. Itulah yang disebut Web Server..

4.Pengertian DNS Server

Domain Name System (DNS) adalah distribute database system yang digunakan untuk pencarian nama komputer (name resolution) di jaringan yang mengunakan TCP/IP (Transmission Control Protocol/Internet Protocol). DNS biasa digunakan pada aplikasi yang terhubung ke Internet seperti web browser atau e-mail, dimana DNS membantu memetakan host name sebuah komputer ke IP address. DNS dapat disamakan fungsinya dengan buku telepon. Dimana setiap komputer di jaringan Internet memiliki host name (nama komputer) dan Internet Protocol (IP) address. Secara umum, setiap client yang akan mengkoneksikan komputer yang satu ke komputer yang lain, akan menggunakan host name. Lalu komputer anda akan menghubungi DNS server untuk mencek host name yang anda minta tersebut berapa IP address-nya. IP address ini yang digunakan untuk mengkoneksikan komputer anda dengan komputer lainnya.

5.Pengertian FTP Server

FtP adalah File Transfer Protocol (FTP) adalah suatu protokol yang berfungsi untuk tukar-menukar file dalam suatu network yang menggunakan TCP koneksi bukan UDP.

Dua hal yang penting dalam FTP adalah FTP Server dan FTP Client.

FTP server adalah suatu server yang menjalankan software yang berfungsi untuk memberikan layanan tukar menukar file dimana server tersebut selalu siap memberikan layanan FTP apabila mendapat permintaan (request) dari FTP client.

FTP client adalah computer yang merequest koneksi ke FTP server untuk tujuan tukar menukar file. Setelah terhubung dengan FTP server, maka client dapat men-download, meng-upload, merename, men-delete, dll sesuai dengan permission yang diberikan oleh FTP server.

Tujuan dari FTP server adalah sebagai berikut :

• Untuk tujuan sharing data

• Untuk menyediakan indirect atau implicit remote computer

• Untuk menyediakan tempat penyimpanan bagi user

• Untuk menyediakan transfer data yang reliable dan efisien

6.Pengertian Proxy Server

Proxy server bekerja dengan menjembatani komputer ke Internet. Program Internet seperti browser, download manager dan lain-lain berhubungan dengan proxy server, dan proxy server tersebut yang akan berkomunikasi dengan server lain di Internet.

3. Alihkan tombol Wi-Fi ke "ON" . Tindakan ini akan mengaktifkan Wi-Fi.

4. Ketuk salah satu nama jaringan. Carilah nama jaringan Wi-Fi yang ingin digunakan untuk menghubungkan perangkat Android.

5. Masukkan kata sandi jaringan ketika diminta. Apabila Anda menggunakan jaringan rumah, tetapi belum menetapkan kata sandi, gunakan kata sandi Wi-Fi yang terdapat di belakang atau bagian bawah perute.

Apabila jaringan yang dipilih tidak mempunyai kata sandi, perangkat Anda secara otomatis akan terhubung setelah Anda mengetuk jaringan tersebut.

6. Ketuk Connect yang terletak di pojok kanan bawah. Jika Anda memasukkan kata sandi dengan benar, perangkat Android tersebut akan terhubung ke jaringan.
