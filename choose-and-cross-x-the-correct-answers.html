6. Choose and cross (x) the correct a1: I'm ... how to swim right now.A. learningB. learnedC. learnsD. learn​


Jawaban:

A. Learning

Pembahasan Present Continuous

Digunakan untuk menyatakan Kegiatan yg sedang berlangsung saat ini. Cirinya NOW, RIGHT NOW, TODAY dll. Rumusnya:

(+) S + is/am/are + V-ing + O(-) S + is/am/are + not + V-ing + O(?) is/am/are + S + V-ing + O?

Penjelasan

A. learning

=> Continuous

=> V-ing

B. learned

=> Past Tense

=> Verb 2

C. learns

=> Present Tense

=> Verb+s

D. learn

=> Present Tense

=> Verb 1

Kesimpulan

Karena ada ciri Present Continuous yaitu right now.

Maka:

(+) S + is/am/are + V-ing + O

=> I'm learning how to swim right now.

Subject => ITobe => am

Jadi I + am => I'm

Verb-ing => learn+ingComplement => how to swim right now

29. look at the picture and Choose the correct answer​


Jawaban:

D. There are three books on the table.

Penjelasan:

Are merupakan bentuk plural (jamak) dari is dan books merupakan bentuk plural (jamak) dari book

D. There are three books on the table

Penjelasan:

Perbedaan There is dan There are: There is digunakan untuk benda tunggal, sedangkan There are digunakan untuk benda jamak (lebih dari satu)

Perbedaan Book dan Books: Book dengan tambahan akhiran S (Books) untuk benda berakhiran jamak (lebih dari satu)
