Penjelasan:

Application Letter

Application letter adalah surat bisnis yang bertujuan untuk melamar pekerjaan kedalam sebuah perusahaan.

Generic Structure dari Application Letter adalah:

1. salutation : Berisi salam pembuka kepada HRD perusahaan atau orang yang bertanggung jawab dalam perekrutan.

2. introduction : Berisi pembukaan. Biasanya berbentuk perkenalan diri atau biodata pelamar.

3. main message or body: Berisi maksud dan tujuan penulis bahwa dia akan melamar sebuah pekerjaan dalam perusahaan tersebut.

4. the close : Berisi kalimat penutupan. Biasanya berbentuk harapan agar si penulis dapat diterima dalam perusahaan tersebut.

5. complimentary close: Berisi salam penutup dan ucapan terimakasih.

Language features dari Application Letter:

1. use the present tense

2. use the verbs to indicate actions and feelings

3. use the future tense for suggesting solutions to problems the job
