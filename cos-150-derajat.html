5. nilai tan 150 derajat cos 330 derajat + cos 150 derajat sin 240 derajat adalah


Nilai tan 150 derajat cos 330 derajat + cos 150 derajat sin 240 derajat adalah ¼. Sudut istimewa pada trigonometri terdiri dari 0⁰, 30⁰, 45⁰, 60⁰ dan 90⁰. Untuk menentukan nilai trigonometri apakah bernilai positif atau negatif tergantung dari kuadrannya. Nilai perbandingan trigonometri pada tiap kuadran  

Kuadran I (0ᵒ s.d 90ᵒ) ⇒ positif semua Kuadran II (90ᵒ s.d 180ᵒ) ⇒ yang bernilai positif hanya sinus dan cosecan Kuadran III (180ᵒ s.d 270ᵒ) ⇒ yang bernilai positif hanya tangen dan cotangen Kuadran IV (270ᵒ s.d 360ᵒ) ⇒ yang bernilai positif hanya kosinus dan secan

Sudut relasi yang digunakan

Kuadran I: (90⁰ – α) Kuadran II: (90⁰ + α) atau (180⁰ – α) Kuadran III: (180⁰ + α) atau (270⁰ – α) Kuadran IV: (270⁰ + α) atau (360⁰ – α) atau (–α)

Catatan

Jika kita gunakan sudut relasi 180⁰ atau 360⁰ maka trigonometrinya tetap

Jika kita gunakan sudut relasi 90⁰ atau 270⁰ maka trigonometrinya berubah, yaitu

sin berubah menjadi cos atau cos berubah menjadi sin tan berubah menjadi cot atau cot berubah menjadi tan cosec berubah menjadi sec atau sec berubah menjadi cosec
Pembahasan

Agar tidak merubah trigonometri, maka kita gunakan sudut relasi 180⁰ atau 360⁰

tan 150⁰ = tan (180⁰ – 30⁰) = –tan 30⁰ = –⅓ √3 (bernilai negatif karena dikuadran II) cos 330⁰ = cos (360⁰ – 30⁰) = cos 30⁰ = ½ √3 (bernilai positif karena dikuadran IV) cos 150⁰ = cos (180⁰ – 30⁰) = –cos 30⁰ = – ½ √3 (bernilai negatif karena dikuadran II) sin 240⁰ = sin (180⁰ + 60⁰) = –sin 60⁰ = – ½ √3 (bernilai negatif karena dikuadran III)

Jadi nilai dari: tan 150⁰ cos 330⁰ + cos 150⁰ sin 240⁰

= [tex](-\frac{1}{3} \sqrt{3}) \: . \: (\frac{1}{2} \sqrt{3}) + (-\frac{1}{2} \sqrt{3}) \: . \: (-\frac{1}{2} \sqrt{3})[/tex]

= [tex]-\frac{1}{2} + \frac{3}{4}[/tex]  

= [tex]-\frac{2}{4} + \frac{3}{4}[/tex]

= [tex]\frac{1}{4}[/tex]

11. Diketahui segitiga ABC dengan AB = 12 cm, AC = 8 cm dan luasnya 24 cm2. Besar cos ∠ adalah ... . Cos 60 derajat atau 150 derajat Cos 30 derajat Cos 30 derajat atau 120 derajat Cos 45 derajat .Cos 150 derajat


Jawab:

Sisi - sisi dan sudut - sudut yang belum diketahui pada segitiga ABC dapat dicari menggunakan aturan sinus dan kosinus trigonometri.

Pembahasan

Dalam trigonometri mempunyai hubungan erat dengan segitiga, baik itu berkaitan dengan keliling mauun luas. Segitiga yang memunyai sudut dan panjang sisi tertentu dapay dihitung keliling dan luasnya menggunakan aturan pada trigonometri. Dalam hal ini dikenal dengan aturan sinus dan aturan kosinus.

Segitiga ABC mempunyai sudut - sudut A, B dan C serta panjang sisi AB, BC dan AC yang dilambangkan dengan:

BC = a

AC = b

AB = c

Diperoleh aturan sinus dan kosinus sebagai berikut:

Aturan sinus

a/sin A = b/sin B = c/sin C

Aturan kosinus

a² = b² + c² - 2bc cos A

b² = a² + c² - 2ac cos B

c² = a² + b² - 2ab cos C

Dari soal a. diketahui:

a. AB = 18 cm, sudut B = 30°, sudut C= 45°

Ingat! besar sudut segitiga adalah 180°, maka besar sudut A adalah:

A + B + C = 180°

A + 30° + 45° = 180°

A + 75° = 180°

A = 180° - 75°

A = 105°

Maka besar sudut A = 105°

Dengan memakai aturan sinus diperoleh:

a/sin A = b/sin B = c/sin C

a/sin 105° = b/sin 30° = 18/sin 45°

Mencari panjang sisi AC

b/sin 30° = 18/sin 45°

b/(1/2) = 18/(1/2 √2)

b(1/2 √2) = 18(1/2)

b√2 = 18

b = 18/√2

b = 18/2 √2

b = 9√2

Maka panjang sisi AC adalah 9√2 cm.

Mencari panjang sisi BC

a/sin 105° = 18/sin 45°

a/sin 105° = 18/(1/2 √2)...................1)

Ingat! sin 105° bukan derajat istimewa, maka sin 105° dapat diubah menjadi:

sin 105° = sin (60° + 45°)

sin 105° = sin 60° cos 45° + cos 60° sin 45°

sin 105° = (1/2√3)(1/2 √2) + (1/2)(1/2 √2)

sin 105° = (1/4 √2)(√3 + 1)

Subtitusikan kembali nilai sin 105° ke persamaan 1)

a/sin 105° = 18/(1/2 √2)

a/(1/4 √2)(√3 + 1) = 18/(1/2 √2)

a(1/2 √2) = 18(1/4 √2)(√3 + 1)

a = 18(1/2)(√3 + 1)

a = 9(√3 + 1)

Maka panjang sisi BC adalah 9(√3 + 1) cm.

∴ Jadi dari soal A didapatkan sudut A = 105°, AC = 9√2 cm, BC = 9(√3 + 1) cm.

Dari soal b. diketahui:

b. BC = 16 cm, sudut A = 30°, sudut C= 120°

Ingat! besar sudut segitiga adalah 180°, maka besar sudut B adalah:

A + B + C = 180°

30° + B + 120° = 180°

B + 150° = 180°

B = 180° - 150°

B = 30°

Maka besar sudut B = 30°

Dengan memakai aturan sinus diperoleh:

a/sin A = b/sin B = c/sin C

16/sin 30° = b/sin 30° = c/sin 120°

Mencari panjang sisi AC

16/sin 30° = b/sin 30°

16/(1/2) = b/(1/2)

16(1/2) = b(1/2)

b = 16

Maka panjang sisi AC adalah 16 cm.

Mencari panjang sisi AB

16/sin 30° = c/sin 120°

16/(1/2) = c/(1/2 √3)

16(1/2 √3) = c(1/2)

16√3 = c

Maka panjang sisi AB adalah 16√3 cm.

∴ Jadi dari soal B didapatkan sudut B = 30°, AC = 16 cm, AB = 6√3 cm.

Dari soal c. diketahui:

c. BC = 12 cm, AC = 16 cm, sudut B = 120°

dengan aturan kosinus dapat diperoleh sebagai berikut:

b² = a² + c² - 2ac cos B

16² = 12² + c² - 2(12)c cos 120°

256 = 144 + c² - 24c(-1/2)

256 = c² + 12c + 144

c² + 12c + 144 - 256 = 0

c² + 12c - 112 = 0

Dengan menggunakan rumus abc diperoleh:

c = -6 ± 2√37

c = -6 + 2√37

Maka panjang AB = -6 + 2√37 cm.

Dengan aturan sinus diperoleh:

12/sin A = 16/sin 120° = (-6 + 2√37)/sin C

Mencari besar sudut A

12/sin A = 16/sin 120°

12/sin A = 16/(1/2 √3)

16 sin A = 12(1/2 √3)

16 sin A = 6√3

sin A = 6√3/16

sin A = 3√3/8

A = sin⁻¹(3√3/8)

A = 40,5°

Maka besar sudut A = 40,5°

Mencari besar sudut C

A + B + C = 180°

40,5° + 120° + C = 180°

160,5° + C = 180°

C = 180° - 160,5°

C = 19,5°

Maka besar sudut C = 19,5°

∴ Jadi dari soal C didapatkan sudut A = 40,5°, sudut C = 19,5° dan AB = -6 + 2√37 cm.

20. cos 150 derajat senilai dengan?


Cos 150 derajat senilai dengan - Cos 30 = [tex]-\frac{1}{2}\sqrt{3}}[/tex].

Pembahasan

Perbandingan sudut dalam trigonometri.

Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu :

Kuadran I : 0° < α < 90° (0 < α < π/2)

Sin (90° - α) = cos α

Cos (90° - α) = sin α

Tan (90° - α) = cotan α

Cotan (90° - α) = tan α

Secan (90° - α) = sec α

Cosec (90° - α) = cosec α

Kuadran II : 90° < α < 180° (π/2 < α < π)

Sin (180° - α) = sin α

Cos (180° - α) = -cos α

Tan (180° - α) = -tan α

Cotan (180° - α) = -cotan α

Secan (180° - α) = secan α

Cosec (180° - α) = -cosec α

Kuadran III : 180° < α < 270° (π < α < 3π/2)

Sin (270° - α) = -cos α

Cos (270° - α) = -sin α

Tan (270° - α) = cotan α

Cotan (270° - α) = tan α

Secan (270° - α) = -sec α

Cosec (270° - α) = -cosec α

Kuadran IV : 270° < α < 360° (3π/2 < α < 2π)

Sin (360° - α) = sin (-α) = -sin α

Cos (360° - α) = cos (-α) = cos α

Tan (360° - α) = tan (-α) = -tan α

Cotan (360° - α) = cotan (-α) = -cotan α

Secan (360° - α) = secan (-α) = -secan α

Cosec (360° - α)= sec (-α) = sec α
