2. ctrl p adalah shortcut key untuk memunculkan kotak dialog


Ctrl p adalah shortcut key untuk memunculkan kotak dialog print. Kotak dialog print juga dapat dimunculkan melalui Office Button – Print. Kotak dialog print dimunculkan saat kita hendak mencetak dokumen.

Pembahasan

Untuk melakukan pencetakan dokumen atau file yang telah kita buat, dapat dilakukan dengan langkah-langkah sebagai berikut:

Buka dokumen yang telah kita buat, jika belum silakan buatlah dokumen terlebih dahulu.Klik Office Butto pilih Print, atau tekan Ctrl + P pada keyoard.Kotak dialog print akan ditampilkan.Pada bagian "Name" pilih nama printer yang akan digunakan."Number of Copies", untuk menentukan jumlah salinan.

Print what untuk menentukan bagian dokumen yang akan dicetakAll pages in range untuk mencetak semua halaman dokumenOdd pages untuk mencetak halaman ganjil dokumen seperti 1,3,5.Even pages untuk mencetak halaman genap dokumen seperti 2,4,6.Properties untuk melakuakn setting ukuran kertas, bentuk pencetakan dan warna.

15. apa fungsi dari shortcut ctrl+[ dan ctrl+]


Menurut saya sih tidak ada fungsi nya... Karena shortcut "CTRL+" hanya bisa digunakan di CTRL+( A - Z ) & CTRL+( 1 - 9 ).Pada software Microsoft Office (Word, Excel, Powerpoint, dsb), shortcut Ctrl+[ berfungsi untuk untuk memperbesar ukuran huruf sebanyak 1 point/ukuran (misal sebuah text awalnya memiliki ukuran 11, jika diblok lalu kita tekan Ctrl+[, maka ukuran text tersebut akan bertambah jadi 12), sedangkan Ctrl+] berfungsi sebaliknya/untuk memperkecil ukuran huruf sebanyak 1 point/ukuran (misal awalnya berukuran 11, setelah kita tekan Ctrl+], maka ukurannya menjadi 10)
