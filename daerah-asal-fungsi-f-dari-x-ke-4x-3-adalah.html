____________________PEMBAHASAN

Okey, Fungsi atau bentuk lainnya f : x ⇒ y adalah suatu aturan yg memasangkan tiap anggota x pada suatu himpunan (daerah asal) dengan nilai y dari himpunan kedua (daerah kawan). Berikut 3 istilah penting dari Fungsi :

Daerah asal : DomainDaerah kawan : KodomainDaerah asal : Range

PENYELESAIAN

→ DIKETAHUI ←

Rumus fungsi = f(x)= 4x-1

Daerah asal = -3,-2,-1,0,1,2

→ DITANYAKAN←

Daerah hasil fungsi = ?

→ PENYELESAIAN ←

1. x = -3 ⇒ f (x) = 4 × -3 -1

                         = -12 - 1

                         = -13

2. x = -2 ⇒ f (x) = 4 × -2 -1

                          = -8 - 1

                          = -9

3. x = -1 ⇒ f (x) = 4 × -1 -1

                         = -4 - 1

                         = -5

4. x = 0 ⇒ f (x) = 4 × 0 -1

                         = 0 - 1

                         = -1

5. x = 1 ⇒ f (x) = 4 × 1 - 1

                        = 4 - 1

                        = 3

6. x = 2 ⇒ f (x) = 4 × 2 - 1

                         = 8 - 1

                         = 7

Jadi, Daerah hasil fungsinya adalah =

[ -13, -9, -5, -1, 3, 7 ]

28. Diketahui fungsi kuadrat f(x) = -2xkuadrat + 4x +3 dengan daerah asal {x|-2 < 2 < x <3.x £ R}


Untuk mengetahui nilai minimum dari fungsi kuadrat f(x) = -2x² + 4x + 3, caranya adalah dengan mencari akar-akar persamaan dari turunan pertama fungsi tersebut, yaitu:

f'(x) = -4x + 4 = 0

Jadi akar-akar persamaan dari turunan pertama fungsi tersebut adalah x = 1. Kemudian kita cek nilai minimum dari fungsi f(x) di x = 1 dengan menggunakan rumus f(x) = a(x - r)² + k, di mana r adalah akar-akar persamaan dari turunan pertama fungsi f(x), dan a, k adalah nilai pada fungsi f(x).

Kita bisa mengganti nilai x = 1 ke dalam fungsi f(x) untuk mengetahui nilai k, yaitu:

f(1) = -2(1 - 1)² + 4(1) + 3 = -2 + 4 + 3 = 5

Dengan demikian, kita bisa menuliskan fungsi f(x) dengan menggunakan rumus f(x) = a(x - r)² + k, yaitu:

f(x) = -2(x - 1)² + 5

Dari rumus di atas, kita bisa mengetahui bahwa nilai minimum dari fungsi kuadrat f(x) = -2x² + 4x + 3 adalah 5.
