27. tentukan daerah asal dan daerah hasil dari fungsi f(x)=x²+4x+4


daerah asal :

Df = {x ∈ R | f(x) ∈ R}
Df = {x ∈ R | x² + 4x + 4 ∈ R}

karena x dapat disubtitusi oleh semua bilangan real, maka :

Df = R
Df = (-∞, ∞)

==========

Daerah hasil :
Rf = {f(x) ∈ R | x ∈ Df}
Rf = {x² + 4x + 4 ∈ R | x ∈ R}

x² + 4x + 4 = (x + 2)²

karena (x + 2)² hasilnya pasti nonnegatif, maka daerah hasilnya adalah :

Rf = {f(x) | f(x) ≥ 0}
Rf = [0, ∞)
