Di dalam logika matematika ini ada yang namanya aljabar Boolean.  

Aljabar Boolean adalah suatu sistem aljabar yang berkaitan dengan operasi logika (seperti and, or, nor, nand) dan variabel-variabel biner yang bisa berupa huruf alfabet. Kali ini kita akan fokus pada operator logika and dan or ya!!

Operator logika and

Biasanya disimbolkan dengan tanda . atau tidak ada tanda nya sama sekali Jika pada kedua variabel berlogika 1, maka akan menghasilkan logika 1 juga

Operasi logika AND

Biasanya disimbolkan dengan tanda + atau v Jika pada kedua variabl berlogika 0, maka akan menghasilkan logika 0 juga

Nah untuk soal di atas kita cari dulu untuk yz

Misalnya y berlogika 1 dan z berlogika 1 maka akan menghasilkan yz berlogika 1 Misalnya y berlogika 1 dan z berlogika 0 maka akan menghasilkan yz berlogika 0 Misalnya y berlogika 0 dan z berlogika 1 maka akan menghasilkan yz berlogika 0 Misalnya y berlogika 0 dan z berlogika 0 maka akan menghasilkan yz berlogika 0

Misalnya x berlogika 1 dan yz berlogika 1 maka akan menghasilkan E = x v yz berlogika 1 Misalnya x berlogika 1 dan yz berlogika 0 maka akan menghasilkan E = x v yz berlogika 1 Misalnya x berlogika 0 dan yz berlogika 1 maka akan menghasilkan E = x v yz berlogika 1 Misalnya x berlogika 0 dan yz berlogika 0 maka akan menghasilkan E = x v yz berlogika 0

25. Ekspresi Boolean & Hukum Aljabar Boolean Buktikan bahwa xyz + xyz’ + xy’z + xy’z’ = x


kita gunakan rumus identitas berikut

x + x' = 1

xy + xz = x (y + z)

x (1) = x

F = xyz + xyz' + xy'z + xy'z'

F = xy (z + z') + xy' (z + z')

F = xy (1) + xy' (1)

F = xy + xy'

F = x (y + y')

F = x (1)

F = x

jadi terbukti bahwa

xyz + xyz' + xy'z + xy'z' = x
