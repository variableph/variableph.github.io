5. Dalam rangka acara syukuran, ibu memesan 36 paket nasi kuning ayam panggang dengan harga Rp17.500 per paket. jika dengan jumlah uang yang sama, ibu ingin mengganti menu pesanan tersebut dengan 45 paket nasi uduk dengan lauk rendang,berapa harga setiap paket lauk rendang?


Dalam rangka acara syukuran, ibu memesan 36 paket nasi kuning ayam panggang dengan harga Rp. 17.500 per paket. jika dengan jumlah uang yang sama, ibu ingin mengganti menu pesanan tersebut dengan 45 paket nasi uduk dengan lauk rendang. Maka harga setiap paket lauk rendang adalah sebesar Rp. 14.000.

Penyelesaian Soal :

Diketahui : harga 1 paket nasi kuning ayam panggang = Rp. 17.500.

                  Jumlah paket nasi kuning ayam panggang = 36 paket

                  Jumlah paket nasi uduk lauk rendang = 45 paket

Ditanya : Harga 1 paket nasi uduk lauk rendang ?

Jawab :

STEP 1

Hitung uang yang dikeluarkan ibu untuk membeli 36 paket nasi kuning ayam panggang dengan cara :

Harga = 36 paket × Rp. 17.500

           = Rp. 630.000

STEP 2

Hitung harga 1 paket nasi uduk lauk rendang yang dibeli ibu dengan cara :

Harga = Rp. 630.000 / 45 paket

          = Rp. 14.000

∅ Kesimpulan harga 1 paket nasi uduk lauk rendang adalah Rp. 14.000.

20. Dalamrangka acara syukuran ibu memesan 36 paket nasi kuning ayam panggang dengan harga Rp17.500,00 per paket. Jika dengan jumlah uang yang sama, ibu inggin mengganti menu pesanan tersebut dengan 45 paket nasi uduk lalu dengan lauk rendang. Berapa harga setiap paket nasi uduk dengan lauk rendang


Harga setiap paket nasi uduk lauk rendang adalah Rp 14.000,00 / paket.

Nilai tersebut diperoleh dengan perhitungan perbandingan.

Pembahasan

Diketahui:

harga 36 paket nasi kuning ayam panggang -> Rp 17.500 / paket

Ibu ingin mengganti menu menjadi 45 paket nasi uduk dengan lauk rendang

Ditanya:

harga setiap  paket nasi uduk dengan lauk rendang ?

Dijawab:

36 paket nasi kuning ayam panggang -> Rp 17.500 / paket

45 paket nasi uduk dengan lauk rendang -> x / paket

36 : 45 = x : 17.500

36 x 17.500 = 45 x

630.000 = 45 x

x = 630.000 / 45

21. Dalam rangka acara syukuran ibu memesan 36 paket nasi kuning ayam panggang dengan harga Rp. 17.500 per paket. Jika dengan jumlah uang yang sama ibu ingin mengganti menu pesanan tersebut dengan 45 paket nasi uduk dengan lauk rendang berapa harga setiap paket nasi uduk dengan lauk rendang


Dalam rangka acara syukuran ibu memesan 36 paket nasi kuning ayam panggang dengan harga Rp17.500 per paket. Jika dengan jumlah uang yang sama ibu ingin mengganti menu pesanan tersebut dengan 45 paket nasi uduk dengan lauk rendang, maka harga setiap paket nasi uduk dengan lauk rendang adalah Rp14.000,-

Pembahasan

Diketahui:

jumlah paket nasi kuning = 36

harga nasi kuning per paket = Rp17.500,-

jumlah paket nasi uduk = 45

Ditanya:

harga nasi uduk per paket = ...... ?

Jawab:

Untuk mencari harga paket nasi uduk maka harus dicari terlebih dahulu jumlah uang yang dikeluarkan ibu untuk membeli paket nasi. Caranya dengan mengalikan jumlah paket nasi kuning yang dipesan dengan harga per paketnya.

Jumlah uang yang dikeluarkan ibu = jumlah paket nasi kuning x harga nasi kuning per paket

Jumlah uang yang dikeluarkan ibu = 36 x Rp17.500,-

Jumlah uang yang dikeluarkan ibu = Rp630.000,-

Jika dengan jumlah uang yang sama yaitu Rp630.000 ibu ingin memesan jumlah nasi uduk yang lebih banyak dari nasi kuning yaitu 45 maka harga per paket dicari dengan cara membagi jumlah uang dengan jumlah paket nasi uduk yang akan dipesan.

harga nasi uduk per paket = jumlah uang ibu : jumlah paket nasi uduk

harga nasi uduk per paket = Rp630.000 : 45

harga nasi uduk per paket = Rp14.000,-

PERBANDINGAN BERBALIK NILAI

adalah perbandingan antara dua besaran di mana jika suatu variabel bertambah, maka variabel lainnya akan berkurang atau sebaliknya.

[tex] \frac{a1}{b2} = \frac{a2}{b1} [/tex]
