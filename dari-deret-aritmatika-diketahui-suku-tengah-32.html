2. deret aritmatika diketahui suku tengah 32.jika jumlah n suku pertama deret itu 672,maka banyak suku deret itu adalah


Mapel: Matematika
Kelas: VIII(SMP)

Pembahasan:

32 ×2 = a + Un
a + Un = 64

Sn = n/2 (2a + (n-1)b)
Sn = n/2 (a + Un)
672 = n/2(64)
n = 672/32
n = 21

Jadi banyak suku pada deret diatas adalah 21 suku

13. Dari deret aritmatika diketahui suku tengah 32. Jika jumlah n suku pertama deret itu 672, banyak suku deret tersebut adalah..


Karena suku tengahnya adalah 32, dapat ditulis:
[tex]\boxed{\frac{a+U_n}{2}=32~~~\to~~~a+U_n&=64}[/tex]
Dan, jumlah deret ada 672.
Maka,
[tex]$\begin{align}S_n&=672 \\ \frac n2(a+U_n)&=672 \\ \frac n2\times64&=672 \\ 32n&=672 \\ n&=\frac{672}{32}\\n&=21\end{align}[/tex]
Maka, banyaknya suku ada 21

Jawab:

Penjelasan dengan langkah-langkah:

andaikan suku terakhir = Ua

Ut = 1/2 (U1 + Ua)

(2 x Ut) - U1 = Ua

Ua = (2 x 48) - 32 = 64

Penyelesaian:

suku pertama (a) = 32

suku tengah (Ut) = 48

Ut = 1/2 (a + Un)

48 = 1/2 (32 + Un)

96 = 32 + Un

Un = 64

Maka, suku terakhirnya adalah 64
