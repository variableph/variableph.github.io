1. Dari penimbangan thorium 234 ternyata massanya 12,8 mg. Jika 48 hari kemudian penimbangannya menghasilkan massa thorium 3,2 mg, maka waktu paruhnya adalah


N = N₀ (1 / 2)^(t / T_1/2)
(1 / 2)^(t / T_1/2) = N / N₀
(1 / 2)^(48 hari / T_1/2) = 3,2 mg / (12,8 mg)
(1 / 2)^(48 hari / T_1/2) = (1 / 2)²
48 hari / T_1/2 = 2
T_1/2 = 24 hari

2. Dari penimbangan thorium 234 ternyata massanya 12,8 mg. Jika 48 hari kemudian penimbangannya menghasilkan massa thorium 3,2 mg, maka waktu paruhnya adalah .....


Waktu paruh yang dibutuhkan thorium 234 untuk meluruh dari 12,8 mg ke 3,2 mg adalah 24 hari.

Penjelasan dengan langkah-langkah:

Diketahui :

Massa akhir = 3,2 mg

Jawab :

Untuk menentukan waktu paruh dari peluruhan thorium 234 ini, kita harus mengetahui terlebih dahulu rumus yang digunakan karena thorium 234 sendiri termasuk kedalam radioaktif sehingga untuk menentukan waktu paruhnya dapat dilakukan dengan cara

m akhir = m awal [tex](\frac{1}{2} )^{\frac{T}{t paruh} }[/tex]

Karena rumus yang digunakan sudah diketahui, maka kita bisa mulai menghitung waktu paruh dari thorium 234 dengan cara berikut

m akhir = m awal [tex](\frac{1}{2} )^{\frac{T}{t paruh} }[/tex]

3,2 mg = 12,8 mg [tex](\frac{1}{2}) ^{\frac{48}{t paruh} }[/tex]

0,25 = [tex](\frac{1}{2}) ^{\frac{48}{t paruh} }[/tex]

[tex](\frac{1}{2}) ^{2}[/tex] = [tex](\frac{1}{2}) ^{\frac{48}{t paruh} }[/tex]

2 = [tex]\frac{ 48 }{t paruh}[/tex]

t paruh = 24 hari

Sehingga waktu paruh yang dibutuhkan thorium 234 untuk meluruh dari 12,8 mg ke 3,2 mg adalah 24 hari.

Besaran merupakan suatu pengukuran dalam fisika dan satuan adalah unit yang terukurnya .

I.Besaran : Sesuatu yang dapat diukur, memiliki nilai dan satuan  

Pembagian Besaran  

1.Berdasar Satuan  

a.Besaran Pokok : Besaran yang satuan sudah terdefinisikan dan berdiri sendiri  

Contoh :  

( J I Wa Ku S M P )  

Jumlah zat ( mol ), Intensitas cahaya ( Cd ), Waktu ( s ), Kuat arus ( A ), Suhu ( K ), Massa ( Kg ) dan Panjang ( m )  

b.Besaran Turunan : Besaran yang satuannya diturunkan dari satuan besaran pokok  

Contoh :  

Luas ( m2 ), Volume ( m3 ), Massa jenis ( Kg/m3 ), Kecepatan ( m/s ), Percepatan ( m/s2 ), Gaya ( N = kgm/s2 ), Energi ( J = Nm = Kgm2/s2 ), Daya ( Watt = J/s = Kgm2/s3 ), Tekanan ( N/m2 = Kg/ms2 ), Momentum ( Kgm/s ), Impuls ( Kgm/s ), Usaha (J)  

2.Berdasar Arah  

a.Besaran Skalar : Besaran yang hanya memiliki nilai saja  

Contoh : Semua besaran pokok termasuk besaran skalar, Luas, Volume, Kelajuan, Perlajuan, Massa Jenis, Energi, Daya dsb  

b.Besaran Vektor : Besaran yang memiliki nilai dan arah  

Contoh : Perpindahan, Kecepatan, Percepatan, Gaya, Momentum dsb  

II.Dimensi  

Merupakan gambaran bagaimana besaran tersebut tersusun atas besaran-besaran pokok.  

Fungsi dimensi : untuk menguji benar / setara tidaknya suatu persamaan ( Persamaan yang benar secara dimensi belum tentu benar secara fisis )  

Dimensi dari besaran – besaran pokok :  

Jumlah zat [ N ], Intensitas cahaya [ J ], Waktu [ T ], Kuat arus [ I ], Suhu [ θ ], Massa [ M ] dan Panjang [ L ]

14. dari penimbangan thorium 234 ternyata massanya 12,8 mg. jika 48 hari kemudian penimbangannya menghasilkan massa thorium 3,2 mg, maka tentukanlah besaran waktu paruhnya


Kelas: XII

Mata Pelajaran: Fisika

Materi: Radioaktivitas

Kata Kunci: Paruh Waktu

 

Jawaban:

 

Dari penimbangan thorium-234 ternyata massanya 12,8 mg. Jika 48 hari kemudian penimbangannya menghasilkan massa thorium 3,2 mg, maka besaran waktu paruhnya adalah 24 hari

 

Pembahasan:

 

Waktu paruh adalah lama waktu rata-rata yang diperlukan suatu unsur radioaktif, agar separuh dari jumlah unsur radioaktif tersebut mengalami peluruhan (radioactive decay) dan berubah menjadi unsur lain karena peluruhan inti atom.

 

Dalam soal ini unsur radioaktif Thorium-234 (Th-234) akan mengalami peluruhan, dan setelah rata-rata waktu tertentu (t) jumlahnya akan berkurang menjadi ½  jumlah awal, setelah dua kali waktu tersebut (2t) akan berkurang menjadi ¼  jumlah unsur awal, dan seterusnya.

 

Sehingga jumlah unsur radioaktif Thorium setelah waktu tertentu (Nt) dapat dihitung dalam rumus:

 

Nt = No(½)^(t/t½)

 

Untuk mencari waktu paruh (t½) unsur radioaktif:

 

t½ = t / (½log Nt/No)

 

 

Dimana:

Nt = jumlah jumlah unsur radioaktif X setelah waktu t

No = jumlah unsur radioaktif X awal

t = lama waktu

t½ = waktu paruh

 

Dalam soal diketahui bahwa:

No =  12,8 mg

t = 48 hari

Nt = 3,2 mg 

 

Sehingga, waktu paruh Thorium-234 adalah:

 

t½ = t / (½log Nt/No)

     = 48 / (½log (3,2/12,8)

     = 48 / (½log ¼)

     = 48 / 2

     = 24 hari 

 

 

Besar waktu paruh ini dapat kita cek dengan barisan berikut:

 

 

 

Setelah 24 hari: ½  jumlah awal meluruh, tersisa ½ x 12,8 mg = 6,4 mg

Setelah 48 hari: ¼ jumlah awal ( ½ jumlah pada hari ke 24) meluruh, tersisa ¼  x 12,8 mg = 3,2 mg

24. Dari penimbangan thorium 234 ternyata massanya 1,28 mg. Jika 48 hari kemudian diketahui massa thorium yang meluruh adalah 0,96 mg, tentukan waktu paruh thorium tersebut!


Penjelasan:

Diketahui:

[tex]Massa\ awal\ Th-234: N_{0} = 1,28\ mg \\Massa\ yang\ meluruh: \Delta N= 0,96\ mg\\Waktu\ peluruhan: t = 48\ hari\\[/tex]

Ditanya:

Waktu paruh Th-234

Penyelesaian:

[tex]Massa\ Th-234\ setelah\ meluruh:\\\Delta N= N_{0}- N_{t} \\0,96 = 1,28 - N_{t}\\N_{t} = 1,28 - 0,96\\N_{t} = 0,32\ mg\\\\Persamaan\ peluruhan\ zat\ radioaktif:\\N_{t}=N_{0} (\frac{1}{2}) ^{\frac{t}{T_{\frac{1}{2} } } } \\0,32 = 1,28 (\frac{1}{2}) ^{\frac{t}{T_{\frac{1}{2} } } }\\\frac{0,32}{1,28}= (\frac{1}{2}) ^{\frac{t}{T_{\frac{1}{2} } } }\\ \frac{1}{4}=(\frac{1}{2}) ^{\frac{t}{T_{\frac{1}{2} } } }\\(\frac{1}{2}) ^{2} = (\frac{1}{2}) ^{\frac{t}{T_{\frac{1}{2} } } }\\[/tex]

Karena basisnya sudah sama, maka eksponennya juga bernilai sama.

[tex]2=\frac{t}{T_{\frac{1}{2} } } \\2=\frac{48}{T_{\frac{1}{2} } } \\\\{T_{\frac{1}{2} }=24\ hari\\[/tex]

Diperoleh hasil bahwa waktu paruh dari Th-234 adalah 24 hari

26. isotop thorium (th-234) memiliki waktu paruh 24 hari, unsur ini dibeli untuk kebutuhan laboratorium sebanyak 5 gram. dalam satu kali penelitian dibutuhkan setidaknya 0,5 gram thorium. jika unsur ini akan digunakan lagi 4 bulan mendatang pernyataan yang benar terkait permasalahannya adalah.... (1 bulan = 30 hari)​


Isotop thorium (th234) memiliki waktu paruh 24 hari, dan elemen ini dibeli hingga 5 gram untuk penggunaan laboratorium. Satu studi membutuhkan setidaknya 0,5 gram thorium. Jika elemen ini  digunakan kembali dalam 4 bulan ke depan, penjelasan yang benar untuk masalah ini adalah bahwa thorium tidak dapat digunakan karena tersisa kurang dari 0,5 gram.

Penjelasan dan Langkah-Langkah:

Waktu paruh suatu zat yang mengalami peluruhan eksponensial adalah waktu yang diperlukan untuk jumlah ini turun menjadi setengah dari nilai aslinya. Konsep ini umum tidak hanya dalam fisika untuk mengukur peluruhan radioaktif materi, tetapi juga di banyak bidang lainnya. Tabel di sebelah kanan menunjukkan penurunan jumlah waktu paruh yang telah terjadi. Atom bahan radioaktif seperti uranium dan radium terurai sendiri menjadi atom unsur lain dari waktu ke waktu, memancarkan radiasi atau energi  dalam bentuk gelombang. Fitur penting dari proses peluruhan radioaktif adalah bahwa setiap zat meluruh dengan kecepatannya sendiri. Oleh karena itu, waktu paruh zat tertentu adalah konstan dan tidak terpengaruh oleh kondisi fisik (suhu, tekanan, dan lain-lain.) yang terjadi di sekitarnya.

Diketahui:

[tex]T(\frac{1}{2}) = 24 hari[/tex]

[tex]N_{0} = 5 gram[/tex]

t = 4 bulan = 120 hari

Ditanya:

[tex]N_{t} = ....?[/tex]

Jawab:

[tex]N_{t} = N_{0} \frac{1}{2}^{(\frac{t}{T(1/2)})}[/tex]

[tex]N_{t} = 5 . \frac{1}{2}^{\frac{120}{24}} \\N_{t} = 5.(\frac{1}{2}^{5})\\N_{t} = 5.\frac{1}{32} \\N_{t} = 0,16 gram[/tex]

Isotop Thorium tidak bisa digunakan lagi karena dalam satu kali penelitian dibutuhkan setidaknya 0,5 gram.
