Diketahui akar-akar persamaan kuadrat 2x² – 4x + 1 = 0 adalah m dan n. Tentukan persamaan  kuadrat baru yang akar-akarnya m + n dan m × n .

Penyelesaian :

2x² – 4x + 1 = 0

a = 2 , b = -4,  c = 1

m + n =  [tex]\displaystyle\frac{-b}{a} = \frac{-(-4)}{2}[/tex] = 2

m . n =  [tex]\displaystyle\frac{c}{a} = \frac{1}{2}[/tex]

Persamaan  kuadrat baru yang akar-akarnya m + n dan m × n

x₁ + x₂ = (m + n) + (m× n)

          = 2 + [tex]\displaystyle\frac{1}{2}[/tex]

          = [tex]\displaystyle\frac{5}{2}[/tex]

x₁ . x₂ = (m + n) × (m× n)

         = 2 × [tex]\displaystyle\frac{1}{2}[/tex]

         = 1

Masukkan ke dalam ruus persamaan kuadrat baru

x² - (x₁ + x₂)x + (x₁ . x₂) = 0

x² − [tex]\displaystyle\frac{5}{2}[/tex] x + 1 = 0      (kesemua ruas dikali 2)

2x² − 5x + 2 = 0

28. Jika α dan β adalah akar-akar persamaan kuadrat 2x2 + 4x – 3 = 0, persamaan kuadrat yang akar-akarnya (2α + 1) dan (2β + 1) adalah ....


Langkah-langkah menentukan persamaan kuadrat baru yaitu :

Menentukan jumlah dan perkalian akar-akar dari persamaan kuadrat awalMenentukan jumlah dan perkalian akar-akar dari persamaan kuadrat baru yang diketahuiMembentuk persamaan kuadrat baru sesuai dengan rumus [tex]x^{2} - ( x_{1} + x_{2} ) + x_{1} \ .\ x_{2} = 0[/tex]

[tex]x_1 + x_2 = - \frac {b} {a}[/tex]

[tex]x_1 . x_2 = \frac {c} {a}[/tex]

Pembahasan

Jika α dan β adalah akar-akar persamaan kuadrat [tex]2x^2 + 4x - 3 = 0[/tex],

a = 2, b = 4, dan c = -3, maka :

α + β = [tex]- \frac {4} {2}[/tex] = -2

α . β = [tex]\frac {-3} {2}[/tex] = [tex]- \frac {3} {2}[/tex]

Cari jumlah dan perkalian akar-akar persamaan kuadrat baru

(2α + 1) + (2β + 1) = 2α + 2β + 1 + 1

                            = 2(α + β) + 2

                            = 2(-2) + 2

                            = -2

(2α + 1) . (2β + 1) = 4αβ + 2α + 2β + 1

                          = 4αβ + 2(α + β) + 1

                          = 4([tex]- \frac {3} {2}[/tex]) + 2(-2) + 1

                          = -6 - 4 + 1

                          = -9

Persamaan kuadrat yang baru adalah [tex]x^{2} - ( -2) + (-9) = 0[/tex] = [tex]x^{2} + 2 - 9 = 0[/tex].
