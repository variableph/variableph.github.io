11. Diketahui fungsi f:R-R dengan fx=3x-1 Tentukan fungsi inversnya


Fungsi invers dari f(x) = 3x - 1 adalah [tex]\sf f^{-1}(x) = \frac{1}{3}(x + 1) [/tex]

[tex]~[/tex]

PEMBAHASAN

[tex]~[/tex]

[tex]\bf Fungsi~Invers[/tex]

Fungsi invers adalah alter ego (kebalikan) dari fungsi biasa. Jika fungsi biasa dinyatakan dengan f(x), maka fungsi inver dinyatakan dengan f-¹(x). Rumus-rumus yang ada pada fungsi invers:

[tex]\sf 1. ~f(x) = ax + b \rightarrow f^{-1}(x) = \frac{x - b}{a} [/tex]

[tex]\sf 2.~ f(x) = \frac{ax + b}{cx + d} \rightarrow f^{-1}(x) = \frac{-dx + b}{cx - a} [/tex]

[tex]\sf 3.~ f(x) = a^{cx} \rightarrow f^{-1}(x) = ^a log x^{\frac{1}{c}} [/tex]

[tex]\sf 4. ~f(x) = ax^2 + bx + c \rightarrow f^{-1}(x) = \frac{-b \pm \sqrt{b^2 - 4a(c - x) }}{2a} [/tex]

[tex]~[/tex]

[tex]~[/tex]

DIKETAHUI

f(x) = 3x - 1

[tex]~[/tex]

[tex]~[/tex]

DITANYAKAN

f-¹(x) = ..?

[tex]~[/tex]

[tex]~[/tex]

JAWAB

[tex]~[/tex]

[tex]\sf f(x) = ax + b \rightarrow f^{-1}(x) = \frac{x - b}{a} [/tex]

[tex]\sf f(x) = 3x - 1\rightarrow f^{-1}(x) = \frac{x + 1}{3} [/tex]

[tex]\sf f(x) = 3x - 1 \rightarrow f^{-1}(x) = \frac{1}{3}(x + 1) [/tex]

[tex]~[/tex]

[tex]~[/tex]

KESIMPULAN

Jadi, fungsi invers dari f(x) = 3x - 1adalah [tex]\sf f^{-1}(x) = \frac{1}{3}(x + 1) [/tex]
