2. Dua kawat terbuat dari bahan yang sama. Panjang kawat kedua 4kali panjang kawat pertama dan diameter kawat kedua 2kali diameter kawat pertama. Jika hambatan kawat pertama adalah R, maka hambatan kawat kedua adalah


Hambatan kawat penghantar

Perbandingan luas penampangnya:
A : A2 = π/4d² : π/4(2d)² = 1 : 4
A2 = 4A

Hambatan pertama:
R = α L / A

Hambatan kedua:
R2 = α L2 / A2
R2 = α 4L / 4A

Perbandingannya:
R/R2 = (α L / A) / (α 4L / 4A)
R/R2 = 1 / 1

R = R2

Jadi, hambatan kawat kedua sama dengan hambatan kawat pertama.

23. Panjang dua buah kawat yang terbuat dari bahan sama memenuhi perbandingan 2:1 diameter kawat kedua ½ kali diameter kawat pertama besarnya hambatan kawat kedua adalah


Besarnya hambatan kawat kedua adalah 8 kali besar hambatan kawat pertama dengan perhitungan sebagai berikut.

Pembahasan

Rumus untuk mencari hambatan pada penghantar adalah:  

R = ρ. L / A, dengan A = ¼ π D², maka rumusnya menjadi

R = ρ. L / (1/4 π D²)  

di mana  

R : hambatan penghantar (Ω)  

ρ : hambatan jenis penghantar (Ω.m)  

L : panjang penghantar (m)  

A : luas penampang penghantar (m²)

D : diameter penghantar (m)

Maka kita dapat melakukan perbandingan

R2 / R1 = (ρ2. L2 / (1/4 π D2²)) / (ρ1. L1 / (1/4 π D1²))

R2 / R1 = (ρ2. L2 / D2²) / (ρ1. L1 / D1²)

R2 / R1 = (ρ2. L2 / D2²) x (D1² / ρ1. L1)

R2 / R1 = (ρ2. L2. D1² / (D2². ρ1. L1), pada jenis kawat yang sama maka nilai hambatan jenisnya sama, sehingga

R2 / R1 = (L2. D1² / (D2². L1), yang ditanyakan adalah nilai hambatan kawat kedua, maka

R2 = R1 (L2. D1² / (D2². L1), masukkan nilai yang diketahui,

R2 = R1 (2 L1. D1² / ((1/2 D1)². L1)

R2 = R1 (2 D1² / (1/4 D1²))

R2 = R1 (2 / 0,25)

R2 = 8 R1

25. Dua buah kawat penghantar terbuat dari bahan yang sama. Jika hambatan kawat pertama adalah R, Panjang kawat kedua adalah dua kali panjang kawat pertama, dan luas permukaannya 4 kali luas permukaan kawat pertama, maka hambatan kawat kedua adalah​


Dua buah kawat penghantar terbuat dari bahan yang sama. Jika hambatan kawat pertama adalah R, Panjang kawat kedua adalah dua kali panjang kawat pertama, dan luas permukaannya 4 kali luas permukaan kawat pertama, maka hambatan kawat kedua adalah …  ½ dari hambatan kawat pertama

Pembahasan:

Besar dari hambatan ada suatu kawat penghantar dapat dihitung dengan rumus:

R = ρL /A

Dimana:

R = resistance atau hambatan kawat,  

ρ = hambatan jenis kawat,  

L = panjang kawat  

A = luas penampang kawat.

Dari rumus ini terlihat bahwa hambatan kawat penghantar bergantung pada jenis, panjang, dan luas penampangnya.  

Besar hambatan berbanding lurus dengan panjang (L) kawat penghantar dan berbanding terbalik dengan luas penampang (A) kawat penghantar.

Dalam soal diketahui:

R1 = R

L2 = 2 L1

A2 = 4 A1

ρ1 = ρ2 = ρ (karena kawan sejenis)

Ditanya: R2

Maka perbandingan hambatan keduanya adalah:

R1 : R2 = (ρ1L1 /A1) : (ρ2L2 /A2)

             = (ρL1 /A1) : (ρL2 /A2)

            = (L1 /A1) : (2 L1 /4 A1)

R1 : R2 = 2 : 1

R2 = ½ R1  

     = ½ R

Jadi hambatan kawat kedua ½ dari hambatan kawat pertama

28. dua buah kawat penghantar terbuat dari bahan yang sama, panjang kawat pertama 2× kawat kedua, jari-jari kawat kedua 2× jari-jari kawat pertama, kawat pertama memiliki hambatan 10 ohm, hambatan kawat kedua adalah...​


Jawaban:

R1:R2 = I/A:1/A

R1:R2 = 2I/(phi)^2 : I/(phi)^2 *coret yang sama*

R1:R2 = 2/1 : 1/4

R1:R2 = 8/1

R2 = 1/8 R1...

R2 = 1/8*10

R2 = 10/8 = 5/4 = 1,25 ohm
