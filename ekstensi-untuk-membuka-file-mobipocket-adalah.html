14. 1. Apakah nama ekstensi file pada tiap sistem operasi berbeda? Jelaskan! 2. Mengapa suatu file harus memiliki ekstensi? 3. Apakah suatu file dapat memiliki lebih dari satu ekstensi? Jelaskan!​


1. Berbeda karena ekstensi file adalah tanda yang membedakan antara satu jenis file dengan jenis file lainnya. Misalnya, file gambar akan memiliki ekstensi jpg, gif, png, dan lain-lain. Sedangkan untuk file video akan memiliki ekstensia mpeg, avi, mp4, wmv, dan lain-lain. File di dalam komputer dengan ekstensi yang berbeda-beda.

2. Ekstensi memberi tahu komputer Anda aplikasi yang dibuat atau bisa membuka file dan ikon mana yang digunakan untuk file tersebut. Misalnya, ekstensi docx memberi tahu komputer Anda bahwa Microsoft Word bisa membuka file dan untuk menampilkan ikon Word saat Anda menampilkannya di file Explorer.

3. Bisa, contohnya beberapa ekstensi pada file video adalah mpg, wmv, mp4, 3gp, avi, flv, KV. Masing-masing ekstensi ini menunjukkan bahwa masing-masing video memiliki jenis pemutar yang berbeda.


15. ekstensi file untuk program database adalah


Ekstensi file untuk program database adalah .dbf yaitu File database. Selain itu, ekstensi lain untuk files database adalah .pdb (program database file), .wdb (microsoft work database file), .db (file database), .mdb (microsoft Access Database File).

Pembahasan

Database atau disebut juga basis data yaitu kumpulan data yang berhubungan dengan suatu objek, topik atau tujuan khusus tertentu, biasanya berupa kumpulan dari tabel.

Berikut ini jenis-jenis aplikasi untuk database, yaitu:

Dbase, yaitu aplikasi database yang dikembangkan oleh perusahaan Ashton Tate, untuk platform Apple dan Unix kemudian digunakan dibawah DOS. Ekstensi filenya “dbf” masih dipakai untuk Arcview dan Foxpro.IBM DB2, yaitu aplikasi database pertama menggunakan bahasa SQL. Microsoft SQL Server, yaitu aplikasi database buatan Microsoft, yang berguna untuk database skala besar dan terpusat ke server.My SQL, yaitu aplikasi database yang bersifat gratis (freeware). Digunakan untuk aplikasi berbasis web dan dikombinasikan dengan bahasa pemrograman berbasis web, yaitu PHP.Oracle, yaitu aplikasi database client-server yang digunakan untuk penyimpanan dan pengelolaan data yang sangat besar dan banyak.Microsoft Access, yaitu aplikasi database buatan Microsoft, untuk penggunaan skala kecil dengan file yang dihasilkan berbentuk file accdb (access database).

25. File ekstensi untuk fotoshop pada saat disimpan adalah dapat berupa file file di bawah ini,kecuali?


Didalam Photoshop, terdapat kurang lebih 21 jenis format file yang dapat dipilih oleh pengguna. Format file tersebut antara lain adalah sebagai berikut

- PSD,PDD (Photoshop Document). Merupakan format penyimpanan asli dari Photoshop

- PSB (Photoshop Big). Digunakan untuk meningkatkan resolusi maksimum

- BMP, RLE, DIB. Digunakan untuk menyimpan gambar tipe bitmap

- GIF. Digunakan untuk menyimpan gambar tipe GIF. Memiliki kombinasi 256 warna. Biasa digunakan untuk logo, line drawing dan icon

- EPS (Encapsutaed Potscript). Digunakan untuk pencetakan atau pertukaran dokumen antar program grafis

- IFF (Interchange File). Digunakan untuk video

- JPEG. Digunakan untuk menyimpan mode warna RGB, CYMK, Grayscale

- JPEG 2000 (JPF,JPX,JP2,J2C,J2K,JPC). Merupakan standar kompresi untuk meningkatkan rasio kualitas dan kompresi

- JPS (JPEG Stereo). Terdiri dari side by side berdasarkan JPEG

- MPO (Multi Picture Format). Digunakan untuk format 3D

- PCX (Personal Computer Exchange). Berberntuk rester.

- Photoshop PDF. Digunakan oleh Adobe Acrobat

- RAW. Digunakan untuk menyimpan warna RGB, CMYK dan Grayscale

- PXR (Pixar). Digunakan untuk pertukaran file dengan Pixar Image Computer

- PNG (Portable Network Graphics). Digunakan untuk transparasi dalam browser

- Portable Bit Map (PBM,PGM,PPM,PNM,PFM,PAM). Digunakan untuk menyimpan informasi dengan kualitas 1 bit hingga 24 bit

- Scitex CT (SCT)

- Targa (TGA,VDA,ICB,VST)

- Tagged Image Format File (TIF,TIFF)

Dengan demikian, jika file ektensi tidak termasuk dalam daftar diatas maka bukan merupakan file ekstensi untuk Photoshop
