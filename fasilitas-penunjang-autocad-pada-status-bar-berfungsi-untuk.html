1. jelaskan fungsi tool bar pada layar autocad


1. Line = Membuat object dengan garis
2. Polyline = Membuat object dengan garis menyatu.
3.Splin = Membuat object dengan garis melengkung.
4.Elipse = Membuat object Elipse.
5. Cirxle = Membuat object Lingkaran.
6. Rectangle = Membuat object Persegi.
7.Region = Menyatukan object.
8.Extrude = Memberikan ketebalan pada object.
9.Move = Memindahkan object.
10.Mirror = Mengcopy object disebelah kiri, kanan, atas, dan bawah.
11.Trim = Memotong object.
12. Chamfer = Membantu object persegi menjadikan sudut object tidak lancip.
13. Fillet = Membuat object menjadikan sudut tidak lancip atau melengkung.
14. Union = Menggabungkan object.
15. Substract = Menghilangkan object 3D.
16. Ofset = Menggadakan object didalam atau diluar object.
17. Extrude Faces = Menambahkan ketebalan yang sudah dibuat.
