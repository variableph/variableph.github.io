12. tuliskan perbedaan folder ,file dan ekstensi file!!​


Jawaban ringkas:

Folder adalah nama tempat/ lokasi yg dapat dipergunakan untuk menyimpan file.

File adalah data komputer dg berbagai tipe yg bermacam-macam.

Ekstensi file adalah keterangan yg menunjukkan tipe file.

Penjelasan panjang:

Folder dapat dibuat di beberapa media penyimpanan, seperti: harddisk, flashdisk, memory card.

Dokumen1.Doc (nama file: Dokumen1 dan ekstensi file: Doc)

28. 1. Apakah nama ekstensi file pada tiap sistem operasi berbeda? Jelaskan! 2. Mengapa suatu file harus memiliki ekstensi? 3. Apakah suatu file dapat memiliki lebih dari satu ekstensi? Jelaskan!​


1. Berbeda karena ekstensi file adalah tanda yang membedakan antara satu jenis file dengan jenis file lainnya. Misalnya, file gambar akan memiliki ekstensi jpg, gif, png, dan lain-lain. Sedangkan untuk file video akan memiliki ekstensia mpeg, avi, mp4, wmv, dan lain-lain. File di dalam komputer dengan ekstensi yang berbeda-beda.

2. Ekstensi memberi tahu komputer Anda aplikasi yang dibuat atau bisa membuka file dan ikon mana yang digunakan untuk file tersebut. Misalnya, ekstensi docx memberi tahu komputer Anda bahwa Microsoft Word bisa membuka file dan untuk menampilkan ikon Word saat Anda menampilkannya di file Explorer.

3. Bisa, contohnya beberapa ekstensi pada file video adalah mpg, wmv, mp4, 3gp, avi, flv, KV. Masing-masing ekstensi ini menunjukkan bahwa masing-masing video memiliki jenis pemutar yang berbeda.


29. apa artinnya ekstensi AFI pada macromedia flash?


AVI ( Audio Video Interleaved )

Diperkenalkan oleh Microsoft pada tahun 1992 sebagai bagian dari teknologi Video for Windows miliknya. File AVI menyimpan data audio dan video pada struktur interleaved. File ini hanya berupa kontainer- dan data audio video dapat dikompres menggunakan berbagai codec. Kualitas dan kapasitas tergantung pada codec dan secara khusus codec yang digunakan adalah MPEG, Divx atau WMV.
