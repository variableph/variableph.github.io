3. jelaskan topologi jaringan yang sering digunakan server


 topologi jaringan yang sering digunakan server adalah Topologi Star

Topologi star merupakan bentuk topologi jaringan berupa percabangan (konvergensi) dari node atau titik tengah ke setiap pengguna.Pada topologi star masing-masing workstation dihubungkan secara langsung ke server atau Hub,Semua link yang ada harus melewati pusat penyaluran data tersebut ke semua simpul atau client yang dipilihnya.

Simpul pusat dinamakan stasiun primer atau server dan untuk yang lainnya dinamakan stasiun sekunder atau client server.Setelah hubungan jaringan dimulai4 oleh server maka setiap client server sewaktu-waktu dapat menggunakan jaringan tersebut tanpa harus menunggu perintah dari server.Pada jaringan ini,Jika salah satu workstation terputus dari jaringan,tidak akan mengganggu kinerja workstation lain karena masing masing langsung terhubung ke hub.Topologi jaringan ini termasuk topologi jaringan dengan biaya menengah.

29. Server Adalah web server adalah Server FTP adalah Server DHCP Adalah


Mapel : TI
Materi : Server

--Soal--

Server Adalah
Web server adalah
Server FTP adalah
Server DHCP Adalah

--Jawaban--

Server adalah suatu sistem komputer yang menawarkan jenis layanan tertentu dalam suatu jaringan.

Web server adalah suatu perangkat lunak (software) yang memberikan perintah berbasis data yang berfungsi menerima permintaan dari http atau https pada klien web terkenal.

Server FTP adalah protokol internet yang berjalan dalam suatu lapisan aplikasi yang berfungsi untuk tukar menukar berbagai data antara klien dan server dalam 1 jaringan.

Server DHCP adalah adalah suatu protocol pada jaringan komputer yang dapat memberikan atau meminjamkan atau memberi akses IP address terhadap host yang berada dalam suatu jaringan secara otomatis.
