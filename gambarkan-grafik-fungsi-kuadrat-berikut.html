FungsI Kuadrat
danGrafiknya

pada gmbr kurva

i) memotong di x1 = -3 dan  x2 = 1
y = a ( x- x1) (x - x2)
y = a (x + 3)(x - 1)

ii) melalui (- 1, - 4),  x = - 1  dan  y= - 4

y = a(x + 3)(x - 1)
-4= a(-1 + 3)(-1 - 1)
-4=  a(2)(-2)
-4 = a(-4)
a= 1

persamaan  grafik
y = a (x + 3)(x - 1) , dengan a = 1
y= x² + 2x -  3

❐ Pers. Kuadrat

_____________________________

Analisis

Grafik memotong sumbu x di titik (-3,0) dan (1,0). Karena pada opsi, a = -1 dan a = 1, maka persamaannya akan menjadi y = x² + bx + c, dengan a = 1.

Substitusi titik potong dgn smb.x

y = (x - x1)(x - x2)

y = (x + 3)(x - 1)

y = x² + 2x - 3 (E)

Jawaban:

C. f(x) = x² + 5x - 6

Penjelasan dengan langkah-langkah:

Diketahui :

x = 0 y = -6

x1= 1 x2= -6

TAHAP I (Mencari a)

y = a ( x - x1 )( x - x2 )

-6 = a ( 0 - 1 )( 0 - (-6) )

-6 = a (-1)(6)

-6 = -6a

1 = a

TAHAP II (Mencari Fungsi)

y = a ( x - x1 )( x - x2)

y = 1 ( x - 1 )( x - (-6) )

y = 1 ( x - 1 )( x + 6 )

y = 1 ( x² + 6x - x - 6 )

y = x² + 5x - 6
