a. Melalui titik P(3,6) dan sejajar dengan garis yang melalui titik A(-4,-5) dan titik B(1,0)

A(-4,-5) dan titik B(1,0)

x1=-4, y1=-5, x2=1 y2=0

m(AB) = y2-y1 / x2-x1

= 0-(-5) / 1-(-4)

= 0+5 / 1+4

= 5/5

= 1

Syarat sejajar m1=m2=1

titik P(3,6)

x1=3 dan y1 = 6

y-y1 = m2 (x-x1)

y-6 = 1 (x-3)

y-6 = x-3

y = x-3+6

y=x+3

-x+y=3

-x+y-3=0

b. Melalui titik M(-6, 0) dan tegak lurus dengan garis yang melalui titik K(3,-2) dan L(-1,6)

K(3,-2) dan titik L(-1,6)

x1=3, y1=-2, x2=-1 y2=6

m(KL) = y2-y1 / x2-x1

= 6-(-2) / -1-3

= 6+2 / -4

= 8/-4

= -2

Syarat tegak lurus m1 x m2 = -1

-2 x m2 = -1

m2 = -1/-2

m2 = 1/2

titik M(-6, 0)

x1=-6 dan y1=0

y-y1 = m2 (x-x1)

y-0 = 1/2 (x-(-6))

y-0 = 1/2 (x+6)

y-0 = 1/2x+6/2

y = 1/2x+6/2

2y=x+6

Soal no. 4
-------------

Oleh karena persamaan garis yang melalui titik (4, 5) dan (4, -5) adalah x = 4, maka garis tersebut sejajar dengan sumbu Y dan tegak lurus dengan sumbu X.

Soal no. 5
--------------

Karena garis K tidak sejajar terhadap sumbu X maupun sumbu Y, maka persamaannya adalah ax + by + c, dimana a, b, c sebarang bilangan real, a dan b tidak nol.

Misal kita ambil a = 1, b = 1.
Oleh karena garis K melalui titik (-3, -5), maka untuk menentukan nilai c perlu kamu lakukan subtitusi x = -3 dan y = -5 ke persamaan x + y = c.

c = x + y = -3 + (-5) = -8

Jadi, persamaan garis K yang terbentuk adalah x + y = -8.

Note: nilai a dan b boleh sebarang, asalkan bukan nol.
