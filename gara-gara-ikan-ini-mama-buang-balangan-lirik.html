2. Kuis Gampang (Liat Mudah Kerjain Susah)5! x 2! = ?Note: Gara-gara Ikan Ini Mama Buang Balanga :( (Ga lah, bercanda) ​


》Faktorial

5! × 2! = 240

Penyelesaian

5! × 2!

= ( 5 × 4 × 3 × 2 × 1 ) × ( 2 × 1 )

= ( 20 × 3 × 2 × 1 ) × 2

= ( 60 × 2 × 1 ) × 2

= 120 × 2

= 240

Jawaban:

5! = (( 5 x 4 ) x 3 x 2 x 1 )

= 5 x 4 = 20

= 20 x 3 = 60

= 60 x 2 = 120

= 120 x 1 = 120

2! = 2 x 1

= 2 x 1 = 2

120 x 2= 240
