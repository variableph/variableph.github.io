(3x - 2) (x - 1) = 0

3x - 2 = 0
3x = 2
x = 2/3
α = 2/3

x - 1 = 0
x = 1
β = 1

α + β = 2/3 + 1
α + β = 5/3 = 1,66

x² - 7x + 2 = 0
a= 1
b = - 7
c = 2

α+β = -b/a = -(-7)/1 = 7
α. β = c/a = 2/1 = 2

α/ β+ β/α = (α² + β²) / (α β)
= {(α+β)² - 2 αβ}  / (αβ)
= {7² - 2(2)} / (2)
= (49 - 4)/(4)
= 45/4
