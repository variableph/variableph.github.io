1. hasil kali dua bilangan ganjil positif berurutan adalah 143 tentukan kedua bilangan ganjil tersebut


misal : bil ganjil I = A  ⇒ bil ganjil II = A + 2

          A .(A + 2) = 143
            A² + 2A  = 143
   A² + 2A - 143 = 0
(A - 11)(A + 13) = 0
A - 11 = 0      atau  A + 13 = 0
        A = 11   atau           A = - 13 (tak memenuhi)
jadi , bil pertama = 11 dan bilangan kedua = A + 2
                                                                               = 11 + 2
                                                                               = 13

3. hasil kali dua bilangan ganjil positif yang berurutan adalah 143.dari dua bilangan itu,bilangan yang terbesar adalah....


Hasil kali dua bilangan ganjil positif yang berurutan adalah 143. dari dua bilangan itu bilangan yang terbesar adalah 13.

Pendahuluan

Persamaan kuadrat adalah persamaan dengan variabel berpangkat maksimal 2 dengan bentuk umum persamaan kuadrat ax² + bx + c = 0.

Pembahasan

Diketahui :

Hasil kali dua bilangan ganjil = 143

Ditanya :

Bilangan terbesar?

Penyelesaian :

Menentukan bilangan terbesar

Misal : bil I = x

           bil II = x + 2

bil I × bil II = 143

x × (x + 2) = 143

x² + 2x = 143

x² + 2x - 143 = 0  (difaktorkan)

(x + 13) (x - 11) = 0

x + 13 = 0

      x = -13   (negatif tidak memenuhi)

atau

x - 11 = 0

     x = 11    (memenuhi)

Maka :

bilangan I = x = 11

bilangan II = x + 2 = 11 + 2 = 13

Jadi bilangan yang terbesar adalah 13.

21. hasil kali dua bilangan ganjil positif berurutan adalah 143, jika bilangan pertama =n,tentukan bilangan kedua!!


semisal n = bilangan ganjil, m = bilangan ganjil berurutan setelah n

jadi bilangan kedua: m = n + 2


n.m = 143

n(n+2) = 143

n²+2n = 143

n²+2n - 143 = 0

bentuk x² + bx +c = 0, ubah ke bentuk (x+p)(x+q), dengan syarat harus memenuhi

p+q = b = 2p.q = c = -143Faktor dari 143 adalah 1, 11, 13, 143. Jadi, pilih p = 13, q = -11.

(n+13)(n-11) = 0


Tinjau akar-akarnya

(n₁+13) = 0

n₁ = -13


n₂-11 = 0

n₂ = 11


Jadi, n = -13 atau n = 11. Pilih n = 11 karena n bilangan ganjil positif. m = n+2, m = 13.
