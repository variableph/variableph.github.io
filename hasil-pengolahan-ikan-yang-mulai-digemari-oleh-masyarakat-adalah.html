21. Dalam suatu kelas terdapat 40 siswa, 20 orang gemar renang, 24 orang gemar futsal dan 4 orang gemar olah raga lain. Perbandingan banyak siswa yang gemar kedua olah raga dengan yang gemar olah raga lain adalah ...​


Materi : Himpunan

S = himpunan semesta ( 40 )

a = himpunan renang ( 20 )

b = himpunan futsal ( 24 )

n = suka keduanya ( n )

ab^c = tidak keduanya ( 4 )

\_____________________/

( a - n ) + ( b - n ) + n = S - ab^c

( 20 - n ) + ( 24 - n ) + n = 40 - 4

44 - n = 36

n = 44 - 36

n = 8 siswa

Semoga bisa membantu
