17. jika PA=11cm QB=5cm dan PQ=20cm tentukan panjang AB


♧ Mapel : Matematika
♧ Tingkat : SMP
♧ Garis Singgung Lingkaran
________________________

Panjang AB (panjang GSPD)
= √( p² - ( R + r )²)
= √( 20² - ( 11 + 5 )²)
= √( 20² - 16² )
= √( 400 - 256 )
= √( 144 )
= 12cm

Keterangan :
- R = Jari jari besar lingkaran
- r = Jari jari kecil lingkaran
- p = Jarak pusat lingkaran
- GSPD = Garis Singgung Persekutuan Dalam
[tex] [/tex]

// R = jari jari lingkaran besar

   r = jari jari lingkaran kecil

   p = jarak titik pusat kedua lingkaran

   d = garis singgung persekutuan dalam

   anggap titik P adalah titik pusat lingkaran yang besar //


// nilai //

R = PA = 11cm


r = QB = 5cm


p = PQ = 20cm


d = AB


// rumus garis singgung persekutuan dalam //

d² = p² - (R + r)²

 d = √p² - (R + r)²


// masukkan nilai

d = √(20cm)² - (11cm + 5cm)²

   = √400cm² - (16cm)²

   = √400cm² - 256cm²

   = √144cm²

   = 12cm


AB = d = 12cm


Jadi panjang AB adalah 12cm

FA = v => EO = OB = DC = v

BC = BO + OC
BC = -OB + OC
BC = -v + u
=> AO = OD = FE = -v + u

= u + (AB + BC) + (AO + OD)
= u + (u + (-v + u)) + ((-v + u) + (-v + u))
= 5u - 3v

2) AB + BC + CR + RA
= (b - a) + (c - b) + (r - c) + (a - r)
= 0

3) AB - DC
= u - v
