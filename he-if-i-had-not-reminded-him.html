6. If he had wanted to sell that house, I would have bought it from him The fact is...


The fact is he didn't want to sell that house.

ELABORATION : Conditionals :

Conditionals merupakan kalimat pengandaian. Conditionals dibagi menjadi 3 :

1st Conditionals :

1st Conditionals dibagi menjadi 2 :

1. Situation : Benar terjadi di masa sekarang, atau masa depan.

Structure :

If clause : using present simple

Result clause : using will + infinitive.

➡ Contoh :

If I don't sleep, I will be sleepy.

[Jika aku tidak tidur, aku akan mengantuk. ]

If clause : If I don't sleep

Result clause : I will be sleepy

2. Situation : General Truth [Conditionals ini juga biasa kita sebut sebagai 0 conditionals (zero conditionals) ]

Sturcture :

If clause : present simple

Result clause : present simple

➡ Contoh ;

If the temperature reaches 100°C, the water boils.

[Jika suhu mencapai 100°C, air akan mendidih. ]

Id clause : If the temperature reaches 100°C

Result clause : the water boils

2nd Conditionals :

Situations : Salah/berlawanan dengan fakta, baik di masa depan ataupun masa sekarang.

Structure :

If clause : using past simple

Result clause : using would + infinitive

➡ Contoh :

Fact : I am not you.

[Fakta : Aku bukan kamu. ]

Conditionals : If I were you, I would not give her a candy.

[Pengandaian : Jika aku adalah kamu, aku tidak akan memberi dia permen. ]

If clause : If I were you

Result clause : I would not give her a candy

Dalam 2nd Conditionals, kita harus menggunakan "were", kita tidak boleh menggunakan " was".

3rd Conditionals :

Situation : Salah/berlawanan dengan fakta pada kejadian di masa lalu.

Structure :

If clause : using past perfect

Result clause : using would + have + V3

➡ Contoh :

[Fact : I didn't go to Paris. ]

[Fakta : Saya tidak pergi ke Paris. ]

[Conditionals : If I had gone to Paris, I would have had many experiences. ]

[Pengandaian : Jika aku telah pergi ke Paris, aku akan sudah punya banyak pengalaman. ]

If clause : If I had gone to Paris

Result clause : I would have had many experiences

EXPLANATION :

Pada soal, diketahui conditionals (pengandaian nya) "If he had wanted to sell that house, I would have bought it from him. " [Jika dia telah mau jual rumah itu, saya akan sudah membelinya untuk dia. ] Karena ini menggunakan 3rd Conditionals, maka faktanya salah pada masa lalu. Sehingga faktanya : Dia (telah) tidak menjual rumah itu, dan saya (telah) tidak membeli rumah itu untuknya.

Sehingga bahasa Inggrisnya :

He didn't want to sell that house. I didn't buy that house.

If conditional sentence merupakan sebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb3

Alasan menggunakan "some" di tempat kosong pertama adalah karena pembicara sedang meminta jumlah tertentu sabun. Dengan menggunakan "some," pembicara menunjukkan bahwa mereka ingin jumlah sabun yang pasti.
