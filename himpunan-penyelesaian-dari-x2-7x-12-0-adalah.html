12. Himpunan penyelesaian dari x2 + 7x + 12 = 0 adalah …


Penjelasan:

x² + 7x + 12 = 0

dengan metode kuadrat sempurna

x² + 7x + 12 = 0

x² + 7x = -12

x² + 7x + (7/2)² = -12 + (7/2)²

(x + 7/2)² = -12 + 49/4

(x + 7/2)² = -48/4 + 49/4

(x + 7/2)² = ¼

(x + 7/2) = √¼

x + 7/2 = ±½

ada 2 penyelesaian, x₁ dan x₂

untuk x₁ :

x₁ + 7/2 = ½

x₁ = ½ - 7/2

x₁ = -3

untuk x₂ :

x₂ + 7/2 = -½

x₂ = -½ - 7/2

x₂ = -4

HP : [{-3, -4}]

18. 4.Tentukan himpunan penyelesaian yang memenuhi pertidaksamaan-pertidaksamaan berikut: ▪4 + 8x < 6x –2 ▪x2+ 7x + 12 ≥ 0


Jawabannya adalah

Untuk 4 + 8x < 6x - 2 himpunan penyelesaiannya adalah x < -3 Untuk [tex]x^2 + 7x + 12 \geq 0[/tex] himpunan penyelesaiannya adalah [tex]x \geq -3[/tex] atau [tex]x \leq -4[/tex]

Pembahasan

Halo adik-adik! Balik lagi di Brainly!! Gimana, masih semangat belajar kah? Nah untuk pertanyaan di atas itu sedikit masuk ke materi tentang pertidaksamaan. Berbeda dengan persamaan yang mana di kalimatnya terdapat tanda sama dengan (=), kalau di pertidaksamaan kalimatnya mengandung tanda kurang dari (<) atau lebih dari (>) atau kurang dari sama dengan (≤) atau lebih dari sama dengan (≥). Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Nomor 1

[tex]4 + 8x < 6x - 2\\8x - 6x < -4 - 2\\2x < -6\\x < -3[/tex]

jadi himpunan penyelesaiannya adalah x < -3

Nomor 2

[tex]x^2 + 7x + 12 \geq 0\\(x + 4)(x + 3) \geq 0[/tex]

Diperoleh pembuat nol penyebut adalah  

[tex]x + 4 = 0\\x = -4[/tex]

atau

[tex]x + 3 = 0\\x = -3[/tex]

dengan meletakkan pembuat nol sebagai batas pada garis bilangan seperti pada lampiran, diperoleh tiga daerah yaitu [tex]x \geq -3[/tex] atau [tex]-4 \leq x \leq -3[/tex] atau [tex]x \leq -4[/tex]

akan diselidiki nilai dari masing-masing daerah

untuk daerah [tex]x \geq -3[/tex], diambil titik x = 0 lalu disubstitusikan ke [tex](x + 4) (x + 3)[/tex] diperoleh [tex](0 + 4) (0 + 3) = 12[/tex] yang berarti pada daerah [tex]x \geq -3[/tex] nilainya adalah positif untuk daerah [tex]-4 \leq x \leq -3[/tex], diambil titik x = -2 lalu disubstitusikan ke [tex](x + 4) (x + 3)[/tex] diperoleh [tex](-3,5 + 4) (-3,5 + 3) = -0,25[/tex] yang berarti pada daerah [tex]-4 \leq x \leq -3[/tex] nilainya adalah negatif untuk daerah [tex]x \leq -4[/tex], diambil titik x = -5 lalu disubstitusikan ke [tex](x + 4) (x + 3)[/tex] diperoleh [tex](-5 + 4) (-5 + 3) = 2[/tex] yang berarti pada daerah [tex]x \leq -4[/tex] nilainya adalah positif

Karena yang dicari adalah [tex] (x + 4)(x + 3) \geq 0[/tex] maka daerah penyelesaian dari pertidaksamaan tersebut adalah [tex]x \geq -3[/tex] atau [tex]x \leq -4[/tex]
