Penjelasan dengan langkah-langkah:

Perpangkatan atau eksponensial adalah perkalian berulang dengan faktor-faktor bilangan yang sama. Bentuk umum perpangkatan adalah  [tex]a^{n}[/tex] ; a adalah bilangan pokok dan n adalah pangkat.

Dalam kehidupan sehari-hari diperlukan cara penulisan sederhana untuk bilangan yang relative besar maupun kecil. Misalnya :

Jarak antara matahari dan neptunus dalam system tata surya adalah 4.500.000.000 km disederhanakan menjadi [tex]4,5.10^{9}[/tex]

Maka, penulisan tersebut menggunakan bilangan berpangkat.

Maka, dapat dijawab dengan cara berikut :

Diketahui : 2 digit terakhir  adalah 77.

Ditanya : 2 digit terakhir a?

Jawab :  adalah digit akhir 77.

            Pemangkatan yang menghasilkan digit akhir 7 adalah 3 dan 7.

Dapat dilihat pada penjelasan dibawah ini :

Pangkat : 0, 1, 2, 3, 4, 5

3             : 1, 3, 9, 7, 1, 3

7             : 1, 7, 9, 3, 1, 7

Maka, perpangkatan 3 menghasilkan 7 pada pangkat 3 dan 7.

Sedangkan, perpangkatan 7 yaitu pangkat 1 dan 5.

Pangkat 777hanya dapat dibagi 7 = (777:7=11)

Maka, a adalah bilangan berdigit akhir 3.

19. Hitung jumlah bilangan kuadrat antara 1000 dan 9999, dengan digit ratusan sama dengan digit satuan dan digit ribuan sama dengan digit puluhan.


Sebuah bilangan tiga digit mempunyai digit ratusan lebih besar dari digit puluhan dan digit ratusan sama dengan digit satuan. Diketahui bahwa selisih bilangan tersebut dengan bilangan yang diperoleh dengan menukar digit satuan dan puluhan bilangan tersebut adalah 72. Jika jumlah digit-digit bilangan tersebut adalah 19, tentukan bilangan tersebut.

Jawaban

karena angka satuan = angka ratusan maka bilangan tersebut adalah aba, dengan a > b.

selisih bilangan tersebut dan bilangan yang sudah ditukar adalah 72, maka:

aab - aba = 72 (100a + 10 a + b) - (100a + 10b + a) = 72 110a + b-101a-10b = 72 9a-9b = 72 a - b = 8... (1)

tetapi 2a + b = 19...(2)jumlahkan persamaan (2) dan (1) didapat :

(2a + b) + (a - b) = 19 +8

3a= 27

a=9

subtitusikan nilai a ke persamaan (1), didapat nilai b = 1

jadi, bilangan yang dimaksud adalah 919


20. Berapakah banyaknya bilangan bulat positif dua digit yang habis dibagi oleh masing-masing digitnya?


Jawab:

14 bilangan

Penjelasan dengan langkah-langkah:

Misalkan x adalah bilangan bulat positif 2 digit, a dan b berturut-turut adalah digit dari x.

Maka:

x = 10a + b

dengan a dan b bilangan cacah, 0 < a < 10, dan 0 ≤ b < 10.

Kasus (1)

x habis dibagi a dinyatakan dengan:

a | 10a + b

⇒ (10a + b) mod a = 0

⇒ b mod a = 0

⇒ b = a·k, k bilangan bulat

⇒ terpenuhi jika b adalah kelipatan dari a

⇒ b = 0 tidak memenuhi, karena pembagian dengan 0 tak terdefinisi

Kasus (2)

x habis dibagi b dinyatakan dengan:

b | 10a + b

⇒ (10a + b) mod b = 0

⇒ 10a mod b = 0

⇒ (2·5·a) mod b = 0

⇒ 10 mod b = 0, atau a mod b = 0, atau 2 mod b = 0, atau 5 mod b

10 mod b = 0
b kelipatan 10 ⇒ tidak mungkin, karena 0 ≤ b < 10a mod b = 0
⇒ a = b·k, k bilangan bulat
⇒ a kelipatan dari b
⇒ karena kasus (1) di atas, untuk kasus ini, a harus sama dengan b   .... (2.1)2 mod b = 0
⇒ b = 2 jika a = 1 .... (2.2)5 mod b = 0
⇒ b = 5 jika a = 1 .... (2.3)

Dari kasus (1) dan pernyataan (2.1), bilangan x yang memenuhi adalah:

11, 22, 24, 33, 36, 44, 48, 55, 66, 77, 88, 99  ⇒ 12 bilangan

Dari pernyataan (2.2) dan (2.3), bilangan x yang memenuhi adalah:

12, 15  ⇒ 2 bilangan

Total = 14 bilangan

KESIMPULAN

∴  Banyaknya bilangan bulat positif dua digit yang habis dibagi oleh masing-masing digitnya adalah 14 bilangan, yaitu 11, 12, 15, 22, 24, 33, 36, 44, 48, 55, 66, 77, 88, dan 99.

25. Digit tengah dari bilangan tiga digit adalah 0 dan jumlah dua digit lainnya adalah 7. Jika dua digit tersebut dibalik maka menjadi bilangan baru yang 99 lebih kecil dari bilangan awalnya. Carilah bilangan tersebut.


bilangan 3 angka, asumsikan urutan bilangannya adalah a,b, dan c, dengan angka tengah adalah 0 berarti b=0
bilangan awal: 100a + 10 + c
(ket : a + c = 7 (persamaan 1))
bilangan kedua (a dan c ditukar): 100c + 10 + a

bilangan awal - bilangan kedua = 99
(100a + 10 + c) - (100c + 10 + a) = 99
100a + 10 + c - 100c - 10 - a = 99
99a -99c = 99
-----------------------:99
a - c = 1 (persamaan 2)

eliminasi persamaan 1 dgn persamaan 2
a + c = 7
a - c = 1
________-
2c = 6
c = 3 (oke ketemu)

masukin c = 3, ke a + c = 7
a + 3 = 7
a = 4

jadi urutan angkanya adalah :
a = 4
b = 0
c = 3

jadi bilangan 3 digit pertama itu 403
dan bilangan 3 digit kedua itu 304
selisihnya 403 - 304 = 99

30. sebuah bilangan terdiri atas dua digit. nilai bilangan itu sama dengan tiga kali jumlah kedua digit itu di tambah 10.digit ke dua di kurang digit pertama sama dengan 5.Bilangan tersebut adalah.....


Digit kedua dikurang digit pertama = 5. Artinya bahwa digit kedua > digit pertama.
Kemungkinan itu terjadi ketika:
(i) 9 - 4 = 5 maka bilangannya 49
(ii) 8 - 3 = 5 maka bilangannya 38
(iii) 7 - 2 = 5 maka bilangannya 27
(iv) 6 - 1 = 5 maka bilangannya 15
(v) 5 - 0 = 5 maka bilangannya 05.

Untuk (v) tidak memenuhi karna bukan dua digit.
