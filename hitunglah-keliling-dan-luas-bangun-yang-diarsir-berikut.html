28. Hitung keliling dan luas bangun yang diarsir berikut


Keliling di arsir =
Keliling persegi panjang- keliling setengah lingkaran(dalam persegi panjang) + keliling lingkaran (luar persegi panjang bagian bawah) + keliling lingkaran (sebelah kanan persegi panjang)

Keliling persegi panjang = 2(p+L) = 2(10) + 2(15) = 20+30 =50
Keliling setengah lingkaran dalam persegi panjang = keliling setengah lingkaran bagian bawah persegi panjang

Keliling lingkaran = 2 x pi x r = 2 x3,14x5 = 31,4 : 2 = 15,7

Keliling setengah lingkaran di sebelah kanan persegi panjang

K= 1/2 x 2 x pi x r
K= 3,14 x 7,5= 23,55

Keliling di arsir = 50-15,7 + 15,7 + 23,55 = 73,55

Luas di arsir = luas persegi panjang - luas setengah lingkaran (bagian dalam) + luas setengah lingkaran (bagian bawah) + luas setengah lingkaran (bagian kiri)

Luas persegi panjang = px L = 10x15 = 150

Luas setengah lingkaran bagian dalam dan bagian bawah
= 3,14 x 5 x 5 x 1/2
=39,25 x 1/2 = 19,625

Luas setengah lingkaran bagian kiri
= 1/2 x 3,14 x 7,5 x 7,5
=88,3125

Luas di arsir = 150 - 19,625 + 19,625 + 88,3125 =238,3125
