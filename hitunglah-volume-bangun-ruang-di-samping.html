15. Hitunglah volume bangun ruang disamping!​


Volume dari bangun ruang Prisma segitiga tersebut adalah 384cm³

PENDAHULUAN

Volume adalah isi dari suatu bangun ruang yang dapat di tampung. Contoh dari bangun ruang yaitu Kubus, Balok, Bola, Prisma, Kerucut, dll.

Rumus Volume Prisma ∆

[tex] \boxed{ \tt V = La \times t }[/tex]

Luas ∆

[tex] \boxed{ \tt L = \frac{1}{2} \times a \times t \: segitiga }[/tex]

Teorema Phytagoras

[tex] \boxed{ \tt {c}^{2} = {a}^{2} + {b}^{2} }[/tex]

Keterangan:

La = Luas alas nya, yaitu segitiga (∆) t = tinggi prisma t ∆ = tinggi pada segitiga a = alas segitigab = tinggi segitigac = sisi miring segitiga

PENYELESAIAN

Diketahui:

sisi miring segitiga = 20cmalas segitiga = 12 cmtinggi prisma = 4cm

Ditanya: Volume prisma?

jawab:

Karna pada alas ∆ tidak di ketahui berapa tingginya dan hanya di ketahui alas ∆ dan sisi miring ∆, maka kita dapat mencari terlebih dahulu tinggi dari ∆ dengan menggunakan Phytagoras.

[tex]{ \tt {c}^{2} = {a}^{2} + {b}^{2} } \\ { \tt {20}^{2} = {12}^{2} + {b}^{2} } \\ { \tt 400 = 144 + {b}^{2} } \\ { \tt 400 - 144 = {b}^{2} } \\ { \tt 256 = {b}^{2} } \\ { \tt \sqrt{256} = b} \\ { \tt 16 = b }[/tex]

Berarti sudah di dapat kan tinggi dari segitiga, yaitu 16cm. Selanjutnya kita dapat mencari berapa Volume pada prisma.

[tex]{ \tt V = La \times t } \\ { \tt V = ( \frac{1}{ \not2} \times \not12 \times 16) \times 4} \\ { \tt V = ( 1 \times 6 \times 16) \times 4} \\ { \tt V = 96 \times 4 } \\ { \tt V = 384 {cm}^{3} }[/tex]

Jadi, Volume prisma segitiga tersebut adalah 384 cm³.

16. hitunglah volume bangun ruang disamping


tabung adalah bangun ruang dengan sisi lengkung

volume tabung = π x r² x t

r = jari-jari

t = tinggi

kubus adalah bangun ruang dengan sisi datar

volume kubus = s x s x s

                        = s³

Pembahasan

diketahui:

jari" tabung = 7/2 cm = 3,5 cm

ditanya volume gabungan = ... ?

                         = ²²/₇ x 3,5 x 3,5 x 8

                         = 7 x 7 x 7

volume tabung + volume kubus = 308 dm³ + 343 dm³                                                                                      = 651 dm³
