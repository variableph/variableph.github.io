3. ikhsan memiliki 3 orang anak . pada suatu hari 3 anaknya terkena flu burung. sampai di rumah sakit diperoleh data bahwa suhu badan 3 anak itu masing" 40celcius , 39 , 5celcius dan 40,6celcius . Ubahlah 3 suhu badan itu dalam derajat reamur dan farenhit


Ikhsan memiliki 3 orang anak. Pada suatu hari anaknya terkena flu. Setelah dibawa ke rumah sakit suhu badan ketiga anak itu masing masing 40ᵒ C, 39,5ᵒ C, 40,6ᵒ C. Ubahlah ketiga suhu badan tersebut dalam derajat reamur dan Fahrenheit. Satuan suhu terdiri dari celcius (C), Fahrenheit (F) dan Reamur (R).

Termometer skala Celcius ⇒ Titik didih air = 100° C dan Titik beku = 0° C   Termometer skala Reamur ⇒ Titik didih air = 80° R dan Titik beku = 0° R   Termometer skala Fahrenheit ⇒ Titik didih air = 212° F dan Titik beku = 32° F  

Perbandingan Celcius dengan Reamur

(C - 0°) : (R - 0°) = (100° - 0°) : (80° - 0°)  

C : R = 100° : 80°  

C : R = 5 : 4  

maka  

C = [tex]\frac{5}{4}[/tex] × R  R = [tex]\frac{4}{5}[/tex] × C  

Perbandingan Celcius dengan Fahrenheit  

(C - 0°) : (F - 32°) = (100° - 0°) : (212° - 32°)

C : (F - 32°) = 100° : 180°  

C : (F - 32°) = 5 : 9  

maka  

C = [tex]\frac{5}{9}[/tex]× (F - 32°)  (F - 32)° = [tex]\frac{9}{5}[/tex] × C ⇒ F = [tex]\frac{9}{5}[/tex] × C + 32°  

Perbandingan Reamur dengan Fahrenheit  

(R - 0°) : (F - 32°) = (80° - 0°) : (212° - 32°)

R : (F - 32°) = 80° : 180°  

R : (F - 32°) = 4 : 9  

maka  

R = [tex]\frac{4}{9}[/tex] × (F - 32°)  (F - 32°) = [tex]\frac{9}{4}[/tex] × R ⇒ F = [tex]\frac{9}{4}[/tex] × R + 32°
Pembahasan

Mengubah derajat Celcius ke derajat Reamur


40ᵒ C = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 40ᵒ  

R = 4 × 8ᵒ

R = 32ᵒ

Jadi 40ᵒ C = 32 ᵒR


39,5ᵒ C = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 39,5ᵒ  

R = 4 × 7,9ᵒ

R = 31,6ᵒ

Jadi 39,5ᵒ C = 31,6 ᵒR


40,6ᵒ C = = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 40,6ᵒ  

R = 4 × 8,12ᵒ

R = 32,48ᵒ

Jadi 40,6ᵒ C = 32,48 ᵒR




Mengubah derajat Celcius ke derajat Fahrenheit


40ᵒ C = .... ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = [tex]\frac{9}{5}[/tex] × 40° + 32°

F = 9 × 8° + 32°

F = 72° + 32°

F = 104°

Jadi 40ᵒ C = 104 ᵒF


39,5ᵒ C = .... ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = [tex]\frac{9}{5}[/tex] × 39,5° + 32°

F = 9 × 7,9° + 32°

F = 71,1° + 32°

F = 103,1°

Jadi 39,5ᵒ C = 103,1 ᵒF


40,6ᵒ C = .... ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = [tex]\frac{9}{5}[/tex] × 40,6° + 32°

F = 9 × 8,12° + 32°

F = 73,08° + 32°

F = 105,08°

Jadi 40,6ᵒ C = 105,08 ᵒF

10. Ikhsan memiliki 3 orang anak. Pada suatu hari ketiga anaknya terkena flu burung. Sampai di rumah sakit diperoleh data bahwa suhu badan ketiga anak itu masing-masing, 40oC, 39,5oC, dan 40,6oC. Ubahlah ketiga suhu badan itu dalam derajat Reamur dan Fahrenheit.


Kelas          : VII
Pelajaran    : Matematika
Kategori      : Perbandingan Skala
Kata Kunci  : suhu, derajat, Celcius, Reamur, Fahrenheit, Kelvin, konversi

Kode : 7.2.5 [Kelas 7 Matematika Bab 5 - Perbandingan]

Diketahui
Suhu masing-masing anak adalah 40°C, 39,5°C, dan 40,6°C

Ditanya
Ketiga suhu tersebut dalam skala Reamur dan Fahrenheit

Penyelesaian

Terdapat empat jenis skala suhu yang umumnya kita kenal, yakni Celcius (°C), Reamur (°R), Fahrenheit (°F), dan Kelvin (K). Keempatnya diberi nama sesuai nama ilmuwan penemunya. Hal yang perlu dicatat khusus untuk skala Kelvin yaitu tidak diberi lambang derajat, merupakan skala suhu mutlak, dan sebagai besaran pokok suhu.

Dari C, R, dan F, hanya Celcius yang terhubung langsung dengan Kelvin. Konversi atau perubahan suhu antara C dan K adalah sebagai berikut:
[tex]K=^{0}C+273[/tex] atau [tex]^{0}C=K-273[/tex].

Perbandingan antara C, R, dan F yaitu 5 : 4 : 9. Sehingga hubungan antara C dan R adalah sebagai berikut:
[tex] \frac{^{0}C}{^{0}R}= \frac{5}{4} [/tex].
Dapat ditulis menjadi [tex]^{0}C=^{0}R\times \frac{5}{4} [/tex]
Atau [tex]^{0}R=^{0}C\times \frac{4}{5}[/tex]

Hubungan antara C dan F adalah sebagai berikut:
[tex] \frac{^{0}C}{(^{0}F-32)}= \frac{5}{9} [/tex].
Dapat ditulis menjadi [tex]^{0}C= \frac{5}{9}(^{0}F-32) [/tex]
Atau [tex]^{0}F= \frac{9}{5}(^{0}C)+32 [/tex]

Hubungan antara R dan F dinyatakan sebagai berikut:
[tex] \frac{^{0}R}{(^{0}F-32)}= \frac{4}{9} [/tex].
Dapat ditulis menjadi [tex]^{0}R= \frac{4}{9}(^{0}F-32) [/tex]
Atau [tex]^{0}F= \frac{9}{4}(^{0}R)+32 [/tex]

(a). Suhu anak pertama 40°C

Konversi C ⇒ R
[tex] \frac{40}{^{0}R}= \frac{5}{4} [/tex] atau
[tex]^{0}R=40 \times \frac{4}{5} [/tex]
Diperoleh suhu dalam derajat Reamur sebesar 32°R.

Konversi C ⇒ F
[tex]^{0}F= \frac{9}{5}(^{0}C)+32 [/tex]
[tex]^{0}F= \frac{9}{5}(40)+32 [/tex]
Diperoleh suhu dalam derajat Fahrenheit sebesar 104°F.

(b). Suhu anak kedua 39,5°C

Konversi C ⇒ R
[tex] \frac{39,5}{^{0}R}= \frac{5}{4} [/tex] atau
[tex]^{0}R=39,5 \times \frac{4}{5} [/tex]
Diperoleh suhu dalam derajat Reamur sebesar 31,6°R.

Konversi C ⇒ F
[tex]^{0}F= \frac{9}{5}(^{0}C)+32 [/tex]
[tex]^{0}F= \frac{9}{5}(39,5)+32 [/tex]
Diperoleh suhu dalam derajat Fahrenheit sebesar 103,1°F.

(c). Suhu anak ketiga 40,6°C

Konversi C ⇒ R
[tex] \frac{40,6}{^{0}R}= \frac{5}{4} [/tex] atau
[tex]^{0}R=40,6 \times \frac{4}{5} [/tex]
Diperoleh suhu dalam derajat Reamur sebesar 32,48°R.

Konversi C ⇒ F
[tex]^{0}F= \frac{9}{5}(^{0}C)+32 [/tex]
[tex]^{0}F= \frac{9}{5}(40,6)+32 [/tex]
Diperoleh suhu dalam derajat Fahrenheit sebesar 105,08°F.

[tex] T_{r1} = \frac {800}{25}[/tex]

fahrenheit

[tex] \frac {T_{c1}}{100} = \frac {T_{F1} -32}{180} [/tex]

[tex] \frac {40}{100} = \frac {T_{F1} -32}{180} [/tex]

[tex] \frac {10}{25} = \frac {T_{F1} -32}{180} [/tex]

[tex] \frac {180 \times 10}{25} = T_{F1} -32[/tex]

PEMBAHASAN MATERI SUHU, KALOR DAN PEMUAIAN

[tex] \boxed {SUHU} [/tex]

         Suhu merupakan ukuran keadaan panas/dingin dari suatu benda. Alat untuk mengukur suhu adalah Termometer. Terdapat 4 jenis termometer yaitu Termometer Reamur, Termometer Celcius, Termometer Fahrenheit dan Termometer Kelvin.


Termometer Reamur :

- Titik atas 80°R

- Titik bawah 0°R


Termometer Celcius

- Titik atas 100°C

- Titik bawah 0°C


Termometer Fahrenheit

- Titik atas 212°F

- Titik bawah 32°F


Termometer Kelvin

- Titik atas 373 K

- Titik bawah 273 K


Perbandingan keempat suhu tsb, sebagai berikut

[tex] \boxed { \frac {T_R}{80} = \frac {T_C}{100} = \frac {T_F -32}{180} = \frac {T_K - 273}{100}} [/tex]


Kalibrasi Termometer

[tex] \boxed { \frac {T_{Xt} - T_{Xo}}{T_X - T_{Xo}} = \frac {T_{Yt} - T_{Yo}}{T_Y - T_{Yo}}} [/tex]


[tex] \boxed {KALOR} [/tex]

         Kalor merupakan suatu bentuk energi yang dihasilkan karena adanya suhu atau temperatur.


         [tex] \boxed {Q = m c \Delta T} [/tex]


Q = Kalor (Joule)

m = massa benda (kg)

c = massa jenis benda (J/kg°C)

ΔT = perubahan suhu (°C)


•> [ASAS BLACK]

         [tex] \boxed {Q_{lepas} = Q_{terima}} [/tex]


[tex] \boxed {PEMUAIAN} [/tex]

         Pemuaian berarti bertambahnya panjang, luas ataupun suatu volume dari benda yang memiliki koefisien muai karena adanya kenaikan suhu.


Pemuaian Panjang memiliki koefisien muai α , Pemuaian Luas memiliki koefisien muai β dan Pemuaian Volume memiliki koefisien muai γ.


Pemuaian Panjang,


         [tex] \boxed {L = Lo ( 1 + \alpha \Delta t)} [/tex]


Pemuaian Luas,

         [tex] \boxed {A = Ao ( 1 + \beta \Delta t)} [/tex]


Pemuaian Volume

         [tex] \boxed {V = Vo ( 1 + \gamma \Delta t)} [/tex]

24. ikhsan memiliki 3 orang anak,pada suatu hari ketiga anak nya terkena flu burung,sampai dirumah sakit dipeloreh data bahwa suhu badan ketiga anak masing masing 40°c,39,5°c,dan40,6°c.ubalah ketiga suhu anak itu dalam derajat reamur dan fahrenheit


Ikhsan memiliki 3 orang anak. Pada suatu hari anaknya terkena flu. Setelah dibawa ke rumah sakit suhu badan ketiga anak itu masing masing 40ᵒ C, 39,5ᵒ C, 40,6ᵒ C. Ubahlah ketiga suhu badan tersebut dalam derajat reamur dan Fahrenheit. Satuan suhu terdiri dari celcius (C), Fahrenheit (F) dan Reamur (R).

Termometer skala Celcius ⇒ Titik didih air = 100° C dan Titik beku = 0° C   Termometer skala Reamur ⇒ Titik didih air = 80° R dan Titik beku = 0° R   Termometer skala Fahrenheit ⇒ Titik didih air = 212° F dan Titik beku = 32° F  

Perbandingan Celcius dengan Reamur

(C - 0°) : (R - 0°) = (100° - 0°) : (80° - 0°)  

C : R = 100° : 80°  

C : R = 5 : 4  

maka  

C = [tex]\frac{5}{4}[/tex] × R  R = [tex]\frac{4}{5}[/tex] × C  

Perbandingan Celcius dengan Fahrenheit  

(C - 0°) : (F - 32°) = (100° - 0°) : (212° - 32°)

C : (F - 32°) = 100° : 180°  

C : (F - 32°) = 5 : 9  

maka  

C = [tex]\frac{5}{9}[/tex]× (F - 32°)  (F - 32)° = [tex]\frac{9}{5}[/tex] × C ⇒ F = [tex]\frac{9}{5}[/tex] × C + 32°  

Perbandingan Reamur dengan Fahrenheit

(R - 0°) : (F - 32°) = (80° - 0°) : (212° - 32°)

R : (F - 32°) = 80° : 180°  

R : (F - 32°) = 4 : 9  

maka  

R = [tex]\frac{4}{9}[/tex] × (F - 32°)  (F - 32°) = [tex]\frac{9}{4}[/tex] × R ⇒ F = [tex]\frac{9}{4}[/tex] × R + 32°
Pembahasan

Mengubah derajat Celcius ke derajat Reamur


40ᵒ C = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 40ᵒ  

R = 4 × 8ᵒ

R = 32ᵒ

Jadi 40ᵒ C = 32 ᵒR


39,5ᵒ C = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 39,5ᵒ  

R = 4 × 7,9ᵒ

R = 31,6ᵒ

Jadi 39,5ᵒ C = 31,6 ᵒR


40,6ᵒ C = = .... ᵒR

R = [tex]\frac{4}{5}[/tex] × C  

R = [tex]\frac{4}{5}[/tex] × 40,6ᵒ  

R = 4 × 8,12ᵒ

R = 32,48ᵒ

Jadi 40,6ᵒ C = 32,48 ᵒR




Mengubah derajat Celcius ke derajat Fahrenheit


40ᵒ C = .... ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = [tex]\frac{9}{5}[/tex] × 40° + 32°

F = 9 × 8° + 32°

F = 72° + 32°

F = 104°

Jadi 40ᵒ C = 104 ᵒF


39,5ᵒ C = .... ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = [tex]\frac{9}{5}[/tex] × 39,5° + 32°

F = 9 × 7,9° + 32°

F = 71,1° + 32°

F = 103,1°

Jadi 39,5ᵒ C = 103,1 ᵒF

F = [tex]\frac{9}{5}[/tex] × C + 32°

F = 73,08° + 32°

F = 105,08°

Jadi 40,6ᵒ C = 105,08 ᵒF
