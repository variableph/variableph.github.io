12. ion X3+ mempunyai konfigurasi elektron 1s2 2s2 2p6. bilangan kuantum elektron terakhor atom X adalah .....


Harga keempat bilangan kuantum terakhir atom S tersebut adalah n = 3, ℓ = 1, m = -1, s = +½. Bilangan kuantum tersebut dapat diketahui jika diketahui konfigurasi elektron terakhir. Konfigurasi dapat ditentukan menggunakan jumlah elektron. Jumlah elektron atom dapat diketahui dari nomor atom an muatan atomnya.

Pembahasan

Bilangan kuantum dari masing-masing elektron penyusun atom terdir dari 4 jenis. Berikut penjelasan terkait empat jenis bilangan kuantum tersebut.

1) Bilangan kuantum utama ( n )

Bilangan kuantum ini menyatakan letak elektron dalam kulit atom atau bilangan yang mewakili nomor kulit,  

jenisnya : K ( n = 1 ), L ( n = 2 ), M ( n = 3 ), N ( n = 4 ), dst.

2) Bilangan kuantum azimuth ( ℓ ) → (n-1)

Bilangan kuantum ini menyatakan letak elektron dalam sub kulit.atau bilangan yang mewakili subkulit atom yang ditempati elektron. Aturan menentukan banyaknya subkulit adalah n – 1 (dengan n adalah nomor kulit),  

Jenis subkulit :  

s = sharp (ℓ = 0);   p = principal (ℓ=1) ; d = diffuse (ℓ = 2);  f = fundamental (ℓ = 3)

3) Bilangan kuantum magnetik (m)

Bilangan kuantum ini menyawakan letak elektron dalam orbital atau bilangan yang mewakili orbital yang ditempati elektron. Aturan banyaknya orbital adalah mulai dari – ℓ  sampai dengan +ℓ. Berikut jenis orbital sesuai subkulitnya:

jika ℓ = 0, maka orbitalnya = 0

jika ℓ = 1, maka orbitalnya mulai dari -1 s/d +1 = -1, 0, +1

jika ℓ = 2, maka orbitalnya mulai dari -2 s/d +2 = -2, -1, 0, +1, +2

jika ℓ = 3, maka orbitalnya mulai dari -3 s/d +3 = -3, -2, -1, 0, +1, +2, +3

4) Bilangan kuantum spin (s)

Bilangan kuantum yang menyatakan arah elektron putaran elektron dalam orbital. Jenisnya:  

+ ½  untuk tanda panah ke atas (↑) => putaran searah jarum jam

- ½  untuk tanda panah ke bawah (↓)=> putaran berlawanan arah jarum jam

Pada kasus di atas

Pada soal diketahui ion X³⁺ dengan konfigurasi elektron 1s² 2s² 2p⁶. Adapun yang ditanyakan adalah harga keempat bilangan kuantum elektron terakhir dari atom X.

Setelah mengetahui beberapa jenis bilangan kuantum kalian dapat menentukan bilangan kuantum elektron terakhir pada atom X tersebut. Langkah awal yang harus dilakukan adalah menentukan konfigurasi elektron atom X. Konfigurasi elektron atom X dapat ditentukan jika diketahui jumlah elektron dalam atom X tersebut. Pada soal diketahui ion X³⁺ memiliki konfigurasi 1s² 2s² 2p⁶ jumlah elektron pada ion X³⁺ tersebut adalah

Elektron X³⁺ = 2 + 2 + 6 = 10

Elektron X = elektron X³⁺ + muatan

                  = 10 + 3  

                  = 13

Konfigurasi elektronnya dapat dituliskan sebagai berikut:

₁₃X = 1s² 2s² 2p⁶ 3s² 3p¹

Elektron terakhir pada atom X tersebut terletak pada orbital 3p¹. Berikut analisis keempat bilangan kuantmnya:

n = 3 (kulit yang ditempati)

ℓ = 1 (subkulit p)

m = -1       [↑ |   |   ]

                -1  0  +1

S = +½ (arah panah ↑)

Jadi dapat disimpulkan keempat bilangan kuantum elektron terakhir dari atom X tersebut adalah n = 3, ℓ = 1, m = -1, s = +½.
