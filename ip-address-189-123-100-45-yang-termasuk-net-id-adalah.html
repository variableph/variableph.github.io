5. Dari Alamat IP 179.123.100.45/16, yang termasuk Net ID adalah....


Net ID dari alamat di atas adalah 179.123. Network ID atau Net ID adalah sebuah identitas yang mana dari pengelompokkan IP Addres sehingga pada beberapa komputer dengan jaringan yang sama secara umum memiliki Net ID yang sama.

Pembahasan

Net ID menjadi bukti pentingnya alamat pada jaringan untuk bisa berkomunikasi dalam jaringan yang sama, berikut beberapa fungsi Net ID lainnya:

Network ID menjadi identitas suatu kelompok alamat. Dalam internet terkenal di Indonesia seperti Indihome secara umum jika memasang suatu jaringan dalam ruang atau jangkauan tertentu akan memiliki Net ID yang sama, hal tersebut membuktikan bahwa jaringan mereka satu namun berbeda Host IDnya saja sebagai penanda berbeda komputer ataupun laptop. Maka tentu saja Net ID menjadi pedoman jaringan agar tidak terjadi salah mengirim atau informasi salah pada jaringan dalam jangka panjang.Menjaga jaringan. Dalam Net ID yang berbeda mencegah interaksi dari jaringan lainnya mengganggu, dari hal inilah pentingnya interaksi antar jaringan dalam satu arah. Maka mencegah dari hal-hal yang tidak diinginkan seperti mencuri informasi maupun hal-hal lainnya yang merugikan.Menunjukan nomor jaringan. Pada nomor jaringan ini agar internet maupun aktivitas lainnya dapat berjalan baik, dari hal inilah kecepatan internet sesuai jalur jaringan masing-masing sehingga meminimalisir hingga mencegah terjadinya kesalahan jaringan yang mana dimiliki. Maka dari ini pentingnya fungsi dari Net ID.

14. Tentukan Host ID dan Net ID dari IP address 10.10.10.1​


Host ID dan Net ID adalah bagian-bagian dari alamat IP yang digunakan untuk membedakan antara perangkat atau host dalam jaringan dengan jaringan itu sendiri.

Untuk IP address 10.10.10.1, dapat ditentukan sebagai berikut:

Net ID: 10.10.10Host ID: .1

Net ID merupakan bagian dari alamat IP yang digunakan untuk menentukan jaringan yang digunakan, dan Host ID merupakan bagian dari alamat IP yang digunakan untuk menentukan perangkat yang terhubung ke jaringan tersebut.

Net ID dapat digunakan untuk menentukan subnet dari jaringan, sementara Host ID digunakan untuk menentukan alamat unik dari perangkat yang terhubung ke jaringan tersebut.

24. bagaimana pendapatmu tentang IP address 192.168.2.3 yang termasuk network id dan host ​


IP Address adalah sebuah alamat pada komputer agar komputer bisa saling terhubung dengan komputer lain, IP Address terdiri dari 4 Blok, setiap Blok di isi oleh angka 0 – 255. Contoh IP Address seperti 192.168.100.1 , 10.57.38.223 , ini adalah IPv4.

IP Address Memiliki 2 bagian, yaitu Network ID dan Host ID , contoh 192.168.100.1 , secara default Net ID nya adalah 192.168.100 dan Host ID nya adalah 1, agar komputer bisa saling terhubung , IP yang digunakan Net ID nya harus sama, dan Host ID nya harus berbeda.
