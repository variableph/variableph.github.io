misal :

mobil M = x

mobil N = y

total tepung

6x + 4y <= 240 ton

total mobil

x + y <= 48 buah

x>=0

y>=0

x + y = 48 <=> y = 48 - x

6x + 4(48-x) = 240

6x + 192 - 4x = 240

2x = 48

x = 24

y = 48 - 24 = 24

(24,24)

Biaya sewa

z = 100.000x + 50.000y

(0,48) ---> z = 100.000(0) + 50.000(48) = 2.400.000

(40,0) ---> z = 100.000(40) + 50.000(0) = 4.000.000

(24,24) ---> z = 100.000(24) + 50.000(24) = 3.600.000

Algoritma untuk menentukan biaya sewa mobil rental yang menyewakan 4 jenis mobilPendahuluanPengertian Algoritma

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan. Misalnya adalah langkah – langkah untuk memasak mie instan, langkah – langkah yang dapat kita temukan di balik bungkus mie instan tersebut juga dapat kita sebut sebagai sebuah algoritma karena merupakan langkah-langkah untuk menyelesaikan permasalahan yaitu memasak mi instan menjadi matang.  

Dalam dunia komputer, algoritma merupakan alur pemrosesan pada sistem komputer yang terbentuk dari kumpulan command atau perintah yang terintegrasi pada setiap aspek komputer (brainware, software, dan hardware).  

Dalam dunia pemrograman, algoritma digunakan untuk membuat software dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

Pengertian dan Pengenalan Program

Apakah program menurut kalian? Program adalah file teks yang berisi pernyataan-pernyataan dalam bahasa pemrograman. Kemudian bagaimanakah cara membuat program? Ada beberapa langkah yang dilakukan, yaitu:

Merancang program. Disini kita mendefinisikan masalah dan menulis algoritmanya untuk penyelesaian masalah tersebut. Memasukkan program ke komputer (Implementasi).  Kompilasi program ke “excecutable”. Excecutable: membuat program dapat dijalankan. Jalankan program. Ulangi hingga benar. Ini disebut proses debugging.Pembahasan

Karena tidak ada jumlah biaya yang pasti pada soal, kita asumsikan biaya sewa setiap mobil adalah sebagai berikut

Avanza = price_avanzaAPV = price_apvInnova = price_innovaMinibus = prive_minibus

Kemudian kita asumsikan jangka waktu menyewa adalah setiap 1 hari.

Maka algoritma untuk menentukan biaya sewa mobil adalah sebagai berikut

StartPilih jenis mobilPilih berapa hari akan menyewa mobilJika mobil yang dipilih adalah Avanza, biaya total = price_avanza * hariJika mobil yang dipilih adalah APV , biaya total = price_apv * hariJika mobil yang dipilih adalah Innova, biaya total = price_innova * hariJika mobil yang dipilih adalah Minibus, biaya total = price_minibus * hariTampilkan biayaSelesai

Contoh program untuk menentukan biaya sewa mobil dalam bahasa C

#include<stdio.h>

#include<conio.h>

void main()

{

int avanza = 1, apv = 2, innova = 3, minibus = 4;

int price_avanza = 100000, price_apv = 120000, price_innova = 130000, price_minibus = 150000;

int biaya, hari;

int opsi;

printf("Masukan pilihan mobil (1 avanza, 2 apv, 3 innova, 4 minibus): ");

scanf("%i",&opsi);

printf("Masukan jumlah hari menyewa: ");

scanf("%i",&hari);

if(opsi == avanza) biaya = price_avanza * hari;

else if(opsi == apv) biaya = price_apv * hari;

else if(opsi == innova) biaya = price_innova * hari;

else if(opsi == minibus) biaya = price_minibus * hari;

printf("Biaya sewa : %i", biaya);

getch();

}

Kesimpulan

Algoritma langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan, kita bisa membuatnya dengan logika berpikir yang sederhana tapi sistematis. Setelah kita dapatkan algoritmanya, membuat program pun akan terasa mudah.

Buatlah algoritma untuk menentukan biaya sewa mobil rental yang menyewakan 4 jenis mobil, yaitu Avanza, APV, Innova, dan Minibus. Biaya sewa perhari dari tiap mobil berbeda beda. Total biaya sewa diperoleh dengan mengalikan biaya sewa mobil dikalikam berapa lama mobil disewa (dalam satuan hari).  Sebelum membahas mengenai algoritma tentang persewaan mobil, mari kita bahas telebih dahulu apa itu algoritma. Pada soal kali ini kita menghadapi masalah untuk membuat perhitungan biaya sewa mobil.

2. Setelah variabel harga sudah ada, maka langkah selanjutnya adalah menghitung berdasarkan lama sewa mobil.

3. Misal, APV disewa selama 2 hari. Maka biayanya adalah Rp 300.000 x 2 hari, dengan hasil Rp 600.000

4. Masukkan data mobil kembali, cek terlebih dahulu jenis mobil kemudian dikali lama sewa.

5. Tampilkan total perhitungan sewa mobil
