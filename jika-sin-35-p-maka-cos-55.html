Diketahui tan 35° = 2/p, maka dapat diketahui bahwa karena tan 35° bernilai positif, maka 2/p adalah positif yang berakibat pada p > 0

Berdasarkan sifat segitiga siku-siku pebentuk trigonometri, diperoleh bahwa pembanding yang digunakan adalah:
[tex]\displaystyle \tan 35^o=\frac yx=\frac2p \\ \downarrow \\ x=p,\ y=2,\ r=\sqrt{x^2+y^2}=\sqrt{p^2+4}[/tex]

Atas jabaran tersebut, diperoleh (dengan 35° berada pada kuadran I, maka semua return pada fungsi bernilai positif):
[tex]\displaystyle \sin35^o=\frac{y}{r}=\frac{2}{\sqrt{p^2+4}} \\ \cos35^o=\frac{x}{r}=\frac{p}{\sqrt{p^2+4}} \\ \tan35^o=\frac{y}{x}=\frac{2}{p} \\ \csc35^o=\frac{r}{y}=\frac{\sqrt{p^2+4}}{2} \\ \sec35^o=\frac{r}{x}=\frac{\sqrt{p^2+4}}{p} \\ \cot35^o=\frac{x}{y}=\frac{p}{2}[/tex]

Menggunakan sifat kuadran, akan diperoleh:
1.)
[tex]\displaystyle \sin55^o=\cos(90^o-55^o)=\cos35^o=\frac{p}{\sqrt{p^2+4}}[/tex]

2.)
[tex]\displaystyle \cos215^o=\cos(180^o+35^o)=-\cos35^o=-\frac{p}{\sqrt{p^2+4}}[/tex]

3.)
[tex]\displaystyle \tan125^o=\tan(90^o+35^o)=-\cot35^o=-\frac{p}{2}[/tex]

4.)
[tex]\displaystyle \sec(-145)^o=\sec145^o=\sec(180^o-35^o)=-\sec35^o=-\frac{\sqrt{p^2+4}}{p}[/tex]

5.)
[tex]\displaystyle \csc305^o=\csc(270^o+35^o)=-\sec35^o=-\frac{\sqrt{p^2+4}}p[/tex]

6.)
[tex]\displaystyle \cot235^o=\cot(270^o-35^o)=\tan35^o=\frac2p[/tex]

y = a ( x- x1) ( x – x2 )

-3 = a ( 2 – 3 ) ( 2 + 1 )

y = a ( x – 3 ) ( x + 1 )

y = 1 ( x – 3 ) ( x + 1 )
