11. buatlah contoh website menggunakan wordpress tentang biodata diri seseorang​


Jawaban:

Langkah 1. Beli Nama Domain

Langkah pertama menuju pembuatan situs pribadi Anda adalah membeli nama domain.

Karena situs ini adalah tentang Anda, pilihan yang jelas adalah menggunakan nama Anda sendiri.

Step 2. Pilih Hosting

Setelah Anda memiliki nama domain Anda, Anda perlu membeli hosting untuk situs web Anda. Anda bisa membeli hosting seharga $5 / month dan dalam kebanyakan kasus, Anda bahkan bisa membeli nama domain dan hosting dari penyedia yang sama. Pastikan untuk memilih penyedia hosting yang menawarkan dukungan pelanggan sehingga Anda bisa mendapatkan bantuan segera jika Anda membutuhkannya.

Langkah 3. Install WordPress

Kebanyakan penyedia hosting saat ini memiliki satu klik installer untuk WordPress. WordPress sendiri cukup mudah dipasang secara manual, setup satu klik mempercepat prosesnya dengan cukup. Apapun host yang Anda pilih, prosesnya relatif sama.

Softaculous wordpress installer

Installer Website

Setelah Anda membeli paket hosting, perusahaan hosting Anda akan mengirimkan email selamat datang yang berisi semua informasi penting tentang hosting Anda. Ini biasanya menyertakan tautan untuk masuk ke cPanel atau dasbor hosting Anda, serta nama pengguna dan kata sandi Anda untuk cPanel.

Langkah 4. Pilih Theme Wordpress Profesional

Situs pribadi Anda perlahan-lahan datang bersama. Anda memiliki domain, Anda telah membeli hosting, dan Anda juga menginstal WordPress. Langkah selanjutnya dalam daftar kami adalah memilih tema untuk situs web Anda.

Wordpress memiliki banyak tema, baik yang gratis dan berbayar, bagaimanapun, tidak semua tema diciptakan sama. Jika Anda serius membuat situs web resume profesional, satu-satunya pilihan yang masuk akal adalah tema WordPress premium.

Best WordPress Themes on ThemeForest

Tema WordPress yang terbaik di ThemeForest.

Dengan memilih tema premium, Anda dapat yakin situs web Anda mengikuti praktik WordPress terbaik dan Anda bahkan bisa mendapatkan dukungan dari pengembang tema jika Anda memerlukan bantuan dalam menyiapkannya.

Selain itu, tema resume WordPress juga memiliki fitur yang diperlukan untuk menampilkan pendidikan, keterampilan, pengalaman masa lalu, dan contoh proyek masa lalu Anda.

Terlebih lagi, Anda bahkan bisa memasukkan prestasi dan penghargaan, serta testimonial dari atasan masa lalu atau profesor perguruan tinggi Anda untuk menunjukkan keahlian Anda.

14. jabarkan tentang aplikasi CMS Joomla​


Jawaban:

Jommla yang awalnya dibuat oleh perusahaan Miro yang berasal dari Australia yang dalam pengembangannya berawal dari poject open source kemudian beralih ke mambo CMS seiring dengan perkembangannya mambo berubah menjadi Joomla yang berjalan sampai sekarang

Joomla salah satu aplikasi CMS atau yang biasa di sebut “Content Management System” yang open source berarti tidak di kenakan biaya sepeserpun alias gratis untuk mendapatkan fasilitas joomla.

Joomla terdiri dari 3 elemen dasar, yaitu “server web” skrip “PHP” dan basisdata “MySQL“. Server web diasumsikan terhubung dengan Internet/Intranet yang berfungsi sebagai penyedia layanan situs. Skrip PHP terdiri dari kode program dalam bahasa PHP dan basisdata merupakan tempat penyimpanan konten. Joomla menggunakan Apache sebagai server web dan MySQL untuk basisdatanya.
