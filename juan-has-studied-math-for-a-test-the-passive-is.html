PEMBAHASANPASSIVE VOICE

↪ Adalah suatu kalimat yang dibentuk dengan cara :

Mengubah object pada kalimat aktif menjadi subject di kalimat pasif. Mengubah subject pada kalinat aktif menjadi by (agent) pada kalimat pasif.

Ada beberapa hal yang harus diperhatikan dalam mengubah kalimat aktif menjadi pasif, diantaranya :

Mengetahui Subject

Perlu diingat bahwa subject letaknya berada didepan kalimat. Biasanya berupa nama orang/benda ataupun pronoun seperti (I/you/they/we/she/he/it).

Mengetahui Object

Object biasanya terletak setelah kata kerja. Perlu diingat bahwa jika kalimatnya tidak memiliki object, maka kalimat tersebut tidak bisa diubah menjadi kalimat pasif.

Mengetahui Jenis Tenses

Hal ini sangat perlu diperhatikan sekali. Kenapa? Karena jenis tenses yang digunakan antara kalimat aktif dan pasif tidak boleh berbeda.

Contoh :

John studies English. (Simple Present)

John => SubjectStudies => Verb-1 + iesEnglish => Object

Dinda is writing a letter. (Present Continuous)

Dinda => SubjectIs => To be Writing => Verb-ingA letter => Object

Maka bentuk pasif dari kalimat tersebut adalah sbb :

English is studied by John.

English => SubjectIs => To beStudied => Verb-3By John => By (agent)

A letter is being written by Dinda.

A letter => SubjectIs => To be BeingWritten => Verb-3By Dinda => By (agent)

KESIMPULAN

Berikut ini adalah jenis tenses yang digunakan pada kalimat di atas :

Simple Future

Rumus Active :

S + Will + Verb-1 + O

➡ Government will build a new palace next year.

Government => SubjectWill => ModalBuild => Verb-1A new palace => ObjectNext year => Adverb of time

Rumus Passive :

S + Will + be + Verb-3 + by..

↪ A new palace will be built by the government.

A new palace => SubjectWill => ModalBeBuilt => Verb-3By the government => By (agent) Jadi jawabannya adalah (C)

Present Perfect

Rumus Active :

S + Have/has + Verb-3 + O

➡ Juan has studied Math for a test.

Rumus Passive :

S + Have/has + been + Verb-3 + by..

↪ Math has been studied by Juan for a test.

Jadi jawabannya adalah (B)

Past Continuous

Rumus Active :

S + To be (was/were) + Verb-ing + O

➡ Tiara was eating popcorn at the cinema.

Rumus Passive :

S + To be (was/were) + being + Verb-3 + by..

↪ Popcorn was being eaten by Tiara at the cinema.

Jadi jawabannya adalah (D)

Jawaban:

A. Does she study math for the test?

Penjelasan:

Bila kita perhatikan kalimatnya, maka di sana terdapat kata 'does'. Nah, prinsipnya adalah, bila terdapat kata 'does', maka subjeknya PASTI kalau nggak he, she, atau it.

Kemudian, setelah subjek (he/she/it) pada kalimat tersebut, maka menggunakan Verb 1 saja tanpa ada s/es. Mengapa? Karena, pemakaian s/es sudah diwakilkan oleh kata 'does'. Sehingga, kata kerja setelah subjek nggak perlu diberi tambahan s/es.

Jadi, penulisannya BUKAN seperti ini, ya.

"Does he/she/it studies math for the test?"

Melainkan seperti ini nih ....

"Does he/she/it study math for the test?"

Nah, karena di soal pilihan yang tepatnya adalah she study, maka kalimatnya yang benar adalah seperti ini.

"Does she study math for the test?"

Maka, pilihan yang tepat adalah opsi A.

20. Juan has studies math for a test.the passive from is


Jawaban:

math has been studied by Juan for test

Pembahasan

Juan has studied math for a test.

Structure Kalimatnya:Subject => JuanHas/Have => hasVerb 3 => studied Object => MathComplement => for a test

Berdasarkan keterangan di atas:

=> Kalimat bentuk aktif, karena Subjek melakukan aksi/tindakan dalam kalimatnya

=> Kalimat bentuk Present Perfect karena has+V3 (has+studied)

Maka Rumus yg digunakan:

Active Voice:

Kalimat aktif, Subject melakukan aksi dalam kalimatnya

(+) S + has/have + V3 + O

=> Juan has studied math for a test.

Passive Voice:

Kalimat pasif, Object dikenai aksi oleh Subject dalam kalimatnya

(+) O + has/have + been + V3 + By S

=> math has been studied by Juan for test

Kesimpulan

Juan has studied math for a test. The passive from is math has been studied by Juan for test.
