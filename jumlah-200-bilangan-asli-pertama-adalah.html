Uₙ = 8 + 8n - 8

Uₙ = 8n

Maka :

Uₙ = 96

Sₙ = ⁿ/₂ ( 2.U₁ + [n - 1].b )

Sₙ = ⁿ/₂ ( 8n + 8 )

Sₙ = n ( 4n + 4 )

16. deret bilangan asli 1+2+3+........+(n-1)+n. Jumlah 200 suku pertama deret bilangan asli adalah


Deret bilangan : 1 + 2 + 3 + .... + (n - 1) + n. Maka jumlah 200 suku pertama (S₂₀₀) adalah 20.100.

Penyelesaian Soal :

Diketahui : Suku pertama (a) = 1

Ditanya : jumlah 200 suku pertama (S₂₀₀) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung nilai beda (b) dengan menggunakan cara sebagai berikut :

b = Uₙ - Uₙ₋₁

Keterangan : b = beda  

                      Un = suku ke-n

Maka perhitungannya yaitu :

b = U₂ - U₁

  = 2 - 1

  = 1

LANGKAH KEDUA (II)

Hitung jumlah 200 suku pertama (S₂₀₀) dengan menggunakan cara sebagai berikut :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

Keterangan : Sₙ = Jumlah suku n pertama

                       Uₙ = suku ke-n

                       a = suku pertama

                       b = beda

                       n = banyaknya suku

Maka perhitungannya yaitu :

Sₙ = [tex]\frac{n}{2}[/tex] (2a + (n - 1) b)

S₂₀₀ = [tex]\frac{200}{2}[/tex] (2 (1) + (200 - 1) 1)

S₂₀₀ = 100 (2 + 199)

S₂₀₀ = 100 (201)

S₂₀₀ = 20.100

∴ Kesimpulan jumlah 200 suku pertama (S₂₀₀) deret tersebut adalah 20.100.

26. Jumlah 25 bilangan asli pertama adalah 325. Nilai dari 104+ 108 112+...+ 200 adalah... Tolong yang bisa bantuannya pakai cara​


Jawaban:

Jumlah 25 bilangan asli pertama adalah:

1 + 2 + 3 + ... + 24 + 25 = 325

Kita ingin mencari jumlah deret bilangan bulat 104, 108, 112, ..., 200.

Kita dapat menghitung selisih antara dua suku berturut-turut pada deret tersebut:

108 - 104 = 4

112 - 108 = 4

...

200 - 196 = 4

Dapat dilihat bahwa selisih antara dua suku berturut-turut pada deret tersebut adalah 4.

Jadi, kita dapat menggunakan rumus untuk menjumlahkan deret aritmatika dengan selisih 4:

Jumlah deret = (jumlah suku/2) x (suku pertama + suku terakhir)

= (24/2) x (104 + 200)

= 12 x 304

= 3648

Jadi, nilai dari 104 + 108 + 112 + ... + 200 adalah 3648.

Jumlah 200 bilangan asli pertama adalah 20.100 (A).

PEMBAHASAN

Barisan aritmatika ialah barisan barisan angka yang memiliki poIa poIa dengan beda yang tetap. Rumus mencari suku ke-n pada barisan aritmatika :

[tex]\boxed{\rm{ Un = a + (n - 1) \: b }}[/tex]

Keterangan

Un = Suku ke-n.a = suku pertama.b = beda.

[tex]~[/tex]

Rumus untuk b pada barisan aritmatika:

[tex]\boxed{\rm{ b = Un - Un - 1}} [/tex]

Keterangan

b = beda.Un = Suku ke n.Un-1 = suku sebelum suku ke-n.

[tex]~[/tex]

Deret Aritmatika ialah deretan jumlah suku suku dari barisan aritmatika atau konteksnya adalah jumlah n suku pertama barisan aritmatika dengan rumus :

[tex]\boxed{\rm{ Sn = \frac{n}{2} (a + Un)}}[/tex][tex]\boxed{\rm{ Sn = \frac{n}{2} (2a + (n - 1) b )}}[/tex]

[tex]~[/tex]

Barisan Geometri ialah barisan dari rasio atas suku suku bilangan yang berurutan. Rumus barisan geometri :

[tex]\boxed{\rm{ Un = ar^{n-1} }}[/tex]

Keterangan

Un = suku ke-n.a = suku pertama.r = rasio.

[tex]~[/tex]

DIKETAHUI

a = 1 (bil. asli).b = 2 - 1 = 1

[tex]~[/tex]

S 200 = ...?

JAWAB

[tex]\rm{ Sn = \frac{n}{2} (a + Un)}[/tex]

[tex]~[/tex]
