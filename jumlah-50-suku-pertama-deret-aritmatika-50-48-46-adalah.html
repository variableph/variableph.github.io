1. jumlah 50 suku pertama deret aritmatika 50+48+46+... adalah?


Jumlah 50 suku pertama deret aritmatika 50 + 48 + 46 + ... adalah?

Barisan Aritmatika adalah  suatu barisan dengan selisih antara dua suku yang berurutan selalu tetap.

Rumus : Un = a + (n-1)b

Deret Aritmatika adalah jumlah suku – suku barisan aritmatika

Rumus : Sn = 1/2 n (a + Un)

              Sn = ¹/₂ n (2a + (n-1)b)

Dengan keterangan

a = suku pertama

b = beda atau selisi tiap sukunya

n = banyak data (banyak suku)

Un = suku ke-n

Sn = Jumlah n suku pertama

Pembahasan

Penjelasan dengan langkah-langkahnya:

dalam deret aritmatika diatas,

Diketahui:

a = 50

b = 48 - 50 = -2

n = 50

Ditanya:

Jumlah 50 suku pertama

Jawab:

Gunakan rumus ↓

  Sn = ¹/₂ n (2a + (n-1)b)

S₅₀ = ¹/₂. 50 (2.50 + (50-1)(-2))

     = 25 (100 - 98)

     = 25 (2)

     = 50


Jadi Jumlah 50 suku pertama dari deret aritmatika 50+48+46+... adalah 50.

Jumlah 50 suku pertama deret aritmatika 50 + 48 + 46 + ... + adalah 50.

Pendahuluan :

[tex] \rm \blacktriangleright Pengertian :[/tex]

Barisan adalah himpunan bilangan yang diurutkan dengan aturan tertentu. Contoh : 1 , 4 , 7 , 10 ,...

Deret adalah penjumlahan dari suku-suku barisan tersebut. Contoh : 1 + 4 + 7 + 10 +...

[tex] \\[/tex]

[tex] \rm \blacktriangleright Pola~Aritmatika [/tex]

[tex]\boxed {Un \: = a + (n - 1)b}[/tex]

[tex]\boxed{Sn = \frac{n}{2} (a + Un)}[/tex]

atau

[tex]\boxed{Sn = \frac{n}{2} (2a + (n - 1)b)}[/tex]

dimana :

Un = suku ke-n

Sn = jumlah suku ke-n

a = suku pertama (U1)

b = beda atau selisih tiap suku (U3-U2=U2-U1)

n = banyak suku

[tex] \\ [/tex]

[tex] \rm \blacktriangleright Pola~Geometri [/tex]

[tex]\boxed {Un \: = a {r}^{n - 1}} [/tex]

[tex]\boxed {Sn \: = \frac{a( {r}^{n} - 1) }{r - 1}} \: untuk \: r > 1 [/tex]

atau

[tex]\boxed {Sn \: = \frac{a(1 - {r}^{n}) }{1 - r}} \: untuk \: r < 1[/tex]

dimana :

Un = suku ke-n

Sn = jumlah suku ke-n

a = suku pertama (U1)

r = rasio (U3:U2 = U2:U1)

n = banyak suku

Pembahasan :

Diketahui :

50 + 48 + 46 + ... +

Ditanya :

Jumlah 50 suku pertama deret aritmatika tersebut?

Jawab :

[tex] \rm a = 50[/tex]

Hitung beda tiap suku :

[tex] \rm b = U_3 - U_2 = U_2 -U_1[/tex]

[tex] \rm b = 46-48 = 48-50[/tex]

[tex] \rm b = -2 = -2[/tex]

[tex] \rm b = -2[/tex]

Gunakan rumus deret Aritmatika :

[tex] \rm S_n = \frac{n}{2}(2a+(n-1)b)[/tex]

[tex] \rm S_{50} = \frac{50}{2}(2\times 50+(50-1)(-2))[/tex]

[tex] \rm S_{50} = 25 (100 + 49(-2))[/tex]

[tex] \rm S_{50} = 25 (100-98)[/tex]

[tex] \rm S_{50} = 25(2)[/tex]

[tex] \rm S_{50} = 50[/tex]

Kesimpulan :

Jadi, jumlah 50 suku deret aritmatika tersebut adalah 50.
