4. jumlah bilangan kelipatan 3 dan 4 antara 200 dan 450 adalah


materi: barisan dan deret

Jumlah bil. kelipatan 3 dan 4, artinya selisih atau beda (b) adalh 3x4.
b= 3x4 = 12.

karena yang diminta adalah antara 200 dan 450, maka U1= 204. Lalu, untuk suku terakhirnya, Un= 444.

sekarang dimasukkan rumus
Un=U1+(n-1)b
444=204+(n-1)12
444=204+12n-12
252=12n
21= n

Jadi, ada 21 bilangan kelipatan 3&4 di antara 200 dan 450.

sekarang dihitung jumlah 21 bilangan tersebut dengan rumus
Sn=n/2 (U1+Un)
Sn= 21/2 (204+444)
Sn= 21/2 (648)
Sn= 6804

Jadi, jawabannya adalah 6804.

S21 = 21/2 (648)

Penjelasan dengan langkah-langkah:

1) Jumlah semua bilangan kelipatan 3 antara 200 dan 450

a = 201

b = 204 - 201 = 3

Un = 447

Un = a+(n-1)b

447 = 201 + (n-1) 3

447 - 201 = 3 (n-1)

246 = 3 (n-1)

n-1 = 246/3

n-1 = 82

n = 82 + 1

n = 83

Sn = (n/2) (a+Un)

S83 = (83/2) (201+447)

S83 = (83/2) (648)

S83 = 83 x 684/2

S83 = 83 × 342

S83 = 28.386

Jadi, jumlah bilangan kelipatan 3 antara 200 dan 450 adalah 28.386

2) Jumlah semua bilangan kelipatan 4 antara 200 dan 450

a = 204

b = 208 - 204 = 4

Un = 448

Un = a+(n-1)b

448 = 204 + (n-1) 4

448 - 204 = 4 (n-1)

244 = 4 (n-1)

n-1 = 244/4

n-1 = 61

n = 61 + 1

n = 62

Sn = (n/2) (a+Un)

S62 = (62/2) (204+448)

S62 = (31) (652)

S62 = 20.212

Jadi, jumlah bilangan kelipatan 4 antara 200 dan 450 adalah 20.212

Jadi jumlah semuanya 28.386 + 20.212 = 48.598

30. jumlah semua bilangan kelipatan 3 dan 4 antara 200 dan 450 adalah​


Semoga bisa membantu ya

Cara mencari banyaknya adalah

Jumlah kelipatan tiga + kelipatan 4 - kelipatan 12 dalam interval

[tex]200 < x < 450[/tex]

kelipatan 3

b = 3

U1 = 201

Un = 447

Un = a + (n - 1)b

447 = 201 + 3n - 3

3n = 447 + 3 - 201

3n = 249

n = 83

Sn =

[tex] \frac{83}{2} (2(201) + (83 - 1)3) \\ \frac{83}{2} (402 + 246) \\ \frac{83}{2} (648) \\ 83 \times 324 = 26.892[/tex]

Kelipatan 4

b = 4

U1 = 204

Un = 448

448 = 204 + 4n - 4

248 = 4n

n = 62

Dengan n = 62

Sn =

[tex] \frac{62}{2} (2(204) + (62 - 1)4) \\ 31(408 + 244) \\ 31 \times 652 = 20.212[/tex]

Kelipatan 12

U1 = 204

Un = 444

444 = 204 + 12n - 12

240 + 12 = 12n

252 = 12n

n = 21

Maka Sn =

[tex] \frac{21}{2} (2(204) + (21 - 1)12) \\ \frac{21}{2} (408 + 240) \\ \frac{21}{2} (648) \\ 21 \times 324 = 6.804[/tex]

Maka totalnya adalah

26.892 + 20.212 - 6.804 = 40.300
