Kelas C biasanya digunakan untuk jaringan Local Area Network atau LAN. Biasanya ini terdapat dalam Warnet-Warnet maupun sebuah sekolah. NetworkID : 24 bit

HostID : 8 bit

Bit Pertama : 192 – 223

Jumlah : 16.384

Range IP : 192.0.0.x.x – 223.255.255.x.x

20. Carilah subneting dari IP 192.168.1.1 /26 : 1. Jumlah subnet 2. Jumlah host/subnet 3. Blok subnet 4. Alamat host & broadcast yang valid


Berikut di bawah ini adalah subneting dari IP 192.168.11/26:

Jumlah subnet adalah 4 Jumlah host/subnet adalah 62Blok subnet adalah 256 - 192 = 64Alamat host adalah 192.168.1.1 s/d 192.168.1.62 dan broadcast yang valid adalah 192.168.1.63

Penjelasan:

Yang dimaksud alamat IP adalah suatu alamat unik yang mengidentifikasi perangkat yang ada di internet atau jaringan lokal. IP adalah kepanjangan dari Internet Protocol, yang merupakan seperangkat aturan yang mengelola dan mengatur format data yang dikirim melalui internet atau jaringan lokal.

Intinya, alamat IP merupakan suatu pengidentifikasi yang memungkinkan agar informasi dikirim antar perangkat di jaringan. Alamat tersebut berisikan tentang informasi lokasi dan membuat perangkat dapat diakses untuk komunikasi. Internet membutuhkan suatu cara untuk membedakan antara komputer, router, dan situs web yang berbeda. Alamat IP menyediakan cara untuk melakukannya dan merupakan bagian penting dari cara kerja internet.

Penjelasan dari soal nomor 1 untuk pencarian jumlah subnet dapat diperoleh dengan rumus jumlah subnet : IP class c 192.168.1.1 dari pengolahan tersebut, maka rumusnya 2X= maka jumlah subnet 2 * 2 = 4.

Penjelasan dari soal nomor 2 untuk pencarian jumlah host/subnet dapat diperoleh melalui pengolahan rumus jumlah host : rumus = 2y -2 = 2 pangkat 6 - 2 = 62. Dari rumus tersebut maka jumlah host/subnet nya adalah sebanyak 62.

Penjelasan dari soal nomor 3 untuk pencarian blok subnet dapat digunakan rumus pengolahan  blok subnet = 256-192 =64 dan 64+64=128 dan 128+64=192 maka 64, 128, 192.

Penjelasan dari soal nomor 4 untuk mengetahui alamat host dan broadcast yang valid alamat host adalah yaitu ip mulai 192.168.1.1 hingga 192.168.1.62 maka ip broadcastnya adalah 192.168.1.62 + 1 yaitu 192.168.1.63.
