23. apa yang dimaksud format dokumen?​


Jawaban:

Doc dan docx adalah dua format file atau ekstensi file yang digunakan di aplikasi Microsoft Word, salah satu aplikasi yang terdapat di paket aplikasi Microsoft Office. Format file doc digunakan di Microsoft Word 2003 dan versi sebelumnya, sedangkan docx digunakan di aplikasi Microsoft Word 2007 dan versi sesudahnya. Salah satu kendala yang ada sementara ini untuk format file terbaru (docx) adalah terkait kompatibilitas.

Dokumen Microsoft Word yang disimpan menggunakan format file docx tidak dapat dibuka di Microsoft Word 2003 dan versi sebelumnya, sedangkan dokumen yang disimpan dalam format file doc dapat dibuka di Microsoft Word 2007 dan versi sesudahnya. Anda perlu menginstal Microsoft Office Compatibility Pack untuk membuka format file docx di versi Microsoft Word 2003 dan versi sebelumnya.
