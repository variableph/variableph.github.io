25. disebut apakah komputer yang bertugas menyimpan informasi halaman web yang pernah diakses sebelumnya?


Disebut apakah komputer yang bertugas menyimpan informasi halaman web yang pernah diakses sebelumnya komputer yang bertugas untuk menyimpan informasi halaman web adalah Proxy Server. Pada saat kita mengakses suatu halaman website, maka secara otomatis komputer kita akan menyimpan beberapa file atau informasi pada komputer kita dalam suatu folder. Selain dalam folder tersebut, apabila kita menggunakan suatu jaringan komputer yang menggunakan proxy, maka data-data tersebut juga tersimpan pada komputer proxy tersebut.

Dengan adanya komputer proxy tersebut, maka akses kita akan semakin cepat, karena apabila kita akan mengakses suatu halaman website yang pernah kita akses sebelumnya, kita hanya akan mengakses proxy tersebut dan tidak harus ke website yang akan kita akses, kecuali halaman website tersebut belum pernah kita akses sebelumnya.

Demikian pembahasan mengenai proxy server, komputer yang berfungsi untuk menyimpan halaman website yang sudah pernah kita akses sebelumnya, perbedaan web server dengan proxy server adalah web server merupakan komputer tempat disimpannya file-file dari website, sedangkan proxy server merupakan komputer yang menyimpan data-data website yang pernah kita kunjungi, apabila kita akan mengakses halaman yang berlum pernah kita kunjungi, maka kita akan diarahkan ke komputer web server.
