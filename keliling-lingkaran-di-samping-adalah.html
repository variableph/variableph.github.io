25. Keliling lingkaran disamping adalah...


Jawab:

B. 66

Penjelasan dengan langkah-langkah:

3.14 x 21 = 65,94

= 66

Semoga membantu

66cm

Penjelasan dengan langkah-langkah:

Rumus keliling lingkaran :

C = 2 x π x r atau C = π x d

Keterangan :

C = Keliling lingkaran

π = phi

r = jari - jari

d = diameter

Jika jari-jari lingkarannya bisa dibagi 7 (bukan kelipatan 7), maka phi nya 22/7

Sedangkan jika jari-jarinya tidak bisa dibagi 7 (bukan kelipatan 7), maka phi nya 3,14

Jawaban soal :

d = 21

C = π × d

C = 22/7 × 21

C = (22 × 21) ÷ 7

C = 462 ÷ 7

C = 66

Jadi, keliling lingkarannya adalah 66cm
