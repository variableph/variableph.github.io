Jawaban:

Un = 3+(n-1)4

u65= 3+(65-1)4

u65= 3+256

= 259

Jawaban :

259

Langkah penyelesaian :

Pola bilangan diatas

= 3, 7, 11, dan seterusnya

Maka, pola ke-65 adalah

Un = a + (n - 1) × b

U65 = 3 + (65 - 1) × 4

U65 = 3 + 64 × 4

U65 = 3 + 256

U65 = 259 ✔

30. jelaskan cara membuat miniatur jembatan dari korek api​


Jawaban:

liat di YouTube

Penjelasan:

harus liat YouTube karena YouTube segalanya

Jawaban:

Sebelum kita mulai pembuatannya, mari kita siapkan dahulu alat dan bahannya.

1. 300 Korek api

2. Lem tembak

3. Kertas

4. Gunting tajam

5. Lem G

6. Benang nilon

langkah langkahnya

1. Susun 13 korek api sejajar, lalu kita satukan dengan merekatkanya. Dan buatlah 4 papan korek api ya.

2. Selanjutnya rekatkan ke 4 papan korek itu dan akan menjadi sebuah jalan

3. Kita buat pilar untuk jembatannya dengan merekatkan korek yang sudah dipotong pentolannya. Rekatkan satu-persatu korek api ya,untuk ukuran disesuaikan dengan keinginan sobat.

4. Susun-susun korek api secara rapi seperti dibawah ini dan rekatkan dengan lem tembak

5. Nah papan-papan itu, buatlah sebanyak 3 lagi. Ini digunakan untuk bagian alasnya. Tidak lupa dipotong rapi ya

6. Saatnya kita buat jembatannya dengan menempelkan korek api di kertas

7. Yang kemudian kita potong korek

8. Selanjutnya kita rekatkan jembatannya dengan alasnya

9. Setelah itu barulah, jalan yang sudah kita buat direkatkan dengan jembatannya.

10. Lanjut, kita rekatkan pilarnya juga.

11. Sesudah itu ambil benang nilonnya, kita hias dan lilitkan dengan koren api.

12. Tidak lupa pada sisi nya kita tutupi dengan papan koren supaya lebih rapi.

13. Selesai, sekarang sudah bisa dipajang miniatur jembatan ini.
