18. pola banyak korek api 3,9,18,30 banyaknya korek api pada pola ke 15 adalah?


ini adalah barisan aritmatika tingkat 2.

Rumus umum Un = An²+Bn+C

 Rumus Cepat:          2A       = constanta

                                  3A + B = beda pertama

                                  A+B+C = suku pertama

 contoh :

rumus suku ke-n dari barisan 3, 9, 18, 30,….. adalah?

jawab :

Diket : a (suku pertama) = 3;

           b (beda pertama) = 6

           c (constanta) = 3

Maka:

Langkah (1) :

2A = constanta

2A = 3

A = 3/2

Langkah (2):                                            

 3A+B     = 6     

 3(3/2)+B   = 6       

             B = 3/2  

Langkah (3) :

A+B+C = 3

3/2+(3/2)+C  = 3

            C = 0

Masukkan ke rumus umum: Un = 3/2n²+3/2n   

maka U15 = 3/2(15)²+3/2(15)= (375/2) + (45/2) = 420/2 =210

24. pola 1=7 batang korek api pola 2=17 batang korek api pola 3=31 batang korek api pola 4=49 batang korek apa berapa banyak korek api pada pola ke 7 adalah?​

Pola 3 = 31
Pola 4 = 49

Dit: Pola ke 7?
Jawab:
Tentukan dulu apakah ini barisan aritmatika atau tidak?
Maka: d = 17 - 7 = 10
d = 31 - 17 = 14
d = 49 - 31 = 18

Karena selisih antara setiap pasangan dari pola berurutan adalah tidak sama, barisannya bukan aritmatik.
Tetapi diketahui juga bahwa di setiap pola, selisihnya bertambah 4 dari selisih sebelumnya.
Maka:
Pola 5 = 18 + 4 + 49 = 22 + 49 = 71
Pola 6 = 22 + 4 + 71 = 26 + 71 = 97
Pola 7 = 26 + 4 + 97 = 30 + 97 = 127
Jadi, pola ke 7 memiliki 127 korek api.
