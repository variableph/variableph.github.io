624 x + 768y = 61,92  (x320)     199.680x + 245.760y = 19814,4
320 x + 300y = 28        (x624)    199.680x + 187.200y = 17.472
                                                                   58.560y = 2.342,4
                                                                         y = 0,04 m2
                                                                           = 400 cm2
320x + 300(0,04) = 28
  320x + 12      = 28
320 x = 16
   x = 0,05 m2
   x = 500 cm2
Jadi, luas A = 500 cm2 dan B = 400 cm2

[tex]Soal[/tex]

Sebuah toko bahan bangunan menyediakan 2 jenis keramik.

Sebanyak 624 keping keramik jenis A dan 768 keping keramik jenis B dapat menutup permukaan 61,92 m2.

Sebanyak 320 keping keramik jenis A dan 300 keping keramik jenis B dapat menutup permukaan 28 m2.

Tentukan luas permukaan setiap keping keramik jenis A dan jenis B (dalam cm) ?

[tex]JAWABAN [/tex]

[tex]{\orange{\boxed{\boxed{\red{A = 499 \: cm² \: \: \: }}}}}[/tex]

[tex]{\orange{\boxed{\boxed{\mathfrak{\underline{\green{ langkah \: langkah}}}}}}}[/tex]

Diketahui : Sebanyak 624 keping keramik jenis A dan 768 keping keramik jenis B dapat menutup permukaan 61,92 m2

Ditanya : luas permukaan setiap keping keramik jenis A dan jenis B (dalam cm) ?

Penyelesaian :

624 A + 768 B = 61192

768 B = 61, 92 - 624 A

96 B = 7,74 - 78 A

12 B = 0,9678 - 9,75 A

B = 0,08065 - 0,8125 A

Maka :

320 + 300 B = 28 m²

320 A + 300 (0,08065 - 0,8125 A )

320 A . 24,195 - 243,75 A = 28

76,25 A = 28 - 24,145

A = 31,805 - 76,25

A = 0,0499 m²

A = 499 cm²

28. bagaimana membuat tungku pembakaran keramik..?


Jawaban:

Jenis tungku keramik

Wood Fired.Gas Fired.Electric.Bottle Kiln.Anagama kiln.Suhu pembakaran keramik

Pembakaran (Firing)

Pembakaran (Firing)Tanah liat merupakan bahan baku utama untuk pembuatan keramik yang akan mengalami proses pembakaran apabila telah melewati temperatur 600 ºC.

Bahan bakar apa saja yang digunakan untuk membakar keramik?

Fungsi tungku pembakaran adalah untuk membakar benda-benda keramik yang disusun di dalamnya dan dibakar dengan menggunakan bahan bakar khusus (kayu, batu bara, minyak, gas, atau listrik) sampai semua panas menyebar dan membakar semua yang ada di dalam tungku itu.

Mengapa keramik harus dibakar pada suhu yang tinggi?

Keramik dibuat dari tanah liat,tanah liat tsb,kemudian dibakar dlm tingkat pembakaran suhu 600^c-1300^c sesuai dgn jenis tanah liatnya,sehingga tanah liat menjadi keras,padat,dan kedap air.

Apakah tanah liat mudah hancur jika tidak melalui proses pembakaran?

Tanah liat mudah hancur jika tidak melalui proses pembakaran. Kerajinan tanah liat biasa dikenal dengan istilahkeramik.
