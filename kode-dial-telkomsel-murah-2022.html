1. buka Control Panel > Network and Internet > View Network Status and Task.
2. Pilih set up a new connection or network.
3. Klik set up a dial-up connection
4. isi dial-up phone number dari ISP yaitu telkomnet@instan lalu klik Next.
5. isi phone number : 080989999. 
6. isi Username telkomnet@instan. dan passwordnya telkom. lalu confirm password telkom.
5. Klik Konek dan Tunggu Beberapa Saat
6. Setelah register selesai maka akan muncul tulisan You are connected to the Internet
