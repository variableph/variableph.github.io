Pembahasan  Perangkat keras (hardware) adalah komponen fisik dari sebuah komputer seperti keyboard, CPU, mouse, RAM (Random Access Memory), Hard disk (HDD), dan lain-lain. Memory bukan termasuk perangkat keras. Namun, termasuk perangkat keras dengan nama khusus misalnya RAM. Perangkat keras komputer terbagi menjadi 4, yaitu perangkat input, perangkat output, perangkat processing, dan media penyimpan. Perangkat input berfungsi untuk memasukkan data dan program ke dalam komputer. Media penyimpan berfungsi untuk menyimpan data atau program komputer. Contoh: Hard disk, CD, DVD, USB Flash disk, dan lain-lain. Mouse berfungsi sebagai alat pointer untuk mengakses melalui layar monitor. Scanner untuk mengubah data hard copy menjadi soft copy yang dapat diolah di komputer. Joystick adalah tuas kendali yang dapat digerakkan ke segala arah sebagai pelengkap permainan video. Mouse dapat digunakan untuk mengoperasikan beberapa perintah seperti click, double klik, right click, dan drag. Drag dioperasikan dengan mengarahkan pointer mouse pada objek, kemudian menekan dan menahan tombol kiri mouse sambil digerakkan ke tempat yang diinginkan lalu dilepaskan. Keyboard berdasarkan fungsi dasarnya dibagi menjadi 4, yaitu Numeric key, function key, typewriter key dan special function key. Numeric key menampilkan angka, tanda baca titik dan operasi hitung. Perangkat output (output device) berfungsi untuk menyampaikan atau menampilkan informasi. Contoh: Monitor, speaker, dan printer.  ROM singkatan dari Read Only Memory merupakan salah satu memori yang ada dalam komputer yang bersifat permanen dan hanya bisa dibaca saja. Contoh: ROM BIOS (program dasar sistem komputer). RAM (Random Access Memory) dan Hard disk adalah media penyimpan data komputer yang dapat berubah tergantung penggunaan. Memory card adalah media penyimpan data digital pada gadget. Monitor merupakan perangkat output berupa gambar digital (softcopy). Printer merupakan perangkat output berupa gambar atau tulisan cetak (hardcopy). Speaker merupakan perangkat output berupa audio. Sedangkan mikrofon merupakan alat masukan (suara).
