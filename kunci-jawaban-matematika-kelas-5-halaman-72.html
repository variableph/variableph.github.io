[tex]= \frac{17}{12}[/tex]

15. kunci jawaban matematika kelas 6 halaman 103 1-5​


Soal pertama jawaban soal a adalah  4.106,67 cm³, soal b adalah 314 cm³, soal c adalah  1.884 cm³, soal d adalah 523,33 cm³, soal e adalah 11.088 cm³, soal f adalah  4.312 cm³.Besar volume kemasan tersebut adalah 13.860 cm³.Besar volume kacang rebus di dalam wadah adalah 924 cm³.Besar jari-jari dari kerucut tersebut adalah 21 cm.Besar volume tabung tanpa kerucut tersebut adalah 3.080 cm³.

Soal tersebut merupakan soal tentang bangun ruang.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal matematika yang membahas tentang bangun ruang. Bangun ruang merupakan bangun tiga dimensi yang memiliki ruang dan terbatasi oleh sisi.

Persamaan mencari volume kerucut

V = π x r² x [tex]\frac{1}{3}[/tex] x t

Dengan:

V = volume kerucutr = jari-jariπ = 3,14 atau [tex]\frac{22}{7}[/tex]t = tinggi kerucut

Penyelesaian soal

Soal Pertama

Soal a

Diketahui:

t = 20 cmr = 14 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = [tex]\frac{22}{7}[/tex] x 14² x [tex]\frac{1}{3}[/tex] x 20V = [tex]\frac{22}{7}[/tex] x 196 x [tex]\frac{1}{3}[/tex] x 20V = 4.106,67 cm³

Soal b

Diketahui:

t = 12 cmr = 5 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = 3,14 x 5² x [tex]\frac{1}{3}[/tex] x 12V = 3,14 x 25 x [tex]\frac{1}{3}[/tex] x 12V = 314 cm³

Soal c

Diketahui:

t = 18 cmr = 10 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = 3,14 x 10² x [tex]\frac{1}{3}[/tex] x 18V = 3,14 x 100 x [tex]\frac{1}{3}[/tex] x 18V = 1.884 cm³

Soal d

Diketahui:

t = 20 cmr = 15 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = 3,14 x 15² x [tex]\frac{1}{3}[/tex] x 20V = 3,14 x 225 x [tex]\frac{1}{3}[/tex] x 20V = 523,33 cm³

Soal e

Diketahui:

t = 24 cmr = 21 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = [tex]\frac{22}{7}[/tex] x 21² x [tex]\frac{1}{3}[/tex] x 24V = [tex]\frac{22}{7}[/tex] x 441 x [tex]\frac{1}{3}[/tex] x 24V = 11.088 cm³

Soal f

Diketahui:

t = 21 cmr = 14 cm

Ditanyakan:

Berapa volume dari kemasan tersebut?

Jawab:

V = [tex]\frac{22}{7}[/tex] x 14² x [tex]\frac{1}{3}[/tex] x 21V = [tex]\frac{22}{7}[/tex] x 196 x [tex]\frac{1}{3}[/tex] x 21V = 4.312 cm³

Soal Kedua

Diketahui:

t = 30 cmr = 21 cm

Ditanyakan:

Berapa volume dari kerucut tersebut?

Jawab:

V = [tex]\frac{22}{7}[/tex] x 21² x [tex]\frac{1}{3}[/tex] x 30V = [tex]\frac{22}{7}[/tex] x 441 x [tex]\frac{1}{3}[/tex] x 30V = 13.860 cm³

Soal Ketiga

Diketahui:

t = 27 cmr = 7 cm

Ditanyakan:

Berapa volume dari kacang rebus tersebut?

Jawab:

V = [tex]\frac{2}{3}[/tex] x [tex]\frac{22}{7}[/tex] x 7² x [tex]\frac{1}{3}[/tex] x 27V = [tex]\frac{2}{3}[/tex] x  [tex]\frac{22}{7}[/tex] x 49 x [tex]\frac{1}{3}[/tex] x 27V = 924 cm³

Soal Keempat

Diketahui:

V = 16.632 cm³t = 36 cm

Ditanyakan:

Berapa jari-jari kerucut tersebut?

Jawab:

V = π x r² x [tex]\frac{1}{3}[/tex] x t16.632 = [tex]\frac{22}{7}[/tex] x r² x [tex]\frac{1}{3}[/tex] x 3616.632 = [tex]\frac{22}{7}[/tex] x r² x 12r² = 441r = 21 cm

Soal Kelima

Diketahui:

t = 30r = 7

Ditanyakan:

Berapa volume tabung tanpa kerucut?

Jawab:

V = π x r² x t - π x r² x [tex]\frac{1}{3}[/tex] x tV = [tex]\frac{22}{7}[/tex] x 7² x 30 - [tex]\frac{22}{7}[/tex] x 7² x [tex]\frac{1}{3}[/tex] x 30V = [tex]\frac{22}{7}[/tex] x 49 x 30 - [tex]\frac{22}{7}[/tex] x 49 x 10V = 4.620 - 1.540V = 3.080 cm³
