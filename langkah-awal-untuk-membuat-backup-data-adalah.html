3. langkah langkah dalam membuat backup file data myob adalah dengan cara?


Berikut ini adalah langkah-langkah yang bisa kamu lakukan dalam membuat backup file data MYOB:

Aktifkanlah data file "MYOB" lalu klik "File" pada menu baris, kemudian klik "Backup" jendela MYOB maka "Premier MYOB Company File Backup" akan tampil. Pilih "Backup Company File Only", kemudian klik.Pilih "Check Company File" for errors, kemudian klik.Klik tombol Continue, lalu klik tombol OK dan simpan file backup ditempat yang diinginkan dengan meng-klik tombol Save.

Pembahasan

Backup merupakan proses menyalin atau duplikasi data dan file dari satu perangkat atau media penyimpanan ke media sekunder lainnya, seperti hardisk, flashdisk, sistem cloud dan media lainnya baik dilakukan secara offline maupun online.

Tujuan backup ialah untuk mengembalikan data apabila data tersebut hilang, baik karena rusak ataupun terhapus, dan untuk mengembalikan data ke titik tertentu pada masa lalu. Fungsi utama backup ialah membuat cadangan dari data-data agar ketika data tersebut rusak atau hilang kita masih memiliki salinannya.

Sistem backup terdiri dari beberapa komponen, yaitu:

Software backup merupakan perangkat lunak yang bertugas untuk menjalankan proses backup.Client backup merupakan komputer yang berisi data yang akan dibackup.Server backup merupakan server yang bertugas untuk mengatur proses backup.Media backup merupakan media seperti tape, magnetic disc, optical disc dan sebagainya.Mesin backup merupakan mesin yang digunakan untuk membackup data.

10. Sebutkan langkah-langkah dalam melakukan backup menggunakan perintah dd


Peringatan: Saat menggunakan perintah dd, jika Anda tidak hati-hati, dan jika Anda tidak tahu apa yang Anda lakukan, Anda akan kehilangan data Anda! Contoh 1. Backup Seluruh Harddisk

Untuk backup seluruh salinan dari harddisk ke harddisk lain yang terhubung ke sistem yang sama, jalankan perintah dd seperti yang ditunjukkan di bawah ini. Dalam perintah ini contoh dd dari harddisk sumber /dev/hda, ke  hardisk target /dev/hdb.

# dd if=/dev/sda of=/dev/sdb “if” mewakili inputfile, dan “of” mewakili file output. Jadi salinan dari /dev/sda akan ada dalam /dev/sdb.Jika ada kesalahan, perintah di atas akan gagal. Jika Anda memberikan parameter “conv = noerror” maka akan terus menyalin jika ada errpr read.

Dalam salinan hard drive ke hard drive menggunakan perintah dd diberikan di bawah ini, pilihan sync memungkinkan Anda untuk menyalin semuanya menggunakan disinkronkan I / O.

# dd if=/dev/sda of=/dev/sdb conv=noerror,sync

Langkah-langkah mengcopy file dengan menggunakan command prompt

Secara ringkas cara mengcopy sebuah file di command prompt adalah dengan menggunakan syntax seperti berikut:

copy (object path) (target path)

keterangan:

copy: perintah untuk mengcopy

object path: nama file yang akan di-copy, lengkap dengan ekstensinya dan path (jika ada)

target path: tempat file akan di-paste

supaya lebih jelas, ikutilah langkah-langkah berikut.

Klik start > run > cmd > enter (pada windows xp) atau klik start > ketikan “cmd” pada kotak “search programs or files” > enter.

Pada window akan muncul tulisan “C:\users\asus>”, artinya Anda sedang berada di dalam drive ‘C:‘ folder ‘Users’ folder ‘asus’.

Tentukan file mana yang akan kita copy, misalkan file ada di drive D:\, maka ketiklah “C:\users\asus>D:” lalu tekan enter, maka akan muncul tulisan “D:\>”, artinya Anda sekarang sudah berada di drive “D:”.

Ketik “dir” untuk mengetahui isi file yang ada di drive “D:”. maka akan muncul sederetan file dan folder (untuk folder biasanya ada tanda “

Jika Anda akan masuk ke dalam folder maka ketiklah “CD”, misalkan jika ingin masuk ke folder “IMK”, ketiklah “CD IMK”.

Jika Anda akan mengakses subfolder, sub-subfolder, sub-sub-sub…dst. Maka berikan tanda backslash (\) setelah nama folder yang pertama dst. Misalnya “CD multimedia\film\indo\favorite”. Maka kita akan langsung berada di folder favorite tanpa langsung mengetik CD beberapa kali.

Jika anda ingin kembali ke luar folder ketiklah “CD..” (titiknya 2 kali) lalu tekan enter.

Sampai sini silahkan Anda latihan dulu untuk berpindah-pindah dari satu drive ke drive yang lain dan dari folder ke folder yang lain.

14. Sebutkan peralatan input yang digunakan untuk memasukkan data backup atau data yang sudah disimpan?


Jawaban:

Keyboard

Kalian pasti ngga asing sama keyboard. Keyboard adalah salah satu perangkat input yang paling penting cuy, karena untuk memasukkan data berupa huruf, angka, dan simbol lainnya. Keyboard juga berfungsi sebagai short-cut/jalan pintas seperti copy, paste, dan lainnya. Tanpa keyboard kalian bakal susah mengoperasikan komputer cuy, rusak satu tombol aja udah repot apalagi rusak semua.

Mouse

Berdasarkan namanya, perangkat input satu ini berbentuk seperti tikus cuy. Bentuknya yang kecil membuat mouse ini mudah digenggam usernya. Terdapat 2 tombol di atasnya dan di bawahnya terdapat laser atau bola kecil. Mouse berfungsi untuk menggerakkan kursor/panah di monitor komputer dan juga bisa membuka folder/aplikasi.

Touchpad

Perlu diketahui bahwa touchpad ini cuma ada di playstore laptop cuy. Touchpad ini mempunyai sensor khusus yang bisa menggerakkan kursor di monitor hanya dengan memakai jari. Fungsinya kurang lebih sama cuy seperti mouse.

Joystick

Buat kalian yang suka main game tapi ngga biasa dengan kontrol mouse dan keyboard, kalian bisa menggunakan joystick. Joystick untuk komputer bentuknya beragam cuy, tapi yang paling mainstream biasanya berbentuk seperti kontroller PlayStation atau Xbox.

Flashdisk

Flashdisk ini berguna untuk kalian yang ingin menyimpan data secara eksternal kalau penyimpanan kalian sudah penuh. Kapasitas flashdisk sendiri juga beragam cuy, ada yang 2 GB, 4 GB, 8 GB, 16 GB dan seterusnya.

Compact disc atau CD

CD terbuat dari plastik yang berbentuk lingkaran tipis dengan lubang kecil di tengahnya. Fungsi CD kurang lebih sama seperti flashdisk. Namun saat ini CD sudah jarang digunakan cuy karena saat ini sudah ada flashdisk yang bentuknya lebih kecil dan serta mudah disimpan dan dibawa kemana pun.

Digitizer

Kalau kalian ingin mengubah data analog menjadi data digital maka kalian perlu perangkat digitizer cuy. Perangkat ini biasanya berbentuk seperti papan yang dilengkapi beberapa tombol di bagian samping. Kalian bisa menggambar dan menulis dari digitizer lalu dimasukkan ke dalam komputer.

CD dan DVD room

Kalau kalian menyimpan data di sebuah CD atau DVD, kalian wajib mempunyai perangkat untuk membacanya yaitu CD/DVD ROM. Perangkat ini biasanya sudah ada di dalam komputer atau laptop. Ada juga yang berbentuk eksternal dengan port usb.

Barcode scanner

Barcode scanner ini banyak dijumpai di supermarket atau minimarket. Fungsinya yaitu untuk mengscan barcode pada sebuah barang dengan menggunakan sinar laser merah. Bentuk barcode juga beragam cuy, ada yang berbentuk kotak, bisa juga berbentuk vertikal. Setelah discan, barcode scanner akan menerjemahkan barcode tersebut menjadi sebuah angka/harga dari barang tersebut lalu dimasukkan ke dalam komputer.

Light Pen

Kalau kalian seorang desainer, maka kalian wajib mempunyai Light pen. Light pen berfungsi mengedit gambar atau mendesain gambar yang ada di monitor. Perangkat ini berbentuk seperti pulpen yang memiliki kabel yang terhubung dengan komputer.

18. Tuliskan langkah-langkah melakukan backup database​

Penjelasan:

Keterangan :

[userdb] Nama user database Anda

[pass] Password database Anda

[dbname] Nama database Anda

[backupfile.sql] Nama file untuk database yang Anda backup nantinya

Pada contoh berikut, database MySQL yang dijalankan ialah melalui aplikasi XAMPP. Maka, terlebih dahulu Anda harus masuk ke direktori C:\xampp\mysql\bin melalui CMD. Berikut langkah-langkahnya :

Jalankan fasilitas CMD melalui RUN –> ketik cmd. Kemudian tekan Enter

Masuk ke direktori C: terlebih dahulu, caranya ketik cd/

Selanjutnya ketik cd xampp/mysql/bin untuk masuk ke direktori bin

Konfigurasi database yang akan di backup pada contoh ini ialah sebagai berikut :

[userdb] root

[pass] Pada contoh ini, database tidak diberikan password. Nantinya, perintah -p [pass] tidak dieksekusi

[dbname] belajar

[backupfile.sql] Nama file backup yang nantinya dibuat ialah belajar

Berikut perintah untuk melakukan backup database :

$ mysqldump -u root belajar > belajar.sql

Catatan : Perintah ini dijalankan ketika Anda masih di direktori C:\xampp\mysql\bin melalui CMD

Apabila berhasil, backup belajar.sql dapat Anda lihat di direktori C:\xampp\mysql\bin. Backup database tersebut siap Anda gunakan di komputer lainnya.

20. MEMBUAT PAPARAN MENGENAI "CARA BACKUP DATA DAN CARA PEMULIHANNYA, SERTA TUTORIAL CARA MELAKUKAN BACKUP DAN PEMULIHANNYA"​


Backup data merupakan hal yang sangat penting untuk dilakukan karena dapat melindungi data penting dari hilang atau rusak. Jika terjadi masalah pada perangkat atau sistem, backup data dapat digunakan untuk mengembalikan data yang hilang atau rusak ke keadaan semula. Berikut adalah paparan mengenai cara backup data dan cara pemulihannya, serta tutorial cara melakukan backup dan pemulihannya.

1. Cara Backup Data

Ada beberapa cara untuk melakukan backup data, antara lain:

- Menggunakan External Hard Drive: Anda dapat membackup data ke hard drive eksternal yang terpisah dari perangkat yang digunakan. Hal ini dilakukan dengan cara menyalin file dan folder ke hard drive tersebut. Pastikan hard drive tersebut memiliki kapasitas yang cukup besar untuk menyimpan semua data yang akan dibackup.

- Menggunakan Cloud Storage: Anda dapat menyimpan data di cloud storage seperti Google Drive, Dropbox, atau iCloud. Data akan disinkronkan secara otomatis ke server penyimpanan online tersebut. Hal ini memungkinkan untuk mengakses data dari mana saja dan kapan saja dengan internet.

- Menggunakan Software Backup: Anda dapat menggunakan software backup seperti Acronis True Image atau EaseUS Todo Backup untuk membuat backup data secara otomatis dan terjadwal. Software backup ini dapat menghemat waktu dan tenaga dalam melakukan backup.

2. Cara Pemulihan Data

Setelah data dibackup, data tersebut dapat dipulihkan dengan mudah jika terjadi masalah pada perangkat atau sistem. Berikut adalah cara pemulihan data:

- Menggunakan External Hard Drive: Anda dapat menghubungkan hard drive eksternal ke perangkat dan menyalin data yang dibackup kembali ke perangkat. Pastikan untuk memilih opsi "restore" atau "copy" saat melakukan proses pemulihan.

- Menggunakan Cloud Storage: Data dapat dipulihkan dengan mengunduh data dari cloud storage. Pastikan untuk mengikuti petunjuk yang diberikan oleh penyedia layanan cloud storage.

- Menggunakan Software Backup: Anda dapat memulihkan data dengan software backup yang digunakan. Pilih opsi "restore" dan ikuti petunjuk yang diberikan.

3. Tutorial Cara Melakukan Backup dan Pemulihan Data

Berikut adalah tutorial cara melakukan backup dan pemulihan data:

- Menggunakan External Hard Drive:

1. Hubungkan hard drive eksternal ke perangkat.

2. Pilih file dan folder yang ingin dibackup.

3. Salin file dan folder tersebut ke hard drive eksternal.

4. Pastikan untuk memilih opsi "safely remove" saat ingin mengeluarkan hard drive eksternal.

5. Untuk melakukan pemulihan, hubungkan hard drive eksternal ke perangkat dan salin file dan folder yang dibackup kembali ke perangkat.

- Menggunakan Cloud Storage:

1. Buat akun di penyedia layanan cloud storage seperti Google Drive atau Dropbox.

2. Pilih file dan folder yang ingin dibackup.

3. Unggah file dan folder tersebut ke cloud storage.

4. Pastikan untuk menyinkronkan data secara teratur.

5. Untuk melakukan pemulihan, unduh data dari cloud storage dan simpan ke perangkat.

- Menggunakan Software Backup:

1. Unduh dan instal software backup seperti Acronis True Image atau EaseUS Todo Backup.

2. Pilih file dan folder yang ingin dibackup.

3. Atur jadwal backup secara teratur.
