16. Untuk menyimpan file yang sudah pernah tersimpan dengan menggunakan nama file yang sama di gunakan perintah..


Jawaban:

perintah save as

Penjelasan:

Perintah save as dapat digunakan antara lain untuk hal-hal berikut ini:

1. Menyimpan file dg nama yg sama, namun berbeda ekstensi filenya.

Contoh: Ada file dengan nama brainly.doc, lalu dilakukan perintah save as dengan disimpan memakai nama brainly.docx

2. Menyimpan file dg nama yg sama, tapi beda lokasi penyimpanannya.

Contoh: Ada file dengan nama brainly.doc di harddisk komputer, lalu dilakukan perintah save as dengan disimpan di dalam flashdisk (masih dengan nama yang sama, yaitu brainly.doc)

3. Perintah save as dapat juga dipakai untuk menyimpan file yg pernah tersimpan dengan penamaan berbeda.

Contoh: Ada file dengan nama brainly.doc, lalu dilakukan perintah save as dengan disimpan memakai nama brainlyhebat.doc

27. 4. Google Colaboratory adalah layanan berbasis Cloud Google yang mereplikasi Cloud. Pernyataan tentang Google Colaboratory yang sesuai berdasarkan Jupyter bacaan Notebook di adalah tersebut​


Google Colaboratory, atau Colab untuk singkat, adalah layanan gratis yang dikembangkan oleh Google yang berbasis pada teknologi Jupyter Notebook. Colab menyediakan lingkungan komputasi berbasis web yang dapat digunakan untuk menulis, menjalankan, dan menyimpan kode Python, serta untuk menyertakan teks, gambar, dan media lainnya dalam format yang sama seperti Jupyter Notebook. Selain Python, Colab juga mendukung beberapa bahasa pemrograman lain seperti R dan Swift.

Salah satu keuntungan dari menggunakan Colab adalah akses gratis ke infrastruktur pengolahan data dan pembelajaran mesin milik Google, seperti TPU (Tensor Processing Unit) dan GPU. Ini memungkinkan pengguna untuk menjalankan komputasi yang intensif dengan biaya yang sangat rendah atau bahkan tanpa biaya sama sekali. Selain itu, Colab juga menyediakan akses mudah ke berbagai library Python yang populer, seperti TensorFlow, PyTorch, dan pandas.

Secara umum, Google Colab adalah sebuah layanan berbasis cloud yang memungkinkan pengguna untuk menulis dan menjalankan kode Python dalam lingkungan Jupyter Notebook dengan akses gratis ke infrastruktur pengolahan data dan pembelajaran mesin milik Google.
