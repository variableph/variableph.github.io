19. CuS + NO 3- → Cu2+ + S + NO ( suasaa asam) lengkapi dan setarakan reaksi redoks berikut dengan metode ion-elektron.


CuS + NO₃⁻  ==> Cu⁺² + S + NO ( suasana asam) dengan metode ion-elektron diperoleh persamaan reaksi setaranya adalah:

3 CuS + 2 NO₃⁻ + 8 H⁺  ==> 3 Cu⁺² + 3 S +  2 NO + 4 H₂O

Penjelasan:

Langkah-langkah penyelesaian dilakukan dengan menggunakan Metode Setengah Reaksi atau ion elektron yaitu:

CuS + NO₃⁻  ==> Cu⁺² + S + NO

Langkah 1 : Menuliskan Reaksi Reduksi dan Oksidasi

Reduksi : NO₃⁻ ==>  NO  (Biloks N = +5 ==> +2)

Oksidasi : CuS  ==>  Cu⁺² +  S    (Biloks S = -2 ==> 0)

Langkah 2 : Menyetarakan unsur yang terlibat dalam reaksi Reduksi dan Oksidasi

Reduksi : NO₃⁻ ==>  NO        (Atom N setara)

Oksidasi : CuS  ==>  Cu⁺² +  S    (Atom S setara)

Langkah 3 : Menyetarakan Jumlah atom Oksigen dan Hidrogen (Oksigen dahulu)

Penyetaraan atom Oksigen dilakukan dengan penambahan molekul H₂O pada sisi yang kekurangan atom Oksigen karena reaksi berjalan pada kondisi ASAM

Reduksi : NO₃⁻ ==>  NO

* Karena disisi kiri terdapat kelebihan 2 atom Oksigen, maka kita tambahkan 2 molekul H₂O disisi kanan menjadi:

Reduksi : NO₃⁻ ==>  NO + 2 H₂O

Selanjutnya kita setarakan atom Hidrogen dengan menambahkan 4 ion H⁺ pada sisi kiri yaitu:

Reduksi : NO₃⁻ + 4 H⁺ ==>  NO + 2 H₂O

Oksidasi : CuS  ==>  Cu⁺² +  S

* Karena tidak ada kandungan Oksigen maka kita biarkan (Sudah Setara).

Langkah 4 : Menyetarakan muatan

Penyetaraan dilakukan dengan penambahan elektron (e⁻)

Reduksi : NO₃⁻ + 4 H⁺ ==>  NO + 2 H₂O

Muatan kiri = -1 + 4 = 0

Muatan kiri = +3

Muatan kanan = 0

maka kita tambahkan 3 e⁻ disisi kiri sehingga muatan sama-sama menjadi 0 menjadi:

Reduksi : NO₃⁻ + 4 H⁺ + 3 e⁻  ==>  NO + 2 H₂O

Oksidasi : CuS  ==>  Cu⁺² +  S

Muatan kiri = 0

Muatan kanan = +2

maka kita tambahkan 2 e⁻ disisi kanan sehingga muatan sama-sama menjadi 0 menjadi:

Oksidasi : CuS  ==>  Cu⁺² +  S + 2 e⁻

Langkah 5 : Menyetarakan Elektron

Penyetaraan elektron kita lakukan dengan melakukan perkalian terlebih dahulu sehingga jumlah elektron pada reaksi reduksi dan Oksidasi nilainya sama yaitu:

Reduksi : NO₃⁻ + 4 H⁺ + 3 e⁻  ==>  NO + 2 H₂O     |x 2|

Oksidasi : CuS  ==>  Cu⁺² +  S + 2 e⁻                     |x 3|

----------------------------------------------------------------------------------------

Reduksi : 2 NO₃⁻ + 8 H⁺ + 6 e⁻  ==>  2 NO + 4 H₂O

Oksidasi : 3 CuS  ==>  3 Cu⁺² + 3 S + 3 e⁻    

----------------------------------------------------------------------------------------  +

Redoks :  3 CuS + 2 NO₃⁻ + 8 H⁺  ==> 3 Cu⁺² + 3 S +  2 NO + 4 H₂O (SETARA)

* Kita coret elektron karena jumlah di sisi kiri dan kanan sama

Langkah 6 : Silahkan Cek ulang dan Hitung Ulang jika langkah benar maka sudah SETARA.

Cara setengah reaksi

LANGKAH PERTAMA (I)

           Oksidasi

Ι-------------------------------Ι

MnO (s) + PbO₂ (s) ⇒ MnO₄⁻ (aq) + Pb²⁺ (aq)

+2 -2       +4 -2            +7 -2              +2

                Ι-------------------------------------Ι

                              Reduksi

LANGKAH KEDUA (II)

Setarakan reaksi dengan menggunakan cara setengah reaksi sebagai berikut :

Langkah 1 : Untuk mempermudah maka buatlah zat yang mengalami oksidasi dan reduksi sebagai berikut :

Reduksi      : PbO₂ ⇒  Pb²⁺

Langkah 2 : Setarakan jumlah koefisien  selain atom O dan H

Oksidasi     : MnO   ⇒ MnO₄⁻

Reduksi      : PbO₂ ⇒  Pb²⁺

Oksidasi     : MnO + 3H₂O  ⇒ MnO₄⁻

Reduksi      : PbO₂ ⇒  Pb²⁺ + 2H₂O

Langkah 4 : Menambah H⁺ diruas yang kekurangan H sebanyak kekurangannya.

Oksidasi     : MnO + 3H₂O  ⇒ MnO₄⁻ + 6H⁺

Reduksi      : PbO₂ + 4H⁺ ⇒  Pb²⁺ + 2H₂O

Oksidasi     : MnO + 3H₂O  ⇒ MnO₄⁻ + 6H⁺ + 5e           ║ × 2

Reduksi      : PbO₂ + 4H⁺ + 2e ⇒  Pb²⁺ + 2H₂O               ║ × 5

25. setarakan reaksi redoks berikut dengan metode bilangan oksidasi dan metode ion elektron (metode setengah reaksi) Cl2+SO2+H2O-->SO4^2- +Cl^- +H^+


Cl₂ + SO₂ + H₂O ⇒ SO₄²⁻ + Cl⁻ + H⁺

Cara Setengah Reaksi

LANGKAH PERTAMA (I)

Tentukan zat yang mengalami oksidasi dan reduksi dengan mengunakan cara berikut :

          Reduksi

Ι-------------------------------Ι

Cl₂ +  SO₂  ⇒  SO₄²⁻ + Cl⁻

0     +4 -2       +6 -2     -1

         Ι-------------Ι

           Oksidasi

LANGKAH KEDUA (II)

Setarakan reaksi dengan menggunakan cara setengah reaksi sebagai berikut :

Langkah 1 : Untuk mempermudah maka buatlah zat yang mengalami oksidasi dan reduksi sebagai berikut :

Oksidasi     : SO₂  ⇒  SO₄²⁻

Reduksi      : Cl₂ ⇒   Cl⁻

Langkah 2 : Setarakan jumlah koefisien  selain atom O dan H

Oksidasi     : SO₂  ⇒  SO₄²⁻

Reduksi      : Cl₂ ⇒  2Cl⁻

Langkah 3 : Menambah H₂O diruas yang kekurangan O sebanyak kekurangannya

Oksidasi     : SO₂ + 2H₂O   ⇒  SO₄²⁻

Reduksi      : Cl₂ ⇒  2Cl⁻

Langkah 4 : Menambah H⁺ diruas yang kekurangan H sebanyak kekurangannya.

Oksidasi     : SO₂ + 2H₂O   ⇒  SO₄²⁻ + 4H⁺

Reduksi      : Cl₂ ⇒  2Cl⁻

Langkah 5 : Setarakan jumlah ion dengan menambah elektron.

Oksidasi     : SO₂ + 2H₂O   ⇒  SO₄²⁻ + 4H⁺ + 2e

Reduksi      : Cl₂  + 2e  ⇒  2Cl⁻

                  ______________________________________ +

                    Cl₂ +  SO₂ + 2H₂O ⇒   2Cl⁻ +  SO₄²⁻ + 4H⁺

Cl₂ + SO₂ + H₂O ⇒ SO₄²⁻ + Cl⁻ + H⁺
