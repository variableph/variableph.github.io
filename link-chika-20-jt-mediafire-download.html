13. Pn=Po(1+r ) =20 jt (1+2%)4 =20 jt (1+0,02)4 =20 jt (1,02)4 =20 jt (1,0824322) =21.648.644 jutaJdi petanyaannya hasil dari 20 jt (1,02)4 itu smpai akhir gimana cranya ya?​


Pn = Po (1 + r)^n

Pn = 20.000.000 (1 + 2%)⁴

Pn = 20.000.000 (1 + 0,02)⁴

Pn = 20.000.000 (1,02)⁴

Pn = 20.000.000 (1,02 x 1,02 x 1,02 x 1,02)

Pn = 20.000.000 (1,0404 x 1,0404)

Pn = 20.000.000 x 1,08243216

Pn = 20.000.000 x 1,082432 dibulatkan

Pn = 21.648.644

Catatan

Jika pangkatnya banyak, biasanya dihitung pakai kalkulator. Jika pakai cara manual akan terlalu rumit dan lama.
