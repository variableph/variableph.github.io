Penjelasan dengan langkah-langkah:

Terdapat 3 persegi panjang jika gambar tsb dibagi. Maka luasnya :

L1 = p x l = 6 x 2 = 12 cm²

L2 = p x l = 8 x 4 = 32 cm²

L3 = p x l = 6 x 2 = 12 cm²

L4 = p x l = 4 x 2 = 8 cm²

Panjang L3 (persegi panjang paling bawah) = Panjang L1 (persegi panjang paling atas)

Luas total = L1 + L2 + L3 + L4 = 12 + 32 + 12 + 8 = 64 cm²
