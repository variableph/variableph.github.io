8. luas permukaan bangun ruang disamping adalah?​


Jawab :

Luas permukaan kerucut :

= πr ( r + s )

= 3,14 x 20 cm x ( 20 cm + s )

= 3,14 x 20 cm x ( 20 cm + ( √r² + t² ))

= 3,14 x 20 cm x ( 20 cm + ( √ 20 cm² + ( 35 - 20 )²))

= 62,8 cm x ( 20 cm + ( √ 20² + 15² ))

= 62,8 cm x ( 20 cm + ( √ 400 + 225 ))

= 62,8 cm x ( 20 cm + ( √ 625 ))

= 62,8 cm x ( 20 cm + 25 cm )

= 62,8 cm x 45 cm

= 2.826 cm²

Luas permukaan bola :

= 4πr²

= 4 x 3,14 x 20 cm x 20 cm

= 12,56 x 20 cm x 20 cm

= 251,2 cm x 20 cm

= 5.024 cm²

Luas permukaan gabungan :

= Lp1 + Lp2

= 2.826 cm² + 5.024 cm²

= 7.850 cm²
