28. Biasanya perintah pada aplikasi ditandai dengan gambar atau icon, dapatkah perintah diaplikasi tidak menggunakan ... menggunakan gambar atau icon tetapi menggunakan keyboard? j


Jawaban:

Tentu saja kita bisa menggunakan keyboard untuk menjalankan perintah tertentu (tidak harus selalu memakai icon).

Penjelasan:

Yang perlu dikuasai adalah: kita harus mengetahui dulu tombol-tombol mana yg ada di keyboard yg bermanfaat untuk menerapkan perintah tertentu. Perhatikan contoh gambar berikut ini (dalam menu Edit Ms. Word 2003):

- gambar melengkung dg anak panah mengarah ke bawah (icon Undo)

dapat dilakukan dengan keyboard memakai tombol Ctrl dan Z

- gambar melengkung dg anak panah mengarah ke atas (icon Redo)

dapat dilakukan dengan keyboard memakai tombol Ctrl dan Y

- gambar gunting (icon Cut)

dapat dilakukan dengan keyboard memakai tombol Ctrl dan X
