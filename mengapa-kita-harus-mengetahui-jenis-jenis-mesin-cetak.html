14. 227(10) dikonversi ke sistem biner

227 dibagi 2 = 113 sisa 1

113 dibagi 2 = 56 sisa 1

56 dibagi 2 = 28 sisa 0

28 dibagi 2 = 14 sisa 0

14 dibagi 2 = 7 sisa 0

7 dibagi 2 = 3 sisa 1

3 dibagi 2 = 1 sisa 1

1 dibagi 2 = 0 sisa 1

Bilangan biner ditulis dari bawah = 11100011(2). Angka 2 menunjukkan bilangan biner.

16. 01011001 dikonversi ke bilangan desimal

1 × 2⁰ = 1

0 × 2¹ = 0

0 × 2² = 0

1 × 2³ = 8

1 × 2⁴ = 16

0 × 2⁵ = 0

1 × 2⁶ = 64

0 × 2⁷ = 0

Bilangan desimal adalah jumlah dari hasil perkalian di atas.

Bilangan desimal
