1. sub menu "mode" pada menu image dipergunakan untuk?Bantuuuu!​


Sub menu "mode" pada menu image dipergunakan untuk mengubah mode warna yang akan digunakan pada image. Menu image sendiri digunakan untuk melakukan setting dan juga perubahan image yang aktif. Menu dan sub menu tersebut ada dalam aplikasi atau software editing yakni adobe photoshop.

Pembahasan

Di sub menu image terdapat banyak pengaturan ataupun pilihan mode warna serta juga terdapat pengaturan lanjutan serta pengaturan levels, brightness & contrast, hue saturation dan masih banyak yang terdapat pengaturan lainya.

Berikut ini isi dari sub menu image:

Mode berfungsi untuk memilih mode warna seperti RGB, Grayscale, CMYK dan yang lainya.Adjustment atau pengaturan berfungsi untuk mengatur warna image, pengaturan warna semua terletak disini mulai dari Levels, Curves, Exposure dan yang lainya.Auto Tone (Shift+Ctlr+L) berfungsi untuk mengubah Tone gambar secara otomatis berdasarkan warna disekitarnya. Auto Contrast (Alt+Shift+Ctrl+L) berfungsi untuk mengubah contrast gambar secara otomatis. Auto Color (Shift+Ctrl+B) berfungsi untuk mengubah warna gambar secara otomatis. Image Size (Alt+Ctrl+I) berfungsi untuk mengubah ukuran gambar yang dipilih. Canvas Size (Alt+Ctrl+C) berfungsi untuk mengubah ukuran kanvas atau lembar kerja. Image Rotation berfungsi untuk memutar gambar baik searah jarum jam ataupun sebaliknya.Crop berfungsi untuk memotong lembar kerja atau gambar sesuai dengan kebutuhan. Trim berfungsi untuk memangkas atau menghilangkan area kerja yang kosong. Reveal All Duplicate berfungsi untuk menduplikat document kita secara cepat, sehingga tidak perlu save as dan mengubah nama file. Apply Image berfungsi untuk mengapply sesuai dengan settingan gambar yang ada. Calculations berfungsi untuk memperhitungkan antara layer satu dengan yang lainya. Variables Apply Date Set Trap Analysis berfungsi untuk menganalisa atau juga bisa untuk mengukur secara manual setia sudut ataupun antara titik satu dengan yang lainya.Pelajari lebih lanjut
