8. sebutkan cara membuka dan menutup lembaran kerja ms. word 2007​


Jawaban:

Microsoft Office 2007 kali ini, kita akan belajar bagaimana cara membuka dan menutup aplikasi Microsoft Office Word 2007. Ada tiga macam cara membuka aplikasi tersebut. Baiklah, silahkan simak paparan di bawah ini :

Cara 1 :

1. Klik tombol "Start"

2. Arahkan kursor/pointer ke "All Programs"

3. Kemudian arahkan kursor ke "Microsoft Office"

4. Setelah itu arahkan pula kursor ke "Microsoft Office Word 2007"

5. Dan klik

6. Sampai tahap ini anda sudah membuka aplikasi MS Word 2007

Cara 2 :

1. Klik tombol "Start"

2. Arahkan kursor ke "Run"

3. Pada dialog box "Run" di kotak isian "open" ketikkan "winword" (tanpa tanda petik)

4. Kemudian klik OK

5. Sampai tahap ini anda sudah membuka aplikasi MS Word 2007

Cara 3 :

1. Double klik shortcut "Microsoft Office Word 2007" di desktop komputer anda

2. Sampai langkah ini anda sudah membuka aplikasi MS Word 2007

Kemudian cara menutup aplikasi Microsoft Office Word 2007 dapat anda simak penjelasan di bawah ini. Ada dua cara menutup aplikasi tersebut, yakni :

Cara 1 :

1. Klik tanda "X" yang berada di pojok kanan atas aplikasi MS Word 2007

2. Maka aplikasi tersebut akan tertutup.

Cara 2 :

1. Klik tombol "Office Button" yang berada di pojok kiri atas aplikasi MS Word 2007

2. Kemudian klik "Close" atau anda juga bisa klik "Exit Word"

3. Sampai tahap ini anda sudah menutup aplikasi MS Word 2007

9. cara keluar dari lembar kerja ms word adalah....​


Jawaban:

Menutup Microsoft Word melalui Tombol Close :

1). Simpan file anda agar tidak hilang saat aplikasi di tutup.

2). Klik tombol Close yang berada di sebelah kanan atas aplikasi MS Word.

3). Aplikasi MS Word telah tertutup.

Menutup dan Keluar dari Ms Word dengan Menggunakan Keyboard :

1). Simpan dokumen yang telah dikerjakan.

2). Tekan tombol ALT dan F4 secara bersamaan.

3). Maka aplikasi MS Word akan ditutup.

Menutup File MS Word melalui Menu File :

1). Simpan dokumen terlebih dahulu.

2). Klik menu File.

3). Klik Close untuk menutup lembar kerja (Workspace).
