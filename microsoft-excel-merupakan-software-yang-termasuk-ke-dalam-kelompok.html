30. Jelaskan dengan singkat fungsi/manfaat software-software berikut ini! 1) Microsoft Word 2) Microsoft Excel 3) Microsoft Powerpoint​


Jawaban:

Software (perangkat lunak aplikasi). Software ini punya fungsi tertentu, misalnya untuk presentasi dan pengolahan data. ... Fungsi software ini biasanya digunakan untuk mengkoordinasikan tiap komponen dan fungsi komputer. Contoh: Windows, DOS, dan Linux

Microsoft Word atau Microsoft Office Word atau Word adalah perangkat lunak pengolah kata andalan Microsoft. Pertama diterbitkan pada 1983 dengan nama Multi-Tool Word untuk Xenix, versi-versi lain kemudian dikembangkan untuk berbagai sistem operasi, misalnya DOS, Apple Macintosh, SCO UNIX, OS/2, dan Microsoft Windows.

Microsoft Excel atau Microsoft Office Excel adalah sebuah program aplikasi lembar kerja yang dibuat dan didistribusikan oleh Microsoft Corporation yang dapat dijalankan pada Microsoft Windows dan Mac OS. Aplikasi ini merupakan bagian dari Microsoft Office System.

Microsoft PowerPoint atau Microsoft Office PowerPoint atau PowerPoint adalah sebuah program komputer untuk presentasi yang dikembangkan oleh Microsoft di dalam paket aplikasi kantoran mereka, Microsoft Office, selain Microsoft Word, Excel, Access dan beberapa program lainnya.
