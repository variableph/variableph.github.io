16. Ketiga muatan listrik q1,q2dan q3 adalah segaris .bila jarak d=20 cm, maka besar dan arah gaya coulomb yang bekerja pada muatan q2 adalah


Besar gaya coulumb yang bekerja pada Q₂ adalah 5 N ke arah Q₁.

[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:2.\:LISTRIK\:STATIS}}[/tex]

Gaya Coulumb merupakan salah satu besaran vektor, karena memiliki nilai dan arah.

Persamaan gaya coulumb adalah :

[tex]\boxed{F=k \frac{Q_{1} Q_{2}}{r^{2}}}[/tex]

Persamaan Besarnya Medan Listrik adalah:

[tex]\boxed{E=\frac{F}{Q} }[/tex]

atau

[tex]\boxed{E=k\frac{Q}{r^{2} }}[/tex]

Persamaan Energi potensial listriknya adalah :

[tex]\boxed{W=QV}[/tex]

[tex]\boxed{Diketahui}[/tex]

Untuk soal versi Lengkap , Lihat pada lampiran.

Q₁ = 20 μC

Q₂ = 10 μC

Q₃ = 40 μC

d₁  = 60 cm = 60 x 10⁻² m  (3d)

d₂ = 60 cm = 60 x 10⁻² m   (3d)

k = 9 x 10⁹ Nm²/C²

[tex]\boxed{Ditanya}[/tex]

Besar dan arah gaya coulumb.

[tex]\boxed{Penyelesaian}[/tex]

Perhatikan gambar pada lampiran .

Besar gaya coulumb yang bekerja pada muatan Q² adalah :

F₂ = F₂₁ - F₂₃

[tex]F_{2} =k\frac{Q_{2}Q_{1}}{d_{1} ^{2} } -k\frac{Q_{2} Q_{3} }{d_{3}^{2} }[/tex]

dimana d₁ = d₃ = 60 cm sehingga bisa juga dibuat :

[tex]F_{2} =k\frac{Q_{2} }{d^{2} } (Q_{1}- Q_{3})[/tex]

[tex]F_{2} =9x10^{9}Nm^{2}/C^{2}\:\frac{10x10^{-6}C}{(60x10^{-2}m)^{2}  } \:(20x10^{-6}C-40x10^{-6}C)[/tex]

[tex]F_{2} =9x10^{9}Nm^{2}/C^{2}\:\frac{10x10^{-6}C}{(3600x10^{-4}m^{2})} \:(-20x10^{-6}C)[/tex]

F₂ = - 5 N     (arah ke Q₁)

Jadi, Besar gaya coulumb yang bekerja pada Q₂ adalah 5 N ke arah Q₁.

[tex]\boxed{Kesimpulan}[/tex]

Besar gaya coulumb yang bekerja pada Q₂ adalah 5 N ke arah Q₁.

17. muatan listrik +q1 = 5 mikro coulomb, -q2= 10 mikro coulomb , dan +q3= 10 mikro coulomb terpisah seperti pada gambar. agar gaya coulomb yang bekerja di muatan q2=0, maka berapakah besar nilai x?​


LISTRIK STATIS

Q₁ = 5 μC

Q₃ = 10 μC

R₁₂ = 4 cm

R₂₃ = x cm

x = ... ?

F₂₁ = F₂₃

k Q₁ Q₂ / R₁₂² = k Q₂ Q₃ / R₂₃²

Q₁ / R₁₂² = Q₃ / R₂₃²

5 / 4² = 10 / x²

x² = 32

x = 4√2 cm ✔

26. Tiga partikel bermuatan listrik q1, q2, dan q3 terletak seperti ilustrasi pada gambar di samping. Muatan listrik q1 = +2,7 µC dan muatan listrik q3 = -40 µ. Jarak muatan q1 dan q3 sejauh 4,1 cm, sedangkan jarak muatan q2 dan q3 sejauh 4 cm. Gaya coulomb yang dialami pada muatan q2 sebesar 30 N. Akibatnya, muatan q2 ditolak oleh muatan q3 tetapi ditarik oleh muatan q1. Jika konstanta coulomb k = 9 x 10^9 Nm/C^2, besar muatan listrik pada q2 adalah....


Muatan listrik q₂ ialah - 2,064 µC  , atau negatif 2,064 µC

Tiap - tiap partikel memiliki muatan listrik, ada yang negatif dan ada yang psotif

Penjelasan dengan langkah-langkah:

Diketahui : q₁ = +2,7 µC

                  q₃ = - 40 µC

                  r₁   = 4,1 cm (jarak antara q₁ dan q₃)

                  r₂  = 4 cm (jarak antara q₃ dan q₂)

                  K  =  9 x 10^9 Nm/C^2

Ditanya : q₂

Dijawab :

Langkah 1

Konversikan satuan kedalam sistim KMS

q₁ = +2,7 µC = +2,7 .10⁻⁶ = +27.10⁻7 C

q₃ = - 40 µC = -40 .10⁻6  = - 4 .10⁻⁷ C

r₁  = 4,1 cm = 4,1. 10⁻² m

r₂  = 4 cm = 4.10⁻² m

K  =  9 x 10^9 Nm/C^2  =  9. 10⁹ Nm/C²

Langkah 2

Mencari jarak antara q₁ dan q₂, dengan penjumlahan r₁ dan r₂. Sebelumnya harus diketahui dulu letak daripada masing-masing partikel dalam suatu garis lurus.

Pada soal, diketahui jika q₁ memiliki muatan listrik postif dan terjadi gaya tarik menarik dengan q₂, sedangkan q₃ memiliki muatan listrik negatif dan terjadi gaya tolak antara q₃ dan q₂, itu berarti q₂ memiliki muatan listrik negatif.

Maka jika digambarkan dalam garis lurus sebagai berikut :

q₂---4 cm------q₃---------4,1 cm----------q₁

Sehingga :

R = r₁ + r₂

  = 4,1. 10⁻² m + 4.10⁻² m

  = 8,1 . 10⁻² m

Langkah 3

Mencari q₂

∑F = 30 N

    = F₁ + F₂

    = K { (q₁ x q₂) /R² } +  K { (q₃ x q₂) / ( r₂ )² }

    = Kq₂ { (q₁ / R²) + (q₃ / [r₂]²) }

    = 9. 10⁹Nm/C².q₂ { ( +27.10⁻⁷ C / [8,1 .10⁻²m]² ) + ( -4.10⁻⁷ C / [ 4.10⁻² m]² }

    = 9. 10⁹Nm/C².q₂ { [( 1 / 243).10⁻¹C/m²] - [ (1 / 4).10⁻² C/m²] }

    = 9. 10⁹Nm/C².q₂ ( 10⁻¹) { ( 1/ 243) - ( 1 / 40 } C/m²

    = 9.10⁸Nm/C².q₂ { 157. 10⁻¹ /972 }

    = 9.10⁷Nm/C².q₂ {157/972}

    = 10⁷Nm/C².q₂ { 157 /108 }

30 N  = 10⁷Nm/C².q₂ { 157 /108 } C/m²

    q₂  = 30 N / [ {10⁷Nm/C².q₂ { 157 /108 } C/m² ]

          = 2,064 / (10⁶) C

          = 2,064 .10⁻⁶ C

          = - 2,064 µC (negatif 2,064 µC )

          = - 2,064 µC

q₂ memiliki muatan listrik negarif karena terjadi gaya tarik menarik antara q₁ yang bermuatan positif dan tolak menolak dengan q₃ yang bermuatan negatif

jika muatan listrik postif bertemu dengan muatan listrik negatif maka akan terjadi gaya tarik menarik sedangkan jika bermuatan sama, yakni negatif dan negatif atau positif dan positif maka akan terjadi tolak menolak
