4. Jika pada seperempat jarak dari muatan q1,menuju ke muatan q2 kuat medan listriknya nol, maka perbandingan q2 :q1 adalah


Jika pada seperempat jarak dari muatan q1 menuju ke muatan q2 kuat medan listriknya nol, maka perbandingan q2 : q1 adalah 9 : 1.

Pembahasan

Syarat untuk mendapatkan kuat medan di titik sama dengan 0 adalah

E1 = E2

dengan

E = k q / r²

di mana  

E : kuat medan (N/C)

k : konstanta Coulomb (9 x 10⁹ N.m²/C²)  

q : muatan (C)  

r : jarak muatan terhadap titik (m)  

maka,  

E1 = E2

k q1 / R1² = k q2 / R2²

q1 / q2 = k. R1² / (R2². K)

q1 / q2 = R1² / R2²

Dari soal, dikatakan bahwa “seperempat jarak dari muatan q1, menuju ke muatan q2 kuat medan listriknya nol”

Maka,  

R1 = ¼ R12

dan  

R2 = 1 – R1

R2 = 1 – ¼ R12

R2 = 3/4 R12

Masukkan nilai tersebut ke dalam rumus,

q1 / q2 = (¼ R12)² / (¾ R12)²

q1 / q2 = (1/16 R12²) / (9/16 R12²)

q1 / q2 = (1/16) / (9/16)

q1 / q2 = (1/16) x (16/9)

q1 / q2 = 1 / 9

Jadi, perbandingan antara q1 dengan q2 adalah 1 : 9. Karena yang ditanyakan adalah q2 : q1, maka perbandingannya menjadi 9 : 1.

Langkah 2

Mencari jarak antara q₁ dan q₂, dengan penjumlahan r₁ dan r₂. Sebelumnya harus diketahui dulu letak daripada masing-masing partikel dalam suatu garis lurus.

Pada soal, diketahui jika q₁ memiliki muatan listrik postif dan terjadi gaya tarik menarik dengan q₂, sedangkan q₃ memiliki muatan listrik negatif dan terjadi gaya tolak antara q₃ dan q₂, itu berarti q₂ memiliki muatan listrik negatif.

Maka jika digambarkan dalam garis lurus sebagai berikut :

q₂---4 cm------q₃---------4,1 cm----------q₁

Sehingga :

R = r₁ + r₂

  = 4,1. 10⁻² m + 4.10⁻² m

  = 8,1 . 10⁻² m

Langkah 3

Mencari q₂

∑F = 30 N

    = F₁ + F₂

    = K { (q₁ x q₂) /R² } +  K { (q₃ x q₂) / ( r₂ )² }

    = Kq₂ { (q₁ / R²) + (q₃ / [r₂]²) }

    = 9. 10⁹Nm/C².q₂ { ( +27.10⁻⁷ C / [8,1 .10⁻²m]² ) + ( -4.10⁻⁷ C / [ 4.10⁻² m]² }

    = 9. 10⁹Nm/C².q₂ { [( 1 / 243).10⁻¹C/m²] - [ (1 / 4).10⁻² C/m²] }

    = 9. 10⁹Nm/C².q₂ ( 10⁻¹) { ( 1/ 243) - ( 1 / 40 } C/m²

    = 9.10⁸Nm/C².q₂ { 157. 10⁻¹ /972 }

    = 9.10⁷Nm/C².q₂ {157/972}

    = 10⁷Nm/C².q₂ { 157 /108 }

30 N  = 10⁷Nm/C².q₂ { 157 /108 } C/m²

    q₂  = 30 N / [ {10⁷Nm/C².q₂ { 157 /108 } C/m² ]

          = 2,064 / (10⁶) C

          = 2,064 .10⁻⁶ C

          = - 2,064 µC (negatif 2,064 µC )

          = - 2,064 µC

q₂ memiliki muatan listrik negarif karena terjadi gaya tarik menarik antara q₁ yang bermuatan positif dan tolak menolak dengan q₃ yang bermuatan negatif
