1. Jarak interval nada C ke nada C berjarakdisebut Jarak interval nada C ke nada D berjarak.disebut Jarak interval nada C ke nada E berjarak ...sebut...​


1. Jarak interval nada C ke nada C disebut interval prime.

2. Jarak interval nada C ke nada D disebut interval second.

3. Jarak interval nada C ke nada E disebut interval terts.

Pembahasan

Interval dalam musik merupakan jarak dua nada, baik jarak ke atas maupun ke bawah. Setiap interval memiliki nama yang berbeda, tergantung dari jarak di antara kedua nada terkait.

Pada umumnya, ada delapan jenis interval dalam sebuah tangga nada diatonis (tangga nada yang tersusun atas tujuh nada). Nama dari masing-masing interval tersebut dan penjelasannya adalah sebagai berikut:

Prime, yaitu interval untuk dua buah nada yang sama. Contoh interval prime dalam tangga nada C mayor adalah C ke C. Second, yaitu interval antara nada satu dengan nada kedua (berjarak selangkah). Contohnya dalam tangga nada C mayor adalah C ke D, F ke G. Terts, yaitu interval dari sebuah nada ke nada ketiga (berjarak dua langkah). Contohnya dalam tangga nada C mayor adalah C ke E, F ke A. Kuarts, yaitu interval dari nada satu ke nada keempat (berjarak tiga langkah). Contohnya dalam tangga nada C mayor adalah C ke F, F ke Bb. Kwint, yaitu interval dari nada satu ke nada kelima (berjarak empat langkah). Contohnya dalam tangga nada C mayor adalah C ke G, D ke A. Sekst, yaitu interval dari nada satu ke nada keenam (berjarak lima langkah). Contohnya dalam tangga nada C mayor adalah C ke A. Septim, yaitu interval dari nada satu ke nada ketujuh (berjarak enam langkah). Contohnya dalam tangga nada C mayor adalah C ke B. Oktaf, yaitu jarak antara nada satu dengan nada kedelapan (berjarak tujuh langkah). Contohnya dalam tangga nada C mayor adalah C ke C'.
