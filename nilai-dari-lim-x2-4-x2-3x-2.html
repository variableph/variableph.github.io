3. nilai lim dari Lim x>1 x3-x2+2x-2 : x2 + 3x-4​


[tex] \lim \limits_{x \to 1} \frac{ {x}^{3} - {x}^{2} + 2x - 2 }{ {x}^{2} + 3x - 4 } = \boxed{ \frac{3}{5}} \\ \\ [/tex]

Pembahasan

[tex] \: \: \: \: \lim \limits_{x \to 1} \frac{ {x}^{3} - {x}^{2} + 2x - 2 }{ {x}^{2} + 3x - 4 } \\ \\ = \lim \limits_{x \to 1} \frac{ (x - 1)( {x}^{2} + 2) }{ (x - 1)(x + 4) } \\ \\ = \lim \limits_{x \to 1} \frac{ ( {x}^{2} + 2) }{ (x + 4) } \\ \\ = \frac{ {1}^{2} + 2}{1 + 4} \\ \\ = \frac{3}{5} \\ \\ [/tex]

19. Nilai dari lim x → 0 (√(x2–2x+1) - √(x2+3x–4)) adalah ….


Nilai dari [tex] \lim_{ x \to 0} =( \sqrt{ {x}^{2} - 2x + 1 } - \sqrt{ {x}^{2} + 3x - 4 } ) [/tex] adalah limit tak terdefinisi

Pembahasan

Limit suatu fungsi f(x) untuk x mendekati nilai a adakah harga yang paling dekat dari f(x) pada saat x mendekati nilai a dari kiri dan dari kanan.

[tex] \lim_{x \to a} f(x) [/tex] terdefinisi jika dan hanya bisa [tex] \lim_{x \to a } f(x) = \lim_{x \to a } [/tex] ,atau limit kiri sama dengan limit kanan.

Langsung saja, simak lebih lanjut

[tex] \lim_{ x \to 0} =( \sqrt{ {x}^{2} - 2x + 1 } - \sqrt{ {x}^{2} + 3x - 4 } ) \\ \\ = x - 1 - \sqrt{ {x}^{2} + 3x - 4 } \\ \\ = 0 - 1 - \sqrt{ {0}^{2} + (3.0) - 4 } \\ \\ = 0 - 1 - \sqrt{ - 4} [/tex]

= Limit tak terdefinisi

Karena fungsi limit diatas tak terdefinisi diantara 0, maka limitnya tak terdefinisi.

Jadi, hasilnya adalah limit tak terdefinisi
