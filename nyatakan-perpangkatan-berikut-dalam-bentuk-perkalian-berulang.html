Jawab:

Lihat Penjelasan

Penjelasan dengan langkah-langkah:

Perpangkatan adalah perkalian berulang suatu bilangan yang sama. Bilangan pokok dalam perpangkatan dinyatakan sebagai basis. Dan banyaknya bilangan pokok yang digunakan dalam perkalian berulang dinyatakan dengan pangkat.

Bentuk umum dari perpangkatan dianyatakan sebagai berikut :

[tex]a^n[/tex] = [tex]a[/tex] x  

Keterangan

[tex]a[/tex] adalah bilangan pokok (basis)

[tex]n[/tex] adalah pangkat (eksponen)

Bilangan berpangkat yang perlu diketahui adalah bilangan berpangkat positif, bilangan berpangkat negatif, dan bilangan berpangkat nol

Sehingga soal tersebut dapat dinyatakan dengan

Soal a

[tex]3^8[/tex] = 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3

       bilangan pokoknya 3

       eskponennya 8

Soal b

[tex](0,83)^4[/tex] = (0,83) x (0,83) x (0,83) x (0,83)

       bilangan pokoknya 0,83

       eskponennya 4

Soal c

[tex]t^3[/tex] = t x t x t

       bilangan pokoknya [tex]t[/tex]

       eskponennya [tex]3[/tex]

Soal d

[tex](-\frac{1}{4} )^4[/tex] =  [tex](-\frac{1}{4} )[/tex] x [tex](-\frac{1}{4} )[/tex] x [tex](-\frac{1}{4} )[/tex] x [tex](-\frac{1}{4} )[/tex]

       bilangan pokoknya [tex](-\frac{1}{4} )[/tex]

       eskponennya [tex]4[/tex]

Soal e

[tex]-(\frac{1}{4} )^4[/tex] =  [tex]-( (\frac{1}{4} )[/tex] x [tex](\frac{1}{4} )[/tex] x [tex](\frac{1}{4} )[/tex] x [tex](\frac{1}{4} )[/tex] )

       bilangan pokoknya [tex](\frac{1}{4} )[/tex]

       eskponennya [tex]4[/tex]
