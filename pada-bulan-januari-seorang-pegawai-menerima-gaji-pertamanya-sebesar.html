18. seorang pegawai menerima gaji sebesar Rp.800.000,00 pada bulan januari 2010.Bila setiap bulan berikut nya pegawai tersebut mendapat kenaikan gaji sebesar 2 1/2% dari gaji bulan januari 2010,tentukan gaji pegawai tersebut pada bulan Desember 2010!


Matematika SMP Kelas 9
Barisan Bilangan dan Deret

Rungkasan Materi :
Deret Aritmetika
Suku ke-n (Un) suatu deret aritmetika ditentukan dengan rumus :
Un = a + (n-1) b
dimana =
a = suku pertama (U1)
n = banyak suku
b = beda

b = U2 - U1
____________

Pembahasan :
============
Soal diatas merupakan bentuk dari suatu deret aritmetika :

Dik :
Gaji awal januari 2010 (U1) = 800.000
Kenaikan gaji setiap bulan (b) = 2 1/2%
Kenaikan gaji (b) :
= (800.000/100) x 2(1/2)%
= 8.000 x 2(1/2)
= 20.000

Jadi beda = 20.000
_______________
Dit : Gaji pada bulan Desember 2010 (U12).

U12 = a + (12-1)b
U12 = a + 11 b
U12 = 800.000 + 11 (20.000)
U12 = 800.000 + 220.000
U12 = 1.020.000

Jadi, gaji pegawai tersebut pada bulan desember 2010 sebesar Rp. 1.020.000,-

Semoga membantu.
