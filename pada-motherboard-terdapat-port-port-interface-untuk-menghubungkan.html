6. Pada setiap pheripheral (peralatan tambahan) yang akan dihubungkan ke sistim komputer akan dibedakan dengan penggunaan type .... True PortMain portPort InterfacePortable​


Peripheral komputer adalah berbagai komponen yang terhubung ke CPU (Central Processing Unit) yang berfungsi sebagai perangkat Input dan Output data kedalam dan keluar CPU yang selanjutnya diproses oleh sistem komputer untuk mengolah data Input yang di-Input oleh periferal Input seperti Mouse, Keyboard, Scaner, Recording Device, WebCam. Data input ini dapat berupa data Dokumen, Suara, Gambar, Gerakan dan Video. Selanjutnya data yang telah dimasukan kedalam komputer akan diolah oleh peralalatan Process untuk menghasilkan data Output yang akan ditampilkan dalam bentuk nyata atau tidak oleh Periferal Output misalnya berupa Dokumen, Suara, Gambar, dan Video yang dihasilkan oleh Periferal sepert Monitor, Printer, Plotter, Speaker, HeadSet, dsb.

Periferal termasuk dalam katagori Hardware yang merupakan bagian dari tiga komponen utama komputer yaitu :

Hardware   : adalah perangkat keras yang dapat berupa benda/komponen  fisik yang secara nyata dapat disentuh dan dilihat.

Software   : adalah perangkat lunak yang berfungsi untuk menjalankan harware berdasarkan perintah yang di-Inputkan oleh Brainware melalui software sehingga hardware dapt digunakan dan difungsikan.

Brainware       : adalah perrngakat akal yang memanajemen atau berfungsi untuk memasukan perintah agar Software dan Hardware dapat beroprasi sesuai dengan fungsinya.

Berdasarkan kegunaannya, periferal komputer terdiri dari 2 (Dua) tipe yaitu sebagai berikut :

Peripheral Utama (Main Periferals) : yaitu tipe periferal yang keberadaannya harus ada pada saat menjalankan komputer misalnya Mouse, Keyboard dan Monitor.

Peripheral Pendukung (Out Sillary Periferals) : yaitu periferal pendukung yang keberadaannya tidak harus ada pada saat menjalankan komputer misalnya Printer, Speaker, WebCam, Scaaner dsb.

Sesuai dengan penjelasan diatas, maka peripheral yang termasuk kedalam Hardware memiliki fungsi tertentu untuk dapat memasukan atau mengeluarkan data ke/dari komputer.

Berikut fungsi dari beberapa peripheral dalam komputer yang dikelompokkan dalam fungsinya(perangkat Masukan/input dan perangkat Keluaran/output)

Perangkat Masukan

 

Perangkat masukan (bahasa Inggris: input device) adalah semua periferal (perangkat keras komputer) yang digunakan untuk memberikan data dan sinyal kendali untuk suatu sistem pemrosesan informasi, misalnya komputer. Perangkat masukan dapat dikelompokkan berdasarkan beberapa hal:

(a) cara masukan, misalnya melalui gerak mekanis atauaudio;

(b) masukan diskret (tekanan tombol) atau kontinu (posisi tetikus); serta

(c) derajat kebebasan gerakan, misalnya dua dimensi pada tetikus standar dan tiga dimensi pada navigator untuk aplikasi CAD. Jenis perangkat masukan utama antara lain adalah papan ketik, perangkat penunjuk, pemindai, perangkat masukan video (kamera, dll.), serta perangkat masukan audio (mikrofon, dll.).

 

Perangkat Pengetikan

 

Keyboard

Keyboard merupakan unit input yang paling penting dalam suatu pengolahan data dengan komputer. Keyboard dapat berfungsi memasukkan huruf, angka, karakter khusus serta sebagai media bagi user (pengguna) untuk melakukan perintah-perintah lainnya yang diperlukan, seperti menyimpan file dan membuka file. Penciptaan keyboard komputer berasal dari model mesin ketik yang diciptakan dan dipatenkan oleh Christopher Latham pada tahun 1868, Dan pada tahun 1887 diproduksi dan dipasarkan oleh perusahan Remington. Keyboard yang digunakanan sekarang ini adalah jenis QWERTY, pada tahun 1973, keyboard ini diresmikan sebagai keyboard standar ISO (International Standar Organization).Jumlah tombol pada keyboard ini berjumlah 104 tuts.Keyboard sekarang yang kita kenal memiliki beberapa jenis port, yaitu port serial, ps2, usb dan wireless.

Jenis port yang ada pada motherboard dan fungsinya​ :

Konektor Power, konektor ini digunakan untuk menghubungkan power  ke motherboard. Konektor Serial Konektor ini biasanya memiliki 9 pin dan biasanya digunakan untuk menghubungkan mouse dan joystick. Fungsi  port ini adalah mengirimkan data satu per satu melalui satu kabel  paralel port. Port paralel digunakan untuk menghubungkan  CPU ke printer, modem eksternal, dan  periferal lain yang berbagi port yang sama. Fitur port paralel yang mengirim dan menerima beberapa bit  saat mengirim port konektor printer, modem, dan hard disk drive saat ini. PS/2, port ini sering disebut juga dengan serial port 2, dan fungsinya hanya untuk menghubungkan keyboard dan mouse. USB (Universal Serial Bus) Konektor ini serbaguna dan dapat digunakan dengan beberapa perangkat dan periferal, sehingga memiliki kemampuan untuk memasukkan konektor USB yang sama dengan port serial. Konektor VGA digunakan untuk menghubungkan monitor ke CPU dan membiarkan monitor menampilkan isi CPU. Konektor ini juga terdapat pada komputer atau laptop bergaya modern. HDMI adalah kabel HD untuk menghubungkan beberapa perangkat elektronik hiburan rumah ke komputer Anda. Perbedaannya adalah HDMI dapat menggantikan fungsi beberapa kabel sekaligus. Port network (RJ-45) / LAN. Port ini berfungsi untuk menghubungkan komputer  dengan komputer lain (jaringan LAN), dan fungsinya untuk menghubungkan CPU ke jaringan komputer. Port sound card Ada tiga jenis (input saluran, keluaran mikrofon) yang dirancang untuk menghubungkan perangkat audio (speaker, mikrofon, TV tuner).Serial Port sebuah suatu port personal komputer yang meiliki fungsi guna mentransmisikan satu bit informasi menjadi satuan waktu.Pelajari Lebih LanjutMateri tentang motherboard brainly.co.id/tugas/7762056Materi tentang komponen motherboard brainly.co.id/tugas/39036Materi tentang slot pada motherboard brainly.co.id/tugas/19669397Detail Jawaban

10. Sebutkan macam-macam port interface dan tuliskan fungsinya


Jawaban:

Berikut ini adalah beberapa macam port interface dan fungsinya:

Port USB (Universal Serial Bus) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti mouse, keyboard, printer, dan lain-lain.

Port HDMI (High Definition Multimedia Interface) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti komputer, TV, proyektor, dan lain-lain yang memiliki resolusi gambar tinggi.

Port VGA (Video Graphics Array) - Digunakan untuk menghubungkan perangkat-perangkat elektronik seperti komputer, TV, proyektor, dan lain-lain yang memiliki resolusi gambar sedang.

Port LAN (Local Area Network) - Digunakan untuk menghubungkan perangkat-perangkat elektronik ke jaringan lokal (misalnya jaringan kantor atau sekolah).

Port Audio - Digunakan untuk menghubungkan perangkat-perangkat audio seperti speaker, headphone, atau microphone.

Port Serial - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan komunikasi data satu per satu (misalnya modul GPS atau RFID).

Port Parallel - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan komunikasi data banyak sekaligus (misalnya printer atau scanner).

Port Firewire (IEEE 1394) - Digunakan untuk menghubungkan perangkat-perangkat elektronik yang membutuhkan kecepatan tinggi dalam mentransfer data (misalnya kamera video atau hard disk eksternal).

24. sebutkan macam macam port interface dan tuliskan fungsinya


1. port Serial
Port yang berfungsi untuk melakukan transfer data dengan kecepatan lebih kecil dengan jenis paralel. sehingga kini port ini kurang populer dipergunakan.

2. port PS/2
Port serial 2, seperti halnya port serial: melakukan transfer data, namun relatif lebih cepat dari port serial sebelumnya. kini masih dipakai untuk menghubungkan komputer dengan mouse atau keyboard jenis PS/2

3. port Paralel
adalah port yang berfungsi mengirim dan menerima beberapa bit data pada satu saat melalui satu set kabel. dipergunakan untuk menghubungkan komputer dengan modem eksternal yang memiliki kabel jenis paralel.

4. port USB
adalah port pengembangan dari port serial dan kini tengah populer digunakan. Bisa digungsikan untuk menghubungakn komputer dengan flashdisk, scanner atau printer yang mendukung kabel USB.

5. port VGA
adalah port yang berfungsi menghubungkan monitor dengan CPU.
