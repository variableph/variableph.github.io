5. pada ulangan matematika diketahui nilai rata rata adalah 58 jika rata rata nilai matematika untuk siswa putra adalah 65 sedangkan untuk siswa putri rata ratanya 54 maka perandingan jumlah siswa putri dan putra pada kelas tersebut adalah


rata2 nilai (X_nilai) = 58
rata2 putra (X_putra) = 65
rata2 putri (X_putri) = 54

jumlah siswa putra(n_putra) : jumlah siswa putri(n_putri) ....................?

[tex]X_{nilai}= \frac{X_{putra}n_{putra}+X{putri}n_{putri}}{n_{putra}+n_{putri}} \\ \\ 58= \frac{65n_{putra}+54n_{putri}}{n_{putra}+n_{putri}} \\ \\ 58n_{putra}+58n_{putri}=65n_{putra}+54n_{putri} \\ \\ 58n_{putri}-54n_{putri}=65n_{putra}-58n_{putra} \\ \\ 4n_{putri}=7n_{putra} \\ \\ \frac{n_{putra}}{n_{putri}}= \frac{4}{7} [/tex]

jadi, perbandingan jumlah siswa putra dan siswa putri adalah 4 banding 7 ^-^rata-rata Putra (L) = 65
Rata-rata Putri (W) = 54
(65L + 54W)/(L + W) = 58
65L + 54W = 58(L + W)
65L + 54W = 58L + 58W
65L - 58L = 58W - 54W
7L = 4W
L/W = 4/7
Putra : putri = 4 : 7

19. pada ulangan matematika, diketahui nilai rata rata kelas adalah 58. jika rata rata nilai matematika untuk siswa laki laki adalah 64 dan rata rata untuk siswa perempuan adalah 56 maka perbandingan banyak siswa laki laki dan perempuan adalah​


Jawab:

Perbandingan jumlah siswa laki - laki dan jumlah siswa perempuan adalah 1 : 3.

Penjelasan dengan langkah-langkah:

Hal pertama yang kita lakukan adalah mendata informasi yang ada. Jumlah anak laki - laki dan perempuan kita umpamakan dengan variabel untuk mempermudah penulisan matematisnya.

Diketahui :

Jumlah Anak Laki - laki = a

Jumlah Anak Perempuan = b

Rata - Rata nilai Anak Laki - laki = 64

Rata - Rata nilai anak perempuan = 56

Maka kita dapat mengetahui total nilai tiap - tiap kelompok anak sebagai berikut :

Total Nilai = Jumlah Anak x Rata - rata nilai anak

Total Nilai Laki - laki = a x 64

Total Nilai Laki - laki = 64a

Total Nilai Perempuan = b x 56

                                     = 56b

Nilai rata-rata seluruh kelas adalah gabungan antara nilai anak kelompok laki - laki dan perempuan dibagi dengan jumlah siswa seluruhnya. Maka secara matematis didapatkan :

Rata - rata Kelas = (Total Nilai Laki - laki + Total Nilai perempuan) : (Jumlah Anak Laki - laki + Jumlah Anak perempuan)

Rata - rata kelas = (64a + 56b) :  (a + b)

Kemudian jumlah anak berpindah ke sisi lainnya maka didapatkan :

Rata - rata kelas x (a+b) = 64a + 56b

58 x (a + b) = 64a + 56b

58a + 58b = 64a + 56b

58b - 56b = 64a - 58a

2b = 6a

b = 3a

Berdasarkan hasil diketahui bahwa jumlah b tiga kali dari jumlah a, maka persamaannya adalah :

Jumlah Siswa Laki - laki : Jumlah Siswa Perempuan = 1 : 3

21. Pada ulangan matematika, diketahui nilai rata-rata kelas adalah 58. Jika rata-rata nilai matematika untuk siswa prianya adalah 65 dan untuk siswa perempuannya adalah 54, perbandingan jumlah siswa pria dan perempuan pada kelas itu adalah .....


Jawab:

Kelas         : IX

Pelajaran   : Matematika

Kategori     : Statistika

Kata Kunci : rata-rata gabungan, perbandingan jumlah siswa

Penjelasan dengan langkah-langkah:

Diketahui

Hasil ulangan matematika 

Nilai rata-rata kelas adalah X = 58

Nilai rata-rata siswa adalah X₁ = 65

Nilai rata-rata siswi adalah X₂ = 54

Ditanya

Perbandingan jumlah siswa dan siswi

Penyelesaian

Misalkan banyaknya siswa (putra) adalah P dan banyaknya siswi (putri) adalah W

Rata-rata gabungan adalah sebagai berikut:

Keterangan

X = rata-rata nilai gabungan

X₁ = rata-rata nilai kelompok siswa (putra)

X₂ = rata-rata nilai kelompok siswi (putri)

P = banyaknya siswa (putra)

W = banyaknya siswi (putri atau wanita)

Substitusikan data-data dari soal

⇔

⇔ Kalikan silang

⇔ 58(P + W) = 65P + 54W

⇔ 58P + 58W = 65P + 54W

⇔Saling pindah ruas

⇔ 65P - 58P = 58W - 54W

⇔ 7P = 4W

⇔

Dapat ditulis sebagai W : P = 7 : 4

Diperoleh perbandingan banyaknya siswi putri dan siswa putra sebesar 7 : 4

25. Pada ulangan matematika, diketahui nilai rata-rata kelas adalah 58. Jika rata-rata nilai matematika untuk siswa laki-laki 64 dan rata-rata untuk siswa perempuan 56, maka perbandingan banyak siswa laki-laki dan perempuan adalah


Jawab:

1 : 3

Penjelasan dengan langkah-langkah:

x = bagian perempuan

1 - x = bagian laki-laki

56x + 64(1 - x) = 58

56x + 64 - 64x = 58

-8x = 58 - 64

-8x = -6

x = -6/-8 = 3/4 (perempuan)

28. Pada ulangan matematika diketahui nilai rata rata suatu kelas adalah 58. Jika rata rata nilai ulangan untuk siswa laki laki adalah 64 dan rata rata nilai ulangan untuk suswa perempuan adalah 56, maka perbandingan banyak siswa laki laki dan perempuan adalah


x = banyak siswa laki laki

y = banyak siswa perempuan

58 = x(64) + y(56) / (x + y)

58(x + y) = 64x + 56y

58x + 58y = 64x + 56y

58y - 56y = 64x - 58x

2y = 6x

x/y = 2/6

x : y = 2 : 6 = 1 : 3


29. Pada ulangan matematika, diketahui nilai rata-rata kelas adalah 58.jika rata-rata nilai matematika untuk siswa laki-laki 64 dan rata-rata untuk siswa perempuan 56 maka perbandingan banyak siswa laki-laki dan perempuan adalah


Jawaban:

1:3

Penjelasan dengan langkah-langkah:

Misalkan

jumlah nilai laki-laki=a

jumlah nilai perempuan=b

jumlah siswa laki-laki=p

jumlah siswa perempuan=q

maka:

[tex] \frac{a + b}{p + q} = 58[/tex]

[tex] \frac{a}{p} = 64[/tex]

[tex] \frac{b}{q} = 56[/tex]

a=64p

b=56q

64p + 56q = 58(p+q)

64p + 56q = 58p + 58q

6p - 2q = 0

3p - q = 0

3p = q

[tex] \frac{p}{q} = \frac{1}{3} [/tex]

jadi perbandingan banyak siswa laki-laki dan perempuan adalah 1:3
