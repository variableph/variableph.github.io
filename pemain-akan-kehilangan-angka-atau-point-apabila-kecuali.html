3□4 = 304 + x, 45□ = 450 + y, □56 = 56 + z
x + y + z = N
Rata rata 3 bilangan = 339
Jumlah 3 bilangan = 339 × 3 = 1017
304 + 450 + 56 + N = N + 810
N = 1017 - 810 = 207 ; maka x = 0 ; y = 7 ; z = 2 ; 2 + 0 + 7 = 9

13. Dalam suatu pertandingan bola basket, 8 orang pemain rata-rata mencetak 14 angka (point). Jika perolehan angka Panji digabungkan, rata-ratanya menjadi 15 angka. Banyak angka yang dicetak Panji adalah


Banyak angka yang dicetak Panji adalah 24.

Pembahasan

Rata-rata atau mean adalah jumlah nilai selutuh data dibagi dengan banyaknya data.

Rumus Rata-rata :

[tex]\boxed{\bold{\bar{x}=\frac{x_1+x_2+x_3+...+x_n}{n}}}[/tex]

[tex]\bar{x}[/tex] = rata-rata

n = banyaknya data

=======================================================

Diketahui :

Jumlah awal pemain = 8

Rata-rata awal = 14

Rata-rata baru = 15

Ditanya :

Angka yang dicetak Panji

Jawab :

Jumlah pemain awal = 8 orang => bila Panji bergabung maka jumlah pemain akhir  menjadi = 8 + 1 = 9 orang.

Misalkan angka yang dicetak panji adalah a.

[tex]\text{Rata-Rata Baru}=\frac{\text{Jumlah awal pemain x rata-rata lama + Angka Panji}}{\text{Jumlah pemain akhir}}[/tex]

[tex]15=\frac{14(8)+a}{9}[/tex]

[tex]15=\frac{112+a}{9}[/tex]

15 (9) = 112 + a

112 + a = 135

a = 135 - 112

a = 24

Jadi banyak angka yang dicetak Panji adalah 24.
