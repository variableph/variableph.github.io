6. Sebuah termometer X dibuat dengan batas-batas ukur bawah 0⁰X dan batas ukur atas 120⁰ X. Jika termometer X menunjukkan angka 36⁰ X, Berapakah angka yang ditunjukkan oleh termometer Celcius?


diketahui:

misal Tb = titik beku air, Ta = titik didih air

[tex]T_{xb}=0^oX\\T_{xa}=120^oX\\T_{cb}=0^oC\\T_{ca}=100^oC[/tex]

ditanya:

suhu 36°X pada skala celcius, Tc

jawab:

[tex]\frac{T_c-T_{cb}}{T_{ca}-T_{cb}}=\frac{T_x-T_{xb}}{T_{xa}-T_{xb}}\\\frac{T_c-0}{100-0}=\frac{36-0}{120-0}\\\frac{T_c}{100}=\frac{36}{120}\\T_c=\frac{3600}{120}\\T_c=30^oC[/tex]

12. Termometer x dengan batas atas 180° x dan batas bawah 20° x. termometer y dengan batas atas 100°y dan batas bawah -20°y. Jika termometer x menunjukkan angka 50 x ,tentukan angka yang ditunjukkan termometer y


Termometer x dengan batas atas 180° x dan batas bawah 20° x. Termometer y dengan batas atas 100°y dan batas bawah -20°y. Jika termometer x menunjukkan angka 50° x , angka yang ditunjukkan termometer y adalah 2,5°y

PEMBAHASAN

Untuk mengkonversi satuan suhu bisa di pakai perbandingan sebagai berikut ini :

°C : (°F-32) : °R : (K-273) = 5 : 9 : 4 : 5  

Untuk skala sembarang yang lain bisa di pakai perbandingan sebagai berikut ini :

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

dimana :

x₁ = titik tetap bawah termometer x

x₂ = titik tetap atas termometer x

y₁ = titik tetap bawah termometer y

y₂ = titik tetap atas termometer y

Mari kita gunakan rumus diatas untuk mengerjakan soalnya.

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

(50 - 20) / (180 - 20) = (y - (-20)) / ( 100 - (-20) )

30/160 = (y+20) / 120

3/16 = (y+20) / 120

y+20 = 3/16 x 120

y+20 = 45/2

y = 45/2 - 20

y = 2,5

Kesimpulan :

50°x = 2,5°y

20. Andi membuat termometer dengan batas bawah 50°X dan batas atas 140°x benda diukur suhunya menggunakan termometer celsius dengan batas bawah 20°c dan batas atas 80°c jika pada termometer celsius menunjukkan angka 40°c berapakah angka yang ditunjukkan oleh termometer x


Penjelasan:

untuk penggunaan batas atas (BA) Dan batas bawah (BB) menyesuaikan apa yang ditanya dan diketahui. itu di ruas kiri yang ditanya adalah °X sehingga BA dan BB nya pakai miliknya °X, dan di ruas kanan yang diketahui adalah °C (40°C) sehingga BA Dan BB nya pakai miliknya °C

diket:

termometer X:

batas bawah = 50°

batas atas = 140°

skala = 140° - 50° = 90°

termometer Celcius:

batas bawah = 20°

batas atas = 80°

skala = 80° - 20° = 60°

tC = 40°C

ditanya:

suhu dalam termometer X (tX)....?

jawab:

(tX - 50°) / 90° = (tC - 20°) / 60°

(tX - 50°) / 3 = (tC - 20°) / 2

(tX - 50°) / 3 = (40° - 20°) / 2

(tX - 50°) / 3 = 20° / 2

(tX - 50°) / 3 = 10°

tX - 50° = (10°)(3)

tX - 50° = 30°

tX = 30° + 50°

tX = 80°

Jadi, suhu yang ditunjukkan pada termometer X adalah 80°X.

22. Termometer x memiliki batas bawah 80° dan batas atas 240°, jika hasil pengukuran termometer x adalah 120°b, maka jika diukur dengan termometer Celcius adalah?​


Jawaban:

Termometer X memiliki batas bawah 80° dan batas atas 240°.  Hasil pengukuran dengan termometer X adalah 120°, maka hasil pengukuran dengan termometer Celcius adalah 25°C​

Penjelasan:

Diketahui:

Batas Bawah Termometer X ([tex]X_{b}[/tex]) = 80°

Batas Atas Termometer X ([tex]X_{a}[/tex]) = 240°

Batas Atas Termometer Celcius  ([tex]C_{a}[/tex])= 100°

Batas Bawah termometer Celcius ([tex]C_{b}[/tex]) = 0°

Ditanya:

Jika T°X menunjukkan suhu 120°, maka T°C=..................?

Penyelesaian:

[tex]\frac{T^{o}X - X_{b} }{Z_{a} - X_{b} }[/tex] =[tex]\frac{T^{o}C - C_{b} }{C_{a} - C_{b} }[/tex]

[tex]\frac{120 - 80}{240 - 80}[/tex] = [tex]\frac{T^{o} C - 0}{100 -0}[/tex]

[tex]\frac{40^{o} }{ 160^{o} }[/tex] = [tex]\frac{T^{o} C}{100^{o} }[/tex]

[tex]T^{o}C =[/tex][tex]\frac{40 x 100}{160}[/tex] = 25° C

28. Sebuah termometer x dengan batas atas 150°x dan batas bawah 10°x. Termometer Y dengan batas atas 100°Y dan batas bawah 30°Y.jika termometer x menunjukkan angka 40x, Tentukan angka yang ditunjukkan termometer Y!


Sebuah termometer X dengan batas atas 150°X dan batas bawah 10°X. Termometer Y dengan batas atas 100°Y dan batas bawah 30°Y. Jika termometer X menunjukkan angka 40°X, maka angka yang ditunjukkan termometer Y adalah 45°Y

PEMBAHASAN

Untuk mengkonversi satuan suhu bisa di pakai perbandingan sebagai berikut ini :

°C : (°F-32) : °R : (K-273) = 5 : 9 : 4 : 5  

Untuk skala sembarang yang lain bisa di pakai perbandingan sebagai berikut ini :

(x - x₁) / (x₂ - x₁) = (y - y₁) / ( y₂ - y₁ )

dimana :

x₁ = titik tetap bawah termometer x

x₂ = titik tetap atas termometer x

y₁ = titik tetap bawah termometer y

y₂ = titik tetap atas termometer y

Okay marilah kita gunakan rumus ini untuk menyelesaikan soal yang di maksud.

Diketahui suatu termometer X dengan titik tetap atas 150°X dan titik tetap bawah 10°X  , ingin di konversi ke termometer Y yang memiliki titip tetap atas 100°Y dan titik tetap bawah 30°Y , maka:

(40 - 10) / (150 - 10) = (y - 30) / ( 100 - 30 )

(y - 30) = 15

y = 45

Kesimpulan :

40°X = 45°Y
