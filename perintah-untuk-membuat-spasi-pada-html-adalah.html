9. Perintah shortcut keyboard untuk membuat jarak tulisan menjadi satu setengah spasi pada ms word, setelah semua area terseleksi adalah


Jawaban Singkat:

CTRL + 5

Penjelasan panjang:

Di Ms. Word terdapat beberapa shortcut keyboard untuk mengatur spasi pada teks yg sudah terseleksi, antara lain:

CTRL + 1 untuk mengatur spasi teks menjadi 1 (single)

CTRL + 2 untuk mengatur spasi teks menjadi 2 (double)

CTRL + 5 untuk mengatur spasi teks menjadi 1 setengah

Perlu dimengerti bahwa menekan tombol CTRL dan tombol 5 harus dilakukan bersamaan. Amatilah lampiran (letak kedua tombol itu ditunjukkan dalam kotak kuning)
