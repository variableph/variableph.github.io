8. perintah yang di gunakan untuk mengubah ukuran kertas header and footer,orientasi pencetakan serta margin halaman web yg akan d cetak adalah​


Perintah yang digunakan untuk melakukan pengaturan kertas pada halaman website yang akan dicetak adalah perintah page setup. Menu page setup terdapat pada browser yang kita gunakan. Menu page setup ini berada di dalam menu file.

Pada page setup, kita bisa mengatur beberapa pengaturan yaitu

Ukuran kertasMargin atas bawahMargin kanan kiriHeader footerJenis orientasi kertas landscape atau potraitPembahasan

Page setup tidak hanya ada pada browser saja, hampir semua aplikasi perkantoran memiliki menu page setup. Dengan menggunakan page setup kita bisa menghemat penggunaan kertas. Karena dengan page setup kita bisa mencetak beberapa halaman website pada satu halaman saja. Beberapa contoh aplikasi perkantoran adalah

Microsoft WordMicrosoft ExcelMicrosoft Power Point

Untuk mengakses menu page setup pada aplikasi perkantoran juga biasanya berada pada menu File.

Pelajari lebih lanjut

Demikian pembahasan mengenai perintah yang digunakan untuk mengatur ukuran kertas dan orientasi kertas. Menu yang digunakan adalah menu Page Setup. Untuk mempelajari lebih lanjut mengenai menu atau browser dapat dibaca pada link berikut
