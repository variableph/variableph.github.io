2. Perintah html dalam form yang digunakan untuk menerima masukan dari pengguna berupa beberapa baris teks, seperti komentar dalam guestbook adalah …


Perintah html (HTML tag) dalam form yang digunakan untuk menerima masukan dari pengguna berupa beberapa baris teks, seperti komentar dalam guestbook adalah <textarea>.
 

Pembahasan

Pada HTML, elemen kontrol UI (user interface) yang digunakan untuk menerima masukan beruba beberapa baris teks (multiline) adalah textarea, dengan sintaks dasar:
<textarea atribut...></textarea>

Pada bagian atribut, yang paling penting adalah name, kemudian id. Ketika form HTML dikirimkan (di-submit), hasil pengirimannya (form submission) dapat diambil dari atribut name.

Sebagai contoh:

Contoh pertama:
<textarea id=“txtKomentar” name=“komentar”>
   Silahkan masukkan komentar Anda.
</textarea>
akan menampilkan kotak masukan untuk komentar, dengan banyak baris dan kolom default (sesuai browser), atau sesuai styling yang ditetapkan pada CSS, atau ditentukan dari javascript.Contoh kedua:
<textarea id=“txtKomentar” name=“komentar”
       rows=“3” cols=“40>
   Silahkan masukkan komentar Anda.
</textarea>
akan menampilkan kotak masukan yang sama dengan sebelumnya, dengan dimensi/ukuran 3 baris dan 40 kolom (40 karakter).

Debian adalah sistem operasi berbasis Linux. Versi pertama dirilis pada tahun 1996. Sistem operasi  ini memiliki  nama kode unik yang berasal dari karakter  film Toy Story seperti Buzz, Rex,  dan Bo. Perintah yang dimiliki oleh Linux Debian adalah "mkdir". Fungsinya untuk membuat  folder baru. Perintah ini dapat digunakan tanpa  terlebih dahulu memasukkan superuser.

Pembahasan

Debian tidak hanya menyediakan sistem operasi untuk perangkat Anda, tetapi juga  ribuan perangkat lunak gratis. Perangkat lunak ada di dalam repositori dan dapat diinstal sesuai kebutuhan. Sistem operasi  berbasis Linux sebenarnya memiliki perintah dasar yang sama. Perintah  dijalankan melalui terminal, seperti prompt perintah Windows.

1.su Perintah

su digunakan oleh pengguna Debian untuk mendapatkan akses root atau menjadi superuser. Setelah memasukkan perintah "su", pengguna akan dimintai kata sandi untuk otentikasi.

2.cd Perintah

cd berguna untuk mengubah direktori atau folder. Perintah ini tersedia setelah pengguna masuk ke sistem root menggunakan perintah pertama. Di terminal,  nama folder target akan muncul setelah "cd".

3.ls

Setelah masuk ke direktori atau folder, yang biasa dilakukan pengguna  adalah melihat isi atau daftar file. Untuk melakukan ini, gunakan perintah "ls". Seperti "cd", nama folder target akan muncul setelah "ls".

20. Perintah html untuk menentukan, tebal garis table adalah


Hypertext Markup Language atau yang lebih sering dikenal dengan HTML adalah bahasa sederhana yang dipergunakan di dalam pembuatan halaman suatu website. Dalam operasional penggunaan html, perintah html yang digunakan untuk menentukan ketebalan garis table adalah Table Border.

Pembahasan

Di era digital saat ini, pembuatan situs web dan aplikasi seluler telah menjadi pusat tren untuk  meningkatkan promosi dan meluncurkan produk dan layanan secara efektif. Oleh karena itu, bekerja sebagai programmer adalah pilihan yang paling diinginkan di industri ini. Pengembang web juga harus mahir dalam bahasa markup khusus untuk membantu  struktur halaman situs web, desain HTML.

HTML atau bahasa markup hypertext adalah bahasa markup yang digunakan untuk menyusun halaman web untuk ditampilkan di browser web. Oleh karena itu, HTML dapat digunakan sebagai landasan pertama untuk membangun kerangka website terstruktur sebelum membahas  tampilan desain dan aspek fungsional.

Atribut border, cellpadding, dan cellpacing digunakan untuk mengubah tampilan tabel, terutama yang berkaitan dengan batas dan spasi antar sel tabel. Atribut border digunakan untuk mengatur ketebalan border tabel. Jika atribut ini tidak ditentukan, browser web  akan menampilkan tabel tanpa bingkai. Nilai  atribut ini adalah angka yang diukur dalam piksel. Jika Anda menentukan batas nilai = "2", browser web  menampilkan batas 2 piksel di bagian atas, bawah, kiri, dan kanan tabel.

26. Perintah untuk menyimpan dokumen slide ke dalam bentuk HTML adalah


Perintah untuk menyimpan dokumen slide (power point) ke dalam bentuk HTML adalah dengan cara menggunakan perintah save as yang terdapat di microsoft power point.  Caranya adalah dengan mengeklik office button yang berada di pojok kiri atas, kemudian arahkan kursos ke dalam tab "Save As", kemudian pilih "Other Format" Nah di dalam dialog tersebut, silahkan kamu pilih tipe file Web Page. Secara otomatis file slide atau power point (ppt) kamu akan berubah menjadi file dengan format HTML dan bisa dibuka dengan menggunakan browser seperti Google Chrome, Firefox, Safari, dan sebagainya.

Pembahasan

format HTML atau kepanjangan dari Hypertext Markup Language merupakan format dokumen multimedia pada tampilan website yang dapat dibuka pada suatu browser. Dengan mengkonversi file power point menjadi format html, maka power point atau slide tersebut hanya bisa dibuka di Browser saja, alisas tidak bisa dibuka di software pengolah presentasi tersebut.
