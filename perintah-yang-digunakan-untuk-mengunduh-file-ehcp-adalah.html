1. Perintah yg di gunakan untuk mengunduh file ehcp adlh


Jawaban:

Perintah yang digunakan untuk mengunduh file ehcp adalah wget -O ehcp.tgz . Setelah perintah ini ditulis, dialnjutkan dengan perintah untuk melakukan extract dengan tar -zxvf ehcp.tgz. Lalu kita harus masuk ke dalam direktori filenya dengan cd ehcp dan install menggunakan perintah ./install.sh

Penjelasan:

EHCP atau Easy Hosting Control Panel adalah suatu layanan software opensource yang berguna untuk melakukan control panel hosting. Yang bisa dilakukan hosting dengan ehcp adalah email, akun dalam suatu ftp, hingga web. Jika menggunakan ehcp, ehpc akan menyediakan beberapa layanan default seperti webserver, web mail, database server, ftp server dan sebagainya. Model kontrol ehcp ini mirip dengan beberapa osftware seperti mysql, squirrel mail, dan roundcube.

5. Untuk menyimpan file yang sudah pernah tersimpan dengan menggunakan nama file yang sama di gunakan perintah..


Jawaban:

perintah save as

Penjelasan:

Perintah save as dapat digunakan antara lain untuk hal-hal berikut ini:

1. Menyimpan file dg nama yg sama, namun berbeda ekstensi filenya.

Contoh: Ada file dengan nama brainly.doc, lalu dilakukan perintah save as dengan disimpan memakai nama brainly.docx

2. Menyimpan file dg nama yg sama, tapi beda lokasi penyimpanannya.

Contoh: Ada file dengan nama brainly.doc di harddisk komputer, lalu dilakukan perintah save as dengan disimpan di dalam flashdisk (masih dengan nama yang sama, yaitu brainly.doc)

3. Perintah save as dapat juga dipakai untuk menyimpan file yg pernah tersimpan dengan penamaan berbeda.

Contoh: Ada file dengan nama brainly.doc, lalu dilakukan perintah save as dengan disimpan memakai nama brainlyhebat.doc

24. Perintah yang digunakan untuk menyimpan file adalah...


Perintah yang dapat dilakukan guna untuk melakukan penyimpanan sebuah file adalah dengan cara menekan tombol control + s.

Pembahasan  

Penyimpanan secara umum pada sebuah aplikasi akan menggunakan tombol control + s(ave). Penyimpanan tersebut akan dapat menyimpan ke sebauh folder dari drive hard disk yang dimana kita miliki, lokasi dari jaringan internet, dengasn menggunakan awan, DVD, hingga ke dalam dekstop serta flash drive. Tetapi didalam melakukan penyimpanan haruslah dengan cara melakukan pengidentifikasian dari sebuah tujuan, apabila akan berbeda dari sebuah folder default, dari proses melakukan penyimpanan tersebut akan sama seperti sebuah bentuk dari tujuan yang dimana kita miliki.  Akan tetapi secara umum program Office akan melakukan penyuimpanan pada folder default.
