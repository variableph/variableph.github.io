Jawaban:

Permainan bola basket dimainkan oleh 5 orang setiap tim.

Penjelasan:

PERMAINAN  BASKET

Tujuan utama dari permainan ini adalah masing-masing tim berusaha memasukkan bola ke keranjang lawan dengan menggunakan tangan sebanyak-banyaknya demi meraih kemenangan

SEJARAH PERMAINAN BASKET

Permainan basket diciptakan oleh James Naismith yang berasal dari Amerika. Permainan basket ini dimainkan oleh dua regu dengan  5 pemain setiap regunya.  

LATIHAN DASAR PERMAINAN BOLA BASKET

Dribbling ( menggiring bola ) Passing and Catching ( mengoper dan menangkap ) Shooting ( menembakkan bola ke arah ring basket ) Pivot ( gerakkan memutar badan ) Jump Stop Rebound

DRIBBLING

Prinsip dalam gerak dasar dribbling antara lain:

Jari – jari tangan terbuka Bola tidak dipukul Tangan menjemput bola Tinggi pantulan bola sebatas pinggang

PASSING

Passing berarti mengoper / mengumpan bola kepada teman  

3 gerak dasar cara pasing dalam permainan basket:

Chest Pass ( operan dari depan dada ) Bounce Pass ( operan pantul )   Overhead Pass ( operan dari atas kepala )

CHEST PASS ( OPERAN DARI DEPAN DADA ):

Operan ini dimulai dari memegang bola di depan dada , salah satu kaki berada di depan, kemudian bola didorong lurus dengan telapak tangan ke arah luar

BOUNCE PASS ( OPERAN PANTUL ):

Sama dengan chest pass, bola di depan dada, salah satu kaki berada di depan, bedanya hanya lemparan di arahkan ke lantai , usahakan titik pantulnya tidak terlalu dekat dengan penerima bola.

OVERHEAD PASS( OPERAN DARI ATAS KEPALA ):

Lemparan dilakukan dengan kedua tangan berada di atas kepala memegang bola, salah satu kaki berada di depan, kemudian bola dilemparkan ke arah teman.

SHOOTING

Dalam permainan bola basket menembakkan bola dengan tujuan memasukkan bola ke ring disebut  shooting.

Ada beberapa cara shooting atau menembak:

Menembak dengan satu tangan (one hand shoot) Menembak dengan dua tangan (two hand shoot) Menembak sambil meloncat (jump shoot) Tembakan melayang (lay-up shoot)

GERAKAN PIVOT ATAU MEMUTAR BADAN

Pivot adalah gerakan memutar badan dengan menggunakan salah satu kaki sebagai poros putaran.

Gerakan ini bertujuan untuk menjauhkan bola dari jangkauan lawan.

Ada 3 Variasi gerakan yang dapat dilakukan:

Pivot kemudian dribbling ( menggiring bola )Pivot kemudian passing ( melempar bola ) Pivot kemudian shooting ( menembak bola )
