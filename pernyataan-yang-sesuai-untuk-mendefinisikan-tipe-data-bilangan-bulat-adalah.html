27. deklarasi banyak variabel dengan tipe data sama,Definisikan?


deklarasi banyak variabel dengan tipe data sama,Definisikan?JawabanPendahuluan

Sebelum kita belajar tentang variabel, ada baiknya bila kita mengenal tipe data terlebih dahulu. Tipe data merupakan bagian program yang paling penting karena tipe data mempengaruhi setiap instruksi yang akan dilaksanakan oleh computer. Misalnya saja 5 dibagi 2 bisa saja menghasilkan hasil yang berbeda tergantung tipe datanya.

Jika 5 dan 2 bertipe integer maka akan menghasilkan nilai 2, namun jika keduanya bertipe float maka akan menghasilkan nilai 2.50. Pemilihan tipe data yang tepat akan membuat proses operasi data menjadi lebih efisien dan efektif.


Pembahasan

Tipe data adalah suatu nilai yang dapat dinyatakan dalam bentuk konstanta atau variabel dan operator. Konstanta menyatakan nilai yang tetap, sedangkan variabel  menyatakan nilai yang dapat berubah-ubah selama eksekusi berlangsung.

Berikut ini adalah contoh tipe data yag sering digunakan.

No Tipe Data Ukuran Range (Jangkauan) Format Keterangan

1 Char          1 byte -128 s/d 127                  %c Karakter/string

2 Int                  2 byte -32768 s/d 32767          %i , %d Integer/bilangan bulat

3 Float          4 byte 3.4E-38 s/d 3.4E+38 %f Float/pecahan

4 Double          8 byte 1.7E-308 s/d 1.7+308 %lf Pecahan presisi ganda

5 Void          0 byte -                                  - Tidak bertipe

Yang perlu diketahui tentang “Deklarasi Variabel”. Variabel adalah suatu pengenal yang digunakan untuk mewakili suatu nilai tertentu dalam program tersebut. Variabel harus dideklarasikan terlebih dahulu sebelum digunakan. Dan nilai variable itu sendiri dapat berubah-ubah selama program dijalankan.

Contoh :

int a;  // untuk mendeklarasikan a yang bertipe integer

float nilai; // untuk mendeklarasikan nilai yang bertipe float  

char nama; // untuk mendeklarasikan nama yang bertipe char

Kesimpulan

Untuk mendeklarasikan variabel yang lebih dari satu namun memiliki tipe data yang sama, misalnya:

int nilai1;

int nilai2;

int nilai3;

Bisa kita sederhanakan menjadi int nilai1, nilai 2, nilai3;
