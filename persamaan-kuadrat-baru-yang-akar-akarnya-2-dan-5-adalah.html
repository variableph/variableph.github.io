8. persamaan kuadrat baru yang akar akarnya 2 kurangnya dari akar akar persamaan 2x²+3x+5=0 adalah​


Jawaban:

Persamaan Kuadrat secara Umum

Jika ada persamaan kuadrat ax² + bx + c = 0, dan memiliki akar x₁ dan x₂, maka akar persamaan kuadrat tersebut memenuhi

x₁ + x₂ = -\cfrac{b}{a}−ab

x₁ x₂ =  \cfrac{c}{a}ac

Persamaan Kuadrat 2x² - 3x - 4 = 0

Jika akar-akarnya adalah x₁ dan x₂, maka

x₁ + x₂ = -\cfrac{(-3)}{2}−2(−3)

x₁ + x₂ = \cfrac{3}{2}23

x₁ x₂ = -\cfrac{(4)}{2}−2(4)

x₁ x₂ = -2

Persamaan Kuadrat yang Baru

Jika persamaan kuadrat yang baru memiliki akar α dan β, dan diketahui akar-akar tersebut dua kurangnya dari akar persamaan kuadrat yang pertama, maka nilai α dan β adalah

α = x₁ - 2

β = x₂ - 2

Untuk membentuk persamaan yang baru, kita harus menentukan nilai -\cfrac{b}{a}−ab serta \cfrac{c}{a}ac yang baru. Misal, kita gunakan nilai a adalah 1.

Nilai -b/a

-\cfrac{b}{a}−ab = α + β

-\cfrac{b}{1}−1b = x₁ - 2 + x₂ - 2

-\cfrac{b}{1}−1b = x₁ + x₂ - 4

-b = x₁ + x₂ - 4

-b = \cfrac{3}{2} -423−4

-b = \cfrac{3}{2} - \cfrac{8}{2}23−28

-b = \cfrac{-5}{2}2−5

b = \cfrac{5}{2}25

Nilai c/a

\cfrac{c}{a}ac = αβ

\cfrac{c}{1}1c = (x₁ - 2)(x₂ - 2)

c= x₁x₂ - 2(x₁ + x₂) + 4

c = -2 - 2(\cfrac{3}{2})(23) + 4

c = -2 -3 + 4

c = -5 + 4

c = -1

Persamaan yang baru menjadi

ax² + bx + c = 0

1x² + \cfrac{5}{2}25 x + (-1) = 0

x² + \cfrac{5}{2}25 x - 1 = 0

----------------------------------------------- kali kedua ruas dengan 2

2x² + 5x - 2 = 0

Persamaan yang baru adalah 2x² + 5x - 2 = 0

Untuk lebih jelasnya, mari kita lihat contoh sederhana. Jika akar-akar persamaan kuadrat 3X^2 - 4x + 5 = 0 adalah P = 1 dan Q = 2, maka persamaan kuadrat baru yang akar-akarnya P + 2 dan Q + 2 adalah (X - 3)(X - 4) = 0. Ini dapat dituliskan sebagai persamaan kuadrat 3X^2 - 4(1 + 2 + 2)X + (1 + 2)(2 + 2) = 0.

28. Persamaan kuadrat 3√2 − 5 − √2 = 0 memiliki akar-akar dan a. Persamaan kuadrat baru yang akar-akarnya 3α dan 3 adalah ….


Sepertinya soal ini kurang sempurna, yang namanya persamaan kuadrat pastilah ada variabel yang berderajat 2. Oleh karna itu soal saya udah agar menjadi persamaan kuadrat terlebih dahulu.

Persamaan kuadrat 3√2 x² − 5x − √2 = 0 memiliki akar-akar α dan β. Persamaan  kuadrat baru yang akar-akarnya 3α dan 3β adalah

Jawab: Persamaan kuadrat barunya adalah  x² - [tex]\frac{5}{2}\sqrt{2}[/tex] - 3 = 0.

Pembahasan

Persamaan kuadrat adalah persamaan polinomial dengan suku terdingginya berderajat 2.

Bentuk umum dari persamaan kuadrat adalah ax² + bx + c, dengan a, b dan c adalah koefisiennya, x² dan x adalah peubahnya.

Untuk mengetahui akar-akar dari persamaan kuadrat dengan cara memfaktorkan, menggunakan rumus abc, atau menjadikan kuadrat sempurna.

Rumus abc ↓

x₁, x₂ = [tex]\frac{-b+-\sqrt{b^2-4ac}}{2a}[/tex]

Pada Bentuk umum dari persamaan kuadrat adalah ax² + bx + c = 0, Penjumlahan akar-akarnya = [tex]-\frac{b}{a}[/tex] , dan

Hasil kali akar-akarnya = [tex]\frac{c}{a}[/tex] .

Rumus diskriminan persamaan kuadrat

D = b² - 4ac

Jika akar-akar dari persamaan kuadrat adalah p dan q, maka bentuk persamaan kuadratnya adalah

x² - (p+q)x + pq = 0

Penyelesaian Soal

Diketahui:

Persamaan kuadrat 3√2 x² − 5x − √2 = 0.

Memiliki akar-akar α dan β.

Ditanya:

Persamaan  kuadrat baru yang akar-akarnya 3α dan 3β adalah ...

Jawab:

Dari Persamaan kuadrat 3√2 x² − 5x − √2 = 0.

a =  3√2

b = -5

c = -√2

α + β = [tex]\frac{5}{3\sqrt{2}}[/tex]

α.β = [tex]\frac{-\sqrt{2}}{3\sqrt{2}}[/tex] = [tex]-\frac{1}{3}[/tex]

3α + 3β = 3(α + β)

             = 3 [tex]\frac{5}{3\sqrt{2}}[/tex]

             = [tex]\frac{5}{\sqrt{2}}[/tex]

             = [tex]\frac{5}{2}\sqrt{2}[/tex]

3α . 3β = 9(α.β)

            = 9 ([tex]-\frac{1}{3}[/tex])

            = -3

Persamaan kuadrat barunya adalah  x² - [tex]\frac{5}{2}\sqrt{2}[/tex] - 3 = 0.

29. tentukan persamaan kuadrat baru yang akar akarnya 1 lebihnya dari akar akar persamaan kuadrat x^2-3x+5=0​

x² - 5x + 9 = 0

Penjelasan:

Bentuk umum persamaan kuadrat

ax² + bx + c = 0

x² - 3x + 5 = 0

α + β = -b/a

α + β = - (-3)/1

α + β = 3

α × β = c/a

αβ = 5/1

αβ = 5

Persamaan kuadrat baru dengan akar-akar x₁ = α + 1, x₂ = β + 1

x₁ + x₂ = (α + 1) + (β + 1)

x₁ + x₂ = α + β + 2

x₁ + x₂ = (3) + 2

x₁ + x₂ = 5 = 5/1

x₁ + x₂ = -b/a

x₁ × x₂ = (α + 1) × (β + 1)

x₁x₂ = αβ + α + β + 1

x₁x₂ = αβ + (α + β) + 1

x₁x₂ = (5) + (3) + 1

x₁x₂ = 9 = 9/1

x₁x₂ = c/a

a = 1, b = - 5, c = 9

Jadi, Persamaan kuadrat baru dengan akar-akar

x₁ = α + 1, x₂ = β + 1 adalah x² - 5x + 9 = 0
