18. Jika ×1 dan ×2 adalah akar-akar persamaan kuadrat ×²-5×+3=0 persamaan kuadrat baru yang akar-akarnya (3×1+1) dan (3×2+1) adalah..?


• Jika akar persamaan kuadrat baru simetris dengan akar persamaan kuadrat lama maka persamaan kuadrat baru didapatkan dengan cara inverskan akar yang baru kemudian substitusikan pada persmaan kuadrat lama.

>> akar-akar persamaan kuadrat baru adalah 3x+1, maka inversnya adalah (x-1)/3
>> substitusikan (x-1)/3 ke persamaan kuadrta lama, maka :
[tex] {x}^{2} - 5x + 3 = 0 \\ ( { \frac{x - 1}{3} })^{2} - 5( \frac{x - 1}{3} ) + 3 = 0 \\ \frac{ {x}^{2} - 2x + 1}{9} - \frac{(5x - 5)}{3} + 3 = 0 \\ \frac{ {x}^{2} - 17x + 43}{9} = 0 \\ {x}^{2} - 17x + 43 = 0[/tex]

b = 3
c = -2

x1 . x2 = c/a = -2

x1 - 1 + x2 - 1 
= x1 + x2 - 2
= -3 - 2
= -5

= x1 . x2 - x1 - x2 + 1

= 2
