13. Diperusahaan "sabar jaya" suatu perusahaan jasa,memiliki 2 tipe karyawan.karyawan tipe A digaji sebesar Rp13500000 per minggu dan karyawan tipe B digaji sebesar Rp27000000 per minggu.pada suatu proyek memerlukan 110 karyawan,tetapi paling sedikit sebanyak 40 karyawan tipe B yang bekerja.selain itu, untuk setiap proyek,aturan perusahaan mengharuskan banyak karyawan tipe B paling sedikit 0,5 dari banyak karyawan tipe A.hitunglah banyak karyawan tipe A dan karyawan tipe B pada perusahaan tersebut ?


Pembahasan :

Misal :
x = karyawan tipe A
y = karyawan tipe B

'Paling sedikit' berarti tandanya ≥

• Proyek memerlukan 110 karyawan
x + y ≤ 110

• Paling sedikit 40 karyawan tipe B
y ≥ 40

• banyak karyawan tipe B paling sedikit 0,5 dari banyak karyawan tipe A
y ≥ 0,5x

• x ≥ 0

Gaji => fungsi sasaran
f(x, y) = 13.500.000x + 27.000.000y


Untuk menggambar grafik
x + y ≤ 110
x = 0 => y = 110 ... (0, 110)
y = 0 => x = 110 ... (110, 0)
Tarik garis yang melalui (0, 110) dan (110, 0) lalu arsir ke bawah

Untuk menggambar grafik
y ≥ 0,5x
x = 0 => y = 0 ... (0, 0)
x = 20 => y = 10 ... (20, 10)
Tarik garis yang melalui (0, 0) dan (6, 3) lalu arsir ke atas

Titik potong garis (x + y = 110) dengan (y = 0,5x)
x + y = 110
x + 0,5x = 110
1,5x = 110 => kali 2
3x = 220
x = 220/3

y = 0,5x
y = 0,5(220/3)
y = 110/3

(220/3, 110/3)

Titik potong (x + y = 110) dengan (y = 40)
x + y = 110
x + 40 = 110
x = 70
(70, 40)

Kemudian kita gambar grafik daerah himpunan penyelesaiannya. Diperoleh titik-titik sudutnya yaitu (0, 110), (0, 40) dan (70, 40)

Kita substitusikan ke fungsi sasaran
f(x, y) = 13.500.000x + 27.000.000y
f(0, 110) = 13.500.000(0) + 27.000.000(110) = 2.970.000.000
=> jumlah karyawan = 110
f(0, 40) = 13.500.000(0) + 27.000.000(40) = 1.080.000.000
=> jumlah karyawan = 40
f(70, 40) = 13.500.000(70) + 27.000.000(40) = 2.025.000.000
=> jumlah karyawan = 110

Jadi

Agar diperoleh pengeluaran maksimum maka banyak karyawan yang harus dipekerjakan adalah 110 karyawan tipe B saja

Sedangkan agar diperoleh pengeluaran minimum maka banyak karyawan yang harus dipekerjakan adalah 70 karyawan tipe A dan 40 karyawan tipe B.

Mengapa yang (0, 40) ga di pakai, padahal lebih minimum, karena perusahaan tersebut memerlukan 110 karyawan

23. Perusahan"SABAR JAYA"suatu perusahaan jasa, memiliki 2 tipe karyawan.karyawan tipe A digaji sebesar Rp 135.000,00 perminggu dan karyawan tipe B digaji sebesar rp 270.000.00 per minggu. Pada suatu proyek memerlukan 110 karyawan, tetapi paling sedikit sebanyak 40 karyawan tipe B bekerja. Selain itu,untuk unruk setiap proyek, aturan perusahaan mengharuskan banyak karyawan tipe B paling sedikit 0,5 dari banyak karyawan tipe A. Hitunglah banyak karyawan tipe A dan karyawan tipe B pada perusahaan tersebut.


Pendahuluan

Pertidaksamaan linear dua variabel merupakan pertidaksamaan dua variabel dengan pangkat tertinggi variabel adalah satu. Bentuk umumnya


ax+by>= c atau ax+by<=c atau ax+by> c atau ax+by<c dimana a, b, c adalah bilangan real


Himpunan penyelesaian pertidaksamaan linear dua variabel dicari berdasarkan daerah penyelesaian grafik PtLDV dengan langkah-langkah


Menggunakan metode uji titik


Gambar grafik ax+by=c


Lalu uji titik (titik diambil secara sebarang)  

Memperhatikan tanda ketaksamaan


Pastikan koefisien x dari PtLDV positif. Jika tidak positif, kalikan PtLDV engan -1


Jika koefisien x dari PtLDV positif, perhatikan tana ketaksamaan. Jika tanda ketaksamaan <= aerah penyelesaian di kiri garis pembatas. Jika tanda ketaksamaan >= daerah penyelesaian di kanan garis pembatas


Solusi sistem Pertidaksamaan linear dua variabel


Menentukan daerah penyelesaian pertidaksamaan I


Menentukan daerah penyelesaian pertidaksamaan II


Menentukan daerah penyelesaian SPtLDV



Pembahasan


Diketahui: Misalkan karyawan A =x


         Karyawan B = y


     X digaji 135.000/minggu


     Y digaji 270.000/minggu


     Paling sedikit berarti tandanya >=


     Proyek membutuhkan 110 karyawan, model matematikanya x+y <=110


     Paling sedikit 40 karyawan B, model matematikanya y>=40


     Aturan perusahaan banyak karyawan B paling sedikit 0,5 banyak karyawan A,  model matematikanya  

    y >= ½ A


Ditanya: banyak karyawan A dan B pada perusahaan


Penyelesaian:


Untuk menggambar grafik


x+y<=110


x=0 ------x+y=110


  0+y=110


      y=110-------- (0,110)


y=0 ------x+y=110


  x+0=110


      x=110-------- (110,0)


Tarik garis melalui (0,110) dan (110,0) lalu arsir ke bawah


y>=40  

Tarik garis horizontal dari y=40 lalu arsis ke atas


Titik potong garis x+y<=110 dengan y>=40


x+y=110


x+40=110


x=110-40


x=70------ (70,40)



Kemudian gambar grafik daerah penyelesaian, diperoleh titik-titik (0,110), (0,40) dan (70,40)


Karena untuk setiap proyek, aturan perusahaan mengharuskan banyak karyawan tipe B paling sedikit 0,5 dari banyak karyawan tipe A.


Sehingga dipilih solusi (70 ,40) yang memenuhi persyaratan, yaitu


y >= ½ x


y >= ½ (70)


y >= 35



Jadi, banyak karyawan A adalah 70 orang dan karyawan B adalah 40 orang
