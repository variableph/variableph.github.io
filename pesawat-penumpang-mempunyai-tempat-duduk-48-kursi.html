Pesawat penumpang mempunyai tempat duduk 48 kursi Setiap penumpang kelas utama boleh membawa bagasi 60 kg sedang kelas ekonomi 20 kg pesawat hanya dapat membawa bagasi 1440 kg harganya Rp150.000 dan kelas ekonomi Rp100.000 pendapatan dari penjualan tiket pada saat pesawat kuno mencapai maksimum jumlah tempat duduk kelas utama haruslah


x + y ≤ 48
60x + 20y ≤ 1440

x ≥ 0
y ≥ 0

x + y = 48
x = 48 - y

60x + 20y = 1440
60(48 - y) + 20y = 1440
2880 - 60y + 20y = 1440
2880 - 1440 = 60y - 20y
1440 = 40y
y = 1440/40
y = 36

x + y = 48
x = 48 - y
x = 48 - 36
x = 12

misal x = 0
60x + 20y = 1440
20y = 1440
y = 72 (tapi karena maksimalnya 48 orang, maka y = 48))

misal y = 0
60x = 1440
x = 24

f(x, y) = 150.000x + 100.000y
f(12, 36) = 150.000(12) + 100.000(36) = 5.400.000
f(0, 48) = 150.000(0) + 100.000(48) = 4.800.000
f(24, 0) = 150.000(24) + 100.000(0) = 3.600.000

Jadi, hasilnya maksimum ketika jumlah tempat duduk di kelas utama 12 kursi.

28. Pesawat penumpang mempunyai tempat duduk 48 kursi. Setiap penumpang kelas utama boleh membawa bagasi 60 kg sedangkan kelas ekonomi 20 kg pesawat hanya dapat membawa bagasi 1440 kg. Harga tiket kelas utama rp150.000,00 dan kelas ekonomi rp100.000,00. Supaya pendapatan dari penjual tiket pada saat pesawat penuh mencapai maksimum, jumlah tempat duduk kelas utama haruslah


.......................... Utama (x) ...... Ekonomi (y)
Jmlh kursi ............. x ..................... y ............... 48
Bagasi ................ 60x ................. 20y ............ 1.440
Harga ............ 150.000x ........... 100.000y .. ???

Model matematika
x + y ≤ 48
x = 0 => y = 48 => (0, 48)
y = 0 => x = 48 => (48, 0)

60x + 20y ≤ 1.440
3x + y ≤ 72
x = 0 => y = 72 => (0, 72)
y = 0 => x = 24 => (24, 0)

Titik potong
3x + y = 72
x + y = 48
--------------- --
2x = 24 => x = 12
x + y = 48 => 12 + y = 48 => y = 36
(12, 36)

Setelah digambar diperoleh titik2 sudutnya : (12, 36), (0, 48) dan (24, 0)
Substitusikan ke harga : 150.000x + 100.000y
(12, 36) => 150.000(12) + 100.000(36) = 5.400.000
(0, 48) => 150.000(0) + 100.000(48) = 4.800.000
(24, 0) => 150.000(24) + 100.000(0) = 3.600.000
Maksimumnya 5.400.000 untuk kelas utama = 12, ekonomi = 36

Jadi agar pendapatannya maksimum jumlah tempat duduk kelas utama harus 12 kursi
