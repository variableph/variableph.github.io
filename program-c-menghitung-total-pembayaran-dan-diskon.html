3. Bu Novita menjual mangga golek dengan harga Rp. 1.100,- per buah. Seorang pembeli akan mendapat diskon sebesar 10% jika total pembelian di atas Rp. 100.000,- Tampilkan total yang harus dibayar seorang pembeli, besarnya diskon yang didapat dan total yang harus dibayarkan sesudah dikurangi dengan diskon. Buatlah Programnya Dengan Bahasa C++


Materi: Bahasa Pemrograman

Algoritma/Penjelasan:

Program dimulai.Deklarasi jumlahMangga, hargaMangga = 1100, totalAwal, besarDiskon = 0, totalAkhir sebagai integer.Input jumlahMangga.Masukkan nilai totalAwal = jumlahMangga * hargaMangga.Jika totalAwal > 100000, maka besarDiskon = jumlahAwal * (10/100) dan totalAkhir = totalAwal - besarDiskon; Selain itu, totalAkhir = totalAwal.Cetak dengan keterangan: biaya yang harus dibayar tanpa diskon adalah totalAwal; Besarnya diskon yang didapat adalah besarDiskon; Biaya yang harus dibayar setelah dikurangi dengan besar diskon adalah totalAkhir.Program selesai.

Kode Program C++:

#include <iostream>

using namespace std;

int main() {

   int jumlahMangga;

   int hargaMangga, totalAwal, besarDiskon = 0, totalAkhir;

   hargaMangga = 1100;

   

   cout << "Berapa buah anda mau membeli mangga golek?: " << endl;

   cin >> jumlahMangga;

   

   totalAwal = jumlahMangga * hargaMangga;

   

   if(totalAwal > 100000) {

       besarDiskon = totalAwal * 10/100;

       totalAkhir = totalAwal - besarDiskon;

   } else {

       totalAkhir = totalAwal;

   }

   cout << "\nBiaya yang harus dibayar tanpa diskon: Rp. " << totalAwal << "\nBesarnya diskon yang didapat: Rp. " << besarDiskon << "\nBiaya yang harus dibayar setelah dikurangi dengan besar diskon: Rp. " << totalAkhir;

}

= 391.500
