(72x + 80y) : (x + y) = 75
72x + 80y = 75(x + y)

75x - 72x = 1200 - 1125
3x = 75

Rata rata nilai siswa kelas 8A adalah 72. Rata rata nilai 15 siswa kelas 8B adalah 80. Jika nilai kelas 8A dan 8B digabungkan, rata ratanya menjadi 75 berapakah banyak siswa kelas 8A?

Rata-rata adalah jumlah semua nilai data yang diamati dibagi dengan banyak data yang diamati.

Rumus rata-rata gabungan

[tex]\overline x_{gab} = \frac{ n_{1} .\overline x_{1}~ + ~n_{2} .\overline x_{2}}{n_{1} ~+ n_{2} }[/tex]

Pembahasan

Dketahui :

Rata-rata kelas 8A (x₁) = 72

Rata-rata kelas 8B (x₂) = 80

Banyak siswa kelas 8B (n B) = 15 siswa

Rata-rata gabungan ([tex]\overline x_{gab}[/tex]) = 75

Ditanya :

Banyak siswa kelas 8A ?

Jawab :

Cara Biasa

[tex]\overline x_{gab} = \frac{ n_{1} .\overline x_{1}~ + ~n_{2} .\overline x_{2}}{n_{1} ~+ n_{2} }[/tex]

75 = [tex]\frac{(A \times 72) + (15 \times 80)}{A + 15}[/tex]

75 = [tex]\frac{72A + 1200}{A + 15}[/tex]

75 (A + 15) = 72A + 1200

75A + 1125 = 72A + 1200

75A - 72A = 1200 - 1125

           3A = 75

             A = [tex]\frac{75}{3}[/tex]

             A = 25

Cara Cepat

Untuk cara cepat bisa dilihat pada lampiran

Jadi banyak siswa kelas 8A adalah 25 siswa

Banyak siswa kelas 8 A adalah 16 orang

Pembahasan

Untuk pembahasan kali ini, kita akan membahas bagian dari ilmu statistika yaitu terkait perhitungan rata-rata. Adapun rumus perhitungan rata-rata ialah

Rata-rata ([tex]\bar{x}[/tex]) = Jumlah data / Banyak data

Berdasarkan soal di atas, dikatakan bahwa rata-rata nilai siswa 8A adalah 75 dan rata-rata nilai 20 siswa 8B adalah 84. Apabila digabungkan rata-ratanya menjadi 80. Untuk mengetahui banyak siswa 8A, kita perlu mengetahui berapa banyak jumlah data nilai siswa 8B dengan menggunakan rumus rata-rata tadi.

Mencari jumlah data nilai siswa 8B:

Rata-rata = Jumlah data / Banyak data

84 = Jumlah data nilai siswa 8B / 20

Jumlah data nilai siswa 8B = 20 x 84 = 1680

Mencari jumlah data nilai siswa 8A

Misalkan, banyak siswa = x

Rata-rata = Jumlah data / Banyak data

75 = Jumlah data nilai siswa 8A / x

Jumlah data nilai siswa 8A = 75x

Mencari banyak siswa 8A dengan menggunakan rumus rata-rata gabungan

Rata-rata = Jumlah data / Banyak data

80 = (75x + 1680) / (20 + x)

80 (20 + x) = 75x + 1680

1600 + 80x = 75x + 1680

80x - 75x = 1680 - 1600

5x = 80

x = 16

Jawaban:

Jumlah siswa kelas 8A = a

Jumlah siswa kelas 8B = b → 15 siswa

Nilai rata - rata a () = 72

Nilai rata - rata b () = 80

Nilai rata - rata gabungan () = 75

secara umum, menentukan nilai suatu rata-rata didefenisikan sebagai hasil dari jumlah keseluruhan nilai data dibagi jumlah banyak siswa atau frekuensi data.

Langkah Pertama → Hitung Jumlah nilai siswa kelas 8b

Jumlah nilai b = banyak siswa 8B x nilai rata-rata

= b x 80

= 80b

= 80 x 15

= 1.200

Jumlah nilai a = banyak siswa 8A x nilai rata - rata

= a x 72

= 72a

Maka, menentukan banyaknya siswa kelas 8A dapat dihitung dengan cara sebagai berikut :

→ Lakukan perkalian silang

72a + 1.200 = 75 (a + 15)

72a + 1.200 = 75a + (75 x 15)

72a + 1.200 = 75a + 1.125 → Gabungkan variabel a

1.200 - 1.125 = 75a - 72a

75 = 3a

= a

25 = a → Banyak siswa kelas 8A

Jadi, Banyak siswa kelas 8A adalah 25 siswa

19. Rata-rata nilai siswa kelas 8A adalah 77. Rata-rata nilai 8 siswa kelas 8A adalah 90. Jika semua nilai digabungkan diperoleh rata-rata 79. Maka banyak siswa kelas 8A adalah....​


Diket :

Rata² 8A = 77

Rata² 8 siswa = 90

Jumlah nilai jika digabung = 79

Ditanya :

Banyak siswa kelas 8A

Jawab :

8A = 77 × a

8 siswa = 8 × 90

= 720

720 + (77 × a) = 79 × (a + 8)

720 + 77a = 79a + 632

720 - 632 = 79a - 77a

88 = 2a

44 = a

Jumlah = 44 + 8

= 52 siswa ✔️

Jadi, banyak siswa kelas 8A adalah 52 siswa.

26. Rata-rata nilai siswa kelas 8A adalah 72 rata-rata nilai 15 siswa kelas 8B adalah 80. jika nolai digabungkan rata-ratanya menjadi 75. Banyaknya siswa Kelas 8A adalah?


[Pembahasan]

Rata-rata nilai siswa kelas 8A adalah 72 rata-rata nilai 15 siswa kelas 8B adalah 80. jika nolai digabungkan rata-ratanya menjadi 75. ditanyakan Banyaknya siswa Kelas 8A.

Misalkan

Jumlah siswa kelas 8A = a
Jumlah siswa kelas 8B = b → 15 siswa

Nilai rata - rata a ([tex] X_{a} [/tex])       = 72
Nilai rata - rata b ([tex] X_{b} [/tex])       = 80
Nilai rata - rata gabungan ([tex] X_{a+b} [/tex]) = 75


secara umum, menentukan nilai suatu rata-rata didefenisikan sebagai hasil dari jumlah keseluruhan nilai data dibagi jumlah banyak siswa atau frekuensi data.
      
Langkah Pertama → Hitung Jumlah nilai siswa kelas 8b

  Jumlah nilai b = banyak siswa 8B x nilai rata-rata
                         = b x 80
                         = 80b
                         = 80 x 15
                         = 1.200
  
  Jumlah nilai a = banyak siswa 8A x nilai rata - rata
                         = a x 72
                         = 72a

Maka, menentukan banyaknya siswa kelas 8A dapat dihitung dengan cara sebagai berikut :
 
 [tex] \frac{72a + 80b}{a+b}= X_{(a+b)} [/tex]
 [tex] \frac{72a+1.200}{a+15}= 75 [/tex]    → Lakukan perkalian silang
 72a + 1.200   = 75 (a + 15)
 72a + 1.200   = 75a + (75 x 15)
 72a + 1.200   = 75a + 1.125 → Gabungkan variabel a
 1.200 - 1.125 = 75a - 72a
                  75 = 3a
                  [tex] \frac{75}{3} [/tex] = a
                  25 = a       → Banyak siswa kelas 8A

Jadi, Banyak siswa kelas 8A adalah 25 siswa
