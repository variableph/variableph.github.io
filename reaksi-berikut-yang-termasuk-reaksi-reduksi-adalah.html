Konsep

5) FeCl3 -> FeCl2  ->  (Biloks)

yang mengalami reaksi reduksi = atom Fe dari +3 menjadi +2

15. Tentukan apakah reaksi berikut termasuk reaksi oksidasi atau reaksi Reduksi?a. HPO3²- → HPO2³-pake penjelasan yaa​


[tex] \sf reaksi \: \: HPO_{3}^{2 - } \to \: HPO_{2}^{3 - } adalah[/tex]

[tex]{\boxed{\underline{ \sf\: reaksi \: reduksi}}}[/tex]

Pembahasan ☆

⇔ Reaksi Reduksi, Oksidasi, Reduktor, dan Oksidator

Pengertian✧Reduksi adalah sebuah peristiwa penurunan bilangan oksidasi di dalam sebuah reaksiReduktor adalah unsur yang mengalami naiknya bilangan oksidasi dalam suatu reaksiOksidasi adalah sebuah peristiwa peningkatan bilangan oksidasi di dalam sebuah reaksiOksidator adalah unsur yang mengalami penurunan bilangan oksidasi dalam suatu reaksi

Dari penjelasan diatas dapat diketahui bahwa penyelesaian soal tersebut sebagai berikut :

[tex]⇒ \sf \: \: biloks \: HPO_{3}^{2 - } [/tex]

Biloks H + Biloks P + Biloks O × 3 = -2

(+1) + Biloks P + (-2) × 3 = -2

(+1) + Biloks P + (-6) = -2

Biloks P - 5 = -2

Biloks P = -2 + 5

Biloks P = +3

[tex]⇒ \sf \: \: biloks \: HPO_{2}^{3 - } [/tex]

Biloks H + Biloks P + Biloks O × 2 = -3

(+1) + Biloks P + (-2) × 2 = -3

(+1) + Biloks P + (-4) = -3

Biloks P - 3 = -3

Biloks P = -3 + 3

Biloks P = 0

Kesimpulan : Dari bukti diatas dapat kita ketahui bahwa biloks P pada 2 senyawa tersebut mengalami Penurunan, dimana biloks p yang semula adalah (+3) menurun menjadi (0) maka dapat disimpulkan bahwa reaksi tersebut merupakan reaksi Reduksi.

21. analisis reaksi berikut apakah termasuk reaksi reduksi atau reaksi oksidasi​


a) MnO2 => MnO4^-

Biloks Mn pada (MnO2) = +4Biloks Mn pada (MnO4^-) = +7=> Reaksi oksidasi

b) Cr2O7^2- => Cr^3+

Biloks Cr pada (Cr2O7^2-) = +6Biloks Cr pada (Cr^3+) = +3=> Reaksi reduksi

c) NO3^- => NO2

Biloks N pada (NO3^-) = +5Biloks N pada (NO2) = +4=> Reaksi reduksi

d) SbO3^3- => SbO4^3-

Biloks Sb pada (SbO3^3-) = +3Biloks Sb pada (SbO4^3-) = +5=> Reaksi oksidasi

e) SO2 => SO4^2-

Biloks S pada (SO2) = +4Biloks S pada (SO4^2-) = +6=> Reaksi oksidasi

f) C2O4^2- => CO2

Biloks C pada (C2O4^2-) = +3Biloks C pada (CO2) = +4=> Reaksi oksidasi
