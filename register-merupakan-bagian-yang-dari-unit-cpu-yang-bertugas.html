11. Tuliskan pendapat anda, Jika pada CPU tidak terdapat register???

Setiap register dapat menyimpan satu bilangan hingga mencapai jumlah maksimum tertentu tergantung pada ukurannya. Register-register dapat dibaca dan ditulis dengan kecepatan tinggi karena berada pada CPU.

Jadi kalau tidak ada register maka komputer akan lambat atau lalot

15. 1. sebutkan 2 operasi cpu yang terjadi di dalam komputer2. apa fungsi register unit3. jelaskan tentang RAMtolong bantu jawab ya​


Jawaban

1. CU (Computer Unit) dan ALU (Arithmetic Logic Unit).

2. Fungsi register unit adalah sebagai tempat penyimpanan sementara untuk menaruh data dan instruksi.

3. RAM itu semacam memori komputer yang berfungsi untuk menyimpan data yang sedang bekerja atau kode mesin.

Pembahasan

Setelah data di-input ke dalam suatu mesin/komputer, data tersebut akan disimpan dulu dalam memori secara sementara dan dilangsungkan ke proses lalu di simpan di memori lagi lalu di keluarkan hasilnya.

Proses - memproses data dari memori

CU (Computer unit) — Berfungsi untuk membawa data ke dalam / dari memori. CU juga berfungsi untuk mengelola data dan input & output.ALU (Arithmetic Logic Unit) — berfungsi untuk melakukan fungsu matematika.Jadi di dalam proses ini, data akan diambil dari memori oleh CU, lalu oleh CU data tsb akan di kelola dan dipisahkan, jika ada data yang memiliki matematika maka akan dibawa ke ALU untuk di proses kalau hanya data sederhana, akan di proses oleh CU. Setelah datanya sudah di proses oleh CU dan ALU data tersebut akan dikumpulkan ke CU dulu untuk dikirim ke memori lagi, baru dikeluarkan hasilnya.

20. Tiga Fungsi dari CPU adalah sebagai Berikut,kecualiA. ALUB. CU (Control Unit)C. RegisterD. Motherboard​


Jawaban:

D. motherboard

Penjelasan:

Fungsi CPU Pada Perangkat Komputer

Beberapa ulasan di atas dalam perangkat CPU maka di sini akan kami sampikan beberapa fungsi dari cpu adalah sebagai berikut :

A. Fungsi Storing

Fungsi Storing dapat menyimpan data dari CPU sehinga seseorang yang akan menggunakan komputer dapat membutuhkan perangkat untuk menyimpannya secara permanen.

B. Fungsi Decoding

Fungsi Decoding adalah sistem CPU dengan program yang dapat dijalankan dengan kode instruksi sehingga dapat diterjemahkan pada CPU yang dapat mengenalinya pada kode biner.

Dengan demikian decoding sebuah proses yang dapat menerjemahkan suatu program ke dalam bahasa yang dimengerti oleh CPU sehingga dapat juga dilakukan dalam semua program.

C. Fungsi Executing

Fungsi Executing merupakan salah satu program yang dieksekusi oleh CPU yang dapat melakukan sesuatu sehinga perhitungan dengan menggunakan komponen CPU.

Maka Fungsi Executing disebut sebagai ALU atau Arithmetical Logical Unit yang bertanggung jawab untuk semua proses komputasi logis di pada komputer dan perangkat yang lainnya.

D. Fungsi Fetching

Fungsi Fetching adalah istilah yang dapat digunakan dalam komputasi sebagai mengambil atau memanggil data sehingga perangkat komputasi data dapat disimpan dengan alamat pada hard disk CPU.

Maka Fungsi Fetching Mirip dengan data dalam prosesor dalam CPU yang diatur pada kumpulan alamat program yang dijalankan pada CPU dengan mengambil data dalam komponen.

sementara Motherboard (juga sering disebut Mainboard atau MoBo) adalah Papan Sirkuit Cetak (Printed Circuit Board, disingkat PCB) yang memuat komponen-komponen utama komputer atau perangkat lainnya. Dengan konektor, papan sirkuit lainnya (daughterboard seperti VGA Card) dapat diselipkan/dicolokkan.

28. Pengertian dari • Instuction Register • Program Counter • General Purpose Register • Memory Address Register • Memory Data Register pada proses register CPU


1.  program counter = register yang menunjukan ke instruksi berikutnya yg harus di ambil dan di jalankan.
2. instuction register   = registrasi yang menyimpan instruksi yg sedang di jalankan.
