9. Untuk membuat drop-down list menggunakan tag? .


Untuk membuat drop-down list menggunakan HTML, Anda dapat menggunakan tag <select> dan tag <option>. Berikut adalah contoh sederhananya:

<select>

 <option value="option1">Option 1</option>

 <option value="option2">Option 2</option>

 <option value="option3">Option 3</option>

</select>

Tag <select> digunakan untuk membuat drop-down list, sedangkan tag <option> digunakan untuk menentukan pilihan yang tersedia dalam drop-down list. Setiap <option> harus diatur dalam tag <select>. Nilai atribut value pada tag <option> akan menentukan nilai yang akan dikirimkan ketika pilihan tersebut dipilih.

Anda juga dapat menambahkan atribut name pada tag <select> untuk memberikan nama pada drop-down list yang dikirimkan ke server ketika form di-submit. Selain itu, Anda juga dapat menambahkan atribut multiple pada tag <select> untuk memungkinkan pengguna memilih lebih dari satu pilihan pada drop-down list.

Contoh lengkapnya adalah sebagai berikut:

<form action="/submit-form">

 <label for="favorite-color">Pilih warna favorit Anda:</label>

 <select name="favorite-color" id="favorite-color">

   <option value="red">Merah</option>

   <option value="green">Hijau</option>

   <option value="blue">Biru</option>

 </select>

 <br>

 <input type="submit" value="Submit">

</form>

Di sini, drop-down list akan menampilkan pilihan "Merah", "Hijau", dan "Biru", dan ketika form di-submit, nilai pilihan yang dipilih akan dikirim ke server bersama dengan nama "favorite-color".
