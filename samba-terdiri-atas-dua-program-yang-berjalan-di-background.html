Program firewall yang berjalan pada background komputer disebut Jawaban Pendahuluan

Program firewall yang berjalan pada background komputer disebut dengan software firewall. Firewal  sendiri merupakan aplikasi yang memiliki fungsi untuk melindungi jaringan komputer dari berbagai macam serangan.

Pembahasan

Setelah mengetahui pengertian dari software firewall dan fungsi dari firewall, maka selanjutnya kita akan membahas lebih lanjut mengenai firewall. Firewall seperti yang telah kita bahas sebelumnya memiliki fungsi untuk melindungi jaringan komputer kita. Selain dalam bentuk software, firewall juga bisa kita temui dalam bentuk hardware. Firewall dalam bentuk hardware dapat digunakan sebagai perangkat tambahan dari modem ke dalam jaringan komputer kita. Untuk komputer yang menggunakan sistem operasi windows, kita dapat melakukan pengaturan firewall kita menggunakan menu control panel, kemudian pilih Windows Firewall.

Jawaban:

Cara Menghapus Latar Gambar di GIMP

Langkah 1: Buka gambar di GIMP yang latar belakangnya ingin Anda hapus menggunakan opsi File> Open.

Tip: Gunakan pintasan Ctrl + O untuk meluncurkan kotak dialog gambar terbuka di Windows dan Command + O di macOS.

Langkah 2: Kemudian, di kotak layer, klik kanan pada gambar dan pilih Add Layer Mask dari menu. Atau, masuk ke Layer> Mask> Add Layer Mask di bar bagian atas untuk menambahkannya.

Langkah 3: Di kotak dialog layer mask yang muncul, centang opsi White (opacity penuh).

Anda akan melihat bahwa kotak kecil yang diisi dengan warna putih muncul di sebelah thumbnail gambar Anda di kotak layer.

Langkah 4: Sekarang kita telah mengisi layer mask dengan warna putih, kita perlu menggunakan warna hitam untuk menghapus latar belakang. Untuk itu, pertahankan warna foreground sebagai hitam di kotak warna di sisi kanan.

Tip : Tekan D untuk masing-masing membuat hitam sebagai latar depan dan putih sebagai warna latar belakang.

Langkah 5: Klik pada alat kuas untuk mengaktifkannya. Kemudian warnai area yang ingin Anda hapus. Anda akan melihat bahwa membelai kuas pada gambar menghilangkan area. Terapkan sapuan kuas pada wilayah yang ingin Anda jadikan transparan.

Penjelasan:

Samba Server adalah program yang dapat menjembatani kompleksitas berbagai platform sistem operasi Linux (UNIX) dengan mesin Windows yang dijalankan dalam suatu jaringan komputer. Samba Server memungkinkan komputer dengan sistem operasi Windows untuk berbagi file dan printer dengan komputer dengan sistem operasi Linux (UNIX) atau sebaliknya. 14. untuk menjalankan atau menghentikan daemon samba, smbd dan nmbd dapat diatur secara otomatis melalui skrip


/etc/rc.d/init.d/smbUntuk menjalankan atau menghentikan daemon samba, smbd dan nmbd dapat diatur secara otomatis dengan menambahkan skrip berikut di /etc/rc.local:

menjalankan:
/etc/init.d/samba start
/etc/init.d/smbd start
/etc/init.d/nmbd start

menghentikan:
/etc/init.d/samba stop
/etc/init.d/smbd stop
/etc/init.d/nmbd stop

15. agar server samba selalu dijalankan setiap kali linux booting dapat menambah perintah baris?


Agar Samba selalu dijalankan setiap kali Linux melakukan booting, kita dapat menambahkan baris-baris di atas pada file /etc/rc.d/rc.local :

nano /etc/rc.d/rc.local
/usr/local/samba/bin/smbd -D
/usr/local/samba/bin/nmbd -D
