3. Sebuah transformator step down terdiri atas kumparan primer Dengan 1200lilitan dan kumparan sekunder 40lilitan. Jika kumparan primer dihubungkan dengan tegangan sebesar 330volt,berapa tegangan pada kumparan sekunder?


Tegangan yang dikeluarkan adalah 11 Volt.

Pembahasan

Diketahui :

Np (lilitan di kumparan primer) = 1200 lilitan

Ns (lilitan di kumparan sekunder) = 40 lilitan

Vp (tegagan di kumparan primer)= 330 Volt

Ditanyakan :

Vs = ?

Jawaban:

Transformator  step down adalah alat yang dihubungkan dengan perangkat elektronik dan sifatnya untuk menurunkan arus listrik.

Rumus yang dipakai adalah:

[tex]\frac{Vp}{Vs} = \frac{Np}{Ns}[/tex]

Keterangan:

Vp = Tegagan di kumparan prime

Np = Lilitan di kumparan primer

Ns = Lilitan di kumparan sekunder

Vs = Tegangan di kumparan sekunder

[tex]\frac{Vp}{Vs} = \frac{Np}{Ns}[/tex]

[tex]\frac{330}{Vs} = \frac{1200}{40} \\\\1200. Vs = 40 \times 330\\Vs = \frac{13200}{1200}\\Vs = 11 Volt.[/tex]

Maka tegagan kumparan sekundernya adalah 11 Volt.

15. sebuah transformator step down terdiri atas kumparan primer dengan 1200 lilitan dan kumparan sekunder dengan 40 lilitan jika kumparan primer dihubungkan dengan tegangan sebesar 330 volt berapa tegangan pada kumparan sekunder


Transformator ( Trafo )

alat untuk mengubah tegangan arus bolak balik menjadi lebih rendah atau lebih tinggi.  

ada dua jenis trafo menurut tegangan:

- trafo step-up

 Np < Ns

 Vp < Vs

 Is < Ip

- trafo step-down

 Np > Ns

 Vp > Vs

 Is > Ip

ada dua jenis trafo menurut effsiensinya :

- trafo tidak ideal :

 ηVp / Vs = ηNp / Ns = Is / Ip  

- trafo ideal :

 Vp / Vs  =  Np / Ns = Is / Ip

dimana :

Vs = tegangan sekunder ( Volt )

Vp = tegangan primer ( Volt )

Ns = jumlah lilitan sekunder

Np = jumlah lilitan primer

_________________________________________________

diketahui:

Np = 1.200 lilitan

Ns = 40 lilitan

Vp = 330 v

ditanya:

Vs

jawab:

Vp / Vs  =  Np / Ns

330 / Vs =  1.200 / 40

        Vs =  330 x ( 1.200 / 40 )

             =  9.900 v

Jadi tegangan sekunder pada trafo sebesar 9.900 volt
