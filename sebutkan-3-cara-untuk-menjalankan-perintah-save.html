25. jelaskan dua cara untuk menjalankan perintah save


Tekan CTRL dan S lalu pilih tempat untuk menaro file tsb lalu save atau tekan simbol di ujung kiri monitor lalu kalik perintah save to pilih tempat file untuk menyimpan nya lalu saveCara yang pertama:
1. Untuk Microsoft 2003 dan 2010, klik menu File yang terdapat di Kiri atas lembar kerja. Sementara untuk Microsoft 2007, klik ikon kotak di pojok kiri atas lembar kerja.
2. Kemudian pilih menu Save atau Save As dipilihan tersebut.
3. Setelah itu, ketik nama untuk file tersebut. Jika telah selesai diketik, klik menu Save.

Cara yang kedua :
Klik Ctrl dan Huruf S secara bersamaan. Setelah muncul jendela Save as, Ketik namamu di kotak nama yang tersedia di jendela Save As. Jika telah selesai, klik menu Save.

Tambahan dari saya :
Save berguna untuk menyimpan file yang telah tersimpan sebelumnya, atau Menyimpan file yang telah diubah dan disimpan kembali.
Save As berguna untuk menyimpan kembali file yang telah tersimpan dengan file yang baru.
