4. Berdasarkan fungsinya, perangkat keras komputer dapat dikelompokkan kedalam ... kategori


Penjelasan:

Berdasarkan fungsinya, peangkat keras komputer dibagi menjadi 5 yaitu: 1. Input divice yaitu alat untuk memasukan data. 2. Process device yaitu unit atau alat Pemrosesan data. 3. Output device yaitu unit atau alat untuk keluaran data. 4. Backing Storage Unit yaitu unit penyimpanan. 5. Periferal yaitu unit tambahan. Pembahasan Perangkat keras komputer adalah alat atau perangkat yang memiliki wujud, dapat dilihat dan diraba secara langsung. Berdasarkan fungsinya, perangakt keras komuter dibagi menjadi: 1. Input divice yaitu alat untuk memasukan data, yaitu keyboar, mouse, touch screen, light pen, digitizer graphics tablet, scanner, webcam dan lainnya. 2. Process device yaitu unit atau alat untuk memproses data yang telah dimasukkan input device, yaitu CPU, VGA , RAM. 3. Output deviceuntuk memproses dala yang teian dimasukkan input device, yaitu CPU, VGA , RAM. 3. Output device yaitu unit atau alat untuk mengeluarkan atau menampilkan hasil data yang telah diproses, yaitu monitor, printer, proyektor dan speaker. 4. Backing Storage Unit yaitu alat tempat menyimpan data pada komputer, yaitu DVD, hardisk, USB Flash Drive. 5. Periferal yaitu unit tambahan pada komputer, seperti kabel LAN untuk jaringan, modem.
