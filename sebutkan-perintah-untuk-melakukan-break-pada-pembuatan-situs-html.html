15. Perintah html untuk menentukan, tebal garis table adalah


Hypertext Markup Language atau yang lebih sering dikenal dengan HTML adalah bahasa sederhana yang dipergunakan di dalam pembuatan halaman suatu website. Dalam operasional penggunaan html, perintah html yang digunakan untuk menentukan ketebalan garis table adalah Table Border.

Pembahasan

Di era digital saat ini, pembuatan situs web dan aplikasi seluler telah menjadi pusat tren untuk  meningkatkan promosi dan meluncurkan produk dan layanan secara efektif. Oleh karena itu, bekerja sebagai programmer adalah pilihan yang paling diinginkan di industri ini. Pengembang web juga harus mahir dalam bahasa markup khusus untuk membantu  struktur halaman situs web, desain HTML.

HTML atau bahasa markup hypertext adalah bahasa markup yang digunakan untuk menyusun halaman web untuk ditampilkan di browser web. Oleh karena itu, HTML dapat digunakan sebagai landasan pertama untuk membangun kerangka website terstruktur sebelum membahas  tampilan desain dan aspek fungsional.

Atribut border, cellpadding, dan cellpacing digunakan untuk mengubah tampilan tabel, terutama yang berkaitan dengan batas dan spasi antar sel tabel. Atribut border digunakan untuk mengatur ketebalan border tabel. Jika atribut ini tidak ditentukan, browser web  akan menampilkan tabel tanpa bingkai. Nilai  atribut ini adalah angka yang diukur dalam piksel. Jika Anda menentukan batas nilai = "2", browser web  menampilkan batas 2 piksel di bagian atas, bawah, kiri, dan kanan tabel.

25. Perintah untuk menyimpan dokumen slide ke dalam bentuk HTML adalah


Perintah untuk menyimpan dokumen slide (power point) ke dalam bentuk HTML adalah dengan cara menggunakan perintah save as yang terdapat di microsoft power point.  Caranya adalah dengan mengeklik office button yang berada di pojok kiri atas, kemudian arahkan kursos ke dalam tab "Save As", kemudian pilih "Other Format" Nah di dalam dialog tersebut, silahkan kamu pilih tipe file Web Page. Secara otomatis file slide atau power point (ppt) kamu akan berubah menjadi file dengan format HTML dan bisa dibuka dengan menggunakan browser seperti Google Chrome, Firefox, Safari, dan sebagainya.

Pembahasan

format HTML atau kepanjangan dari Hypertext Markup Language merupakan format dokumen multimedia pada tampilan website yang dapat dibuka pada suatu browser. Dengan mengkonversi file power point menjadi format html, maka power point atau slide tersebut hanya bisa dibuka di Browser saja, alisas tidak bisa dibuka di software pengolah presentasi tersebut.

29. sebutkan perintah dari HTML/code yang kamu ketahui beserta fungsinya

fungsi  : untuk membuat (memilih) ukuran teks.

Sintak  : <H1>             </H1>

Contoh

<HTML>

<HEAD><TITLE>Contoh Heading</TITLE></HEAD>

<BODY>

<H2> Teks dengan Heading 2</H2>

<H3> Teks dengan Heading 3</H3>

<H4> Teks dengan Heading 4</H4>

<H5> Teks dengan Heading 5</H5>

<H6> Teks dengan Heading 6</H6>

</BODY>

</HTML>


<HR>

Perintah ini digunakan untuk membuat garis horizontal penuh layar
