1. sederhanakan dlm bentuk pangkat positif b pangkat -6 per b pangkat -3


Penyelesaian :
Ditanya :
Sederhanakan dalam bentuk pangkat positif
b^(-6) / b^(-3)

Dijawab :
Bentuk Umum Pangkat :
a^b / a^c = a^(b-c)
a^(-b) = 1/a^b
1/a^(-b) = a^b

b^(-6) / b^(-3) →
= b^{-6-(-3)}
= b^(-6+3)
= b^(-3)
= 1/b³

10. Sederhanakan dalam bentuk pangkat positif. A.2m^-4 × m^-3 B. 6^7/6^3 C.b^-3/b^-3 D.1/a^3bc^-4


Pembahasan

Sederhanakanlah dalam bentuk pangkat positif.

a]  2m⁻⁴ x m⁻³

b] [tex]\frac{6^{7}}{6^{3}}[/tex]

c] [tex]\frac{b^{-3}}{b^{-3}}[/tex]

Pada soal bagian d] ada dua kemungkinan dalam penulisan :

d] [tex]\frac{1}{a^{3}bc^{-4}}\text{ atau }\frac{1}{a^{3}}bc^{-4}[/tex]


Dalam menyelesaikan bentuk perpangkatan dari persoalan diatas, perlu untuk menyimak beberapa hal dari sifat pangkat berikut ini.

[tex]\boxed{\boxed{\begin{array}{l}\bold{I) \ \ \ a^{m} \ x \ a^{n}= a^{(m+n)}}\\\\ \bold{II) \ \ a^{m} \ : \ a^{n}=\frac{a^{m}}{a^{n}}=a^{(m-n)}}\\\\\bold{III) \ a^{-m}= \frac{1}{a^{m}}}\\\\\bold{IV) \ \frac{1}{a^{-m}}=a^{m}} \end{array}}}[/tex]


Penyelesaian Soal Bagian a]

2m⁻⁴ x m⁻³ ⇒ 2m⁽⁻⁴⁺⁽⁻³⁾⁾  → Sifat I

2m⁻⁴ x m⁻³ = 2m⁽⁻⁴⁻³⁾

                  = 2m⁻⁷           → Sifat III

                  = [tex]\frac{2}{m^{7}}[/tex]

[tex]\boxed{Jawaban : \ 2m^{-4} \ x \ m^{-3}=\frac{2}{m^{7}} }[/tex]


Penyelesaian Soal Bagian b]

[tex]\frac{6^{7}}{6^{3}}[/tex] ⇒ 6⁽⁷⁻³⁾  → Sifat II

        = 6⁴

[tex]Jawaban : \boxed{\frac{6^{7}}{6^{3}}=6^{4}}[/tex]


Penyelesaian Soal Bagian c]

[tex]\frac{b^{-3}}{b^{-3}}[/tex] ⇒ b⁻³ x b³ → Sifat II-IV

      = b⁽⁻³⁺³⁾

      = b⁰ → Ingat bahwa apapun Bilangan yang berpangkat 0 hasilnya tetap 1.

      = 1

Jawaban : [tex]\boxed{\frac{b^{-3}}{b^{-3}}\ = 1}[/tex]


Penyelesaian Soal Bagian d]

Kemungkinan Pertama : [tex]\frac{1}{a^{3}bc^{-4}}[/tex]

Dari Pernyataan yang ada pada soal "Mengubah dalam bentuk pangkat positif" maka perhatikan tiap variabel yang ada.

variabel a → berpangkat 3 (Positif)

              b → berpangkat 1 (positif)

              c → berpangkat -4 (negatif)

Maka, yang perlu diubah adalah bentuk variabel c menjadi berpangkat positif.

[tex]\frac{1}{a^{3}bc^{-4}}[/tex] ⇒ [tex]\boxed{\frac{1}{a^{3}bc^{-4}}\ = \ \frac{c^{4}}{a^{3}b} }[/tex] → Simak Sifat IV

            = [tex]\frac{1}{a^{3}b}[/tex] x c⁴

            = [tex]\frac{1 \ x \ c^{4}}{a^{3}b}[/tex]

            = [tex]\frac{c^{4}}{a^{3}b}[/tex]

Jawaban : [tex]\boxed{\frac{1}{a^{3}bc^{-4}}\ = \frac{c^{4}}{a^{3}b}}[/tex]


Kemungkinan kedua : [tex]\frac{1}{a^{3}}bc^{-4}[/tex]

variabel a → berpangkat 3 (positif)

             b → berpangkat 1 (positif)

             c → berpangkat -4 (negatif)

[tex]\frac{1}{a^{3}}bc^{-4}[/tex] ⇒ [tex]\frac{1 \ x \ b}{a^{3}}\ x \ \frac{1}{c^{4}}[/tex]  → Sifat III

                = [tex]\frac{b}{a^{3}c^{4}}[/tex]

Jawaban : [tex]\boxed{\frac{1}{a^{3}}bc^{-4}= \ \frac{b}{a^{3}c^{4}}}[/tex]

Pembahasan :

Sifat - sifat eksponensial

Misalkan a € R, b € R m dan n bilangan bulat positif, berlaku sifat - sifat berikut

[tex]1.) \: {a}^{m} \times {a}^{n} = {a}^{m + n} \\ \\ 2.) \: {a}^{m} \div {a}^{n} = {a}^{m - n } \\ \\ 3.) \: ({ {a}^{n} })^{m} = {a}^{m \times n} \\ \\ 4.) \: {(ab)}^{m} = {a}^{m} {b}^{m} \\ \\ 5.) \: {a}^{0} = 1 \\ \\ 6.) \: {a}^{ - m} = \frac{1}{ {a}^{m} } \\ \\ 7.) \: \sqrt[m]{ {a}^{n} } = {a}^{ \frac{n}{m} } [/tex]

Penyelesaian :

Bagian A

▪ Sifat dari 1

[tex] {2 m}^{ - 4} \times {m}^{ - 3} \\ \\ = 2 \times {m}^{ - 4 + ( - 3)} \\ \\ = 2 {m}^{ - 7} \\ \\ = \frac{2}{ {m}^{7} } [/tex]

Bagian B

▪ Sifat dari 2

[tex] \frac{ {6}^{7} }{ {6}^{3} } \\ \\ = {6}^{7 - 3} \\ \\ = {6}^{4} [/tex]

Bagian C

▪ Sifat dari 2 dan 5

[tex] \frac{ {b}^{ - 3} }{ {b}^{ - 3} } \\ \\ = {b}^{ - 3 - ( - 3)} \\ \\ = {b}^{0} \\ \\ = 1 [/tex]

Bagian D

▪ Sifat dari 1 dan 6
[tex] \frac{1}{ {a}^{3} } b {c}^{ - 4} \\ \\ = \frac{1}{ {a}^{3} } \times b \times \frac{1}{ {c}^{4} } \\ \\ = \frac{b}{ {a}^{3} {c}^{4} } [/tex]

[tex] a^{-n} = \frac{1}{a^n} [/tex]

[tex] \sqrt[n]{ {a}^{m} }  =  {a}^{ \frac{m}{n} } [/tex]

[tex] a^0 = 1 [/tex]

25. Ubah ke bentuk pangkat positif jika perlu sederhanakan a. 5-² b. 2a-¹b² per c³ c.6×-³ per 3×y-¹z-²


Bentuk pangkat positif dari bentuk pangkat negatif seperti yang tersebut pada soal adalah :

a. [tex]5^{-2}=\frac{1}{25}[/tex]

b. [tex]\frac{2a^{-1}b^{2}}{c^{3}}=\frac{2b^{2}}{ac^{3}}[/tex]

c. [tex]\frac{6x^{-3}}{3xy^{-1}z^{-2}}=\frac{2yz^{2}}{x^{4}}[/tex]

PembahasanMengubah bentuk pangkat negatif ke bentuk pangkat positif.

Bentuk pangkat merupakan bentuk lain dari perkalian berulang pada angka yang sama. Besarnya nilai pangkat menunjukkan banyaknya perkalian.

Contoh :

3³ adalah bentuk lain dari 3 × 3 × 3

a⁵ adalah bentuk lain dari a × a × a × a × a

Konsep di atas berlaku secara umum untuk bilangan berpangkat positif. Nah, bagaimana dengan bentuk pangkat negatif dan pangkat nol?

3⁻³ tentu tidak bisa dikatakan bentuk lain dari -3 × (-3) × (-3)

Untuk itu bilangan berpangkat negatif adalah bentuk bilangan berpangkat khusus yang tidak sama cara menghitung hasilnya dengan bilangan berpangkat positif.

Pada dasarnya bilangan berpangkat negatif adalah bentuk lain dari bentuk pecahan.

Contoh :

[tex]2^{-1}[/tex] adalah bentuk lain dari [tex]\frac{1}{2}[/tex]

[tex]x^{-7}[/tex] adalah bentuk lain dari [tex]\frac{1}{x^{7}}[/tex]

Dan dalam kesempatan kali ini kita akan membahas soal tentang cara mengubah bentuk pangkat negatif ke bentuk pangkat positif.

Diketahui :

a. [tex]5^{-2}[/tex]

b. [tex]\frac{2a^{-1}b^{2}}{c^{3}}[/tex]

c. [tex]\frac{6x^{-3}}{3xy^{-1}z^{-2}}[/tex]

Ditanya :

Bentuk pangkat positifnya dan jika perlu disederhanakan.

Penyelesaian :

a. [tex]5^{-2}[/tex]

   [tex]=\frac{1}{5^{2}}[/tex]

   [tex]=\frac{1}{25}[/tex]

b. [tex]\frac{2a^{-1}b^{2}}{c^{3}}[/tex]

   [tex]=\frac{2b^{2}}{c^{3}a}[/tex]

   [tex]=\frac{2b^{2}}{ac^{3}}[/tex]

c. [tex]\frac{6x^{-3}}{3xy^{-1}z^{-2}}[/tex]

   [tex]=\frac{2x^{-3}}{xy^{-1}z^{-2}}[/tex]

   [tex]=\frac{2}{x^{4}y^{-1}z^{-2}}[/tex]

   [tex]=\frac{2y}{x^{4}z^{-2}}[/tex]

   [tex]=\frac{2yz^{2}}{x^{4}}[/tex]

Demikian penjelasan dan penyelesaian soal di atas.
