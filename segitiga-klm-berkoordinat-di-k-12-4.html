11. Segitiga klm dengan k(6,4),l(−3,1),m(2,−2) didilatasi dengan pusat (−2,3) dan faktor skala 4. koordinat bayangan △klm adalah ⋯⋅


Koordinat bayangan segitiga klm didilatasi dengan pusat (-2, 3) dan faktor skala  4 adalah K' (30,7), L' (-6, -5), M'(14, -17).

Penjelasan dengan langkah-langkah

Rumus dilatas

x' = a + k(x - a)

y' = b + k(y - b)

Keterangan :

a, b = titik pusat dilatasi

x, y = titik yang didilatasi

k = faktor skala

Diketahui :

K (6,4)L (-3, 1)M (2, -2)didilatasi dengan pusat (-2, 3)Faktor skala 4

Ditanya : koodinat bayangan KLM

Jawab :

Titik K (6,4)

x' = a + k(x - a)

x' = -2 + 4(6 - (-2))

x' = -2 + 4(8)

x' = -2 + 32

x' = 30

y ' = b + k(y - b)

y' = 3 + 4(4 - 3)

y' = 3 + 4(1)

y' = 7

Jadi, K' (30,7)

Titik L (-3, 1)

x' = a + k(x - a)

x' = -2 + 4(-3-(-2))

x' = -2 + 4(-1)

x' = -2 - 4

x' = -6

y ' = b + k(y - b)

y' = 3 + 4(1 - 3)

y' = 3 + 4 (-2)

y' = 3 - 8

y' = -5

Jadi, titik L' (-6, -5)

Titik M(2, -2)

x' = a + k(x - a)

x' = -2 + 4(2 - (-2))

x' = -2 + 4 (4)

x' = -2 + 16

x' = 14

y ' = b + k(y - b)

y' = 3 + 4(-2 - 3)

y' = 3 + 4 (-5)

y' = 3 - 20

y' = -17

Jadi, M'(14, -17)

Sehingga koordinat bayangan segitiga KLM adalah K' (30,7), L' (-6, -5), M'(14, -17).

23. Diketahui segitiga KLM, jika KM = 12 cm, LM = 10 cm, dan m< K =30° maka luas segitiga KLM adalah ...cm²​


Jawaban:

Untuk menyelesaikan masalah ini, kita dapat menggunakan rumus luas segitiga:

Luas segitiga KLM = 1/2 x KL x LM x sin(K)

Namun, untuk menggunakan rumus ini, kita perlu mengetahui panjang sisi KL dan besar sudut K.

Berikut langkah-langkah untuk menyelesaikan masalah ini:

Hitung panjang sisi KL menggunakan teorema kosinus.

Diketahui:

KM = 12 cm

LM = 10 cm

Sudut K = 30 derajat

Maka, dapat kita hitung sisi KL dengan teorema kosinus:

KL² = KM² + LM² - 2 x KM x LM x cos(K)

KL² = 12² + 10² - 2 x 12 x 10 x cos(30)

KL² = 244

KL = √244 = 2√61

Jadi, sisi KL memiliki panjang sebesar 2√61 cm.

Hitung luas segitiga KLM menggunakan rumus luas segitiga.

Luas segitiga KLM = 1/2 x KL x LM x sin(K)

Luas segitiga KLM = 1/2 x 2√61 x 10 x sin(30)

Luas segitiga KLM = √61 x 5

Jadi, luas segitiga KLM adalah sebesar √61 x 5 satuan luas. Kita juga dapat memperkirakan nilai numerik dari luas ini, yaitu sekitar 22,82 satuan luas.

27. Segitiga KLM direfleksikan terhadap garis x = 0 (sumbu y) menghasilkan segitiga K'L'M' dengan titik koordinat bayangan K'(-1, 1), L'(-1, 4), dan M'(6, 4). Titik koordinat KLM yang tepat adalah..


Jawab:

K(0,1), L(-1,4), dan M(12,4)

Penjelasan dengan langkah-langkah:

Karena segitiga KLM direfleksikan terhadap garis x=0, maka titik K akan dipantulkan ke sebelah kiri sumbu y sehingga menjadi K'(-1,1). Kemudian titik L dan M dipantulkan juga terhadap sumbu y sehingga menjadi L'(-1, 4) dan M'(6, 4).

Kita dapat menghitung pergeseran pada sumbu x dengan cara mengambil selisih koordinat titik pada gambar asli dan gambar bayangan, yaitu:

pergeseran x untuk titik K: -1 - x_K = -1, sehingga x_K = 0

pergeseran x untuk titik L: -1 - x_L = -2, sehingga x_L = -1

pergeseran x untuk titik M: 6 - x_M = -6, sehingga x_M = 12

Jadi, titik koordinat KLM yang tepat adalah K(0,1), L(-1,4), dan M(12,4).
