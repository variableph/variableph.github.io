8. setarakan persamaan reaksi berikut​


Jawaban

a. 2NaOH + H₂SO₄ ==> Na₂SO₄ + 2H₂O

b. 3Ca(OH)₂ + 2H₃PO₄ ==> Ca₃(PO₄)₂ + 6H₂O

c. N₂ + 3H₂ ==> 2NH₃

d. Fe₂O₃ + 6HCl ==> 2FeCl₃ + 3H₂O

e. CuCl₂ + 2NaOH ==> Cu(OH)₂ + 2NaCl

f. MnO₂ + 4HCl ==> MnCl2 + 2H₂O + Cl₂

g. Fe₂O₃ + 3CO ==> 2Fe + 3CO₂

h. (NH₄)₂SO₄ + 2NaOH ==> Na₂SO₄ + 2H₂O + 2NH₃

i. 3Ba(OH)₂ + P₂O₅ ==> Ba₃(PO₄)₂ + 3H₂O

j. 4NH₃ + 5O₂ ==> 4NO + 6H₂O

Penjelasan

Untuk menyetarakan persamaan reaksi, maka :

1. Hitung atom "X" pada ruas kanan dan ruas kiri. Apabila sudah sama, cek atom "Y" dan seterusnya.

2. Jika jumlah atom "X" belum sama antar dua ruas, maka senyawa/molekul yang menyertainya perlu disesuaikan agar terpenuhi => "atom X di ruas kanan = atom X di ruas kiri"

Untuk pembahasan fase molekul/senyawa tidak dicantumkan untuk mempercepat pengerjaan pada kolom jawaban, tetapi untuk penulisan fase pada lembar jawab silakan diikuti dari subscript di soal (semial aq/l/s/g)

Huruf a

2NaOH + H₂SO₄ ==> Na₂SO₄ + 2H₂O

atom Na = 2 (NaOH dikali 2)

atom S = 1

atom H = 4 (H2O dikali 2)

atom O = 6

Huruf b

3Ca(OH)₂ + 2H3PO₄ ==> Ca3(PO₄)₂ + 6H₂O

atom Ca = 3 (Ca(OH)2 dikali 3)

atom P = 2 (H3PO4 dikali 2)

atom H = 12 (H2O dikali 6)

atom O = 14

Huruf c

N₂ + 3H₂ ==> 2NH₃

atom N = 2 (NH3 dikali 2)

atom H = 6 (H2 dikali 3)

Huruf d

Fe₂O₃ + 6HCl ==> 2FeCl₃ + 3H₂O

atom Fe = 2 (FeCl3 dikali 2)

atom Cl = 6 (HCl dikali 6)

atom H = 6 (H2O dikali 3)

atom O = 3

Huruf e

CuCl₂ + 2NaOH ==> Cu(OH)₂ + 2NaCl

atom Cl = 1 (NaCl dikali 2)

atom Na = 1 (NaOH dikali 2)

atom Cu = 1

atom H = 2

atom O = 2

Huruf f

MnO₂ + 4HCl ==> MnCl₂ + 2H₂O + Cl₂

atom Mn = 1

atom Cl = 4 (HCl dikali 4)

atom H = 4 (H2O dikali 2)

atom O = 2

Huruf g

Fe₂O₃ + 3CO ==> 2Fe + 3CO₂

atom O = 6 (CO2 dikali 3 dan CO dikali 3)

atom Fe = 2 (Fe dikali 2)

atom C = 3

Huruf h

(NH₄)₂SO₄ + 2NaOH ==> Na₂SO₄ + 2H₂O + 2NH₃

atom N = 2 (NH3 dikali 2)

atom Na = 2 (NaOH dikali 2)

atom S = 1

atom H = 10 (H2O dikali 2)

atom O = 6

Huruf i

3Ba(OH)₂ + P₂O₅ ==> Ba₃(PO₄)₂ + 3H₂O

atom Ba = 3 (Ba(OH)2 dikali 3)

atom P = 2

atom H = 6 (H2O dikali 3)

atom O = 11

Huruf j

4NH₃ + 5O₂ ==> 4NO + 6H₂O

atom H = 12 (NH3 dikali 4 dan H2O dikali 6)

atom N = 4 (NO dikali 4)

atom O = 10

23. setarakan persamaan reaksi berikut​


aFe2O3 + bH2SO4 -> cFe2(SO4)3 + dH2O

koefisien :
Fe : 2a = 2c
O : 3a + 4b = 12c + d
H : 2b = 2d
S : b = 3c

misal a = 1,
maka b = 3, c = 1, d = 3

jadi, persamaan reaksi tsb menjadi :
Fe2O3 + 3H2SO4 -> Fe2(SO4)3 + 3H2O
