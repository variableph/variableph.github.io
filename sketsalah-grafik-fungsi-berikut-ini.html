11. tentukan fungsi eksponen dari sketsa grafik berikut ​


Tentukan fungsi eksponen dari sketsa grafik berikut

a. f(x) = 4ˣ b. f(x) = 6⁻ˣ

Suatu grafik fungsi eksponen jika memotong sumbu y di titik (0, 1) maka bentuk fungsinya adalah

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Pembahasan

Grafik pada gambar a

Memotong sumbu y di (0, 1) dan melalui titik (2, 16)

Karena memotong sumbu y di (0, 1) maka bentuk fungsinya adalah  

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Melalui titik (2, 16) artinya

f(2) = 16

a² = 16

a² = 4²

a = 4

Jadi fungsi eksponen untuk grafik pada gambar a adalah

f(x) = 4ˣ

Grafik pada gambar b

Memotong sumbu y di (0, 1) dan melalui titik (–2, 36)

Karena memotong sumbu y di (0, 1) maka bentuk fungsinya adalah  

f(x) = aˣ, dengan a > 0 dan a ≠ 1

Melalui titik (–2, 36) artinya

f(–2) = 36

a⁻² = 36

[tex]\frac{1}{a^{2}} = \frac{36}{1}[/tex]

[tex]\frac{a^{2}}{1} = \frac{1}{36}[/tex]

a² = [tex]\left(\frac{1}{6} \right)^{2}[/tex]

a = [tex]\frac{1}{6}[/tex]

Jadi fungsi eksponen untuk grafik pada gambar b adalah

[tex]f(x) = \left(\frac{1}{6} \right)^{x}[/tex]

[tex]f(x) = (6^{-1})^{x}[/tex]

[tex]f(x) = 6^{-x}[/tex]

sketsalah grafik fungsi y = 2x² + 9x

Jawaban

Pendahuluan

sebelum menggambar kita harus perhatikan terlebih dahulu

karakteristik grafik berdasarkan nilai determinan

1) Jika D > 0 maka persamaan kuadrat memiliki dua akar real berbeda sehingga grafik akan memotong sumbu x di dua titik

2) Jika D = 0 maka persamaan kudrat memiliki dua akar real kembar atau grafik menyinggung sumbu x

3) Jika D < 0 maka persamaan kuadrat memiliki akar yang imaginer atau tidak memotong sumbu x

karakteristik grafik berdasarkan nilai a,

1) jika a > 0 maka grafik terbuka keatas

2) jika a < 0 maka grafik terbuka kebawa

3) jika a = 0 bukan persamaan kuadrat

grafik fungsi kuadrat memotong sumbu x apabila y = 0, dan memotong sumbu y apabila x = 0

untuk menentukan sumbu simetri dapat menggunakan rumus

xₐ = ⁻ᵇ/₂ₐ

untuk menentukan titik puncak atau titik balik dapat menggunakan rumus

(xₐ, yₐ) = (⁻ᵇ/₂ₐ, -d/₄ₐ)

22. Sketsalah grafik fungsi berikut ini y= 8x²-16x+6


Sketsa grafik y = 8x² - 16x + 6 dapat dilihat dilampiran.

Hal-hal yang harus diperhatikan dalam membuat sketsa grafik fungsi kuadrat:

Jika a > 0 maka grafik terbuka ke atas, dan jika a < 0 maka grafik terbuka ke bawah Titik potong dengan sumbu koordinat (memotong sumbu x disaat y = 0 dan memotong sumbu y disaat x = 0) Titik puncak ([tex]\frac{-b}{2a}[/tex], [tex]\frac{-D}{4a}[/tex]) Persamaan sumbu simetri Xp = [tex]\frac{-b}{2a}[/tex]

Pembahasan

Karena nilai a nya lebih besar dari 0 maka grafik terbuka ke atas.

-Memotong sumbu x saat y = 0

8x² – 16x + 6 = 0

(4x – 2) (2x – 3) = 0

4x – 2 = 0 atau 2x – 3 = 0

4x = 2 atau 2x = 3

x = ½ atau x = 3/2

(1/2, 0), (3/2, 0)

-Memotong sumbu y saat x = 0

Y = 8x² – 16x + 6

Y = 8.0² – 16.0 + 6 

Y = 6

(0, 6)

-Titik puncak (xp, yp)  

Xp = [tex]\frac{-b}{2a}[/tex] = -(-16)/16 = 1

Yp = 8.1² – 16.1 + 6 = 8 – 16 + 6 = -2

(1, -2)

Titik  = (1/2, 0), (3/2,0), (0,6), (1, -2)
