Materi : Barisan dan Geometri

Pola barisan bertingkat

2, 6, 12, 20, ... [ a = 2 ]

4, 6, 8, [ b = 4 ]

2, 2 [ c = 2 ]

Suku pertama = 2 , beda = +4, +6, +8, ...

Maka rumus pola :

Un = a + b( n - 1 ) + ½( n - 1 )( n - 2 )c

Un = 2 + 4( n - 1 ) + ( 2/2 )( n² - 3n + 2 )

Un = 2 + 4n - 4 + n² - 3n + 2

[ Un = n² + n ]

U16 = 16² + 16 = 256 + 16 = 272

U19 = 19² + 19 = 361 + 19 = 380

Jika Un = 552

n² + n - 552 = 0

{ a = 1, b = 1, c = - 552 }

Metode Rumus ABC / Kuadratik

n = b ± √[ b² - 4ac ] /2a

n = 1 ± √[ 1² - 4(1)(-552) ] / 2

n = 1 ± √[ 1 + 2.208 ] / 2

n = 1 ± √2.209 / 2

n = 1 ± 47 / 2

Himpunan Penyelesaian : [ positif ]

n = 1 + 47 / 2 = 24 [ Dipakai

n = 1 - 47 / 2 = -23 [ Tidak dipakai ]

Kesimpulan

bahwa suku ke berapakah Rn adalah 24.
