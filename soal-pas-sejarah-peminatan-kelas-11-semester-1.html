20. Soal matriks matematika peminatan kelas 11


Jawab :

Berdasarkan matriks tersebut, diperoleh persamaan :

• a + 3 = 5 - a
a + a = 5 - 3
2a = 2
a = 2/2
a = 1

• 8 + b = 4 - b
b + b = 4 - 8
2b = -4
b = -4/2
b = -2

• -1 + c = c - c
-1 + c = 0
c = 0 + 1
c = 1

• d + (-9) = -13 - d
d - 9 = -13 - d
d + d = -13 + 9
2d = -4
d = -4/2
d = -2

Maka, nilai dari a+b+c+d adalah

a + b + c + d = 1 + (-2) + 1 + (-2)
a + b + c + d = 1 - 2 + 1 - 2
a + b + c + d = -1 - 1 = -2
