7. Apa yang kalian ketahui tentang start berdiri start melayang start jongkok


Penjelasan:

1) Start berdiri adalah jenis start yang digunakan dalam cabang lari jarak menengah dan lari jarak jauh, dengan panjang lintasan 800 meter, 1500 meter, 5000 meter dan 10.000 meter.

2) Start melayang adalah jenis start yang digunakan hanya pada nomor lari estafet.

3) Start jongkok biasanya dilakukan dalam lari jarak pendek, yaitu 100 meter, 200 meter hingga 400 meter.

28. Pada perlombaan atletik lari terdapat berbagai jenis lomba seperti lari jarak pendek, menengah, jauh, dan maraton. Urutan teknik dasar start pada lomba lari tersebut adalah .... a. Start jongkok, start berdiri, start berdiri, start berdiri b. Start jongkok, start berdiri, start melayang, start berdiri c. Start berdiri, start jongkok, start melayang, start berdiri d. Start melayang, start jongkok, start berdiri, start berdiri e. Start berdiri, start melayang, start jongkok, start berdiri


Jawaban:

Jawaban yang tepat adalah A. Start jongkok, start berdiri, start berdiri, start berdiri.

Penjelasan:

Pada perlombaan atletik lari, urutan teknik dasar start pada berbagai jenis lomba adalah sebagai berikut:

Lari jarak pendek: Start jongkok atau disebut juga start blok.Lari jarak menengah: Start berdiri atau disebut juga start tegak.Lari jarak jauh: Start melayang atau disebut juga start gelinding.Maraton: Start berdiri.

Sehingga jawaban yang paling tepat adalah A. Start jongkok, start berdiri, start berdiri, start berdiri.
