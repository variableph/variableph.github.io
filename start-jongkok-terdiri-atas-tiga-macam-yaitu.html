9. lari jarak pendek menggunakan star jongkok dan star jongkok terdiri dari tiga macam yaitu start pendek(short pendek)star menengah(medium start)dan start panjang(long start).jelaskan ke tiga macam cara melakukan star jongkok​


Jawaban:

3 Macam Teknik Start Jongkok: Pendek, Menengah, dan Panjang

- Teknik start jongkok merujuk pada posisi permulaan pelari ketika mengawali lomba yang menempatkan diri dengan posisi jongkok (crouching) di garis mula (start).

Seperti dikutip dari buku Kepelatihan Atletik Jalan dan Lari (2018) karya Suratmin, teknik awalan crouching start atau start jongkok umumnya digunakan dalam perlombaan lari jarak pendek.

Start jongkok terdiri atas tiga macam, yaitu start jongkok pendek, menengah, dan panjang.

Menguasai teknik awalan atau start dalam olahraga lari, khususnya lari jarak pendek, bisa menentukan prestasi seorang pelari dalam perlombaan.

Perbedaan yang mencolok antara lari jarak pendek dengan jarak menengah terletak pada posisi start karena lari jarak menengah secara umum menggunakan awalan berdiri (standing start).

1. Start Pendek (Short Start/Bunch Start)

Kaki terbuka selebar bahu, kaki depan menginjak papan start depan, kaki belakang menginjak papan start belakang.

Lutut kaki belakang berada pada posisi sejajar dengan jari-jari kaki depan atau bahkan lebih depan kurang lebih satu jengkal dari telapak kaki depan.

2. Start Medium

letak lutut kaki belakang berada sejajar dengan tungkai kaki depan atau sedikit lebih kebelakang dari kaki depan. Posisi ini juga masih bisa menghasilkan kecepatan yang tinggi.

3. Start Panjang

Posisi lutut kaki belakang berada sekitar 30 cm dari kaki depan atau menyesuaikan. Yang jelas, posisi lutut kaki belakang selalu berada dibelakang kaki depan.

1. Shot start/bunch start (start pendek)
Pada saat berjongkok lutut kaki belakan berada di depan ujung kaki yang lain, apabila dalam sikap berdiri ujung kaki kaki belakang akan terletak kira-kira di samping tumit atau lakukan telapak kaki kedepan.

Jenis start ini dapat menghasilkan percepatan tinggi, maka banyak pelari kenamaan yang memakainya. Tetapi bagi pemula (anak-anak) biasanya kurang sesuai karena dengan posisi kedua kaki yang saling berdekatan itu, peranan kedua tangan akan terasa lebih berat dalam menahan berat badan, lebih-lebih setelah aba-aba "siap". Itulah sebabnya maka Short start hanya sesuai bagi atlet-altet yang sudah terlatih (senior).

2. Medium Start (start menengah)
Pada saat berjongkok lutut kaki belakang kira-kira berada disamping lekukkan telapak kaki depan. Dalam posisi berdiri, posisi ujung kaki belakang berada sedikit di belakang tumit kaki depan. Para pelari senior masih banyak yang memakai jenis start ini, karena medium start juga dapat menghasilkan percepatan yang cukup tinggi. Bagi pemula biasanya sangat sesuai memakai start ini.

3. Long start (start panjang)
Pada sikap berjongkok lutut kaki belakang berada di samping atau kira-kira segaris dengan tumit kaki depanatau letak lutut lebih mundur lagi. Pada saat berdiri kedua kaki saling berjauhan, yaitu ujung kaki belakang terletak  sekitar dua jangkal dari tumit kaki depan.

Start ini kurang menguntungkan, karena dengan posisi kedua kaki saling berjauhan itu, kurang memberikan percepatan dalam melakukan luncuran kedepan. Pelari yang berkaki panjang (tinggi) biasanya sesuai memakai start ini

24. start jongkok dibagi dalam tiga bagian jelaskan!​


Jawaban:

Start jongkok terdiri atas tiga macam, yaitu start jongkok pendek, menengah, dan panjang.

Penjelasan:

1.start pendek

Pelari jarak pendek akan mengambil jarak sekitar 0,40 meter atau 16 inci dari garis start sampai bagian depan balok.

Untuk melakukan tipe start pendek, lutut kaki belakang berada di depan ujung kaki lainnya dalam posisi berdekatan.

2.Start menengah

Dalam menerapkan jenis start jongkok menengah para pelari mengambil jarak sekitar 0,53 meter atau 21 inci dari garis start hingga bagian depan balok.

3.Start panjang

Tipe start panjang membuat pelari mengambil jarak sekitar 21 inci atau 0,53 meter antara garis start dengan balok bagian depan, sedangkan jarak antar starting block berkisar 26 inci atau 0,66 meter.
