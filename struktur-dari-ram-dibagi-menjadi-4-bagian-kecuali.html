1. Dibagi berada bagian struktur Ram? Sebutkan strukturnya


Struktur RAM dibagi menjadi 4 bagian, yaitu:
-Input storage,berfungsi untuk menampung input atau masukan yang dimasukkan dari alat input
-Program Storage, bagian dari RAM yang berfungsi sebagai penyimpan intruksi program yang akan diakses.
-Working Storage, merupakan bagian dari memori yang bertugas menyimpan data yang akan diolah dan hasil pengolahan.
-Output Storage, berfungsi untuk menampung hasil akhir dari data yang akan di tapilkan ke perangkat output

RAM adalah penyimpan utama yang berfungsi untuk menyimpan instruksi-instruksi dan data yang akan diolah serta hasil pengolahan. 

RAM bersifat volatile, artinya data yang tersimpan akan hilang dengan sendirinya jika daya listriknya tidak ada. 

struktur RAM terbagi menjadi 4 bagian : 
1. Input storage, digunakan untuk menampung masukan yang dimasukkan melalui piranti masukan. 
2. ProgRAM storage, digunakan untuk menyimpan instruksi-instruksi progRAM yang akan dijalankan. 
3. Working storage, digunakan untuk menyimpan data yang akan diolah dan hasil pengolahan. 
4. Output storage, digunakan untuk menampung hasil akhir dari pengolahan yang akan disajikan ke piranti keluaran. 

Jenis-jenis RAM 
1. SIMM (Single In-line Memory Module) 
2. DIMM (Dual In-line memory Module) 
3. DDR SDRAM (Double Data Rate Syncronous Dynamic RAM) 
4. DR DRAM (Direct RAMbus Dynamic RAM)

4. 1. Jelaskan Fungsi dari processor!2. Sebutkan dan jelaskan bagian-bagian processor!3. Jelaskan pengertian RAM dan fungsinya!4. Sebutkan jenis-jenis RAM​


Penjelasan:

1.PROCESSOR

Komputer adalah alat modern yang terbentuk dari komponen - komponen perangkat keras (hardware) dan perangkat lunak (software). Salah satu perangkat keras yang ada pada komputer adalah processor

2.-Control unik yang dimana adalah bagian yang bekerja untuk melakukan pengontrolan terhadap tugas yang dimiliki oleh sebuah komputer. Dan bagian dari control unit ini memiliki sebuah peranan yang dimana sangatlah penting didalam melakukan pengontrolan daripada seluruh perangkat yang dimana akan terhubung dengan sebuah komputer. Fungsi yang dimiliki oleh control unit adalah untuk melakukan penyesuaian dari seluruh perangkat yang ada.

-Arithmetic logic unit yang dimana adalah bagian dari processor yang memiliki fungsi untuk menjadi bagian yang melakukan seluruh perhitungan daripada operasi komputer. Arithmetic logic unit ini sendiri akan dapat melakukan pengerjaan tugas perhitungan dengan menggunakan aritmatika dan juga logika.

-Register yang dimana adalah alat untuk melakukan penyimpanan yang dimana memiliki sebuah kecepatan yang sangatlah tinggi. Selain itu, register memiliki fungsi untuk melakukan penyimpanan dari seluruh perintah yang pada saat itu sedang diproses oleh komputer.

-CPU Interconnection yang dimana memiliki fungsi untuk menghubungkan bagian Control Unit, Arithmetic Logic Unit, Register dan juga berbagai macam sistem lainnya.

3.RAM adalah berasal dari singkatan Random Access Memory, RAM yaitu suatu memori tempat penyimpanan data sementara, ketika komputer dijalankan dan dapat diakses secara acak (random).

Fungsi RAM adalah mempercepat pemeproses data pada PC atau komputer. 

4.1.DRAM (Dynamic RAM) adalah jenis RAM harus sering di refresh oleh CPU agar data yang terkandung didalamnya tidak hilang.

2.SDRAM (Synchronous Dynamic RAM) adalah jenis RAM yang paling umum digunakan pada PC Pentium I,II dan III. RAM ini disinkronisasi oleh clock sistem dan memiliki kecepatan lebih tanggi dari pada DRAM serta dapat digunakan teritama dalam cache.

3.SRAM (Statik RAM) adalah jenis memory yang tidak perlu penyegaran oleh CPU agar data yang terdapat didalamnya tetap tersimpan dengan baik.

4.RDRAM (Rambus Dynamic RAM) adalah pada jenis memory ini yang lebih cepat dan lebih mahal dari pada SDRAM. Memory ini bisa digunakan pada sistem yang menggunakan Pentium 4.

5.FPM DRAM (Fast Page Mode DRAM) adalah merupakan bentuk asli dari DRAM. Laju transfer maksimum untuk cache L2 mendekati 176 MB per sekon.

 6.EDO DRAM (Extented Data Out DRAM) adalah memory ini sekitar 5% lebih cepat dibandingkan dengan FPM. Laju transfer maksimum untuk cache L2 mendekati 264 MB per sekon.

7.Double Data Rate RAM (DDR RAM), dinamis untuk generasi Pentium IV (DDR 400). Memiliki banyak tipe dan umumnya dikelompokkan berdasarkan bandwidth dan dipengaruhi besarkan clock yang diberikan memori itu nantinya. Tipe yang ditawarkan PC 1600 / 2100 / 2700 / Untuk pemilihan jenis memori sangat dipengaruhi kinerja processor, karena perbedaan antar internal bus pada processor dengan internal bus pada memori akan berakibat fatal bagi kecepatan proses pengolahan aplikasi data, untuk itu perbandingan keduanya haruslah sebanding

1 CONTROL UNIT (CU)

Control unit merupakan bagian prosesor yang bekerja mengontrol jalanya tugas komputer.

Tugas-Tugas Control Unit (CU)

*Mengontrol jalannya komputer

*Mengontrol antara perangkat satu dengan perangkat lainya

*Mengambil Intruksi di memori utama

*Mengirimkan perintah ke ALU, apabila ada perintah perhitungan aritcmetic

*Mengawasi kerja ALU

*Mengirimkan hasil proses dari ALU ke Register

2 ARITHMETIC LOGIC UNIT (ALU)

ALU merupakan bagian prosesor yang berfungsi sebagai otak penghitung operasi komputer.

Tugas – Tugas Arithmetic Logic Unit (ALU)

*Melakukan perhitungan aritmatika (Penjumlahan, Pengurangan, Perkalian, Pembagian)

*Melakukan perhitungan kode biner (Konversi)

*Melakukan perbandingan (Lebih dari, kurang dari, sama dengan, lebih dari sama dengan, kurang dari sama dengan)

3 REGISTER

Register merupakan alat penyimpanan yang memiliki kecepatan sangat tinggi.

Tugas Tugas Register

*Menyimpan perintah yang sedang diproses

*Menampilkan hasil proses ke interface

*Apabila ada perintah aritmatika atau logika, perintahnya diberikan kepada ALU terlebih dahulu dengan bantuan CU.

4 CPU INTERCONNECTION

CPU Interconnection adalah bagian prosesor yang menghubungkan Control Unit (CU), Arithmetic Logic Unit (ALU), Register dan Bus-bus yang menghubungkan dengan sistem lainya. Register juga menghubungkan prosesor dengan piranti input dan output.

3.) RAM (Random Access Memory) adalah tempat penyimpanan sementara pada komputer yang isinya dapat diakses dalam waktu yang tetap, tidak memperdulikan letak data tersebut dalam memori atau acak. jadi Ram ini BERFUNGSI sebagai penyimpanan sementara saja saat anda menggunakan komputer atau software tertentu.

4.) Jenis RAM komputer memang memiliki banyak varian. Mulai dari DRAM, SDRAM, RDRAM, SRAM, EDORAM, FPM DRAM RAM, dan masih ada lagi.

14. jelaskan bagian-bagian RAM!


1.Printed Circuit Board (PCB)
Fungsi -> Di setiap lapisan terpasang sirkuit atau jalur, sebagai pengaliran listrik dan data

2. Contact Point
Fungsi -> Menghubungkan informasi antara motherboard ke modul memori

3. DRAM (Dynamic Random Access memory)
Fungsi -> Menampung data dalam jangka waktu yang singkat dan harus di-refresh secara periodik

4. Chip Packaging
Fungsi ->Lapisan luar untuk membentuk fisik dari masing-masing memory chip

5. Dual In-Line Package (DIP)

Fungsi -> digunakan saat memori terinstal langsung pada PCB motherboard.

6. TSOP (Thin Small Outline Package)

Ciri-ciri : Bagian ini termasuk dalam komponen surfacemount. Bentuk dan ukuran fisiknya yang sangat tipis dan kecil tidak sesuai dengan namanya.

7. CSP (Chip Scale Package)

Fungsi -> Apabila bagian komponen yang diatas menggunakan kaki/pin sebagai penghubung BGA (Ball Grid Array) yang tempatnya di bawah komponen.

21. sebutkan dan jelaskan 4 struktur RAM. ....


4 Struktur RAM yaitu :

Input Storage

Bagian ini digunakan dalam memuat masukan - masukan yang diterima dari input devices. Masukan / input sendiri dapat berupa perintah suatu  program maupun hasil pengolahan data.

Program StorageWorking Storage

Program dan working storage sama - sama menyimpan masukan data akan tetapi perbedaanya terletak pada jenis inputan.

Program storage akan menyimpan masukan yang berupa perintah suatu program ,sedangkan working storage akan menyimpan yang berupa data dan hasil olahannya

Output Storage

Bagian ini berfungsi untuk memuat hasil olahan data hingga final untuk dimunculkan pada perangkat output.

Pembahasan

RAM / Random Access Memory adalah media penyimpanan primer yang ada dan karenanya, RAM sendiri terhubung langsung media processor dan tentunya terlibat dalam pemrosesan  data sehingga sangat berpengaruh terhadap kecepatan akses suatu device. Salah satu kekurangan RAM sebagai salah satu media penyimpanan primer adalah sangat mudah kehilangan data jika mendadak terjadi situasi power cut.

RAM sendiri memiliki beberapa jenis diantaranya :

Dynamic Random Access MemorySychronous Dynamic Random Access MemoryDouble Rate Random Access MemoryRambus Dynamic Access MemoryStatic Random Access Memoryxtended Data Out Random Access Memory
