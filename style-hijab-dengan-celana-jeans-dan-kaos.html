Diketahui :

Celana jeans dan 3 buah kaos Rp. 145,000

2 celana jeans dan sebuah kaos Rp. 165,000

Ditanyakan :

Harga sebuah Celana jeans dan sebuah kaos?

Jawab :

Soal ini membahas tentang SPLDV atau Sistem persamaan linear dua variabel, didalam SPLDV ada metode substitusi, eliminasi, grafik, determinan, dll.

Pada soal ini kita gunakan metode Eliminasi , caranya yaitu :

misal :

x = celana jeans

y = kaos

maka , model matematika nya seperti ini :

x + 3y = 145,000

2x + y = 165,000

Metode Eliminasi :

x + 3y = 145,000 |×2

2x + y = 165,000 |×1

2x + 6y = 290,000

2x + y = 165,000

----------------------------- -

5y = 125,000

y = 125,000 : 5

y = 25,000

substitusi kan y = 25,000 ke :

2x + y = 165,000

2x + 25,000 = 165,000

2x = 165,000 - 25,000

2x = 140,000

x = 140,000 : 2

x = 70,000

Diperoleh harga x(celana jeans) dan y(kaos)

maka,

Harga sebuah Celana jeans dan kaos yaitu :

x = harga celana jeans Rp. 70,000

y = harga kaos Rp. 25,000

total sebuah Celana jeans dan kaos adalah :

= x + y

= 70,000 + 25,000

=Rp. 95,000

PENDAHULUAN

Persamaan Garis Lurus adalah suatu bentuk persamaan matematika yang melibatkan 2 variabel dan masing masing variabel berpangkat 1.

Ada beberapa cara untuk menyelesaikan Persamaan Linear 2 Variabel :

Metode GrafikMetode EliminasiMetode SubtitusiMetode Gabungan (Eliminasi - Subtitusi)

PEMBAHASAN SOAL

Misal :

Kaos = x

Celana jeans = y

_

2x + y = 165.000

x + 2y = 225.000

• Eliminasi x

2x + y = 165.000 |×1

x + 2y = 225.000 |×2

2x + y = 165.000

2x + 4y = 450.000

________________ _

-3y = -285.000

y = 95.000 → Harga 1 celana jeans

• Subtitusikan nilai y ke salah satu persamaan

2x + y = 165.000

2x + 95.000 = 165.000

2x = 70.000

x = 35.000 → Harga 1 kaos

Sistem Persamaan Linear Tiga Variabel (SPLTV)

• Diketahui:

Misal,

Hijab = x

Hodie = y

Celana Jeans = z

Maka,

3x + y + 2z = Rp 430.000 ..... (1)

2x + 2y + z = Rp 435.000 ..... (2)

2x + y + z = Rp 300.000 ...... (3)

• Ditanya:

Harga dari masing-masing barang tersebut berupa hijab, hodie, dan celana jeans?

• Jawab:

Eliminasi nilai x pada pers. (2) dan (3)

2x + 2y + z = Rp 435.000

2x + y + z = Rp 300.000

_____________________-

y = Rp 135.000

Eliminasi nilai x pada pers. (1) dan (3)

3x + y + 2z = Rp 430.000 | × 2 | 6x + 2y + 4z = Rp 860.000

2x + y + z = Rp 300.000 | × 3 | 6x + 3y + 3z = Rp 900.000

______________________-

-y + z = - Rp 40.000 .... pers. (4)

Substitusikan nilai y = Rp 135.000 ke pers. (4)

-y + z = - Rp 40.000

- Rp 135.000 + z = - Rp 40.000

z = - Rp 40.000 + Rp 135.000

z = Rp 95.000

Substitusikan nilai y dan z ke pers. (3)

2x + y + z = Rp 300.000

2x + Rp 135.000 + Rp 95.000 = Rp 300.000

2x = Rp 300.000 - Rp 135.000 - Rp 95.000

2x = Rp 70.000

x = Rp 70.000 / 2

x = Rp 35.000

Jadi, harga dari masing-masing barang tersebut adalah

Hijab = Rp 35.000

Hodie = Rp 135.000

Celana jeans = Rp 95.000

                         = Rp232.500
