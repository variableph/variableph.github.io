2. suatu perusahaan taksi memasang tarif seperti grafik berikut. alia pergi kerumah nenek yang berjarak 22 km dgn menggunakan taksi berikut. berapa tarif taksi yang harus dibayar alia?


Kelas         : 8
Mapel         : Matematika
Kategori     : Bab 2 - Fungsi
Kata kunci : tarif taksi, soal UN

Kode : 8.2.2 [Kelas 8 Matematika Bab 2 - Fungsi]

Soal : 

Suatu perusahaan taksi memasang tarif seperti grafik berikut. Alia pergi kerumah nenek yang berjarak 22 km dgn menggunakan taksi berikut. berapa tarif taksi yang harus dibayar Alia ...
A.  Rp 66.000
B.  Rp 73.000
C.  Rp 132.000
D.  Rp 143.000

Pembahasan : 

Diketahui :
Biaya tarif jarak 2 km = Rp 13.000
Biaya tarif jarak 4 km = Rp 19.000

Ditanya : 
Biaya tarif taksi menempuh jarak 22 km ?

Jawab :

Jarak merupakan x 
Biaya tarif taksi adalah f(x)

Kita gunakan rumus fungsi → f(x) = ax + b

f(2) = 13.000 →    2x + b = 13.000
f(4) = 19.000 →    4x + b = 19.000
                             _____________ -
                              -2x     = -6000
                                     x = -6000 / -2
                                     x = 3000
Subtitusikan
2x + b = 13.000
2 (3000) + b = 13.000
6000 + b = 13.000
            b = 13.000 - 6000
            b = 7000

Rumus fungsi → f(x) = 3000 x + 7000

Menentukan tarif taksi kerumah nenek dengan jarak 22 km

f(x) = 3000 x + 7000
f(22) = 3000 (22) + 7000
         = 66.000 + 7.000
         = 73.000

Jadi  tarif taksi yang harus dibayar Alia adalah Rp 73.000


Semoga bermanfaat

21. suatu perusahaan taksi memasang tarif seperti grafik brkt. tlg bantu yaa


19-13= 6
22×6= Rp132.000,00 (c)
Bab Gradien dan Persamaan Garis

Garis tersebut melalui titik (2,13) dan titik (4,19)
Persamaan garisnya
(y - y1)/(y2 - y1) = (x - x1)/(x2 - x1)
(y - 13)/(19 - 13) = (x - 2)/(4 - 2)
(y - 13)/6 = (x - 2)/2
