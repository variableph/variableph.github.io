19. Suatu zat suhunya 80derajat jika diukur menggunakan termometer skala Fahrenheit suhu zat tersebut


176°F

*Pembahasan*

Skala suhu ada 4 :

• Celcius (°C)

• Fahrenheit (°F)

• Reamur (°Re)

• Kelvin (K)

Kelvin merupakan skala suhu dalam satuan Internasional atau "SI"

• °C = 0° - 100°

• °F = 32° - 212°

• °Re = 0° - 80°

• K = 273 - 373

Adapun rumus konferensi skala suhu : ↓↓↓

Celcius (°C) → Reamur (°R) = ⅘.C

→ Fahrenheit (°F) = 9/5.C + 32

→ Kelvin (K) = C + 273

Reamur (°R) → Celcius (°C) = 5/4.R

→ Fahrenheit (°F) = 9/4.R + 32

→ Kelvin (K) = 5/4.R + 273

Fahrenheit (°F) → Celcius (°C) = 5/9.(F - 32)

→ Reamur (°R) = 4/9.(F - 32)

→ Kelvin (K) = 4/9.(F - 32) + 273

Kelvin (K) → Celcius (°C) = K - 273

→ Reamur (°R) = ⅘.(K - 273)

→ Fahrenheit (°F) = 9/5.(K - 273) + 32

Diketahui

80°C

Ditanya

C.Tentukan skala yang ditunjukan oleh Fahrenhait

Jawab

[tex]F = \dfrac {9}{5}.C + 32\\F = \dfrac {9}{5}.80 + 32\\F = 9.16 + 32\\F = 144 + 32\\F = 176°\\[/tex]
