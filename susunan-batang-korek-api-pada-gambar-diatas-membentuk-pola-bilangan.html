7. jika pola gambar disusun dengan batang korek api,maka banyak batang korek api pada pola ke-40 adalah...


Jika pola gambar disusun dengan batang korek api, maka banyak batang korek api pada pola ke-40 adalah 121. Barisan aritmatika adalah barisan yang antar dua suku berdekatannya memiliki selisih yang tetap. Untuk menentukan suku ke n pada barisan aritmatika adalah

Un = a + (n – 1)b

Rumus jumlah n suku pertama

Sn = [tex]\frac{n}{2}[/tex] (2a + (n – 1)b) Sn = [tex]\frac{n}{2}[/tex] (a + Un)

Keterangan

a = suku pertama b = beda ⇒ b = U₂ – U₁ = U₃ – U₂ = ....

Pembahasan

Perhatikan gambar pada lampiran

Banyak korek api pada  

Pola ke 1 = 4 Pola ke 2 = 7 Pola ke 3 = 10 Pola ke 4 = 13

Dan seterusnya

Terlihat pola banyak korek api tersebut membentuk barisan aritmatikan (karena selalu bertambah 3)

4, 7, 10, 13, ...

a = 4 b = 7 - 4 = 3

Maka banyak batang korek api pada pola ke-40 adalah

U₄₀ = a + 39b

U₄₀ = 4 + 39(3)

U₄₀ = 4 + 117

U₄₀ = 121

Pada soal ini kita akan mempelajari tentang barisan aritmatika

Barisan Aritmatika adalah barisan bilangan yang memiliki pola yang tetap di mana polanya berdasarkan operasi penjumlahan atau pengurangan

Rumus :

b = beda

= Suku ke-n

= Suku ke n - 1

= a + (n - 1)b

= Suku ke n

a = suku ke-1

b = beda

Pembahasan :

Diketahui :

Barisan aritmatika :

7, 11, 15,...

Ditanya :

Banyak batang korek api pada pola ke-51 () = ?

Dijawab :

Untuk mendapatkan siku ke-1, ke-2, dan ke-2, maka kita hitung jumlah korek api pada gambar

dari barisan aritmatika berikut ini :

7, 11, 15, ...

suku pertama (a) = 7

= 11

= 15

b =

b =

b = 15 - 11

b = 4

Selanjutnya kita cari pola ke 51 / suku ke-51 ()

a = 7

b = 4

n = 51

= a + (n - 1)b

= 7 + (51 - 1)(4)

= 7 + (50)(4)

= 7 + 200

= 207

∴ Jadi banyak batang korek api pada pola ke-51 adalah 207 (B)
