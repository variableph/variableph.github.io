Jawaban:

Ini adalah materi lanjutan dari Action Script yang terdapat pada Macromedia Flash 8. Jika pada materi sebelumnya, pembahasannya adalah action script general, maka kali ini saya akan menjelaskan tentang action script movie atau frame pada Macromedia Flash 8.

Beberapa command yang saya pelajari yaitu “Play”, “Stop”, “Next Frame”, “Previous Frame”, “Go to And Play”, dan “Go to And Stop”. Penggunaan Action script ini biasanya digunakan pada tombol atau button pada animasi atau project yang dibuat.

Play

Command “play()” pada objek tombol berfungsi untuk mulai memainkan suatu animasi.

Contoh penulisan command :

on(release)

{

play();

}

Stop

Command “stop()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan.

Contoh penulisan command :

on(release)

{

stop();

}

Previous Frame

Command “prevFrame()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan lalu kembali ke posisi 1 frame sebelumnya. Animasi akan berjalan mundur frame per frame sebanyak 1 frame tiap kali tombol dengan command “prevFrame()” ditekan.

Contoh penulisan command :

on(release)

{

prevFrame();

}

Next Frame

Command “nextFrame()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan lalu maju ke posisi 1 frame selanjutnya. Animasi akan berjalan maju frame per frame sebanyak 1 frame tiap kali tombol dengan command “nextFrame()” ditekan.

Contoh penulisan command :

on(release)

{

nextFrame();

}

Go to And Play

Command “gotoAndPlay()” pada objek tombol berfungsi untuk melanjutkan animasi atau project yang sedang berjalan, menuju ke frame tertentu yang telah ditentukan. Dan animasi akan langsung berjalan secara otomatis. Masukkan frame tujuan didalam tanda kurung.

Contoh penulisan command dengan frame tujuan frame ke-50 :

on(release)

{

gotoAndPlay(50);

}

Go to And Stop

Command “gotoAndStop()” pada objek tombol berfungsi untuk melanjutkan animasi atau project yang sedang berjalan, menuju ke frame tertentu yang telah ditentukan. Tetapi animasi tidak akan berjalan secara otomatis, melainkan akan berhenti pada frame yang telah ditentukan tadi. Command ini biasanya digunakan untuk kembali ke menu utama project. Masukkan frame tujuan didalam tanda kurung.

Contoh penulisan command dengan frame tujuan frame ke-10 :

on(release)

{

gotoAndStop(10);

}

Itu tadi adalah penjelasan lanjutan mengenai Action Script yang telah saya pelajari, untuk implementasinya, ditunggu dalam posting selanjutnya. Semoga bermanfaat, sekian dan terima kasih.

29. Untuk render frame 4 sampai dengan frame 500 dari 1000 frame maka mengunakan


Dalam render frame 4 sampai dengan frame 500 dari 1000 frame maka mengunakan menggunakan blender. Dengan blender maka kita dapat melakukan render frame 3D.

Pembahasan

Rendering adalah hasil dari proses perhitungan semua elemen fisik, pencahayaan, efek, dan elemen lainnya untuk menghasilkan  gambar atau output animasi yang realistis. Gambar duplikat umumnya adalah gambar grafis digital atau  raster. Rendering adalah hasil dari program komputer yang memproses gambar fotorealistik atau non-fotorealistik dari model 2D atau 3D. Dalam proses pengolahan citra, rendering merupakan proses  akhir dan umumnya membutuhkan waktu yang lama. Waktu rendering tergantung pada durasi dan kualitas visual dari rangkaian gambar yang diedit.
