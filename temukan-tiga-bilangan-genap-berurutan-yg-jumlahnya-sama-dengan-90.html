28. Temukan tiga bilangan genap berurutan yg jumlahnya sama dengan 90 dan cotohnya

bilangan pertama = a

bilangan kedua = b = (2 + a)

bilangan ketiga = c = (2 + a + 2) = (4 + a)

maka :

a + (2 + a) + (4 + a) = 90

3a + 6 = 90

3a = 90 - 6

3a = 84

a = 84/3

a = 28

b = 2 + a

b = 2 + 28

b = 30

c = 4 + a

c = 4 + 28

c = 32

jadi tiga bilangan genap berurutan yang jumlahnya sama dengan 90 adalah 28 , 30 , dan 32
