13. tentukanlah daerah asal dan daerah hasil fungsi berikut​


Ingat :

Daerah asal disebut Domain

Daerah hasil fungsi disebut Range

Penyelesaian :

f(x) = 2x + 3

Karena nilai sembarang x dapat masuk di fungsi tersebut maka:

Daerah asal (Domain)  =  {x | x , x ∈ Real} Daerah hasil (Range)   =  {y | y , y ∈ Real}

----------------

f(x) = x² - 2x - 8

Karena nilai sembarang x dapat masuk di fungsi tersebut maka:

Daerah asal (Domain)  =  {x | x , x ∈ Real} Daerah hasil (Range)   =  {y | y , y ∈ Real}

----------------

f(x) = x² - 1  2 ≤ x ≤ 6

Daerah asal (Domain)  =  {x | 2 ≤ x ≤ 6, x ∈ Real}

Daerah hasilnya kita masukkan nilai x terkecil dan terbesar di fungsinya:

f(2) = 2² - 1 = 3  

f(6) = 6² - 1 = 35

Daerah hasil (Range)   =  {x | 3 ≤ x ≤ 35, x ∈ Real}

----------------

[tex]f(x) = \frac{2}{x( x - 5 )}[/tex]

Penyebut suatu pecahan tidak boleh sama dengan 0 karena dapat menyebabkan tidak terdefinisi, maka:

x ( x - 5) ≠ 0

x ≠ 0 atau x ≠ 5

Daerah asal (Domain)  =  {x | x ≠ 0 ∨ x ≠ 5, x ∈ Real} Daerah hasil (Range)   =  {x | x ∈ Real}

daerah asal dari fungsi linier tersebut adalah {xl 1 ≤ x < 7, x ∈ R }
daerah hasil dari fungsi kinier tersebut adalah {xl 3 ≤ x < 8, x ∈ R}
rumus fungsi dari fungsi linier tersebut
f(x) = ax + b
f(1) = a(1) + b
3 = a + b ... persamaan 1

f(7) = a(7) + b
8 = 7a + b ... persamaan 2

eliminasi persamaan 1 dan 2

7a + b = 8
a + b = 3
--------------- -
6a = 5
a = 5/6

a + b = 3
5/6 + b = 3
b = 3 - 5/6
b = 13/6

jadi rumus fungsinya adalah
f(x) = 5/6 x + 13/6

30. Tentukan daerah asal dan daerah hasil dari fungsi yang disajikan pada grafik berikut


Daerah asal = batas grafik dilihat dari sumbu X.
Di sebelah kiri, grafik dibatasi oleh x = -3.
Di sebelah kanan, grafik tidak terbatas (karena ada anak panah di ujung kanan grafik). Berarti x = tak hingga.

Jadi daerah asal grafik = [ -3 , tak hingga )

Daerah hasil = batas grafik dilihat dari sumbu Y.
Batas bawah grafik berada di titik y = -5.
Batas atas grafik tidak terhingga (karena anak panah di ujung grafik), sehingga y = tak hingga.

Jadi daerah hasil grafik = [-5 , tak hingga)
