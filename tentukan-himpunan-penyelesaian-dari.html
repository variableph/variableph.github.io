13. tentukan Himpunan Penyelesaian dari |4x-16 | -14=tentukan Himpunan penyelesaian dari | 20-2x|-14​


Jawab:

Tentukan himpunan penyelesaian persamaan nilai mutlak berikut

A. |2x - 3| = 6

b. |3x - 1| = 2

c. |x + 3| = |2x - 1|

d. |3x - 2| = |x + 1|

e. |x - 6| + |3 + 2x| = 20

f. |x + 4| + |x + 2| = 12

g. 2/3 |5x + 2| = 4 - 4/3 |5x + 2|

Pembahasan :

a. |2x - 3| = 6

untuk x ≤ 3/2

(2x - 3) = -6

2x = -6 + 3

2x = -3

x = -3/2

atau

untuk x ≥ -3/2

(2x - 3) = 6

2x = 6 + 3

2x = 9

x = 9/2

HP = {-3/2 , 9/2}

b. |3x - 1| = 2

(3x - 1)² = 2²

9x² - 6x + 1 = 4

9x² - 6x - 3 = 0 (sama2 dibagi 3)

3x² - 2x - 1 = 0

(3x + 1) (x - 1) = 0

3x + 1 = 0

3x = -1

x = -1/3

atau

x - 1 = 0

x = 1

HP = {-1/3 , 1}

c. |x + 3| = |2x - 1|

(x + 3)² = (2x - 1)²

x² + 6x + 9 = 4x² - 4x + 1

x² - 4x² + 6x + 4x + 9 - 1 = 0

-3x² + 10x + 8 = 0 (kesemua ruas dikali negatif)

3x² - 10x - 8 = 0

(3x + 2) (x - 4) = 0

3x + 2 = 0

3x = -2

x = -2/3

atau

x - 4 = 0

x = 4

HP = {-2/3 , 4}

d. |3x - 2| = |x + 1|

(3x - 2)² = (x + 1)²

9x² - 12x + 4 = x² + 2x + 1

9x² - x² - 12x - 2x + 4 - 1 = 0

8x² - 14x + 3 = 0

(4x - 1) (2x - 3) = 0

4x - 1 = 0

4x = 1

x = 1/4

atau

2x - 3 = 0

2x = 3

x = 3/2

HP = {1/4 , 3/2

e. |x - 6| + |3 + 2x| = 20

untuk x ≤ -3/2

-(x - 6) - (3 + 2x) = 20

-x + 6 - 3 - 2x = 20

-3x + 3 = 20

-3x = 20 - 3

-3x = 17

x = -17/3

untuk -3/2 ≤ x ≤ 6

(x - 6) - (3 + 2x) = 20

x - 6 - 3 - 2x = 20

-x = 20 + 9

x = -29

untuk x ≥ 6

x - 6 + 3 + 2x = 20

3x - 3 = 20

3x = 20 + 3

3x = 23

x = 23/3

x = 23/3 memenuhi dalam x ≥ 6

Jadi HP = {23/3}

f. |x + 4| + |x + 2| = 12

untuk x ≤ 2

-(x + 4) - (x + 2) = 12

-x - 4 - x - 2 = 12

-2x - 6 = 12

-2x = 12 + 6

-2x = 18

x = 18/-2

x = -9

Untuk 2 ≤ x ≤ 4

(x + 4) - (x - 2) = 12

x + 4 - x + 2 = 12

0 = 12 - 4 - 2

0 = 6 (tidak memenuhi)

untuk x ≥ 4

x + 4 + x + 2 = 12

2x + 6 = 12

2x = 12 - 6

2x = 6

x = 6/2

x = 3

HP = (-9 , 3}

g. 2/3 |5x + 2| = 4 - 4/3 |5x + 2|

untuk ≤ -2/5

-2/3 (5x + 2) = 4 - 4/3 [-(5x + 2)]

-2/3 (5x + 2) = 4 - 4/3 (-5x - 2) [kesemua ruas dikali 3]

23. Tentukan himpunan penyelesaian


Kategori Soal : Persamaan atau Pertidaksamaan Linear 
Kelas : X (1 SMA) 
Pembahasan :
Soal belum lengkap. Kemungkinan soal sebagai berikut.
Diketahui persamaan linear 
3x + 6 = 5x + 14 dan x ∈ bilangan bulat.
⇔ 3x - 5x = 14 - 6
⇔ -2x = 8
⇔ x = -4
Jadi, himpunan penyelesaian dari persamaan linear tersebut adalah {x|x = -4, x ∈ B}
Atau 
Diketahui pertidaksamaan linear
3x + 6 ≤ 5x + 14
⇔ 3x - 5x ≤ 14 - 6
⇔ -2x ≤ 8
⇔ x ≥ 8/(-2)
⇔ x ≥ -4
Jadi, himpunan penyelesaian dari pertidaksamaan linear tersebut adalah {x|x ≥ -4, x ∈ B}

Semangat!
