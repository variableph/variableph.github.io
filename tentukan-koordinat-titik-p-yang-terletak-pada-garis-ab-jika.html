   a. Panjang PR

   b. Panjang PQ

   c.  Panjang proyeksi PR pada PQ

   d. proyeksi PR pada PQ

  Diketahui : P(5, 1, 5), Q(1,4,5) dan R(3,2,1)

  Ditanya : Panjang PR, Panjang PQ, panjang proyeksi PR pada PQ dan proyeksi vector PR pada PQ

   a. Panjang PR

      PR = R-P

            = (3-5)+(2-1)+(1-5)

            = (-2, 1, -4)

     |PR| = √(-2)^2+(1)^2+(-4)^2 )

            = √(4+1+16)

            = √21

        Jadi, Panjang PQ adalah √21 satuan

   b. Panjang PQ

       PQ = Q – P

             = (1-5) + (4-1) +(5-5)

             = (-4, 3, 0)

      |PQ| = √(-4)^2+(3)^2+(0)^2 )

             = √(16+9+0)

             = √25

             = 5

      Jadi, Panjang PQ adalah 5 satuan

 c. Panjang proyeksi PR pada PQ

     [tex](\frac{PR.PQ}{|PQ|})[/tex]

    [tex]=\frac{(-2, 1, -4).(-4, 3, 0)}{5}\\ =\frac{(-2.-4)+(1.3)+(-4.0)}{5}\\ =\frac{6+3-4}{5} \\ =\frac{5}{5}[/tex]

    =1

    Jadi, panjang proyeksi PR pada PQ adalah 1 satuan

 d. Proyeksi vektor PR pada PQ

     =[tex]\frac{(PQ.PQ}{|PQ|^2}.(PQ})[/tex]

     [tex]=\frac{(-2.-4)+(1.3)+(-4.0)}{5^2}{(-4, 3, 0)}\\ \\=\frac{8+3-0}{25} {(-4, 3, 0)}\\ \\=\frac{11}{25} {(-4, 3, 0)}\\=(\frac{11}{25}{.-4}, \frac{11}{25}{.3}, \frac{11}{25}{.0})\\ = (\frac{44}{25}, \frac{33}{25}, {0})[/tex]

29. Tentukan koordinat titik p yang terletak pada garis ab jika:a. a(-1, 8), b(2,5), dan ap : pb = 4:-1b. a(1, 3), b(5, -1), dan ap : pb = 3:1​


a. Jika a(-1,8), b(2,5), dan ap : pb = 4:-1, maka koordinat titik p adalah (3,4).

b. Jika a(1, 3), b(5, -1), dan ap : pb = 3:1​, maka koordinat titik p adalah (4,0).

Koordinat tersebut diperoleh dengan menggunakan rumus perbandingan ruas garis atau perbandingan vektor.

Penjelasan dengan langkah-langkah:

Ingat rumus perbandingan ruas garis atau perbandingan vektor berikut:

[tex]AP:PB=m:n\rightarrow\bar{p}=\frac{n\cdot\bar{a}+m\cdot\bar{b}}{m+n}[/tex]

Vektor yang digunakan dalam rumus tersebut merupakan vektor posisi, seolah-olah nilai-nilai x dan y pada koordinat titik menjadi elemen-elemen vektor. Ini disebabkan pengurangan titik tersebut dengan titik asal (0,0), sebagai titik pangkal vektor posisi, yang akan menghasilkan titik itu kembali, tetapi dalam bentuk vektor.

Untuk poin a:

Mari tentukan titik koordinat dari p.

[tex]\bar{p}=\frac{(-1)\cdot\bar{a}+4\cdot\bar{b}}{4+(-1)}\\=\frac{-\bar{a}+4\bar{b}}{3}\\=\frac{1}{3}(-\bar{a}+4\bar{b})\\=\frac{1}{3}\left[-\left(\begin{array}{ccc}-1\\8\end{array}\right)+4\left(\begin{array}{ccc}2\\5\end{array}\right)\right]\\=\frac{1}{3}\left[\left(\begin{array}{ccc}1\\-8\end{array}\right)+\left(\begin{array}{ccc}8\\20\end{array}\right)\right]\\=\frac{1}{3}\left(\begin{array}{ccc}9\\12\end{array}\right)\\=\left(\begin{array}{ccc}3\\4\end{array}\right)[/tex]

Jadi, p memiliki koordinat (3,4).

Mari tentukan titik koordinat dari p.

[tex]\bar{p}=\frac{1\cdot\bar{a}+3\cdot\bar{b}}{3+1}\\=\frac{\bar{a}+3\bar{b}}{4}\\=\frac{1}{4}(\bar{a}+3\bar{b})\\=\frac{1}{4}\left[\left(\begin{array}{ccc}1\\3\end{array}\right)+3\left(\begin{array}{ccc}5\\-1\end{array}\right)\right]\\=\frac{1}{4}\left[\left(\begin{array}{ccc}1\\3\end{array}\right)+\left(\begin{array}{ccc}15\\-3\end{array}\right)\right]\\=\frac{1}{4}\left(\begin{array}{ccc}16\\0\end{array}\right)\\=\left(\begin{array}{ccc}4\\0\end{array}\right)[/tex]

Jadi, p memiliki koordinat (4,0).
