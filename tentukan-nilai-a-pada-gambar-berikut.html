20. tentukan nilai a pada gambar berikut


Kelas         : 8
Mapel        : Matematika
Kategori    : Bab 5 - Teorema Pythagoras
Kata kunci : nilai a, pythagoras

Kode : 8.2.5 [Kelas 8 Matematika Bab 5 - Teorema Pythagoras]

Penjelasan :

Gambar setiga siku-sikunya ada pada lampiran
Untuk mencari nilai a pada soal tersebut kita menggunakan pythagoras dan persamaan kuadrat.

Diketahui : 

AB = a + 4
BC = 3a + 2
AC = 3a + 4

Ditanya :

Nilai a ?

Jawab : 

                          AC² = AB² + BC²
                 (3a + 4)²   = (a + 4)² + (3a + 2)²
    (3a + 4)(3a + 4)     = (a + 4)(a + 4) + (3a + 2)(3a + 2)
9a² + 12a + 12a + 16 = a² + 4a + 4a + 16 + 9a² + 6a + 6a + 4
    9a² + 24a + 16       = a² + 8a + 16 + 9a² + 12a + 4
     9a² + 24a + 16     = 10a² + 20a + 20
9a² - 10a² + 24a - 20a + 16 - 20 = 0
    -a² + 4a - 4 = 0       (kesemua ruas dikali negatif)
     a² - 4a + 4 = 0        (difaktorkan)
 (a - 2) (a - 2) = 0
   a - 2 = 0
        a = 2

Jadi nilai a adalah 2

Panjang AB = a + 4
                    = 2 + 4
                    = 6

Panjang BC = 3a + 2
                    = 3 (2) + 2
                    = 6 + 2
                    = 8

Panjang AC = 3a + 4
                    = 3 (2) + 4
                    = 6 + 4
                    = 10
