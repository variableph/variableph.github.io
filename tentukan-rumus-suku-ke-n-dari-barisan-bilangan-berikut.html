29. tentukan rumus suku ke n dari barisan bilangan berikut 3,10,21,36,55


a=3
b=7
Un= a+(n-1)b
Un=3+(n-1)7
=3+7n-7
=-4+7n3..........10........21...........36.........55
......7...........11...........15............19
...........4............4............4

a + b + c = 3
3a + b = 7
2a = 4
a = 4/2
a = 2

3a + b = 7
3(2) + b = 7
6+ b = 7
b = 7 - 6
b = 1

a + b + c = 3
2 + 1 + c = 3
3 + c = 3
c = 3 - 3
c = 0

Substitusikan kedalam rumus Un

Un = an^2 + bn + c
Un = 2n^2 + (1)n + 0
Un = 2n^2 + n

Jadi rumus suku ke-n dari barisan bilangan tersebut adalah Un = 2n^2 + n
