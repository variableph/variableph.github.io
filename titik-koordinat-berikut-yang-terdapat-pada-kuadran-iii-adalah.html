18. Tentukan koordinat Cartesius dari titik-titik berikut! A. P(6, 135 (Kuadran II) B. P(4, 150) (Kuadran II) C. P(3, 225) (Kuadran III)


Penjelasan dengan langkah-langkah:

a)

[tex]x = r. \cos( \alpha ) \\ x = 6( \cos(135) ) \\ = 6( - \frac{1}{2} \sqrt{2} ) = - 3 \sqrt{2} [/tex]

[tex]y= r. \sin( \alpha ) \\ y= 6( \sin(135) ) \\ = 6( \frac{1}{2} \sqrt{2} ) = 3 \sqrt{2} [/tex]

maka koordinat cartesius

[tex]( - 3 \sqrt{2} ,3 \sqrt{2} )[/tex]

b)

[tex]x = r \cos( \alpha ) \\ x = 4( \cos(150) ) \\ = 4( - \frac{1}{2} \sqrt{3} ) = - 2 \sqrt{3} [/tex]

[tex]y = r. \sin( \alpha ) \\ y = 4( \sin(150) ) \\ = 4( \frac{1}{2}) = 2 [/tex]

maka koordinat cartesius = (-2✓3 , 2)

c)

[tex]x = r \cos( \alpha ) \\ x = 3( \cos(225) \\ = 3( - \frac{1}{2} \sqrt{2} ) = - \frac{3}{2} \sqrt{2} [/tex]

[tex]y = r \sin( \alpha ) \\ y = 3( \sin(225) ) \\ = 3( -\frac{1}{2} \sqrt{2} ) = - \frac{3}{2} \sqrt{2} [/tex]

maka koordinat cartesius

[tex] = ( - \frac{3}{2} \sqrt{2} , - \frac{3}{2} \sqrt{2} )[/tex]

Perlu Diketahui Juga! :

Kuadran I selalu (postif,positif) contoh : (5,5)

Kuadran II selalu (negatif,positif) contoh : (-5,5)

Kuadran III selalu (negatif,negatif) contoh : (-5,-5)

Kuadran IV selalu (positif,negatif) contoh : (5,-5)

Lalu mengapa beberapa koordinat mendapat 2 Kuadran?

Karna mereka berada di tengahnya, biasanya koordinat yang memiliki 2 Kuadran adalah koordinat yang memiliki angka "0" atau Kosong/Nol

contoh : (0,9) (7,0) (-2,0)
