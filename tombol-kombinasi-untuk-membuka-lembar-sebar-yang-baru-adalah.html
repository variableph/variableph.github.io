Jawaban:

Tekan tombol Alt, kemudian tekan tombol F, berikutnya tekan tombol N, dan terakhir tekan tombol N sekali lagi (Alt -> F -> N -> N).

Tekan tombol Ctrl dan tombol N secara bersamaan (Ctrl + N), cara ini adalah merupakan cara yang cepat untuk membuat sebuah dokumen baru yang masih kosong pada program Microsof Word.
