3. tombol manajemen berkas dokumen pada microsoft office excel berbentuk bulat dengan logo di tengahnya disebut?


Pada soal yang ditanyakan masih terlalu umum, namun penulis memahami bahwa manajemen berkas yang dimaksud adalah bagian tab File. Sehingga yang berbentuk bulat dan logo di tengahnya adalah "Recent Workbooks".

Pembahasan

Di dalam Microsoft Excel, manajemen berkas terletak pada tab "File", dimana bagian dari File tersebut adalah:

Info: Adalah info dasar mengenai file yang sedang dibukaNew: Untuk membuat halaman baru.Open: Untuk membuka file excel, termasuk di dalamnya adalah Recent Workbooks yang bentuknya bulat dan berlogo.Save As: Untuk menyimpan file.Print: Untuk Print out document.Share: Untuk berbagi file.Export: Untuk mengekspor file ke bentuk lainnya.Close: Untuk menutup dokumen.

Microsoft excel adalah bagian dari Microsoft Office yang fungsi utamanya adalah untuk melakukan pengolahan angka dan data.

Demikian, semoga membantu!
