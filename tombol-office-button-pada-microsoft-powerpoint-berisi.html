16. tombol office button pada ms powerpoint 2007 berisi?


Office button pada Microsoft Powerpoint 2007 berisi :

NewOpenSaveSave asPrint PrepareSendPublishClosePembahasan

New, perintah ini berguna untuk membuat dokumen powerpoint yang baru tanpa menutup dokume sebelumnya. Dapat pula digunakan shortcut Ctrl + N untuk menggunakan perintah ini.

Open, perintah ini berguna untuk membuka dokumen yang sama - sama berformat pptx atau ppt. Biasanya akan terdapat menu tambahan open recent file yang akan memudahkan pengguna membuka dokumen presentasi terakhir. Shortcut yang dapat digunakan untuk perintah ini adalah Ctrl + O

Save, perintah ini berguna untuk menyimpan perubahaan - perubahan atas dokumen presentasi. Walau microsoft powerpoint 2007 sudah dilengkapi autosaving akan lebih aman untuk dilakukan save. SHortcut yang dapat digunakan untuk perintahini adalah Ctrl + S

Save as, perintah ini berguna untuk menyimpan dokumen presentasi dengan nama dan format file tertentu. Format file yang ditawarkan antara alain pptx, powerpoint show, powerpoint 97-2003, pdf dan format lain yang ditentukan sendiri.

Print, perintah ini digunakan untuk mencetak dokumen presentasi. Umumnya akan ditawarkan untuk modek pencetakan, apakah 1 slide 1 halamaan, 4 slide 1 halaman hingga 9 slides dalam 1 halaman, dan dapat juga diatur ukuran pages beserta colour.

Prepare, perintah ini digunakan untuk melakukan pengaturan yang berkaitan dengan keperluan identitas dokumen presentasi seperti author, password ( bila perlu), dan title.

Send, perintah ini digunakan untuk mengirimkan dokumen presentasi terkait melalui email maupun fax

Publish, perintah ini digunakan untuk melakukan publikasi ke blog

Close, perintah ini digunakan untuk menutup software Microsoft Powerpoint

17. Tombol Office Button, pada Microsoft Power Point 2013 berisi.


Jawaban:

Tombol Office Button pada Microsoft PowerPoint 2013 adalah tombol yang terletak di sudut kiri atas layar aplikasi PowerPoint. Tombol ini memiliki banyak fungsi yang berguna untuk memudahkan pengguna dalam mengelola dan mengedit presentasi.

Berikut adalah daftar fitur yang terdapat dalam tombol Office Button pada Microsoft PowerPoint 2013:

New: Membuka jendela baru untuk membuat presentasi baru.Open: Membuka jendela untuk mencari dan membuka presentasi yang sudah ada.Save: Menyimpan presentasi yang sedang dibuat atau diedit ke dalam file.Save As: Menyimpan presentasi yang sedang dibuat atau diedit ke dalam file dengan nama yang berbeda.Print: Membuka jendela untuk mencetak presentasi.Close: Menutup presentasi yang sedang dibuat atau diedit.Info: Menampilkan informasi tentang presentasi, seperti nama file, ukuran file, dan tanggal pembuatan.Help: Membuka jendela bantuan untuk mencari informasi tentang cara menggunakan aplikasi PowerPoint.Options: Membuka jendela untuk mengubah pengaturan aplikasi PowerPoint, seperti warna latar belakang, ukuran font, dan lainnya.Exit: Menutup aplikasi PowerPoint.
