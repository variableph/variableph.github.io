3. tombol Quick acces toolbar pada microsoft power point 2007 berisi?​


Jawaban:

Tombol perintah Save, fungsinya adalah menyimpan file dari dokumen word yang sedang kita buka. Tombol perintah Undo, fungsinya adalah membatalkan perubahan pada file dokumen yang sedang dikerjakan. Tombol perintah Redo, fungsinya berkebalikan dengan fungsi dari tombol undo, yaitu membatalkan perintah yang dilakukan tombol undo sebelumnya. Tombol perintah New, fungsinya untuk membuka file dokumen MS Word yang baru. Tombol perintah Open, fungsinya untuk membuka file yang pernah dibuat, yang tersimpan di media penyimpanan. Tombol perintah Quick Print, fungsinya untuk akses perintah cepat mencetak dokumen. Tombol perintah Print Preview, fungsinya untuk memperlihatkan preview tampilan dokumen sebelum file tersebut dicetak. Tombol perintah Spalling & Grammar, fungsinya untuk cek ejaan dan tata bahasa dari teks yang dibuat pada dokumen. Tombol perintah Draw Table, fungsinya untuk menyisipkan tabel pada dokumen MS Word yang dibuat. Kyk gini yaa
