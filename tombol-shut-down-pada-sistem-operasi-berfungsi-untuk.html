a. Pastikan kabel power komputer unit dan monitor sudah terhubung ke jaringan listrik

b. Tekan tombol power di monitor komputer.

2. Fungsi istilah

a. Shutdown : keluar dari komputer yang aktif dan mematikan komputer

b. Sleep komputer : mematikan sementara komputer tanpa mematikan harddisk.

c. Restart : menyalakan ulang komputer tanpa mematikan dan langsung dihidupkan kembali

d. Update and Shutdown adalah komputer dalam keadaan update dan mematikan komputer

e. Hibernate : mematikan komputer dengan setting program yang terbuka akan tetap tersimpan dan ketika dinyalakan kembali komputer langsung ke kondisi ketika belum di hibernate.

28. Tombol yang digunakan untuk mematikan komputer yang menggunakan sistem operasi Windows 10 adalah ...a. Hibernateb. Restartc. shut down d. sleep​


Tombol untuk mematikan komputer mengunakan windowa 10 adalh shut down (c). Shut down adaalh tombol atau perintah untuk mematikan komputer. Shut down dapat dimunculkan melalui start menu. Selain itu shut down dapat ditampilkan melalui powwer user menu (WinX). Pilihan shut down juga dapat ditampilkan dengan menekatn tombol Alt+F4 pada keyboard.

Pembahasan

Untuk mematikan komputer pada sistem operasi windows 10 ada beberapa cara yaitu:

Melalui menu start, yaitu dengan mengklik simbol windows atau tekan tombol windows di keyboar, lalu pilih power kemudian pilih shut down.Melalui Power User Menu (WinX), yaitu klik kanan pada menu start atau tekan tombol Win + X pada keyboard, untuk membuka Power User Menu, lalu pilih Shut down or sign out ⇒ Shutdown.Melalui shut down windows, yaitu klik pada dekstop kemudian tekan tombol Alt+F4 pada keyboard pilih shut down lalu tekan OK atau Enter pada keyboard.Mengubah tombol power, yaitu buka Control Panel, pilih Hardware & Sound, kemudian pada pilihan Power Options, pilih Change what the power buttons do, ubah pilihan pada  When I press the power button dari Sleep menjadi Shut Down, lalu, klik tombol Save changes.
