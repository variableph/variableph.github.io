1. Ubahlah sudut-sudut berikut ke satuan radian 112°


Kategori soal : matematika
Kelas : 10 SMA
Materi : sudut
Kata kunci : derajat dan radian

Pembahasan :

1 putaran = 2π radian, maka:
1 putaran = 360°
1° = 1° x 2π/360°
1° = 1° x 2π/360°
1° = 0,0055π radian
1° = 0,0174 radian

Pada soal diketahui :
112° = 112° x 2π/360°
= 112° x π/180°
= 0,62 x π
= 0,62 x 3,14
= 1,95 radian

30. Ubahlah sudut sudut berikut kedalam satuan radian 75° 148° 256° 324°


75° = 75° / 180° x π = 5 / 12 π

148° = 148° / 180° x π = 37 / 45 π

256° = 256° / 180° x π = 64 / 45 π atau 1 19 / 45 π

324° = 324° / 180° x π = 9 / 5 Π atau 1 4/5 π

penjelasan

rumus satuan radian

n° = n° / 180°

Jawaban:

75° = 5/12 π rad

148° = 37/45 π rad

256° = 1 19/45 π rad

324° = 1 4/5 π rad

langkah-langkah:

A).

75° = 75°/180°

= (75 ÷ 15) / (180 ÷ 15)

= 5/12 π rad

B).

148° = 148°/180°

=(148 ÷ 4) / (180÷4)

= 37 / 45 π rad

C).

256° = 256° / 180°

= (256 ÷ 4) / (180 ÷ 4)

= 64 / 45

= 1 19/45 π rad

D).

324° = 324° / 180°

= (324 ÷ 36) / (180 ÷ 36)

= 9/5

= 1 4/5 π rad
