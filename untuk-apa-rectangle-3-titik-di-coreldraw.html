d. Klik Menu Edit pilih Paste

- Standart Bar
a. Pilih Objek yang Anda inginkan
b. Klik Standart Bar
c. Untuk menggandakannya

- Dengan menggunakan fungsi Ctrl + C dan Ctrl V

- Menggunakan tanda plus ( + ) di Num Luck

- Menggunakan Ctrl + D (Duplicate)
