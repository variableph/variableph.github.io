4. Sebutkan 5 perintah yang ada pada toolbar Header and Footer!


1. Perintah untuk menyisipkan teks otomatis (Insert Auto Text)
2. Perintah untuk menyisipkan nomor halaman (Insert Page Number)
3. Perintah untuk menampilkan nomor halaman (Insert Number of Pages)
4. Perintah untuk memformat nomor halaman (Format Page Number)
5. Perintah untuk menyisipkan tanggal (Insert Date)

5. sebutkan lima perintah yang ada pada toolbar header and footer


Header atau footer adalah teks atau grafik seperti nomor halaman, tanggal, atau logo perusahaan yang biasanya dicetak pada bagian atas atau bawah setiap halaman dalam dokumen. Header dicetak di dalam margin atas sedangkan footer dicetak di bagian margin bawah.


Header dan footer yang sama dapat digunakan pada seluruh dokumen, atau header dan footer yang berbeda untuk setiap bagian dokumen. Sebagai contoh, menggunakan logo pada header halaman pertama, dan kemudian menyertakan nama file dokumen di dalam header halaman berikutnya.
