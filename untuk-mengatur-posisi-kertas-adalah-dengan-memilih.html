Mengatur posisi kertas dapat dilakukan dengan memilih C. orientation. Orientation terdapat pada menu Page Layout. Pada orientation ada dua pilihan yaitu potrait merupakan posisi kertas yang sering digunakan dalam lembar kerja vertikal, dan landscape kertas yang berposisi horizontal.

Pembahasan

Posisi kertas pada saat membuat dokumen sangatlah penting. Posisi kertas ada dua yaitu posisi potrait atau vertikal dan landscape atau horizontal. Untuk mengatur posisi kertas kita dapat menggunakan menu Page Layout dan pilih icon menu orientation. Namun pada menu Page Layout tidak hanya terdapat orientation, masih banyak sub menu atau icon menu lainnya.

Berikut ini icon menu dan fungsi pada menu Page Layout:

Margin, berfungsi untuk mengatur batas pinggir halaman pada kertas.Orientation, berfungsi untuk mengatur posisi kertas.Size, berfungsi untuk mengatur ukuran kertas.Column, berfungsi untuk membuat halaman menjadi berbentuk kolom.Watermark, berfungsi untuk memberikan tanda air (tulisan kabur) di belakang dokumen.Page Color, berfungsi untuk memberikan warna pada halaman atau kertas.Page Border, berfungsi untuk memberikan dan mengatur garis pinggir halaman atau bingkai.Ident Left, berfungsi untuk mengatur batas kiri tulisan.Ident Right, berfungsi untuk mengatur batas kanan.Spacing Before, berfungsi untuk mengatur jarak antara paragraph dengan paragraph sebelumnya.Spacing After, berfungsi untuk mengatur antara jarak paragraph dengan paragraph setelahnya.

Sedangkan insert adalah menu yag digunakan untuk menyisipkan gambar, audio, dan lainya pada lembar kerja.
