Pembahasan

Microsoft excel ialah salah satu program pada sebuah komputer dan juga laptop yang kita pakai dalam kehidupan sehari-hari untuk mengolah data berupa angka. Pada saat menjalankan microsoft excel kita akan dimudahkan untuk menghitung data dengan berbagai rumus yang memiliki fungsi berbeda seperti:

Rumus SUM untuk menjumlahkan data pada cell di data excel kita.Rumus Average digunakan untuk mencari nilai rata-rata.Rumus Count Untuk menghitung jumlah data.Rumus Max digunakan untuk mencari angka tertinggi dari suatu deretan data excel.

Komputer memiliki beberapa komponen mulai dari perangkat keras hingga perangkat lunak, CPU salah satu komponen perangkat keras yang ada pada komputer dan memiliki peran untuk menjalankan perintah dari perintah dari informasi yang diambil dari komponen komputer lainnya seperti keyboard, tuas , hingga pemindai.

Jawaban:

B. Pengiriman data attach file

Penjelasan:

Attachment merupakan media yg disediakan pada sebuah web atau email utk dapat mengirimkan dokumen baik berupa gambar, data, musik, & dokumen yang lain yg kemudian akan diunggah ke internet.  

Dgn adanya attachment sangat memudahkan kita apabila misalnya ingin bertukar informasi atau mengirim dokumen penting, sebab yg butuh hanyalah mengupload dokumen tersebut ke server, sehingga orang lain sudah dapat mendownloadnya dari lokasi lain.

Fungsi Attachment ialah perintah mengirim file lewat email. Kadang disaat kita mengirimkan surat kita mau mengikut sertakan lampiran surat yg lain, maka surat yg kita buat jadi lebih komplit akan tetapi tidak terlalu panjang. Dalam lampiran attachment surat ini, ada sekian banyak dokumen yg dapat kita lampirkan juga, contohnya : document word maupun power point.
