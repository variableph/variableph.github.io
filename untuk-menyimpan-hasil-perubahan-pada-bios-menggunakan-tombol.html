7. Pada BIOS AMI (American Megatrends Incorporated,)Untuk dapat masuk ke dalam menu BIOS kita di haruskan untuk menekan tombol ?​


Jawaban:

tombol Del

Penjelasan:

Mungkin terdapat perbedaan antara BIOS di laptop maupun di komputer (meskipun sama-sama AMI BIOS). Namun di contoh pada gambar lampiran berikut ini terdapat instruksi:

Press Del to Run Setup (artinya: tekan tombol Del untuk masuk setup/ BIOS)

Press F1 to load default values and continue (artinya: tekan tombol F1 untuk loading standar dan melanjutkan)

Jadi kesimpulannya, tombol Del di keyboard bisa ditekan untuk memasuki menu AMI BIOS.
