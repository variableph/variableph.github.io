1. perintah untuk restart service pada mail server​


Pada dasarnya untuk merestart service pada linux bisa menggunakan perintah

sudo service {nama-service} restart

Jadi jika yang dimaksud pada pertanyaan untuk direstart adalah dovecot, maka ganti {nama-service} jadi dovecot.

Cara lain itu menggunakan systemctl

sudo systemctl restart {nama-service}

8. 1.untuk memonitor jaringan seorang admin jaringan dapat menggunakan software? 2.perintah untuk merestart SSH server adalah?


Jawaban:

 

1. Untuk memonitor jaringan seorang admin jaringan dapat menggunakan software?

 

Nmap, The Dude atau Look@LAN

 

2. Perintah untuk merestart ssh server adalah?

 

untuk sistem operasi Linux: ketik /etc/init.d/ssh restart

 

untuk sistem operasi Windows: buka task manager, cari service yang merpakan ssh server, klik kanan dan pilih restart

 

Pembahasan:

 

Seorang administrator jaringan harus memiliki perangkat untuk memantau jaringan komputer. Hal yang dipantau oleh program ini adalah transmisi data keluar masuk jaringan, kondisi server dan pengguna yang terhubung ke jaringan, serta apakah ada komputer yang seharusnya tersambung namun terputus.

 

Beberapa program untuk monitoring jaringan misalnya adalah:

 

Nmap

 

Nmap adalah software pemantau keamanan, digunakan untuk menemukan host dan layanan di jaringan komputer, sehingga membangun "peta" (map) dari jaringan. Software ini ditulis oleh Gordon Lyon, dan awalnya tersedia untuk sistem Linux, namun kemudian dibuat versi Windowsnya.

 

Fitur dari Nmap antara lain host discovery (untuk menentukan hist yang terhubung dalam jaringan), port scanning (membaca port komputer yang terhubung dengan jaringan), dan deteksi sistem operasi yang terhubung jaringan.

 

Look@LAN

 

Softeware ini digunakan untuk mengetahui IP Address pada komputer jaringan, status konektivitas dengan jaringan, sistem operasi yang digunakan oleh komputer pada jaringan tersebut,hostname, netBIOS name, netBIOS user, SNMP status dan Trap serta menginformasikan pada komputer server, host yang sedang online/offline.

 

The Dude

 

Perangkat lunak ini memiliki fitur pengawasan jaringan sama seperti program lainya, ditambah dengan kemampuan untuk menggambarkan jaringan dengan bentuk diagram.

 

Perangkat lunak ini secara otomatis akan membuat diagram yang menunjukkan seluruh struktur dari semua host yang terkandung dalam berbagai alamat yang ditentukan. Anda dapat menambahkannya secara manual ke perangkat apa pun atau mengonfigurasinya ke perangkat yang sudah Anda miliki.

 

Dengan aplikasi ini, anda dapat mengontrol aktivitas berbagai komputer di jaringan dan mengetahui statusnya kapan saja.

 

Kelas: XI

Mata pelajaran: Teknologi Informasi

Materi: Jaringan Komputer

Kata Kunci: Software Jaringan, SSH Server

24. Cara mengetahui nama localhost mysql server


Jawaban:

Jika sudah masuk ke klien baris perintah cobalah ini:

mysql> select user();

Ini akan menampilkan sesuatu yang mirip dengan ini:

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.41 sec)

contoh diatas login sebagai rootdari localhost.

Jika ingin mengetahui nomor port host lokal di mana Mysql berjalan,gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'port';

mysql> SHOW VARIABLES WHERE Variable_name = 'port';

+---------------+-------+

| Variable_name | Value |

+---------------+-------+

| port | 3306 |

+---------------+-------+

1 row in set (0.00 sec)

kueri diatas akan memberi nomor port tempat MySQL dijalankan.

Jika ingin mengetahui nama host Mysql Anda, gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'hostname';

mysql> SHOW VARIABLES WHERE Variable_name = 'hostname';

+-------------------+-------+

| Variable_name | Value |

+-------------------+-------+

| hostname | Dell |

+-------------------+-------+

1 row in set (0.00 sec)

Kueri diatasakan memberi Anda nama host untuk mysql.

Jika ingin mengetahui nama pengguna Mysql, gunakan kueri ini pada klien baris perintah:

MySQL -

select user();

mysql> select user();

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.00 sec)

Kueri diatas akan memberi Anda nama pengguna untuk mysql.

ex: cd C:\xampp\mysql\bin

kemudian jalankan mysql dengan perintah "mysql -u root"

   ex: create database sekolah;

2. Menghapus Database: drop database 'namadatabase';
   ex: drop database sekolah;

3. Menampilkan Daftar Database: show databases;

4. Menggunakan/Mengaktifkan Database: use 'namadatabase';
     ex: use sekolah;

5. Membuat Table :
CREATE TABLE nama_tbl(
kolom1 tipekolom1(ukuran),
kolom2 tipekolom2(ukuran),
...
);
ex:
CREATE TABLE Siswa(
NIS Char(10) NOT NULL PRIMARY KEY,
NAMA Varchar(40),
Umur Int Not Null Default 0,
Kelamin Enum('Pria','Wanita')
);
*Jangan Lupa Tanda "," Koma Pada Akhir

7. Melihat Deskripsi Tabel: desc 'namatabel';
