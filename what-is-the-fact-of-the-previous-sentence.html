Jawaban:

Penjelasan:

Conditionals :

Conditionals merupakan kalimat pengandaian. Conditionals dibagi menjadi 3 :

1st Conditionals :

1st Conditionals dibagi dua,

1. Situations : Benar terjadi di masa sekarang, atau mungkin terjadi di masa depan.

Structure :

If -clause : simple present

Result -clause : will + infinitive

Contoh :

If I practice every day, I will win the match.

Jika aku berlatih setiap hari, aku akan memenangkan pertandingan.

f I practice every day : If -clause

I practice : simple present

I will win the match : Result -clause

I will win : will + infinitive

2. Situation : General Truth

Structure :

If -clause : simple present

Result -clause : simple present

Contoh :

If the temperature reaches 100°C, the water boils.

Jika suhu mencapai 100°C, air akan mendidih.

If the temperatur reaches 100°C : If -clause

the water boils : Result -clause

the temperatur reaches : simple present

the water boils : simple present

2nd Conditional :

Situations : Salah/tidak terjadi, baik di masa depan/masa sekarang.

Sturcture :

If -clause : simple past

Result -clause : would + infinitive

Contoh :

Fact : I don't watch TV.

Fakta : Saya tidak menonton TV.

Conditional : If I watched TV, I wouldn't have time to study.

Pengandaian : Jika aku menonton TV, aku gak akan punya waktu untuk belajar.

If I watched TV : if clause

I wouldn't have time to study : Result clause

I watched TV : simple past

I wouldn't have time : would + invinitive

3rd Conditional :

Situation : Salah/tidak terjadi di masa lalu.

Structure :

If clause : past perfect

Result clause : would + have + V3

Contoh :

Fact : I didn't study.

Fakta : Aku tidak belajar.

Conditional : If I had studied , I would have understood the questions.

Pengandaian : Jika aku telah belajar, aku akan mengerti soal soal nya.

If I had studied : If clause

I would have understood the questions : Result clause

I had studied : past perfect

would have understood : would + have + V3

Pada Soal :

a. If my car were in better condition, I would make a long trip to Geo Park Coletuh now.

Ini merupakan 2nd Conditional, karena diketahui pada bagian If clause, menggunakan past simple yaitu "my car were". Sehingga, faktanya salah/berlawanan dengan conditional nya (pengandaiannya).

Jika mobilku dalam kondisi yang baik, aku akan pergi dalam perjalanan jauh ke Geo Park Ciletuh sekarang.

Karena ini conditional nya, sehingga fakta nya adalah Mobilku dalam kondisi buruk, dan aku tidak pergi dalam perjalanan jauh ke Geo Park Ciletuh.

Yaitu :

My car is in the bad condition ; or

I don't make a long trip to Geo Park Ciletuh now.

b. If Ananda had not been ill, she would have joined the study tour to Djokjakarta.

Jika Ananda tidak sakit, dia akan sudah ikut study tour ke Djokjakarta.

Ini merupakan 3rd conditional, karena if clause nya menggunakan "past perfect" dan result clause nya menggunakan "would + have + V3".

Yaitu : If Ananda had not been ill -If clause.

She would have joined the study tour to Djokjakarta. -Result clause

had not been : past perfect

would have joined : would + have + V3

3nd conditional merupakan pengandaian yang salah di masa lalu.

Sehingga, seharusnya faktanya :

1. Ananda telah sakit, atau

Ananda was ill.

Dia tidak mengikuti study tour ke Djokjakarta.

She didn't join the study tour to Djokjakarta.

Jawaban:

The fact of the sentence above is they passed the test  because they studied harder last night

Penjelasan:

Kalimat pertama:

If Juminten went there , she would meet me.

Ini adalah conditional type 2, dimana pada titik saat ini (present), kita mengandaikan sesuatu seandainya terjadi, tapi sebenarnya tidak terjadi (pada saat ini)

Jadi faktanya adalah:  Juminten saat ini tidak pergi ke sana, jadi dia tidak bertemu saya. (jadi menggunakan kalimat present tense)

Kalimat ke-2:

They would not have passed the test, if they had not studied harder last night

Ini adalah conditional type-3, yang dipakai untuk mengandaikan sesuatu sudah terjadi pada suatu titik di masa lampau, dan faktanya sesuatu itu tidak terjadi di masa lampau.

Dalam soal ini, diandaikan seandainya mereka tidak belajar lebih serius tadi malam (jadi di masa lampau, bukan saat ini), maka mereka tidak akan lulus (akan gagal), (ini di masa lampau juga, misalnya kita mengucapkan kalimat ini siang hari setelah ulangan dilaksanakan)

Faktanya, mereka belajar lebih keras tadi malam, dan mereka lulus (semua di masa lampau, maka dipakai kalimat past tense)

PENDAHULUAN

Conditional sentence adalah bentuk kalimat yang berguna untuk menyatakan suatu perkiraan untuk terjadi di masa depan. Terdiri dari empat tipe yaitu tipe 0, 1, 2, dan 3.

[tex]\\[/tex]

CONDITIONAL SENTENCE TYPE 0

Kalimat pengandaian yang digunakan jika suatu kondisi sudah pasti akan terjadi. Biasanya termasuk salah satu kebenaran umum, ataupun hal lain yang sudah terbukti secara ilmiah.

rumus:

if + (simple present), (simple present)

contoh:

If you freeze water, it becomes solid.

[tex]\\[/tex]

CONDITIONAL SENTENCE TYPE I

Adalah kalimat pengandaian yang digunakan jika suatu kondisi berkemungkinan besar untuk terjadi di masa depan.

rumus:

if + (simple present), (simple future)

contoh:

If they come late, teachers will punish them.

[tex]\\[/tex]

CONDITIONAL SENTENCE TYPE II

Kalimat pengandaian yang digunakan ketika hipotesis suatu kondisi hanya memiliki kemungkinan yang kecil untuk terjadi di masa depan.

rumus:

if + (was/were), (modal v²) + (bare infinitive)

contoh:

If the teachers were going to the other class, I would go to the library.

[tex]\\[/tex]

CONDITIONAL SENTENCE TYPE III

Merupakan sebuah kalimat pengandaian yang digunakan ketika sebuah kondisi tidak akan terwujud di masa depan.

rumus:

if + (past perfect), (modal v²) + have + (past participle)

contoh:

If they had read the instruction, they should have been here in time.

[tex]\\[/tex]

PEMBAHASAN

(a) If Juminten went there, she would met me. (kalau Juminten pergi ke sana, dia akan bertemu denganku)

The fact of the sentence above is Juminten doesn't go there. (fakta dari kalimat di atas adalah Juminten tidak pergi ke sana, jadi dia tidak bertemu dengan si pengucap kalimat itu)

(b) They would not have passed the test, if they had not studied harder last night. (mereka tidak akan mampu melewati ujiannya, jika mereka tidak belajar sungguh-sungguh kemarin malam)

The fact of the sentence above is they studied harder last night. (fakta dari kalimat di atas adalah mereka telah belajar sungguh-sungguh kemarin malam, sehingga mereka bisa melewati ujiannya)

[tex]\\[/tex]

KESIMPULAN

The fact of each sentence is:

Juminten doesn't go there. They studied harder last night.

Find The Facts Of The Conditional Sentences Type 3JawabanPendahuluan

Conditional sentences type 3 adalah suatu kalimat pengandaian yang digunakan untuk  menggambarkan masa lalu yang tidak mungkin terjadi dan membayangkan hasil dari keadaan tersebut saat ini. Bisa juga disebut unreal sentences, fakta dari kalimat berlawanan dengan kalimat tersebut.

Formula untuk conditional sentences type 3 :

If Clause (condition)                               Main Clause (Result)

If + Past  Perfect                                     S+had+ Verb 3

If + Past Perfect                                      S+would have + Verb 3

Pembahasan

Find the facts of the conditional sentences below carefully!

1. If I had had enough time, I would have come to your wedding party.

Fact : I didn't have enough time and I didn't come to your wedding party.

2. If I had worked hard, I would have got the promotion for manager position.

Fact : I didn't work hard and I didn't promote for manager position.

3. If I hadn't gone to the mountain, I wouldn't have had an accident.

Fact : I went to the mountain and I have an accident.

4. If I had checked my Whatsapp message, I would have contacted you to go home.

Fact : I didn't check my whatsapp and I didn't contact you.

5. If mother had gone to Jakarta, she would have met her old friends.

Fact : Mother didn't go to Jakarta and she didn't meet her old friends.

6. If Sania had brought 2 raincoats, she would have lent it you.

Fact : Sania didn't buy 2 raincoats and she didn't lend you one.

7. If brother had visited Singapore, he would have taken some beautiful pictures.

Fact : Brother didn't visit Singapore and didn't take come beautiful pictures.

8. If she had read your message, she would have come to meet you soon.

Fact : She didn't read your message and she didn't come to meet you soon.

9. Nagita would have sent the application letter if she had not been busy.

Fact : Nagita didn't send the application because she was busy.

10. Miss Laila would have appreciated your work if you had done it by yourself.

Fact : Miss Laila didn't appeciate your work because you didn't do it yourself.

Kesimpulan

Pada kalimat conditional sentences diatas, fakta yang terjadi berlawanan dengan kalimat pengandaian. Untuk membentuk conditional sentences type 3 kita perlu memahami mengenai perubahan verb, khususnya perubahan verb ke bentuk verb 3.

CONDITIONAL SENTENCE

write the fact of the following conditional sentence​

1. If the doctor didn't operate him as soon as possible, he would passed away.

In fact the doctor operates him as soon as possible so he does not pass away

2. If he didn't invite his friends, they wouldn't come to his party

In fact he invites his friends so they come to his party.

In fact they would not lose the way because they knew my address

PEMBAHASAN:

Conditional Sentence

if conditional sentence merupakansebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

Contoh:  

If I go to Palembang I will meet Sarah  If we have a yacht, we will sail the seven seas If he have more time, he will learn karate If they tell their father, he will be very angry She will spend a year in the USA if it is easier to get a green card If I live on a lonely island, I will run around naked all day

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

Contoh:  

If I were you, I would forgive him If we had a yacht, we would sail the seven seas If he had more time, he would learn karate If they told their father, he would be very angry She would spend a year in the USA if it were easier to get a green card If I lived on a lonely island, I would run around naked all day

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb3

Contoh :  

If Mimi had done her homework, she would have not got a punishment. If we had had a yacht, we would have sailed the seven seas If he had had more time, he would have learnt karate If they had told their father, he would have been very angry She would have spent a year in the USA if it had been easier to get a green card If I had lived on a lonely island, I would have run around naked all day
