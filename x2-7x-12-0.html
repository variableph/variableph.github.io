27. Himpunan penyelesaian dari x2 + 7x + 12 = 0 adalah …


Penjelasan:

x² + 7x + 12 = 0

dengan metode kuadrat sempurna

x² + 7x + 12 = 0

x² + 7x = -12

x² + 7x + (7/2)² = -12 + (7/2)²

(x + 7/2)² = -12 + 49/4

(x + 7/2)² = -48/4 + 49/4

(x + 7/2)² = ¼

(x + 7/2) = √¼

x + 7/2 = ±½

ada 2 penyelesaian, x₁ dan x₂

untuk x₁ :

x₁ + 7/2 = ½

x₁ = ½ - 7/2

x₁ = -3

untuk x₂ :

x₂ + 7/2 = -½

x₂ = -½ - 7/2

x₂ = -4

HP : [{-3, -4}]
