7. tanda > termasuk operator ... *


Materi: Pemrograman - Operator

Jawaban:

Tanda > termasuk operator pembanding (comparison operator).

Penjelasan:

Dalam bahasa pemrograman, tanda > digunakan sebagai operator pembanding berbagai kondisi atau variabel. Tanda > termasuk dalam operator pembanding atau disebut juga comparison operator.

Contoh penerapannya pada C++:

//Program dimulai

#include <iostream>

int main() {

   int angka1 = 15;

   int angka2 = 20;

   cout << (angka2 > angka1); //Menghasilkan output true/benar

   return 0;

}

//Program selesai

8. AND termasuk operator..​


Jawaban:

Operator Logika

Penjelasan:

Operator logika adalah operator yang digunakan untuk melakukan operasi dimana nilai yangdihasilkan dari operasi tersebut hanya berupa nilai benar (true) atau salah (false). Nilai ini disebut dengan nilai boolean. Dalam bahasa C++, nilai benar direpresentasikan dengan nilai bilangan selain 0 (biasanya nilai 1), sedangkan salah direpresentasikan dengan bilangan 0. Dalam bahasa C++ modern telah mendukung tipe bool, nilai benar direpresentasikan dengan nilai true dan nilai salah direpresentasikan dengan nilai false.

13. Yang termasuk operator biner​


Jawaban:

1. Operator Aritmatika

adalah operator yang digunakan untuk melakukan operasi aritmatika seperti penjumlahan, pengurangan, perkalian, bembagian, dll

2. Operator Logika

adalah operator yang digunakan untuk melakukan operasi dimana nilai yangdihasilkan dari operasi tersebut hanya berupa nilai benar (true) atau salah (false). Nilai ini disebut dengan nilai boolean.

3. Operator Relasional

merupakan operator yang digunakan untuk menentukan relasi atau hubungan dari dua buah operand.

4. Operator Bitwise

merupakan operator yang digunakan untuk melakukan operasi-operasi yang berhubungan dengan manipulasi bit. Operator bitwise hanya dapat dilakukan pada operand yang bertipe char dan int saja karena ini berkoresponden dengan tipe byte dan word didalam bit.

19. Operator untuk peningkatan nilai bertambah 1 termasuk dalam operator.. A. Operator bitwiseB. Operator logikaC. Operator unaryD. Opetator binaryE. Operator tenary​


Materi: Operator

Jawaban:

Operator untuk peningkatan nilai bertambah 1 termasuk dalam operator C. Operator unary.

Penjelasan:

Operator Unary adalah jenis operator yang akan memodifikasi suatu nilai variabel dengan hanya satu operand atau input tunggal. Contohnya increment (peningkatan nilai bertambah 1). Contoh:

int x = 0;x++;

Maka, x sekarang bernilai 1.

Jadi, opsi yang benar adalah C. Operator unary.
